         Windows XP (32- och 64-bitars)

         Windows Server 2003 (32- och 64-bitars)

         Windows Vista ( (32- och 64 bitars)

II. KÄNDA PROBLEM OCH LÖSNINGAR

Allmänt

 

         Vissa USB 2.0-höghastighetskort levereras med osignerade drivrutiner. Om du har köpt ett USB 2.0-kort och har en osignerad drivrutin kan det uppstå problem vid installationen av drivrutinen eller i kommunikationen mellan enheten och HP-programmet. Vi rekommenderar därför att du alltid använder en Microsoft WHQL-certifierad drivrutin till din USB-maskinvara.

         Plug and Play-drivrutinspaket finns på Internet på följande webbadress:

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

·         Funktionaliteten på system som bara uppfyller minimisystemkraven kan vara begränsad. Om du använder mycket bilder eller videofilmer måste du ha ett system som uppfyller de rekommenderade systemkraven för att systemet ska ha godtagbar prestanda och pålitlighet.

·         Felsökning av installation och konfiguration

Om enheten inte kan identifieras:

  • Kontrollera att kabeln är en kompatibel USB 2.0-höghastighetskabel.
  • Kontrollera att kabeln har anslutits ordentligt. Ta bort USB-kabeln, kontrollera dess ände och jämför den med USB-porten. Se till att den inte är upp och ned. Tryck in USB-kabelns ena ände helt i USB-porten på HP-enheten och den andra i uttaget på datorns baksida.
  • Kontrollera att den externa enheten är påslagen. Externa enheter känns inte igen om de inte är påslagna.
  • Kontrollera att USB-kabeln inte är längre än 2 meter.
  • Kontrollera att USB-porten är aktiverad. Vissa datorer levereras med inaktiverade USB-portar.

Uppgradera Windows

·         HP rekommenderar att du avinstallerar HP-enheter innan du uppgraderar till en senare Windows-version.  Om du t.ex. uppgraderar från Windows XP till Windows Vista bör du avinstallera HP-enheten med hjälp av avinstalleraren eller Lägg till/ta bort program, och sedan uppgradera till det nya operativsystemet. När du har uppgraderat operativsystemet kan du återinstallera HP-enheten med den senaste programvaran från följande webbadress:

http://www.hp.com/support/LJP1100series 

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

Andra program

·         Microsoft Wordpad visar inte komprimerade JPEG-filer i RTF-filer. Om du vill visa dessa filer måste du använda ett annat program, t.ex. Microsoft Word.

Om du vill ha en uppdaterad lista med kända problem med HP LaserJet-utskriftssystemet går du till supportsidorna för HP LaserJet Professional P1100/P1560/P1600-serien på följande webbadresser:

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

III. INSTALLATIONSANVISNINGAR

I en Windows-miljö identifierar CD-ROM-skivans installationsprogram automatiskt operativsystemets språk och version.

INSTALLATION med HP Smart Install PÅ WINDOWS – REKOMMENDERAT INSTALLATIONSÄTT

Obs! Du måste ha administratörsbehörighet för att installera utskriftssystemet i Windows XP eller Windows Vista.

1.       Anslut USB-kabeln till datorn och skrivaren.

2.       HP Smart Install-programmet bör startas automatiskt. Om HP Smart Install inte har startat efter 30 sekunder går du igenom stegen 3–5.

3.       Välj Kör på Start-menyn i Aktivitetsfältet.

4.       Klicka på bläddra. En ny CD-ROM-enhet med namnet HP Smart Install visas. Bläddra till enheten.

5.       Markera filen SISetup.EXE, klicka på Öppna och sedan på OK.

6.       Följ anvisningarna.

 

Nätverksinstallation av PROGRAM med HP Smart Install.

1.       Skriv enhetens IP-adress i webbläsarens adressfält.

2.       Klicka på fliken HP Smart Install.

3.       Klicka på knappen Ladda ner.

4.       Följ anvisningarna.

 

INSTALLERA PROGRAM PÅ WINDOWS med CD-ROM

Obs! Du måste ha administratörsbehörighet för att installera utskriftssystemet i Windows XP eller Windows Vista.

Installation av utskriftssystem från en CD-ROM-skiva

1.       Anslut inte USB- eller parallellkabeln till datorn.

2.       Avsluta alla program.

3.       Sätt in CD-ROM-skivan.

4.       Om installationsprogrammet för HP-utskriftssystemet inte har startat efter en halv minut genomför du stegen 5 till 7.

5.       Välj Kör på Start-menyn i Aktivitetsfältet.

6.       Navigera till CD-ROM-skivans rotkatalog.

7.       Markera filen SISetup.EXE, klicka på Öppna och klicka sedan på OK.

8.       Följ anvisningarna.

 

Ytterligare installationsanvisningar för HP Laserjet P1100w-serien.

HP LaserJet Professional P1100w-serien stöder Wi-Fi 802.11 b/g.  Innan skrivaren installeras bör du försäkra dig om att den trådlösa routern eller åtkomstpunkten har konfigurerats rätt.  Ha det trådlösa nätverket konfigurationsuppgifter tillgängliga.

 

Inaktivera HP Smart Install för att använda en annan drivrutin

Om du vill använda en drivrutin via USB istället för den värdbaserade standarddrivrutinen som följde med HP Smart Install och CD-skivan (ex. PCL5E eller XPS), måste du först inaktivera HP Smart Install på datorn eller skrivaren.

1.       Anslut USB-kabeln till datorn och skrivaren.

2.       HP Smart Install-programmet bör startas automatiskt. Avsluta programmet utan att installera den värdbaserade drivrutinen.

3.       Sätt in CD-ROM-skivan

4.       Installationsprogrammet bör startas automatiskt. Avsluta programmet utan att installera den värdbaserade drivrutinen. 

5.       Navigera till CD-ROM-skivans rotkatalog.

6.       Markera filen HPSmartInstallConfig.EXE, klicka på Öppna och sedan på OK.

7.       Det finns två alternativ i det här verktyget. Det ena alternativet stänger av HP Smart Install på datorn. Installeraren kommer fortfarande att kunna köras om skrivaren är ansluten till andra datorer. Det andra alternativet stänger av HP Smart Install på skrivaren. Om detta alternativ väljs går det inte att köra HP Smart Install på någon annan dator – om installeraren inte återaktiveras med detta verktyg.

 

 

Inaktivera HP Smart Install på nätverksprodukter

Nätverksadministratören kan inaktivera HP Smart Install på nätverksskrivare.

1.       Skriv IP-adressen till enheten i webbläsarens adressfält.

2.       Klicka på fliken Inställningar.

3.       Klicka på menyn Skrivarinställningar.

4.       Om du vill inaktivera HP Smart Install via USB ändrar du alternativet Aktivera HP Smart Install via USB till Nej.

5.       Om du vill inaktivera Hp Smart Install via nätverket ändrar du alternativet Visa nedladdningsalternativ i HP Smart Install till Nej.

Installera via Lägg till skrivare

HP LaserJet-skrivaren kan också installeras via Windows-funktionen Lägg till skrivare, som finns i mappen Skrivare. Funktionen kallas även guiden Lägg till skrivare eller APW. Instruktionerna i guiden Lägg till skrivare skiljer sig något åt beroende på vilken version av Windows du använder. Om du vill installera via guiden Lägg till skrivare måste du först inaktivera HP Smart Install enligt instruktionerna ovan.

 

Installera med guiden Lägg till skrivare i WinXP 

1.         Kontrollera att HP LaserJet P1100/P1560/P1600-skrivaren inte har anslutits till uttaget och att den inte är ansluten till datorn.

2.         Från Start-menyn går du in i Kontrollpanelen och klickar sedan på Skrivare och fax.

3.         Klicka på Lägg till skrivare och klicka sedan på Nästa på sidan Välkommen till guiden Lägg till skrivare.

4.         Följ anvisningarna på skärmen.

5.         Markera alternativet Lokal skrivare när du blir ombedd att ange om produkten är en lokal skrivare eller en nätverksskrivare (detta är det förvalda alternativet). Avmarkera kryssrutan Identifiera och installera Plug and Play-skrivaren automatiskt.  Klicka på Nästa.

6a.         Om du installerar via en USB-anslutning och inte har installerat en HP LaserJet P1100/P1560/P1600-skrivare tidigare finns ännu ingen USB-port. Detta är normalt. Välj LPT1: och klicka på Nästa.

6b.         Om du installerar via en USB-anslutning och tidigare har installerat en HP LaserJet P1100/P1560/P1600-skrivare finns en USB-port installerad. Välj denna USB-port och klicka på Nästa

Obs!    Tänk på att ansluta HP LaserJet P1100/P1560/P1600-skrivaren till samma fysiska USB-port som du tidigare hade den ansluten till.

7.         Markera "Diskett finns" och ange sedan sökvägen till den HP LaserJet P1100/P1560/P1600-drivrutin som ska installeras.

8.         Om flera alternativ visas markerar du din skrivarmodell och klickar sedan på Nästa

9.         Följ anvisningarna på skärmen.

10.        Anslut HP LaserJet P1100/P1560/P1600-skrivaren till datorn när du ombeds att göra det.

 

Obs!    Om du inte uppmanas att ansluta HP LaserJetP1100/P1560/P1600-skrivaren vid detta tillfälle ska du ansluta den till datorn när installationen slutförts.

 

11.        Följ anvisningarna på skärmen tills installationen har slutförts.

 

När installationen har slutförts kan det finnas två exemplar av skrivardrivrutinen på datorn. I så fall bör du radera den kopia som skapades först. Den har vanligtvis samma namn som skrivarmodellen. Följ stegen nedan för att kontrollera hur många drivrutiner som skapades och ta bort den första versionen

 

1.         Peka på Inställningar i Start-menyn, och klicka sedan på Skrivare.

2.         Om du har både "HP LaserJet P1100/P1560/P1600-serien" och " HP LaserJet P1100/P1560/P1600-serien (kopia 2)" ska du inte ta bort den som har "kopia 2" i namnet. Ta istället bort den första drivrutinen som heter "HP LaserJet P1100/P1560/P1600-serien"

 

Nu kan du skriva ut med hjälp av drivrutinen "HP LaserJet P1100/P1560/P1600-serien (kopia 2)".

Skriva ut med PCL5e

PCL5e stöds bara av HP LaserJet Professional P1600-serien.

PCL5e-drivrutinen finns på Internet på följande webbadress:

http://www.hp.com/support/LJP1600series

 

Skriva ut med XPS

XPS-drivrutinen finns på den programskiva som levererades med skrivaren samt på webben på följande webbadress:

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

Använd guiden Lägg till skrivare för att installera XPS-drivrutinen.
