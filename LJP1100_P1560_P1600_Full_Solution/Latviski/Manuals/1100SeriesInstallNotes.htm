Atspçjojiet HP Smart Install, lai izmantotu citu, nevis resursdatora draiveri

Ja vçlaties izmantot citu draiveri ar USB, izòemot standarta resursdatora draiveri, kas iekïauts HP Smart Install komplektâcijâ, un kompaktdisku (izò. PCL5E vai XPS), vispirms jâatspçjo HP Smart Install datorâ vai printerî.

2.       HP Smart Install programmai vajadzçtu darboties automâtiski. Izejiet no ðîs programmas, neinstalçjot resursdatora draiveri.

3.       Ievietojiet kompaktdisku

4.       Instalçðanas programmai vajadzçtu darboties automâtiski. Izejiet no ðîs programmas, neinstalçjot resursdatora draiveri. 

5.       Atrodiet kompaktdiskâ UTIL direktoriju.

6.       Atlasiet failu HPSmartInstallConfig.EXE, noklikðíiniet uz Open (Atvçrt) un pçc tam uz OK (Labi).

7.       Ðajâ utilîtâ bûs divas opcijas. Viena opcija izslçgs HP Smart Install datorâ, tâ joprojâm darbosies, ja printeris ir pievienots citiem datoriem. Cita opcija izslçgs HP Smart Install printerî. Ja atlasîta ðî opcija, printeris nepalaidîs HP Smart Install datorâ, ja tâ ir atkârtoti iespçjota, izmantojot ðo utilîtu.
