I. INTRODUCCIÓN

OBJETIVO DE ESTE DOCUMENTO

Este documento proporciona instrucciones de instalación específicas y otra información que debería saber antes de instalar y utilizar el sistema de impresión. Este software HP LaserJet funciona con los siguientes sistemas operativos:

·         Windows XP (32 bits y 64 bits)

·         Windows Server 2003 (32 bits y 64 bits)

·         Windows Vista (32 bits y 64 bits)

·         Windows Server 2008 (32 bits y 64 bits)

 

Nota: Este software de HP LaserJet no es compatible con Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000 y Windows Server 2000.

Para obtener instrucciones sobre cómo instalar otros sistemas, consulte las notas de instalación relacionadas, los archivos léame u otra información del producto para dicho sistema en concreto.

 

En este archivo se proporciona información específica sobre la instalación, la compatibilidad y la solución de problemas. Para obtener más información sobre las funciones del dispositivo y del controlador, consulte la guía del usuario y la ayuda en línea que se incluye con cada componente del software.

DESTINATARIOS DE ESTE SOFTWARE.

Todos los usuarios necesitan este software del sistema de impresión para utilizar las impresoras HP LaserJet Professional series P1100/P1560/P1600.

II. PROBLEMAS CONOCIDOS Y SOLUCIONES

General

 

·         Algunas tarjetas USB 2.0 de alta velocidad se envían con controladores de host sin firmar. Si ha adquirido una tarjeta complementaria USB 2.0 y dispone de un controlador sin firmar, es posible que tenga problemas con la instalación del controlador o con la comunicación entre el dispositivo y el software de HP. Se recomienda utilizar siempre un controlador de host con la certificación Microsoft WHQL con el hardware USB.

·         Los paquetes de controladores de impresión Plug and Play están disponibles en la Web, en la siguiente dirección URL:

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

·         Es posible que la funcionalidad de los sistemas que sólo cumplan los requisitos mínimos se vea limitada. El uso de muchos vídeos o imágenes requiere un sistema que cumpla los requisitos recomendados del sistema para ofrecer una fiabilidad y un rendimiento aceptables.

·         Solución de problemas en la configuración y la instalación

Si no se reconoce el dispositivo:

  • Compruebe que el cable es compatible con USB 2.0 de alta velocidad.
  • Asegúrese de que el cable está conectado correctamente. Extraiga el cable USB, compruebe el extremo del cable y, a continuación, hágalo coincidir con el patrón del puerto USB. Asegúrese de que no está al revés. Introduzca completamente uno de los extremos del cable USB en el puerto USB del dispositivo HP y el otro extremo en la parte posterior del equipo.
  • Compruebe que el periférico está encendido. Los periféricos no se reconocen si están apagados.
  • Asegúrese de que el cable USB no mide más de 2 metros (6,5 pies) de longitud.
  • Compruebe que el puerto USB está habilitado. Algunos equipos se suministran con los puertos USB deshabilitados.

Actualización de Windows

·         HP recomienda que desinstale los dispositivos HP antes de actualizarlos a una versión compatible de Windows más reciente.  Por ejemplo, al actualizar Windows XP a Windows Vista, desinstale el dispositivo HP mediante el desinstalador o la opción Agregar o quitar programas y, a continuación, instale el nuevo sistema operativo. Una vez finalizada la actualización del sistema operativo, vuelva a instalar el dispositivo HP con el software más reciente desde la siguiente URL:

http://www.hp.com/support/LJP1100series 

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

III. INSTRUCCIONES DE INSTALACIÓN

Cuando se utilice en un entorno de Windows, el software de instalación o configuración del CD-ROM detecta la versión y el idioma del sistema operativo que esté ejecutando.

INSTALACIÓN DE SOFTWARE EN WINDOWS con HP Smart Install – método de instalación RECOMENDADO

NOTA: Para instalar el sistema de impresión en un sistema operativo Windows XP o Windows Vista, es necesario disponer de privilegios de administrador.

1.       Conecte un cable USB en el equipo y en la impresora.

2.       El programa HP Smart Install debería ejecutarse automáticamente. Si el programa HP Smart Install no ha empezado después de 30 segundos, complete los pasos del 3 al 5.

3.       Seleccione Ejecutar del menú Inicio de la barra de tareas.

4.       Haga clic en la opción de navegación. Encontrará disponible una nueva unidad de CD-ROM denominada HP Smart Install. Desplácese a esta unidad.

5.       Seleccione el archivo SISetup.EXE, haga clic en Abrir y, a continuación, haga clic en Aceptar.

6.       Siga las indicaciones.

 

Instalación de SOFTWARE con HP Smart Install a través de la red

1.       En un navegador web, introduzca la dirección IP del dispositivo en la línea de dirección del navegador.

2.       Haga clic en la pestaña HP Smart Install.

3.       Haga clic en el botón Descargar.

4.       Siga las indicaciones.

 

INSTALACIÓN DE SOFTWARE EN WINDOWS con CD-ROM

NOTA: Para instalar el sistema de impresión en un sistema operativo Windows XP o Windows Vista, es necesario disponer de privilegios de administrador.

Instalación de un sistema de impresión desde un CD-ROM

1.       No conecte el cable USB/paralelo al sistema del equipo.

2.       Cierre todos los programas.

3.       Inserte el CD-ROM.

4.       Si el programa instalador del sistema de impresión de HP no ha empezado después de 30 segundos, complete los pasos del 5 al 7.

5.       Seleccione Ejecutar del menú Inicio de la barra de tareas.

6.       Desplácese al directorio raíz de la unidad de CD-ROM.

7.       Seleccione el archivo Setup.EXE, haga clic en Abrir y, a continuación, haga clic en Aceptar.

8.       Siga las indicaciones.

 

Instrucciones adicionales para la instalación de la impresora HP Laserjet serie P1100w

La impresora HP LaserJet Professional serie P1100w es compatible con Wi-Fi 802.11 b/g.  Asegúrese de que su direccionador inalámbrico o punto de acceso se ha configurado correctamente antes de comenzar a instalar la impresora.  Conserve a mano la información sobre la configuración de la red inalámbrica por si la necesitara.

 

Desactivar HP Smart Install para utilizar un controlador distinto del basado en host

Si desea utilizar un controlador a través de USB distinto del controlador basado en host estándar incluido con HP Smart Install y el CD (por ejemplo, PCL5E o XPS), en primer lugar tendrá que desactivar HP Smart Install del equipo o impresora.

1.       Conecte el cable USB con el equipo y la impresora.

2.       El programa HP Smart Install debería ejecutarse automáticamente. Salga de este programa sin instalar el controlador basado en host.

3.       Inserte el CD-ROM.

4.       El programa de instalación debería ejecutarse automáticamente. Salga de este programa sin instalar el controlador basado en host. 

5.       Diríjase al directorio UTIL del CD.

6.       Seleccione el archivo HPSmartInstallConfig.EXE, haga clic en Abrir y, a continuación, haga clic en Aceptar.

7.       Encontrará dos opciones en esta utilidad. Una de las opciones desactivará HP Smart Install del equipo; continuará ejecutándose si se conecta la impresora a otros equipos. La otra opción, desactivará HP Smart Install de la impresora. Si selecciona esta opción, la impresora no ejecutará HP Smart Install en ningún equipo a menos que la active de nuevo a través de esta utilidad.

 

 

Desactivar HP Smart Install en productos de red

Si el administrador lo solicita, es posible desactivar HP Smart Install en impresoras de red.

1.       En un navegador web, introduzca la dirección IP del dispositivo en la línea de dirección del navegador.

2.       Haga clic en la pestaña Configuración.

3.       Haga clic en el menú Configuración de impresión.

4.       Para desactivar HP Smart Install a través de USB, establezca la opción Activar HP Smart Install a través de USB en No.

5.       Para desactivar HP Smart Install a través de la red, establezca Mostrar la opción de descarga HP Smart Install en No.

Instalación a través de la adición de impresoras

También es posible instalar la impresora HP LaserJet a través de la función Agregar impresora de Microsoft que se encuentra en la carpeta Impresoras. Esta función se conoce como Asistente para agregar impresoras o APW. Las instrucciones del APW variarán según las distintas versiones de Windows. Para llevar a cabo la instalación a través de APW, en primer lugar deberá desactivar HP Smart Install siguiendo las instrucciones que se han detallado anteriormente.

 

Programa de instalación de APW deWinXP 

1.         Asegúrese de que la impresora HP LaserJet serie P1100/P1560/P1600 no está enchufada y no está conectada al equipo.

2.         En el menú Inicio, vaya a Panel de control y haga clic en Impresoras y faxes.

3.         Haga clic en Agregar impresora y, a continuación, haga clic en Siguiente en la pantalla “Asistente para agregar impresoras”.

4.         Siga las instrucciones que aparecen en pantalla.

5.         Cuando el asistente le pregunte si se trata de una impresora local o de red, seleccione Impresora local (opción seleccionada de manera predeterminada). Quite la marca de la casilla de verificación Detectar e instalar mi impresora Plug and Play automáticamente.  Pulse Siguiente.

6a.         Si está realizando la instalación a través de USB y no ha instalado la impresora HP LaserJet serie P1100/P1560/P1600 anteriormente, todavía no habrá un puerto USB instalado. Esto es normal. Seleccione LPT1: haga clic en Siguiente.

6b.         Si está realizando la instalación a través de USB y ha instalado y conectado la impresora HP LaserJet serie P1100/P1560/P1600 anteriormente, habrá un puerto USB instalado. Seleccione el puerto USB y haga clic en Siguiente

Nota:    Asegúrese de que conecta la impresora HP LaserJet serie P1100/P1560/P1600 al mismo puerto USB físico al que previamente lo tenía conectado.

7.         Seleccione "Utilizar disco" y especifique la ubicación del controlador HP LaserJet serie P1100/P1560/P1600 que se instalará.

8.         Si se presentan varias selecciones, elija el modelo de su impresora y haga clic en Siguiente.

9.         Siga las instrucciones que aparecen en pantalla.

10.        Cuando el asistente le pida que conecte la impresora, enchúfela y conecte la impresora HP LaserJet serie P1100/P1560/P1600 al equipo.

 

Nota:    Si en este momento no le solicita que conecte la impresora HP LaserJet serie P1100/P1560/P1600, deberá enchufarla y conectarla al equipo una vez finalizada la instalación.

 

11.        Siga las instrucciones que aparecen en la pantallas hasta que finalice la instalación.

 

Una vez finalizada, es posible que tenga dos copias del controlador de impresora creado. En este caso, elimine la primera que se ha creado. Normalmente, adoptará el mismo nombre que el modelo de la impresora. Para comprobar la cantidad de controladores creados y eliminar el primero, siga estos pasos:

 

1.         En el menú Inicio, vaya a Configuración y haga clic en Impresoras.

2.         Si tiene "HP LaserJet serie P1100/P1560/P1600" y "HP LaserJet serie P1100/P1560/P1600 (copia 2)", no elimine la que incluye "copia 2" en el nombre. Elimine el primer controlador creado, denominado "HP LaserJet serie P1100/P1560/P1600"

 

En este momento, ya podrá imprimir utilizando el controlador "HP LaserJet serie P1100/P1560/P1600 (copia 2)".
