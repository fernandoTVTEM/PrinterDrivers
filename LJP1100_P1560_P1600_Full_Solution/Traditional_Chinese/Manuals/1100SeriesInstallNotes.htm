I. 簡介

本文件宗旨

本文件旨在提供安裝與使用列印系統之前必須瞭解的安裝說明及其他資訊。 這套 HP LaserJet 軟體搭配下列作業系統：

·         Windows XP（32 位元和 64 位元）

·         Windows Server 2003（32 位元和 64 位元）

·         Windows Vista（32 位元和 64 位元）

·         Windows Server 2008（32 位元和 64 位元）

 

註： 此 HP LaserJe 軟體不支援 Windows 95、Windows 98、Windows ME、Windows NT、Windows 2000 及 Windows Server 2000。

如需安裝其他系統的相關說明，請參閱該特定系統之相關安裝注意事項、讀我檔案或其他產品資訊。

 

本檔案提供特定的安裝、相容性及疑難排解資訊。 如需有關裝置功能與驅動程式功能的詳細資訊，請參閱各軟體元件隨附的裝置使用指南及線上說明。

本軟體適用對象

所有使用者都需要此列印系統軟體才能夠操作 HP LaserJet Professional P1100/P1560/P1600 印表機系列。

列印系統元件概述

此光碟提供適用於 IBM 相容電腦系統與 Apple Macintosh（及相容）系統的軟體。 如需特定驅動程式或軟體元件的詳細資訊，請參閱相關線上說明。

II. 已知問題與暫時解決方法

一般

 

·         某些高速 USB 2.0 卡隨附未經簽署的主機控制器驅動程式。 如果您已經購買 USB 2.0 擴充卡，並且具有未經簽署的驅動程式，則您可能會遇到驅動程式安裝問題，或者裝置與 HP 軟體之間出現通訊問題。 建議 USB 硬體一律使用經過 Microsoft WHQL 認證的主機控制器驅動程式。

·         網站上會提供隨插即用的列印驅動程式套件，其 URL 如下：

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

·         系統上僅符合基本系統需求的功能可能會受到限制。 若要使用大量影像或視訊，必須配備符合建議系統需求的系統，才可達到合意的效能與穩定性。

·         疑難排解設定與安裝的問題

若無法辨識裝置：

  • 檢查纜線是否符合高速 USB 2.0 的規格。
  • 確認纜線連接無誤。 拔下 USB 纜線、查看線頭，並與 USB 連接埠的型樣相比對。 確定未上下顛倒。 將 USB 纜線的一端完全插入 HP 裝置的 USB 連接埠，再將另一端插入電腦背面的連接埠。
  • 檢查週邊設備的電源是否開啟。 週邊設備的電源若未開啟，便無法辨識。
  • 確認 USB 纜線的長度未超過 2 公尺或 6.5 英呎。
  • 檢查是否已啟用 USB 連接埠。 某些個人電腦會在出廠時停用 USB 連接埠。

升級 Windows

·         HP 建議您在升級為受支援的新版 Windows 之前，先解除安裝 HP 裝置。  例如，從 Windows XP 升級至 Windows Vista 時，可先使用解除安裝程式或「新增/移除程式」解除安裝 HP 裝置，然後再升級為新的作業系統。 當作業系統完成升級後，請使用下列 URL 所提供的最新軟體重新安裝 HP 裝置：

III. 安裝說明

在 Windows 環境中操作時，光碟安裝程式或安裝軟體會對您所執行之作業系統的版本與語言進行偵測。

以 HP 智慧型安裝在 WINDOWS 上安裝軟體 － 建議的安裝方式

註： 在 Windows XP 或 Windows Vista 作業系統上安裝列印系統必須具備管理員權限。

1.       將 USB 電線插入電腦和印表機。

2.       HP 智慧型安裝程式應可自動執行。 如果 HP 智慧型安裝程式未在 30 秒後啟動，請完成步驟 3 至 5。

3.       選取工作列之「開始」功能表中的「執行」。

4.       按一下瀏覽。 將會提供稱為 HP 智慧型安裝的新 CD ROM 光碟機。 瀏覽至此光碟機。

5.       選取 SISetup.EXE 檔案，按一下「開啟」，然後按一下「確定」。

6.       依照提示執行。

 

使用 HP 智慧型安裝在網路上安裝軟體

1.       在 Web 瀏覽工具的位址行輸入裝置的 IP 位址。

2.       按一下「HP 智慧型安裝」標籤。

3.       按一下「下載」按鈕。

4.       依照提示執行。

 

使用 CD ROM 在 WINDOWS 上安裝軟體

註： 在 Windows XP 或 Windows Vista 作業系統上安裝列印系統必須具備管理員權限。

由光碟安裝列印系統

1.       請勿將 USB/並列電纜線插入電腦系統。

2.       關閉所有程式。

3.       放入光碟。

4.       HP 列印系統安裝程式若未在 30 秒後啟動，請執行步驟 5 至 7。

5.       選取工作列之「開始」功能表中的「執行」。

6.       瀏覽至光碟機的根目錄。

7.       選取 Setup.EXE 檔案，按一下「開啟」，然後按一下「確定」。

8.       依照提示執行。

 

有關安裝 HP LASERJET P1100W 印表機系列的其他說明

HP LaserJet Professional P1100w 印表機系列印表機支援 Wi-Fi 802.11 b/g。  請確認您在開始安裝印表機之前，已經成功地設定無線路由器或存取點。  請備妥您的無線網路組態設定資訊，以備不時之需。

 

停用 HP 智慧型安裝以使用主機支援者以外的驅動程式

如果您想要使用 USB 支援的驅動程式，而不是主機支援的驅動程式（內含於 HP HP 智慧型安裝和 CD，例如 PCL5E 或 XPS），您需要先停用電腦或印表機上的 HP 智慧型安裝。

1.       將 USB 電線插入電腦和印表機。

2.       HP 智慧型安裝程式應可自動執行。 不需安裝主機支援的驅動程式即可結束此程式。

3.       放入 CD-ROM 光碟

4.       安裝程式應可自動執行。 不需安裝主機支援的驅動程式即可結束此程式。 

5.       瀏覽至 CD 上的 UTIL 目錄。

6.       選取 HPSmartInstallConfig.EXE 檔案，按一下「開啟」，然後按一下「確定」。

7.       此公用程式中將有兩個選項。 其中一個選項將會關閉電腦上的 HP 智慧型安裝程式，如果印表機連接另一部電腦，則會繼續執行。 另一個選項將會關閉印表機上的 HP 智慧型安裝。 如果選取此選項，印表機將不會執行任何電腦上的 HP 智慧型安裝，除非使用此公用程式重新啟用。

 

 

停用網路產品上的 HP 智慧型安裝

在系統管理員要求下，可在網路印表機停用 HP 智慧型安裝。

1.       在 Web 瀏覽工具的位址行輸入裝置的 IP 位址。

2.       按一下「設定」標籤。

3.       按一下列印設定功能表。

4.       若要停用 USB 支援的 HP 智慧型安裝，請將「啟用 USB HP 智慧型安裝」選項設定為「否」。

5.       若要停用網路支援的 HP 智慧型安裝，請將「顯示 HP 智慧型安裝下載選項」設定為「否」。

透過「新增印表機」進行安裝。

HP LaserJet 印表機也可以透過 Microsoft「印表機」資料夾中的「新增印表機」功能進行安裝 。 此功能也稱為「新增印表機精靈」或 APW。 APW 的指示說明在不同的 Windows 版本中可能略有不同。 若要透過 APW 安裝，您需要先依上述說明停用 HP 智慧型安裝。

 

WinXP APW 安裝 

1.         確認 HP LaserJet P1100/P1560/P1600 印表機系列尚未插入，也沒有連接電腦。

2.         從「開始」功能表，指向「控制台」，按一下「印表機和傳真」。

3.         按一下「新增印表機」，然後按一下「歡迎使用新增印表機精靈」螢幕上的「下一步」。

4.         遵照螢幕上的指示進行。

5.         當詢問您本產品是本機或網路印表機時，請選擇「本機印表機」（這是預設選擇）。 取消核取「自動偵測並安裝我的隨插即用印表機」核取方塊。  按「下一步」。

6a.         如果您透過 USB 進行安裝，而且之前從未安裝 HP LaserJet P1100/P1560/P1600 印表機系列，就不會安裝 USB 連接埠。 這是正常的情形。 選取「LPT1:」 並按「下一步」。

6b.         如果您透過 USB 進行安裝，而且之前已安裝及連接 HP LaserJet P1100/P1560/P1600 印表機系列，就會安裝 USB 連接埠。 選取此 USB 連接埠並按「下一步」

註：    確認您用來連接 HP LaserJet P1100/P1560/P1600 印表機系列的 USB 連接埠與之前所連接的連接埠相同。

7.         選取「從磁片安裝」，然後指定要安裝 HP LaserJet P1100/P1560/P1600 系列驅動程式的位置。

8.         如果出現多種選擇，請選取您的印表機機型，然後按「下一步」。

9.         遵照螢幕上的指示進行。

10.        當要求連接印表機時，請將 HP LaserJet P1100/P1560/P1600 印表機系列插入並連接到您的電腦。

 

註：    此時若系統沒有要求您連接 HP LaserJet P1100/P1560/P1600 印表機系列，則應在安裝完成時插入並連接到您的電腦。

 

11.        遵照螢幕上的指示進行，直到安裝完成。

 

當安裝完成時，您可能建立了兩份印表機驅動程式。 若出現這情形，您應該刪除所建立的第一份驅動程式。 一般來說，驅動程式的名稱會和印表機機型的名稱相同。 若要檢查驅動程式的建立份數並刪除其中的第一份，請遵循下列步驟

 

1.         從「開始」功能表指向「設定」，按一下「印表機」。

2.         如果您同時擁有「HP LaserJet P1100/P1560/P1600 系列」及「HP LaserJet P1100/P1560/P1600 系列 (Copy 2)」，請勿刪除名稱中含「Copy 2」者。 反之，請刪除第一個以「HP LaserJet P1100/P1560/P1600 系列」名稱建立的驅動程式。

 

此時，您已經可以使用「HP LaserJet P1100/P1560/P1600 系列 (Copy 2)」驅動程式進行列印。

利用 PCL5e 來列印

僅 HP LaserJet Professional P1600 印表機系列支援 PCL5e。

PCL5e 驅動程式位於下列 URL 的網站上：

http://www.hp.com/support/LJP1600series

 

利用 XPS 來列印

XPS 驅動程式位於印表機隨附的軟體光碟中，也可在網站上找到，其 URL 如下：

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

使用「新增印表機精靈」來安裝 XPS 驅動程式。 

 

解除安裝程式

安裝列印系統之後，可使用解除安裝程式移除所有的 HP 列印系統元件。

啟動 HP LaserJet Professional P1100 印表機系列適用的解除安裝程式

1.       從「開始」功能表的 HP LaserJet Professional P1100 系列程式群組中，按一下解除安裝圖示。

2.       列印系統會引導您完成移除所有列印系統元件的程序。

啟動 HP LaserJet Professional P1560/P1600 印表機系列適用的解除安裝程式

1.       從「開始」功能表的 HP LaserJet Professional P1560-P1600 系列程式群組中，按一下解除安裝圖示。

2.       列印系統會引導您完成移除所有列印系統元件的程序。
