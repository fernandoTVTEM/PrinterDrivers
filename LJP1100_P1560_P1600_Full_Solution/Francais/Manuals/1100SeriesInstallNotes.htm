I. INTRODUCTION

OBJECTIF DE CE DOCUMENT

Ce document propose des instructions d'installation spécifiques ainsi que tout ce que vous devriez savoir avant d'installer et d'utiliser le système d'impression. Le logiciel de l'imprimante HP LaserJet fonctionne avec les systèmes d'exploitation suivants :

·         Windows XP (32 bits et 64 bits)

·         Windows Server 2003 (32 bits et 64 bits)

·         Windows Vista (32 bits et 64 bits)

·         Windows Server 2008 (32 bits et 64 bits)

 

Remarque : Le logiciel de l'imprimante HP LaserJet ne prend pas en charge Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000 et Windows Server 2000.

Pour obtenir des instructions sur l'installation d'autres systèmes, reportez-vous aux notes d'installation correspondantes, aux fichiers LisezMoi ou à toute autre information produit relative à un système en particulier.

 

Ce fichier propose des informations spécifiques relatives à l'installation, à la compatibilité et au dépannage. Pour obtenir des informations supplémentaires sur les fonctions du périphérique et du pilote, reportez-vous au guide de l'utilisateur concernant le périphérique et à l'aide en ligne proposée avec chaque composant logiciel.

Tous les utilisateurs souhaitant utiliser les imprimantes HP LaserJet Professional série P1100/P1560/P1600 ont besoin de ce logiciel du système d'impression.

II. PROBLEMES CONNUS ET SOLUTIONS

Informations générales

 

·         Certaines cartes USB 2.0 haute vitesse sont fournies avec des pilotes de contrôleur hôte non signés. Si vous avez acheté une carte USB 2.0 additionnelle et que vous disposez d'un pilote non signé, il se peut que vous rencontriez des problèmes d'installation du pilote ou de communication entre le périphérique et le logiciel HP. Nous vous recommandons de toujours utiliser le pilote de contrôleur hôte certifié WHQL Microsoft avec votre matériel USB.

·         Des packs de pilotes d'impression Plug and Play sont disponibles sur Internet à l'adresse suivante :

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

·         Sur les systèmes qui répondent uniquement à la configuration minimale requise, les fonctionnalités risquent d'être limitées. Si vous utilisez un grand nombre d'images ou de vidéos, votre système doit répondre à la configuration recommandée pour présenter des performances et une fiabilité acceptables.

·         Dépannage de la configuration et de l'installation

Si votre périphérique n'est pas reconnu :

  • Vérifiez que votre câble est bien compatible avec la norme USB 2.0 haute vitesse.
  • Assurez-vous que votre câble est correctement branché. Retirez le câble USB, regardez son extrémité, puis connectez-le au port USB. Assurez-vous qu'il n'est pas branché à l'envers. Insérez l'une des extrémités du câble USB dans le port USB du périphérique HP et l'autre extrémité à l'arrière de l'ordinateur.
  • Vérifiez que le périphérique est sous tension. Les périphériques ne sont pas reconnus s'ils sont hors tension.
  • Veillez à ce que votre câble USB ne dépasse pas 2 mètres de long.
  • Vérifiez que le port USB est activé. Certains PC sont livrés avec des ports USB désactivés.

Mise à niveau de Windows

·         Il est recommandé de désinstaller les périphériques HP avant de procéder à la mise à niveau vers une version compatible de Windows plus récente.  Ainsi, lorsque vous procédez à une mise à niveau de Windows XP vers Windows Vista, désinstallez le périphérique HP à l'aide du logiciel de désinstallation ou de l'option d'ajout/suppression de programmes, puis effectuez la mise à niveau vers le nouveau système d'exploitation. Une fois la mise à niveau du système d'exploitation entièrement effectuée, réinstallez le périphérique HP en utilisant la dernière version du logiciel disponible à l'adresse suivante :

http://www.hp.com/support/LJP1100series 

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

III. INSTRUCTIONS D'INSTALLATION

Lorsque vous évoluez dans un environnement sous Windows, le logiciel de configuration ou d'installation contenu sur le CD-ROM détecte la version et la langue du système d'exploitation sur lequel vous travaillez.

INSTALLATION DU LOGICIEL SOUS WINDOWS A L'AIDE DE L'INSTALLATION INTELLIGENTE HP  METHODE D'INSTALLATION RECOMMANDEE

REMARQUE : Vous devez disposer de privilèges d'administrateur pour installer le système d'impression sous Windows XP ou Windows Vista.

1.       Connectez un câble USB à l'ordinateur et à l'imprimante.

2.       Le programme d'installation intelligente HP s'exécute automatiquement. S'il ne démarre pas dans les 30 secondes, passez aux étapes 3 à 5.

3.       Sélectionnez Exécuter dans le menu Démarrer de la barre des tâches.

4.       Cliquez sur parcourir. Un nouveau lecteur de CD-ROM nommé Installation intelligente HP. Allez jusqu'à ce lecteur.

5.       Sélectionnez le fichier SISetup.EXE, cliquez sur Ouvrir, puis sur OK.

6.       Suivez les invites qui s'affichent.

 

Installation du logiciel à l'aide de l'installation intelligente HP sur le réseau.

1.       Dans la barre d'adresse du navigateur Web, saisissez l'adresse IP du périphérique.

2.       Cliquez sur l'onglet Installation intelligente HP.

3.       Cliquez sur le bouton Télécharger.

4.       Suivez les invites qui s'affichent.

 

INSTALLATION DU LOGICIEL SOUS WINDOWS à l'aide du CD-ROM

REMARQUE : Vous devez disposer de privilèges d'administrateur pour installer le système d'impression sous Windows XP ou Windows Vista.

Installation du système d'impression à partir d'un CD-ROM

1.       Ne connectez pas le câble USB/parallèle à l'ordinateur.

2.       Fermez tous les programmes.

3.       Insérez le CD-ROM.

4.       Si le programme d'installation du système d'impression HP ne démarre pas dans les 30 secondes, passez aux étapes 5 à 7.

5.       Sélectionnez Exécuter dans le menu Démarrer de la barre des tâches.

6.       Cherchez le répertoire racine dans l'arborescence du CD-ROM.

7.       Sélectionnez le fichier Setup.EXE, cliquez sur Ouvrir, puis sur OK.

8.       Suivez les invites qui s'affichent.

 

Instructions supplémentaires pour installer l'imprimante HP LaserJet série P1100w

L'imprimante HP LaserJet Professional série P1100w prend en charge le Wi-Fi 802.11 b/g.  Assurez-vous que votre routeur ou votre point d'accès sans fil ont été installés correctement avant de commencer à installer votre imprimante.  Gardez les informations sur la configuration du réseau sans fil à portée de main au cas où vous en auriez besoin.

 

Désactiver l'installation intelligente HP pour utiliser un pilote qui ne soit pas basé sur hôte

Pour utiliser un pilote via USB autre que le pilote basé sur hôte standard inclus dans l'installation intelligente HP et le CD (par exemple, PCL5E ou XPs), vous devez commencer par désactiver l'installation intelligente HP sur l'ordinateur ou l'imprimante.

1.       Connectez le câble USB à l'ordinateur et à l'imprimante.

2.       Le programme d'installation intelligente HP s'exécute automatiquement. Quittez ce programme sans installer le pilote basé sur hôte.

3.       Insérez le CD-ROM.

4.       Le programme d'installation doit s'exécuter automatiquement. Quittez ce programme sans installer le pilote basé hôte. 

5.       Cherchez le répertoire UTIL sur le CD.

6.       Sélectionnez le fichier HPSmartInstallConfig.EXE, cliquez sur Ouvrir, puis sur OK.

7.       Cet utilitaire propose deux options. La première option désactive l'installation intelligente HP sur l'ordinateur. Ce programme est tout de même exécuté si l'imprimante est reliée à d'autres ordinateurs. La seconde option désactive l'installation intelligente HP sur l'imprimante. Si cette option est sélectionnée, l'imprimante n'exécute pas l'installation intelligente HP sur l'ordinateur quel qu'il soit, sauf si vous réactivez le programme à l'aide de cet utilitaire.

 

 

Désactiver l'installation intelligente HP sur les produits réseau

L'installation intelligente HP peut être désactivée sur les imprimantes réseau si l'administrateur le demande.

1.       Dans la barre d'adresse du navigateur Web, saisissez l'adresse IP du périphérique.

2.       Cliquez sur l'onglet Paramètres.

3.       Cliquez sur le menu Paramètres d'impression.

4.       Pour désactiver l'installation intelligente HP via USB, définissez l'option Activer l'installation intelligente HP USB sur Non.

5.       Pour désactiver l'installation intelligente HP via le réseau, définissez l'option Afficher l'option de téléchargement de l'installation intelligente HP sur Non.

Installation à l'aide de l'Assistant Ajout d'imprimante

L'imprimante HP LaserJet peut également être installée à l'aide de la fonction Ajout d'imprimante du dossier Imprimantes de Microsoft. Cette fonction s'appelle également Assistant Ajout d'imprimante. Les instructions de l'Assistant Ajout d'imprimante peuvent varier légèrement en fonction des versions de Windows. Pour installer via APW, commencez par désactiver l'installation intelligente HP en suivant les instructions ci-dessus.

 

Installation avec l'Assistant Ajout d'imprimante pour Windows XP 

1.         Vérifiez que l'imprimante HP LaserJet série P1100/P1560/P1600 n'est pas branchée et n'est pas connectée à votre ordinateur.

2.         Dans le menu Démarrer, sélectionnez Panneau de configuration, puis cliquez sur Imprimantes et télécopieurs.

3.         Cliquez sur Ajouter une imprimante, puis sur Suivant dans l'écran Assistant Ajout d'imprimante.

4.         Suivez les instructions à l'écran.

5.         Lorsque vous êtes invité à choisir une imprimante locale ou réseau, sélectionnez Imprimante locale (option sélectionnée par défaut). Désactivez la case à cocher Détection et installation automatique de l'imprimante Plug-and-Play.  Appuyez sur Suivant.

6a.         Si vous procédez à une installation via un port USB et que vous n'avez pas installé au préalable d'imprimante HP LaserJet série P1100/P1560/P1600, aucun port USB n'est installé pour l'instant. Tout cela est normal. Sélectionnez LPT1: et cliquez sur Suivant.

6b.         Si vous procédez à une installation via un port USB et que vous n'avez pas installé au préalable d'imprimante HP LaserJet série P1100/P1560/P1600, aucun port USB n'est installé pour l'instant. Sélectionnez ce port USB, puis cliquez sur Suivant.

Remarque :    Vérifiez que vous branchez l'imprimante HP LaserJet série P1100/P1560/P1600 sur le même port USB que celui sur lequel vous l'aviez précédemment branchée.

7.         Sélectionnez Disque fourni, puis spécifiez l'emplacement du pilote de l'imprimante HP LaserJet série P1100/P1560/P1600.

8.         Si plusieurs choix s'offrent à vous, sélectionnez le modèle de votre imprimante, puis cliquez sur Suivant.

9.         Suivez les instructions à l'écran.

10.        A l'étape de connexion de l'imprimante, branchez l'imprimante HP LaserJet série P1100/P1560/P1600 et connectez-la à votre ordinateur.

 

Remarque :    Si aucun message ne vous invite à connecter l'imprimante HP LaserJet série P1100/P1560/P1600, branchez l'imprimante et connectez-la à votre ordinateur à la fin de l'installation.

 

11.        Suivez les instructions à l'écran jusqu'à ce que l'installation soit terminée.

 

Une fois l'installation terminée, il se peut que vous disposiez de deux copies du pilote. Dans ce cas, vous devez supprimer la première copie. Habituellement, cette copie porte le même nom que votre modèle d'imprimante. Pour vérifier combien de pilotes ont été créés et effacer le premier, procédez de la manière suivante :

 

1.         Dans le menu Démarrer, sélectionnez Paramètres, puis cliquez sur Imprimantes.

2.         Si vous disposez de deux pilotes appelés respectivement HP LaserJet série P1100/P1560/P1600 et HP LaserJet P1100/P1560/P1600 (Copie 2), ne supprimez pas le pilote dont le nom mentionne Copie 2. Supprimez plutôt le premier pilote créé et appelé HP LaserJet série P1100/P1560/P1600.

 

A ce stade, vous pouvez imprimer à l'aide du pilote HP LaserJet série P1100/P1560/P1600 (Copie 2).

2.       Le système d'impression vous guide dans la suppression de tous les composants du système d'impression.

1.       A partir du menu Démarrer, dans le groupe de programmes de l'imprimante HP LaserJet Professional série P1560-P1600, cliquez sur l'icône de désinstallation.

2.       Le système d'impression vous guide dans la suppression de tous les composants du système d'impression.
