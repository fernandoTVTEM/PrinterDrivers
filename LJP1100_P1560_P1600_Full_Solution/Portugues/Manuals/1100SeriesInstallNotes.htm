I. INTRODUÇÃO

OBJETIVO DESTE DOCUMENTO

Este documento fornece instruções específicas de instalação e outras informações que você deve conhecer antes de instalar e usar o sistema de impressão. Este software da HP LaserJet funciona com os seguintes sistemas operacionais:

·         Windows XP (32 bits e 64 bits)

·         Windows Server 2003 (32 bits e 64 bits)

·         Windows Vista (32 bits e 64 bits)

·         Windows Server 2008 (32 bits e 64 bits)

 

Nota: este software da HP LaserJet não oferece suporte aos sistemas operacionais Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000 e Windows Server 2000.

Para obter instruções sobre a instalação de outros sistemas, consulte as notas de instalação associadas ou outras informações sobre produtos referentes a esse sistema específico.

 

Este arquivo fornece informações específicas sobre instalação, compatibilidade e solução de problemas. Para obter informações adicionais sobre recursos do dispositivo e do driver, consulte o guia do usuário do dispositivo e a ajuda on-line incluída em cada componente de software.

Todos os usuários precisam deste software de sistema de impressão para operar as impressoras HP LaserJet Professional séries P1100/P1560/P1600.

II. PROBLEMAS CONHECIDOS E SOLUÇÕES

Geral

 

·         Certas placas USB 2.0 de alta velocidade são fornecidas com drivers de controladores de host não assinados. Se você comprou uma placa add-in USB 2.0 e tem um driver anônimo, talvez tenha problemas na instalação do driver ou problemas de comunicação entre o dispositivo e o software da HP. É recomendável sempre usar um driver controlador de host Microsoft WHQL Certified com o seu hardware USB.

·         Pacotes de drivers para impressoras Plug and Play estão disponíveis na seguinte URL:

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

·         A funcionalidade em sistemas que atendem apenas aos requisitos mínimos do sistema pode ser limitada. O uso de muitas imagens ou vídeos exige um sistema que atenda aos requisitos de sistema recomendados para que haja desempenho e confiabilidade aceitáveis.

·         Solução de problemas de configuração e instalação

Se o seu dispositivo não for reconhecido:

  • Verifique se o cabo é um USB 2.0 de alta velocidade compatível.
  • Verifique se o cabo está conectado corretamente. Remova o cabo USB, examine sua extremidade e compare-a com o padrão da porta USB. Verifique se ela não está invertida. Insira por completo uma extremidade do cabo USB na porta USB do dispositivo HP e a outra extremidade na parte posterior do computador.
  • Verifique se o periférico está ligado. Os periféricos não serão reconhecidos se estiverem desligados.
  • Certifique-se de que o cabo USB não tenha mais de 2 m ou 6,5 m de comprimento.
  • Verifique se a porta USB está ativada. Alguns PCs são fornecidos com as portas USB desativadas.

Atualização do Windows

·         A HP recomenda a desinstalação de dispositivos HP antes de atualizar para uma versão mais recente do Windows.  Por exemplo, ao atualizar do Windows XP para o Windows Vista, desinstale o dispositivo HP usando o Desinstalador ou Adicionar ou remover programas e faça upgrade para o novo sistema operacional. Depois que a atualização do sistema operacional tiver sido concluída, reinstale o dispositivo HP usando o software mais recente obtido no seguinte URL:

http://www.hp.com/support/LJP1100series 

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

Outros aplicativos

III. INSTRUÇÕES DE INSTALAÇÃO

Ao operar em um ambiente Windows, o software de instalação ou de configuração do CD-ROM detecta a versão e o idioma do sistema operacional que você está executando.

INSTALAÇÃO DE SOFTWARES NO WINDOWS com o HP Smart install – Método de instalação RECOMENDADO

NOTA: São necessários privilégios de administrador para instalar o sistema de impressão no sistema operacional Windows XP ou Windows Vista.

1.       Conecte um cabo USB ao computador e à impressora.

2.       O programa HP Smart Install deve ser executado automaticamente. Se o HP Smart Install não tiver iniciado após 30 segundos, complete as etapas de 3 a 5.

3.       Selecione Executar a partir do menu Iniciar em sua barra de tarefas.

4.       Clique em procurar. Uma nova unidade de CDROM chamada HP Smart Install ficará disponível. Acesse essa unidade.

5.       Selecione o arquivo SISetup.EXE, clique em Abrir e em OK.

6.       Siga os prompts.

 

Instalação de SOFTWARES usando o HP Smart install pela rede

1.       Em um navegador da Web, digite o endereço IP do dispositivo na linha de endereço do navegador.

2.       Clique na guia HP Smart Install.

3.       Clique no botão Download.

4.       Siga os prompts.

 

INSTALAÇÃO DE SOFTWARES NO WINDOWS com CD-ROM

NOTA: São necessários privilégios de administrador para instalar o sistema de impressão no sistema operacional Windows XP ou Windows Vista.

Instalação do sistema de impressão a partir de um CD-ROM

1.       Não conecte o cabo USB/paralelo ao sistema de computador.

2.       Feche todos os programas.

3.       Insira o CD-ROM.

4.       Se o programa instalador do sistema de impressão HP  não tiver iniciado após 30 segundos, complete as etapas de 5 a 7.

5.       Selecione Executar a partir do menu Iniciar em sua barra de tarefas.

6.       Vá até o diretório raiz na unidade de CD-ROM.

7.       Selecione o arquivo Setup.EXE, clique em Abrir e em OK.

8.       Siga os prompts.

 

Instruções adicionais para a instalação da Impressora HP Laserjet série P1100w

A Impressora HP LaserJet Professional série P1100w oferece suporte a Wi-Fi 802.11 b/g.  Verifique se o roteador sem fio ou o ponto de acesso foram configurados com êxito antes de começar a instalar a impressora.  Tenha as informações de configuração da rede sem fio à mão, caso necessário.

 

Desabilite o HP Smart Install para usar um Driver que não seja baseado em host

Caso deseje usar um driver em USB, em vez do driver baseado em host padrão fornecido com o HP Smart Install e o CD (por exemplo, PCL5E ou XPS), desabilite o HP Smart Install no computador ou na impressora.

1.       Conecte o cabo USB ao computador e à impressora.

2.       O programa HP Smart Install deve ser executado automaticamente. Feche o programa sem instalar o driver baseado em host.

3.       Insira o CD-ROM

4.       O programa de instalação deve ser executado automaticamente. Feche o programa sem instalar o driver baseado em host. 

5.       Vá até o diretório UTIL no CD.

6.       Selecione o arquivo HPSmartInstallConfig.EXE, clique em Abrir e em OK.

7.       Haverá duas opções neste utilitário. Uma opção desativará o HP Smart Install no computador, mas ele ainda será executado se a impressora estiver conectada a outros computadores. A outra opção desativará o HP Smart Install na impressora. Se essa opção for selecionada, a impressora não executará o HP Smart Install em nenhum computador, a menos que ele seja habilitado novamente usando este utilitário.

 

 

Desabilitar o HP Smart Install em produtos de rede

O HP Smart Install pode ser desabilitado em impressoras na rede se o administrador solicitar.

1.       Em um navegador da Web, digite o endereço IP do dispositivo na linha de endereço do navegador.

2.       Clique na guia Configurações.

3.       Clique no menu Configurações de impressão.

4.       Para desabilitar o HP Smart Install em USB, configure a opção Habilitar HP Smart Install em USB para Não.

5.       Para desabilitar o HP Smart Install na rede, configure a opção Mostrar HP Smart Install baixado para Não.

Instalando por meio do recurso Adicionar impressora

A impressora HP LaserJet também pode ser instalada a partir do recurso Adicionar impressora da Microsoft, na pasta Impressoras. Ele também é conhecido como Assistente para adicionar impressora ou APW. As instruções do APW irão variar um pouco entre as diferentes versões do Windows. Para instalar usando o APW, desabilite o HP Smart Install usando as instruções acima.

 

Instalação APW WinXP 

1.         Certifique-se de que a impressora HP LaserJet séries P1100/P1560/P1600 não esteja plugada e conectada ao computador.

2.         No menu Iniciar, aponte para Painel de controle e clique em Impressoras e aparelhos de fax.

3.         Clique em Adicionar impressora e em Avançar na tela “Bem-vindo ao assistente para adicionar impressora”.

4.         Siga as instruções exibidas na tela.

5.         Quando for perguntado se a impressora é local ou em rede, selecione Impressora local (a seleção padrão). Desmarque a caixa de seleção Detectar e instalar automaticamente a impressora Plug and Play.  Pressione Avançar.

6a.         Se a instalação for executada via USB e não houver uma impresora HP LaserJet séries P1100/P1560/P1600 instalada, então ainda não haverá uma porta USB instalada. Isso é normal. Selecione LPT1: e clique em Avançar.

6b.         Se a instalação for executada via USB e houver uma impresora HP LaserJet séries P1100/P1560/P1600 instalada e conectada, então há uma porta USB instalada. Selecione esta porta USB e clique em Avançar.

Nota:    Verifique se a impressora HP LaserJet séries P1100/P1560/P1600 está conectada na mesma porta USB física da impressora conectada anteriormente.

7.         Selecione "Com disco" e especifique o local do driver da HP LaserJet séries P1100/P1560/P1600 a ser instalado.

8.         Se várias opções forem exibidas, selecione o modelo da sua impressora e clique em Avançar.

9.         Siga as instruções exibidas na tela.

10.        Se você receber um prompt para conectar a impressora, plugue e conecte a impressora HP LaserJet séries P1100/P1560/P1600 ao computador.

 

Nota:    Se você não receber um prompt para conectar a impressora HP LaserJet séries P1100/P1560/P1600, deverá plugá-la e conectá-la ao computador quando a instalação estiver concluída.

 

11.        Siga as instruções exibidas na tela até que a instalação esteja concluída.

 

Quando a instalação estiver concluída, você poderá fazer duas cópias do driver da impressora criado. Se for o caso, você deverá excluir o primeiro driver criado. Geralmente ele tem o mesmo nome do modelo da impressora. Para verificar quantos drivers foram criados e para excluir o primeiro driver, siga as etapas abaixo

 

1.         No menu Iniciar, aponte para Configurações e clique em Impressoras.

2.         Se você tiver "HP LaserJet séries P1100/P1560/P1600" e "HP LaserJet séries P1100/P1560/P1600 (Cópia 2)", não exclua a opção com "Cópia 2" no nome. Em vez disso, exclua o primeiro driver criado, denominado "HP LaserJet séries P1100/P1560/P1600"

 

Nesse ponto, você estará pronto para usar o driver "HP LaserJet séries P1100/P1560/P1600 (Cópia 2)".

Imprimindo com o PCL5e

O PCL5e só é compatível com a Impressora HP LaserJet Professional P1600.

O driver PCL5e está localizado na seguinte URL:

http://www.hp.com/support/LJP1600series

 

Imprimindo com o XPS

O driver XPS está localizado no CD do software fornecido com a impressora e na seguinte URL:

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

Use o Assistente para adicionar impressora de modo a instalar o driver XPS. 

 

DESINSTALADOR

Após a instalação do sistema de impressão, use o programa de desinstalação para remover todos os componentes do sistema de impressão da HP.

Inicialização do Desinstalador da Impressora HP LaserJet Professional série P1100

1.       No menu Iniciar, no grupo de programas da HP LaserJet Professional série P1100 , clique no ícone de desinstalação.

2.       O sistema de impressão o orientará na remoção de todos os componentes do sistema de impressão.

Inicialização do desinstalador da Impressora HP LaserJet Professional séries P1560/P1600

1.       No menu Iniciar, no grupo de programas da HP LaserJet Professional séries P1560-P1600 , clique no ícone de desinstalação.

2.       O sistema de impressão o orientará na remoção de todos os componentes do sistema de impressão.
