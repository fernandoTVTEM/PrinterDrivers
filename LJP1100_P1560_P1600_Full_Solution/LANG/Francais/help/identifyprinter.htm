Identification de l'imprimante

Si le programme d'installation ne détecte pas votre imprimante sur le réseau, vérifiez les points suivants :

  • Assurez-vous que l'imprimante est correctement connectée et sous tension.
  • Déterminez si un pare-feu est activé sur l'ordinateur. Si un pare-feu a été activé sur l'ordinateur, celui-ci peut empêcher la détection de l'imprimante par le programme d'installation. Désactivez temporairement le pare-feu sur l'ordinateur avant de poursuivre l'installation. Si vous disposez d'un pare-feu activé (tel que ZoneAlarm©, BlackICE©, etc.), consultez la documentation du constructeur du pare-feu pour obtenir des instructions sur sa désactivation. HP recommande vivement d'installer les éléments suivants sur l'ordinateur : (1) un périphérique matériel qui fournit une protection de pare-feu entre Internet et l'ordinateur, tel qu'une passerelle, et (2) un programme de détection de virus mis à jour.

    • Informations supplémentaires sur les pare-feu

Une fois que vous avez vérifié les points ci-dessus, sélectionnez une des méthodes suivantes pour rechercher votre imprimante :

  • Choisir dans la liste des imprimantes détectées. Vous pouvez effectuer une sélection à partir d'une liste d'imprimantes détectées disponibles sur le réseau local. Lorsque vous cliquez sur Suivant, l'utilitaire recherche les imprimantes réseau et les affiche dans une liste. Vous pouvez ensuite sélectionner l'imprimante souhaitée dans la liste.

  • Spécifier une imprimante par son adresse. Vous pouvez rechercher une imprimante spécifique par son adresse. Lorsque vous cliquez sur Suivant, vous êtes invité à identifier l'imprimante par son adresse matérielle LAN, son adresse IP ou son nom d'hôte IP.

  • Entrer les paramètres d'une imprimante ne se trouvant pas sur le réseau. Sélectionnez cette option si vous configurez un port pour une imprimante non connectée au réseau. Pour plus d'informations sur cette option, consultez la documentation livrée avec l'imprimante.
