Vérification de la connexion de l'imprimante

Si vous vous connectez à l'imprimante sans fil, vérifiez que l'indicateur sans fil de celle-ci est activé (voir la documentation livrée avec l'imprimante pour obtenir de l'aide sur l'identification de l'indicateur sans fil de l'imprimante). Sélectionnez ensuite l'une des options affichées sur l'écran décrit ci-dessous.

Si vous vous connectez à l'aide d'un câble réseau, vérifiez les voyants au niveau du port réseau, situé à l'arrière de l'imprimante :

Si le voyant de liaison est ... Alors ...

Allumé et vert

 Sélectionnez l'une des options affichées sur l'écran décrit ci-dessous. Pour connaître l'adresse matérielle (MAC), l'adresse IP ou le nom de périphérique (nom d'hôte IP) de l'imprimante, imprimez une page de configuration réseau. Pour obtenir les instructions d'impression d'une page de configuration réseau, consultez la documentation qui accompagne l'imprimante.

Eteint
  • Vérifiez que le câble réseau est connecté à l'imprimante et au point d'accès sans fil (WAP) ou à l'ordinateur.
  • Vérifiez que le point d'accès sans fil (WAP) est mis sous tension.
  • Vérifiez que l'imprimante est mise sous tension.
  • Vérifiez que l'ordinateur est connecté au réseau.

Lorsque vous avez vérifié les éléments cités plus haut, choisissez un mode pour rechercher à nouveau l'imprimante :

  • Sélectionnez Rechercher toutes les imprimantes disponibles pour que l'assistant d'installation lance une nouvelle recherche.
  • Sélectionnez Spécifier une imprimante par son adresse pour entrer manuellement l'adresse matérielle (MAC), l'adresse IP ou le nom d'hôte IP (nom de périphérique) de l'imprimante.
  • Sélectionnez Entrer les paramètres d'une imprimante ne se trouvant pas sur le réseau si vous configurez un port pour une imprimante qui n'est pas connectée au réseau. Pour plus d'informations sur cette option, consultez la documentation livrée avec l'imprimante.
