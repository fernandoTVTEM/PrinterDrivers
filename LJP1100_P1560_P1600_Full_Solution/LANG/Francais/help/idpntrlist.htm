Imprimantes trouvées

L'écran Imprimantes trouvées vous permet de sélectionner une imprimante qui a été détectée sur le réseau local ou d'obtenir des options supplémentaires pour l'installation des imprimantes connectées via le réseau, via un câble USB ou via un câble parallèle.

  • Installer une imprimante réseau détectée. Sélectionnez l'imprimante de votre choix dans la liste contenant toutes les imprimantes (ou les périphériques multifonction) détectées sur le réseau local, puis cliquez sur Suivant. Cliquez sur Nouvelle recherche pour lancer une autre recherche.
  • M'aider à trouver mon imprimante réseau. Sélectionnez cette option si l'imprimante réseau de votre choix ne figure pas dans la liste ou si vous souhaitez installer une imprimante qui sera directement connectée à votre ordinateur via un câble USB ou parallèle.  Dans le cas d'imprimantes réseau, cette option fournit des options de recherche supplémentaires, comme la recherche par adresse IP. 

Remarque : Pour configurer des imprimantes qui ne se trouvent pas sur le réseau local, reportez-vous à Ajout d'imprimantes sur les grands réseaux IP.

Vous pouvez identifier les imprimantes de la liste avec les champs suivants. L'adresse matérielle constitue probablement le meilleur moyen pour identifier votre imprimante.

Modèle Identifie le modèle de l'imprimante (HP LaserJet série 5000, par exemple). Le message Inconnu s'affiche si le modèle n'a pas pu être identifié, si l'imprimante est hors tension ou si aucune imprimante n'est connectée. Dans le cas de serveurs d'impression externes HP Jetdirect, les messages suivants s'affichent si une imprimante est connectée mais n'a pas pu être identifiée :
  • Parallèle : imprimante/traceur Centronics
  • USB (unidirectionnel) : périphérique unidirectionnel
  • USB (bidirectionnel) : périphérique bidirectionnel
Adresse matérielle (également appelée adresse de station ou adresse MAC). Permet d'identifier de façon unique un périphérique sur le réseau local. Dans le cas d'imprimantes HP et de serveurs d'impression externes HP Jetdirect disposant de plusieurs ports, un numéro de port (Port 1, Port 2, Port 3) peut être ajouté. Cette adresse figure sur la page de configuration (test) du serveur d'impression.
Adresse IP Permet d'identifier de façon unique un périphérique situé sur un réseau TCP/IP. L'adresse IP est attribuée par l'administrateur réseau. Les adresses IPv4 et IPv6 s'affichent dans cette liste.  Si une imprimante a plusieurs adresses, elle s'affiche une fois pour chaque adresse pour vous permettre de sélectionner l'interface à utiliser. 

Remarque : Si un serveur d'impression HP Jetdirect est mis sous tension et qu'il n'est pas configuré avec une adresse IP valide au bout de deux minutes, une adresse AutoIP (169.254.x.x) ou une adresse IP par défaut (192.0.0.192) lui est automatiquement attribuée. HP conseille de modifier l'adresse IP par défaut. Si une adresse par défaut identique est utilisée pour plusieurs imprimantes, vérifiez l'adresse matérielle pour identifier l'imprimante de votre choix.
Nom d'hôte IP Le nom d'hôte IP est un nom convivial et facilement identifiable que vous pouvez utiliser à la place de l'adresse IP de l'imprimante. Vous pouvez l'utiliser lorsqu'un serveur de noms a été configuré sur le réseau pour mapper les noms d'hôtes à leurs adresses IP. Si un nom d'hôte a été attribué et configuré sur l'imprimante (par exemple, imprimante1.marketing.com), vous pouvez spécifier le nom d'hôte pour identifier l'imprimante.

Remarque : Si une entrée de champ est tronquée et que vous souhaitez voir le champ en entier, redimensionnez la largeur du champ. Pour cela, placez le curseur à la limite du titre du champ, puis cliquez et faites glisser la souris pour modifier la taille du champ.

Pour obtenir des instructions sur l'impression d'une page de configuration réseau, reportez-vous à la documentation fournie avec l'imprimante.

Si une imprimante sélectionnée est protégée par un nom de communauté SNMP ou un mot de passe Jetdirect, vous serez invité à fournir les informations appropriées.
