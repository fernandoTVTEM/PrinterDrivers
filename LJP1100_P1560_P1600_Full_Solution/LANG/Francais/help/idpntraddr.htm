Paramètres de l'imprimante

L'écran Paramètres de l'imprimante permet de spécifier les paramètres d'une imprimante non située sur le réseau.

Vous pouvez entrer une ou plusieurs des adresses suivantes pour l'imprimante, puis cliquer sur Suivant. La saisie de plus d'une adresse permet d'assurer que le port créé sera connecté à l'imprimante spécifiée.

  • Nom d'hôte IP. Un nom d'hôte IP est un nom convivial (ou simple à identifier) qui peut être utilisé à la place de l'adresse IP de l'imprimante.
  • Adresse matérielle. Une adresse matérielle LAN (également appelée adresse de station ou adresse MAC) identifie de manière unique un périphérique.
  • Adresse IP (IPv4). Une adresse IP (Protocole Internet) identifie de manière unique chaque périphérique présent sur un réseau TCP/IP. Elle est généralement attribuée par un administrateur réseau. Cet utilitaire permet uniquement de configurer manuellement des paramètres IPv4 sur le périphérique.

Remarque : Pour configurer des imprimantes non situées sur le réseau local, voir la section Ajout d'imprimantes sur les grands réseaux IP.
