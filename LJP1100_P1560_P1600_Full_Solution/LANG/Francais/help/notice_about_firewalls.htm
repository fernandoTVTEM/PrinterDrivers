Pare-feu détecté

De manière à communiquer au travers du pare-feu Windows, le programme d'installation s'ajoute lui-même à la liste d'exceptions d'application du pare-feu selon les spécifications Microsoft. Ceci permet au programme d'installation de communiquer avec les imprimantes sur le réseau pour l'installation.

Cliquez sur Suivant pour rechercher les imprimantes sur le réseau et poursuivre l'installation. Sinon, cliquez sur Annuler pour quitter le programme d'installation.

Si vous disposez d'un pare-feu activé (tel que ZoneAlarm©, BlackICE©, Sygate Security Agent© ou Norton Personal Firewall©, etc.), consultez la documentation du fournisseur pour plus d'informations sur la désactivation du pare-feu.

Lorsque le pare-feu logiciel est désactivé, HP recommande fortement que les éléments suivants soient installés sur l'ordinateur :

  • Périphérique assurant une protection de type pare-feu entre Internet et l'ordinateur (par exemple, une passerelle).
  • Programme de détection des virus mis à jour.

Pour obtenir des informations générales sur les problèmes de communication, consultez la documentation fournie avec l'imprimante et visitez le site d'assistance à l'adresse http://www.hp.com/support.

Les pare-feux génèrent fréquemment des problèmes de communication. Lorsqu'un pare-feu est activé sur un ordinateur en réseau, plusieurs problèmes sont susceptibles de se produire :

    Lors de l'installation du logiciel :

  • Le programme d'installation de l'imprimante HP ne repère pas cette dernière sur le réseau. Un message d'erreur signale que le programme d'installation n'a pas pu détecter d'imprimante sur le réseau. Pour éviter ce problème, désactivez temporairement votre pare-feu lors de l'installation de l'imprimante HP. Consultez la documentation du fournisseur pour plus d'informations sur la désactivation du pare-feu.

    Une fois le logiciel installé :

  • En cas de changement d'adresse IP de l'imprimante HP, l'ordinateur ne peut plus communiquer avec cette dernière. Tous les travaux d'impression, de numérisation, de télécopie et de transfert d'images échouent et l'état du produit indique que l'imprimante HP est déconnectée. Dans un réseau domestique, il est normal que l'adresse IP change parfois. Lorsque le pare-feu logiciel est désactivé, l'ordinateur reçoit la nouvelle adresse et peut continuer à communiquer avec l'imprimante HP. Pour éviter ce problème, désactivez le pare-feu ou choisissez une adresse IP statique (si vous êtes un utilisateur averti des environnements en réseau). Pour connaître la procédure suggérée visant à configurer l'imprimante avec une adresse IP statique, visitez le site Web HP (http://www.hp.com/support).
  • Si vous disposez d'un périphérique HP tout-en-un, vous ne pouvez pas utiliser les logements de carte mémoire de l'appareil pour transférer des fichiers vers l'ordinateur. En effet, les fichiers présents sur la carte photo ne sont pas accessibles à partir du lecteur réseau connecté ou du logiciel de transfert d'images.

Pour plus d'informations sur les pare-feux, consultez le site Web HP.

Désactivation du pare-feu Windows dans Windows Vista :

  1. Cliquez sur Démarrer > Réseau.
  2. Cliquez avec le bouton droit sur Réseau, puis sélectionnez Propriétés.
  3. Cliquez sur Pare-feu Windows dans le coin inférieur gauche.
  4. Sélectionnez Activer ou désactiver le Pare-feu Windows.
  5. Cliquez sur Continuer dans la boîte de dialogue de contrôle d'accès d'utilisateur.
  6. Sélectionnez l'option Désactivé pour désactiver le pare-feu, puis cliquez sur OK. Le pare-feu Windows est désactivé.

Désactivation du pare-feu de connexion Internet sous Windows XP :

  1. Dans la barre des tâches située en bas de l'écran, cliquez sur Démarrer, puis sur Panneau de configuration.
  2. Cliquez sur la catégorie Connexions réseau et Internet. (Si Connexions réseau et Internet n'est pas visible, cliquez sur Basculer vers l'affichage des catégories sous Panneau de configuration dans la partie gauche de la fenêtre du Panneau de configuration.)
  3. Cliquez sur Connexions réseau.
  4. Cliquez avec le bouton droit de la souris sur Accès à distance, Réseau local ou Internet haut débit selon votre connexion à Internet, puis cliquez sur Propriétés dans le menu contextuel.
  5. Dans l'onglet Paramètres avancés, sous Pare-feu de connexion Internet, décochez la case Protéger mon ordinateur et le réseau, puis cliquez sur OK. Le pare-feu Windows XP est désactivé.
  6. Dans l'assistant d'installation, cliquez sur Suivant pour continuer l'installation de l'imprimante.
  7. Pour activer le pare-feu de connexion Internet une fois l'imprimante installée, suivez les étapes décrites avant l'utilisation de l'option Protéger mon ordinateur et le réseau. Cochez la case Protéger mon ordinateur et le réseau.

Remarque : N'oubliez pas de réactiver le pare-feu Windows une fois l'installation terminée.
