Ajout d'imprimantes sur les grands réseaux IP

Si vous ajoutez des imprimantes (ou périphériques multifonction) à travers un routeur ou une passerelle (en d'autres termes, des imprimantes qui ne font pas partie du réseau local), leur identification par cet utilitaire risque de s'avérer difficile :

  • Il se peut que les routeurs ne transmettent pas au travers de l'adresse matérielle, MAC ou de station de l'imprimante.
  • Si le réseau local comporte plusieurs nouvelles imprimantes, l'adresse IP par défaut (192.0.0.192) ne permet pas d'identifier une imprimante sans ambiguïté. Vérifiez l'adresse matérielle pour identifier l'imprimante correcte.

Pour installer une imprimante à l'aide de cet utilitaire, procédez de la manière suivante :

  1. Effectuez l'une des opérations suivantes :
    • Déplacez physiquement l'imprimante sur le réseau local de l'ordinateur ou
    • Exécutez cet utilitaire sur un ordinateur connecté au réseau local de l'imprimante.
  2. Identifiez l'imprimante conformément aux instructions de cet utilitaire et configurez des paramètres IP qui correspondent à l'emplacement définitif de l'imprimante sur le réseau. Si vous devez rechercher l'adresse MAC, vous pouvez la trouver en imprimant une page de configuration (test) (à partir de l'imprimante ou du serveur d'impression externe) pour vous assurer de sélectionner l'adresse MAC ou matérielle correcte.
  3. Si vous avez déplacé l'imprimante (ou le serveur d'impression HP Jetdirect externe), remettez-la à son emplacement définitif sur le réseau.
  4. Pour installer cette imprimante sur d'autres ordinateurs, effectuez l'une des opérations suivantes :
    • Exécutez cet utilitaire pour configurer chaque ordinateur ou serveur qui imprimera directement sur cette imprimante ou
    • Pour les clients Windows qui imprimeront sur cette imprimante via un système partagé, exécutez l'utilitaire Ajout d'imprimante du système client dans le dossier Imprimantes ou utilisez un programme d'installation d'imprimante partagée créé pour cette imprimante.
