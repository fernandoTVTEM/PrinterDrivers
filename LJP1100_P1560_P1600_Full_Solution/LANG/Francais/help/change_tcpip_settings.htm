Modification des paramètres TCP/IP

Vous pouvez spécifier de nouveaux paramètres TCP/IP (IPv4) pour cette imprimante. Il est recommandé de consulter l'administrateur réseau avant de modifier des paramètres.

Remarque : L'utilitaire d'installation détecte la configuration du système et vous recommande des paramètres TCP/IP (IPv4 et IPv6), mais vous pouvez modifier les paramètres IPv4 si vous le souhaitez (cet utilitaire permet uniquement de configurer manuellement des paramètres IPv4). Les paramètres définis ne s'appliquent pas tant que vous n'avez pas terminé l'installation de l'imprimante.

·         Si l'imprimante n'a pas de paramètres TCP/IP corrects :
Entrez une adresse IP (IPv4) correcte pour l'imprimante, un masque de sous-réseau correct et une adresse IP correcte pour la passerelle dans les champs appropriés. Nous vous recommandons de consulter l'administrateur réseau avant d'attribuer des valeurs.

Cliquez sur Paramètres suggérés pour que l'utilitaire d'installation entre automatiquement les paramètres TCP/IP.

Utiliser le nom du port par défaut (Windows NT, 2000 et XP). Cochez cette case pour utiliser les paramètres de port par défaut de l'imprimante. Si cette case n'est pas cochée, la boîte de dialogue Nom du port s'affiche à la suite de l'écran de l'assistant.

ATTENTION : Si l'imprimante doit être connectée à d'autres réseaux, l'unicité des paramètres TCP/IP suggérés par l'utilitaire n'est plus garantie et ceux-ci risquent d'entrer en conflit avec ceux d'autres périphériques. Soyez prudent en cas de connexion à plusieurs réseaux TCP/IP.

·         Si l'imprimante a déjà des paramètres TCP/IP corrects :
Les paramètres TCP/IP actuels s'affichent.

ATTENTION : La modification de paramètres TCP/IP corrects risque d'empêcher d'autres utilisateurs d'accéder à l'imprimante. Consultez l'administrateur réseau avant de modifier des paramètres corrects.

Remarque : Si les paramètres TCP/IP sont protégés par un mot de passe défini à l'aide d'un logiciel de gestion, tel que HP Web Jetadmin, les champs de saisie sont grisés.

Utiliser le nom du port par défaut (Windows NT, 2000 et XP uniquement). Cochez cette case pour utiliser les paramètres de port par défaut de l'imprimante. Si cette case n'est pas cochée, la boîte de dialogue Nom du port s'affiche à la suite de l'écran de l'assistant.
