Impressoras encontradas

A tela Impressoras encontradas permite selecionar uma impressora localizada na rede local ou obter opções adicionais de configuração para a instalação de impressoras com conexão paralela, via rede ou USB.

  • Instalar uma impressora localizada na rede Selecione a impressora desejada na lista de todas as impressoras (ou dispositivos multifuncionais) localizadas na rede local e clique em Avançar. Clique no botão Pesquisar de novo para fazer outra busca por impressoras na rede.
  • Ajude-me a encontrar minha impressora. Selecione essa opção se a sua impressora de rede desejada não estiver listada ou se você quiser instalar uma impressora que será conectada diretamente ao seu computador usando um cabo USB ou paralelo.  Para impressoras de rede, essa opção fornecerá opções adicionais de pesquisa, como a busca de uma impressora por Endereço IP. 

Nota: para configurar impressoras não localizadas na rede local, consulte Adição de impressoras em grandes redes IP.

As impressoras na lista são identificadas pelos campos a seguir. O Endereço do hardware pode ser a forma mais simples de identificar a impressora:

Modelo Identifica o modelo da impressora (por exemplo, HP LaserJet série 5000). Desconhecido aparecerá se o modelo não puder ser identificado, se a impressora estiver desligada ou se não houver impressora conectada. Para servidores de impressão externos HP Jetdirect, o seguinte será exibido se uma impressora estiver conectada, mas não puder ser identificada:
  • Paralelo: Impressora/plotadora Centronics
  • USB (unidirecional): Periférico unidirecional
  • USB (bidirecional): Periférico bidirecional
Endereço do hardware (também chamado de Endereço da estação ou Endereço MAC). Identifica exclusivamente um dispositivo na rede local. Para impressoras HP e servidores de impressão externos HP Jetdirect com várias portas de impressora, um número de porta (Porta 1, Porta 2, Porta 3) pode ser acrescentado. Esse endereço é listado na página de configuração (teste) do servidor de impressão.
Endereço IP (Protocolo Internet) Identifica exclusivamente um dispositivo em uma rede TCP/IP e é atribuído por um administrador de rede. Essa lista mostrará endereços IPv4 e IPv6.  Se uma impressora tiver vários endereços, ela aparecerá uma vez para cada endereço, permitindo que você selecione a interface que deseja usar. 

Nota: se um servidor de impressão HP Jetdirect for ligado, mas não for configurado com um endereço IP válido dentro de dois minutos, um Endereço IP automático (169.254.x.x) ou um Endereço IP padrão (192.0.0.192) será automaticamente atribuído. A HP recomenda que o endereço IP padrão seja alterado. Se houver várias impressoras com o mesmo endereço padrão, verifique o Endereço do hardware para identificar a impressora correta.
Nome do host IP Um nome de host IP é um nome amigável (ou de fácil identificação) que pode ser usado no lugar do endereço IP da impressora. Ele pode ser usado quando um servidor de nomes na rede foi configurado para mapear nomes de host aos seus endereços IP correspondentes. Se um nome de host tiver sido atribuído e configurado na impressora (por exemplo, impressora1.marketing.com), você poderá especificar esse nome de host para identificá-la.

Nota: se uma entrada de campo estiver truncada e você quiser visualizar o campo inteiro, redimensione a largura desse campo. Para redimensionar, posicione o cursor no limite do título do campo e clique e arraste em seguida.

Para obter instruções sobre como imprimir uma página de configuração de rede, consulte a documentação que acompanha a impressora.

Se uma impressora selecionada estiver protegida por um Nome de comunidade SNMP ou uma senha Jetdirect, será necessário especificar esse Nome de comunidade ou senha.
