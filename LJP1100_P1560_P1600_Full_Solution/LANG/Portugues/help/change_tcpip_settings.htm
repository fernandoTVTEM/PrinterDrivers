Alterar configurações de TCP/IP

Você pode especificar novas configurações de TCP/IP (IPv4) para esta impressora. Você deve consultar o Administrador de rede antes de alterar quaisquer configurações.

Nota: O utilitário de instalação detectará as configurações do sistema e apresentará as configurações de TCP/IP recomendadas (ambas IPv4 e IPv6). No entanto, você pode alterar as configurações de IPv4 se desejar (esse utilitário pode ser utilizado somente para configurar manualmente as configurações de IPv4). Os valores de parâmetros atribuídos não serão configurados na impressora até que você tenha concluído a instalação.

·         Se a impressora não tiver configurações de TCP/IP válidas:
Digite um endereço IP válido (IPv4) para a impressora, uma máscara de sub-rede e um endereço IP válidos para o gateway, nos campos fornecidos. Você deve consultar o Administrador de rede antes de atribuir valores.

Clique em Sugerir configurações para que o utilitário de instalação insira automaticamente os parâmetros TCP/IP.

Caixa de seleção "Usar nome de porta padrão" (Windows NT, 2000 e XP). Marque essa caixa para usar as configurações de porta padrão para esta impressora. Se essa caixa não estiver marcada, a caixa de diálogo Nome da porta será a próxima tela do assistente.

ATENÇÃO: Se a impressora estiver destinada a uma conexão com outras redes, as configurações de TCP/IP sugeridas pelo utilitário poderão não ser exclusivas globalmente e poderão provocar conflitos com outros dispositivos. Tenha cuidado ao se conectar a outras redes TCP/IP.

·         Se a impressora já tiver configurações de TCP/IP válidas:
As configurações de TCP/IP atuais serão exibidas.

ATENÇÃO: A alteração de configurações de TCP/IP válidas pode impedir o acesso de outros usuários. Consulte o Administrador da rede antes de alterar configurações válidas.

Nota: Se os parâmetros de TCP/IP estiverem protegidos por senha por meio de um software de gerenciamento, como o HP Web Jetadmin, os campos de entrada estarão acinzentados.

Caixa de seleção "Usar nome de porta padrão" (Windows NT, 2000 e XP somente). Marque essa caixa para usar as configurações de porta padrão para esta impressora. Se essa caixa não estiver marcada, a caixa de diálogo Nome da porta será a próxima tela de assistente.
