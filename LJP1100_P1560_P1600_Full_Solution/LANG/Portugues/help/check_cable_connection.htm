Como verificar a conexão da impressora

Se você estiver utilizando uma conexão sem fio para a impressora, verifique se o indicador de conexão sem fio associado está aceso (consulte a documentação que acompanha a impressora para obter ajuda sobre a identificação do indicador de conexão sem fio da impressora).  Em seguida, selecione uma das opções na tela abaixo.

Se você estiver conectando com um cabo de rede, verifique as luzes da porta de rede na parte posterior da impressora:

Se a luz de conexão estiver… Então...

Acesa e verde

 Em seguida, selecione uma das opções na tela abaixo.  Para localizar o endereço de hardware (endereço MAC), o endereço IP ou o nome do dispositivo (nome de host IP) da impressora, imprima uma página de configuração da rede. Para obter instruções sobre como imprimir uma página de configuração da rede, consulte a documentação fornecida com a impressora.

Apagada
  • Verifique se o cabo de rede está firmemente conectado à impressora e ao Wireless Access Point (Ponto de acesso sem fio) (WAP) ou ao computador.
  • Verifique se o WAP está ativado.
  • Verifique se a impressora está ligada.
  • Verifique se o computador está conectado à rede.

Após verificar os itens listados acima, escolha um método para procurar novamente a impressora:

  • Selecione Procurar novamente para que o assistente de instalação faça outra procura.
  • Selecione Especificar uma impressora pelo endereço para inserir manualmente o endereço de hardware (endereço MAC), o endereço IP ou o nome de dispositivo (nome de host IP) da impressora.
  • Selecione Inserir configurações para uma impressora que não está na rede se estiver configurando uma porta para uma impressora que não esteja conectada na rede. Para obter mais informações sobre esta opção, consulte a documentação que acompanha a impressora.
