偵測到防火牆

為了穿過 Windows 防火牆進行通訊，安裝程式會依 Microsoft 規格，將自己加入防火牆例外清單中。 這樣可以允許安裝程式與網路上的印表機溝通安裝資訊。

按一下「下一步」，搜尋網路上的印表機，並繼續安裝。 否則按「取消」結束安裝程式。

如果您已啟用防火牆（例如 ZoneAlarm©、BlackICE©、Sygate Security Agent© 以及 Norton Personal Firewall© 等等），請參閱您防火牆製造廠商的說明文件，以取得有關如何停用防火牆的指示說明。

不論軟體防火牆何時停用，HP 都強烈建議下列各項務必安裝在您的電腦上：

  • 在網際網路與電腦之間提供防火牆防護的硬體裝置，例如通訊閘。
  • 最新病毒偵測程式。

如需通訊疑難排解的一般資訊，請參閱印表機隨附的說明文件，並造訪支援網站，網址為 http://www.hp.com/support。

防火牆經常會造成通訊問題。如果在網路電腦上啟用防火牆，可能會發生下列問題：

   軟體安裝期間：

  • HP 印表機安裝程式無法在網路上找到 HP 印表機。 錯誤訊息會顯示安裝程式無法在網路上偵測到任何印表機。 若要避免這個問題，可暫時在 HP 印表機軟體安裝程序期間停用防火牆。 請參閱您防火牆製造廠商的說明文件，以取得有關如何停用防火牆的指示說明。

   軟體安裝完成後：

  • 如果 HP 印表機的 IP 位址經過變更，您的電腦將無法與 HP 印表機進行通訊。 所有的列印、掃瞄、PC 傳真及影像傳輸等等工作都將無法進行，並且產品狀態會顯示 HP 印表機的連線已經中斷。 在家用網路環境中，IP 位址有時會變更是正常的現象。 如果軟體防火牆未啟用，新的位址會傳送至電腦，這會允許電腦和 HP 印表機繼續進行通訊。 若要避免這項問題，可停用防火牆，或者，如果您是熟悉網路的進階使用者，您可以選擇改用靜態 IP 位址。 如需如何將印表機設定為使用靜態 IP 位址的建議，請參閱 HP 網站，網址為 http://www.hp.com/support。
  • 如果您使用 HP 多功能事務機，您將無法在 HP 多功能事務機上使用記憶卡擴充槽，將檔案傳輸至您的電腦。 這是因為相片卡上的檔案無法從對應的網路磁碟或影像傳輸軟體存取。

若要瞭解更多關於防火牆的資訊，您可以造訪 HP 網站。

如何在 Windows Vista 中停用 Windows 防火牆：

  1. 按一下「開始」>「網路」。
  2. 在「網路」上按一下滑鼠右鍵，然後選擇「內容」。
  3. 在左下角按一下「Windows 防火牆」。
  4. 選擇「Windows 防火牆開啟/關閉」。
  5. 在使用者存取控制對話方塊中選擇「繼續」。
  6. 選擇「關閉」選項來停用防火牆，然後按一下「確定」。 Windows 防火牆現已停用。

如何停用 Windows XP 上的網際網路連線防火牆 (ICF)：

  1. 在畫面底部的工具列上，按一下「開始」，然後按一下「控制台」。
  2. 按一下「網路和網際網路連線」類別目錄。（如果「網路和網際網路連線」並未顯示，請按一下「控制台」視窗左側「控制台」下方的「切換到類別目錄檢視」）。
  3. 按一下「網路連線」。
  4. 請用滑鼠右鍵按一下您用來連接至網際網路的「撥號」、「區域網路」或「高速網際網路」連線，然後再按一下內容功能表中的「內容」。
  5. 在「進階」索引標籤的「網際網路連線防火牆」下方，取消選取或停用「以限制或防止來自網際網路對這台電腦的存取來保護我的電腦」，然後按一下「確定」。Windows XP 防火牆現已停用。
  6. 在設定精靈中按一下「下一步」來繼續安裝印表機。
  7. 若要在印表機安裝完成後啟用「網際網路連線防火牆」，請遵循上述步驟前往「以限制或防止來自網際網路對這台電腦的存取來保護我的電腦」選項。然後重新選取「以限制或防止來自網際網路對這台電腦的存取來保護我的電腦」選項。

注意：安裝完成後，請務必重新啟用 Windows 防火牆。
