在大型 IP 網路上新增印表機

如果您正跨越路由器或通訊閘﹝不在區域網路上﹞新增印表機﹝或多功能裝置﹞，透過此公用程式識別印表機可能會有困難：

  • 路由器可能不會傳送印表機的硬體、MAC 或工作站等等的位址。
  • 區域網路上裝配有多部新的印表機時，若使用預設的 IP 位址﹝192.0.0.192﹞來識別印表機，則無法保證可以選擇到正確的印表機。請檢查硬體位址，以識別正確的印表機。

若要使用此公用程式設定印表機，請依照下列步驟進行操作：

  1. 執行下面任一個步驟：
    • 將印表機實際連線至電腦所在的區域網路上，或
    • 在連線至您印表機所在區域網路的電腦上使用此公用程式。
  2. 透過此公用程式來識別印表機，並設定符合印表機最終網路位置的 IP 參數。若需找出 MAC 位址，請﹝從印表機或外接式列印伺服器﹞列印一份組態設定﹝測試﹞頁，以確認您是否選擇了正確的 MAC 位址或硬體位址。
  3. 如有需要，請將印表機﹝或 HP Jetdirect 外接式列印伺服器﹞移回其最終的網路位置。
  4. 若要在其他電腦上安裝印表機，請依照下列其中一個方法進行操作：
    • 對於會直接列印到此印表機的電腦或伺服器，請使用此公用程式來設定各電腦或伺服器，或
    • 對於會透過共用系統列印到此印表機的 Windows 用戶端，請用該用戶端系統上「印表機」資料夾的「新增印表機」公用程式，或採用為此印表機建立的共用印表機安裝程式。
