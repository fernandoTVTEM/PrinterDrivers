印表機設定值

「印表機設定值」畫面可讓您指定不在網路上之印表機的設定值。

為所需的印表機選取並輸入一個或以上的下列位址，接著按「下一步 」。 輸入一個以上的位址可以協助確認所建立的連接埠將連線至指定的印表機。

  • IP 主機名稱。 IP 主機名稱是一個好記（或是容易識別）的名稱，可用來取代印表機的 IP 位址。
  • 硬體位址。 LAN 硬體位址（亦稱為工作站位址或 MAC 位址）會唯一識別裝置。
  • IP 位址 (IPv4)。 IP（網際網路通訊協定）位址可唯一識別 TCP/IP 網路上的每個裝置。 它通常由網路管理員指派。這個公用程式只能用來手動設定裝置上的 IPv4 的設定值。

注意： 若要設定區域網路之外的印表機，請參閱在大型 IP 網路上新增印表機。
