找到印表機

「找到印表機」畫面可以讓您選擇從區域網路偵測到的印表機，或是取得其他安裝網路、USB 或平行連接印表機的設定選項。

  • 安裝偵測到的網路印表機。 從區域網路中偵測到的所有印表機（多功能裝置）清單中選擇您想要的印表機，然後按「下一步」。 按一下「再次搜尋」按鈕，在網路上再次執行印表機搜尋。
  • 協助我尋找印表機。 如果您想要的網路印表機不在清單中，或是您想安裝一台使用 USB 或並列電纜線直接連接到電腦的印表機，請選擇此選項。 對於網路印表機，此選項將可提供其他搜尋功能，例如：依照 IP 位址搜尋印表機。 

註： 若要設定不是位於區域網路上的印表機，請參閱在大型 IP 網路上新增印表機。

清單中的印表機是依照下列欄位來識別。 「硬體位址」可能是最容易識別印表機的方式：

機型 識別印表機機型（例如，HP LaserJet 5000 系列）。 「未知」會在機型無法識別、印表機關閉或沒有連接印表機時顯示。 針對 HP Jetdirect 外接式列印伺服器，下列會在印表機已連接但無法識別時顯示：
  • 平行： Centronics 印表機/繪圖機
  • USB（單向）： 單向的週邊設備
  • USB（雙向）： 雙向的週邊設備
硬體位址 （也稱作「工作站位址」或 MAC 位址）。 裝置在區域網路上的唯一識別。 若是 HP 印表機及擁有多個印表機連接埠的 HP Jetdirect 外接式列印伺服器，可能還會附加埠號（連接埠 1、連接埠 2、連接埠 3）。 此位址列在列印伺服器的組態（測試）頁上。
IP 位址 （網際網路通訊協定）裝置在 TCP/IP 網路上的唯一識別且由網路管理員指定。 IPv4 與 IPv6 位址都將顯示於此清單中。 如果印表機有多個位址，則每個位址都將顯示一次，讓您選擇想要使用的介面。 

註： 當 HP Jetdirect 列印伺服器的電源開啟，並且未在兩分鐘內設定有效的 IP 位址，便會自動指定「自動 IP」位址 (169.254.x.x) 或預設 IP 位址 (192.0.0.192)。HP 建議變更預設 IP 位址。 如果有多台印表機的預設位址相同，請檢查「硬體位址」以識別正確的印表機。
IP 主機名稱 IP 主機名稱是可以用來代替印表機 IP 位址的易記（或易於識別）名稱。 當在網路上設定了一個名稱伺服器以對應主機名稱到其相對的 IP 位址時，即可使用此方法。 如果已經在印表機上指定且設定了主機名稱（例如 printer1.marketing.com），則您可指定主機名稱來識別印表機。

註： 如果有欄位項目遭到截斷，而您想要檢視整個欄位，則可調整欄位的寬度。 若要調整大小，請將滑鼠放在欄位標題的邊緣，然後按一下並拖曳。

關於列印網路組態頁的指示，請參閱隨附於印表機的文件。

如果選定的印表機已受到 SNMP 群體名稱或 Jetdirect 密碼的保護，則會出現提示要求您提供群體名稱或密碼。
