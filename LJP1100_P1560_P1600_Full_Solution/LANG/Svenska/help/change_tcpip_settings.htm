Ändra TCP/IP-inställningar

Du måste ange nya TCP/IP-inställningar (Ipv4) för den här skrivaren. Innan du ändrar några inställningar bör du rådgöra med nätverksadministratören.

 

Obs! Installationsprogrammet upptäcker dina systeminställningar och kommer att tillhandahålla de rekommenderade TCP/IP-inställningarna (både IPv4 och IPv6), men du kan ändra IPv4-inställningarna om du vill (detta program kan bara användas för att konfigurera IPv4-inställningar manuellt).

·        Om skrivaren inte har några giltiga TCP/IP-inställningar:
Ange en giltig IP-adress (IPv4) för skrivaren samt en giltig nätmask och en giltig IP-adress för gatewayen i de befintliga fälten.

Klicka på Föreslå inställningar om du vill att TCP/IP-parametrarna ska anges automatiskt av installationsprogrammet.

Använd kryssrutan för standardportnamn (endast i Windows NT, 2000 och XP). Kryssa för den här rutan om du vill använda standardportinställningarna för din skrivare. Om du inte markerar den här kryssrutan kommer dialogrutan Portnamn att visas som nästa skärm.

Varning! Om skrivaren kommer att vara ansluten till andra nätverk kan det hända att de TCP/IP-inställningar som föreslås av programmet inte är globalt unika utan kan hamna i konflikt med andra enheter. Var försiktig när du ansluter till andra TCP/IP-nätverk.

·        Om skrivaren redan har giltiga TCP/IP-inställningar:
De aktuella TCP/IP-inställningarna visas.

Varning! Genom att ändra de giltiga TCP/IP-inställningarna kan du förhindra att andra användare kommer åt enheten. Rådgör med din nätverksadministratör innan du ändrar de giltiga inställningarna.

Obs! Om TCP/IP-parametrarna har lösenordsskyddats av någon styrande programvara, som t.ex. HP Web Jetadmin, kommer fälten att vara nedtonade.

Använd kryssrutan för standardportnamn (endast i Windows NT, 2000 och XP). Kryssa för den här rutan om du vill använda standardportinställningarna för din skrivare. Om du inte markerar den här kryssrutan kommer dialogrutan Portnamn att visas som nästa skärm.
