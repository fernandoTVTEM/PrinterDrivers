En brandvägg upptäcktes

För att kunna kommunicera genom Windows-brandväggen kommer installationsprogrammet att läggas till i brandväggens undantagslista enligt Microsofts specifikationer. Installationsprogrammet kommer då att kunna kommunicera med skrivare i nätverket för installation.

Klicka på Nästa för att söka efter skrivare i nätverket och fortsätta med installationen. Klicka annars på Avbryt för att avsluta installationsprogrammet.

Om en brandvägg är aktiverad på datorn, t.ex. ZoneAlarm©, BlackICE©, Sygate Security Agent© eller Norton Personal Firewall©, finns anvisningar för hur du inaktiverar den i dokumentationen som medföljde brandväggen.

Om den programvarubaserade brandväggen inaktiveras bör följande vara installerat på datorn:

  • En maskinvaruenhet som ger brandväggsskydd mellan Internet och datorn, t.ex. en gateway.
  • Ett uppdaterat antivirusprogram.

Allmän information om kommunikationsfelsökning finns i dokumentationen som medföljde skrivaren och på HP:s supportwebbplats på http://www.hp.com/support.

Brandväggar orsakar ofta kommunikationsproblem. Om en brandvägg är aktiverad på en nätverksansluten dator kan följande problem uppstå:

    Under programvaruinstallationen:

  • Installationsprogrammet för HP-skrivaren kan inte hitta skrivaren i nätverket.  Ett felmeddelande visas som anger att inga skrivare kan hittas i nätverket.  Du kan undvika detta problem genom att tillfälligt inaktivera brandväggen under installationen av HP-skrivaren. Anvisningar för hur du inaktiverar brandväggen finns i dokumentationen som medföljde brandväggen.

    Efter programvaruinstallationen:

  • Om HP-skrivarens IP-adress ändras kan datorn inte kommunicera med skrivaren.  Alla utskrifter, skanningar, datorfax och bildöverföringar misslyckas, och statusen visar att HP-skrivaren är bortkopplad.  I ett hemmanätverk är det normalt att IP-adressen ändras ibland.  Om programvarubrandväggen inte är aktiverad skickas den nya adressen till datorn, så att datorn och HP-skrivaren kan fortsätta att kommunicera.  Du kan undvika detta problem genom att inaktivera brandväggen. Erfarna användare med vana att konfigurera nätverksinställningar kan välja att använda en statisk IP-adress i stället.  Förslag på hur du konfigurerar skrivaren med en statisk IP-adress finns på HP:s webbplats på http://www.hp.com/support.
  • Om du har en HP all-in-one-enhet kan du inte använda minneskortplatserna för att överföra filer till datorn.  Det beror på att filerna på fotokortet inte är tillgängliga från den mappade nätverksenheten eller bildöverföringsprogrammet.

Mer information om brandväggar finns på HP:s webbplats.

Så här inaktiverar du Windows-brandväggen i Windows Vista:

  1. Klicka på Start > Nätverk.
  2. Högerklicka på Nätverk och klicka på Egenskaper.
  3. Klicka på Windows-brandväggen i hörnet längst ned till vänster.
  4. Klicka på Aktivera eller inaktivera Windows-brandväggen.
  5. Klicka på Fortsätt i dialogrutan Användaråtkomstkontroll.
  6. Markera alternativet Av för att inaktivera brandväggen och klicka på OK.  Windows-brandväggen inaktiveras.

Så här inaktiverar du en brandvägg för Internetanslutning i Windows XP:

  1. Klicka på Start i Aktivitetsfältet längst ned på skärmen och klicka sedan på Kontrollpanelen.
  2. Klicka på kategorin Nätverks- och Internet-inställningar. (Om kategorin Nätverks- och Internet-inställningar inte visas klickar du på Växla till kategoriläge under Kontrollpanelen till vänster i Kontrollpanelens fönster.)
  3. Klicka på Nätverksanslutningar.
  4. Högerklicka på den fjärranslutning, nätverksanslutning eller höghastighetsanslutning som du använder för att ansluta till Internet och klicka på Egenskaper på snabbmenyn.
  5. Avmarkera eller inaktivera Skydda min dator och mitt nätverk under Brandvägg för Internet-anslutning på fliken Avancerat och klicka på OK. Windows XP-brandväggen inaktiveras.
  6. Klicka på Nästa i installationsguiden för att fortsätta installationen av skrivaren.
  7. Aktivera Brandvägg för Internet-anslutning när skrivaren har installerats genom att följa stegen ovan fram till alternativet Skydda min dator och mitt nätverk. Markera alternativet Skydda min dator och mitt nätverk.

Obs! Glöm inte att aktivera Windows-brandväggen igen efter installationen.
