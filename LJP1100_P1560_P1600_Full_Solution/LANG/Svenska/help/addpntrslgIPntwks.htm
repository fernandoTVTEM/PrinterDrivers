Lägga till skrivare i stora IP-nätverk

Om du lägger till skrivare (eller multifunktionsenheter) via en router eller gateway (inte i ditt lokala nätverk) kan det vara svårt att identifiera skrivarna med denna funktion:

  • Det kan hända att routern inte skickar vidare skrivarens maskinvaru-, MAC- eller nätverksadress.
  • Om det finns flera nya skrivare i nätverket finns det dock ingen garanti för att rätt skrivare väljs om man använder standard-IP-adressen (192.0.0.192) vid sökningen. Kontrollera maskinvaruadressen för att korrekt kunna identifiera skrivaren.

Så här använder du programmet för att installera en skrivare:

  1. Gör något av följande:
    • Flytta skrivaren fysiskt till datorns lokala nätverk eller
    • Använd detta hjälpprogram på en dator som är ansluten till skrivarens lokala nätverk.
  2. Identifiera skrivaren med detta hjälpprogram och konfigurera IP-parametrar som stämmer överens med skrivarens slutliga nätverksplats. Om du behöver ta reda på MAC-adressen kan du hitta den genom att skriva ut en konfigurationssida (testsida), antingen från skrivaren eller den externa skrivarservern, och kontrollera att du väljer rätt MAC-adress eller maskinvaruadress.
  3. Vid behov kan du flytta tillbaka skrivaren (eller den externa HP Jetdirect-skrivarservern) till dess slutliga nätverksplats.
  4. Gör något av följande för att installera denna skrivare på andra datorer:
    • För andra datorer eller servrar som skriver ut direkt på denna skrivare, använder du detta hjälpprogram till att konfigurera varje dator eller server, eller
    • Windows-klienter som vill skriva ut på den här skrivaren via ett delat system kan använda klientsystemets program Lägg till skrivare i mappen Skrivare eller använda ett installationsprogram för delad skrivare som skapats för den här skrivaren.
