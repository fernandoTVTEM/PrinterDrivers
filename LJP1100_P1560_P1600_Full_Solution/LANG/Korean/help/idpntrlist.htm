프린터 찾음

프린터 찾음 화면에서는 로컬 네트워크에서 검색한 프린터를 선택하거나 네트워크, USB 또는 병렬 연결 프린터를 설치하기 위한 추가 설치 옵션을 얻을 수 있습니다.

  • 검색한 네트워크 프린터 설치 로컬 네트워크에서 검색한 모든 프린터(또는 다기능 장치) 목록에서 원하는 프린터를 선택하고 다음을 누르십시오. 네트워크에서 다른 프린터를 검색하려면 다시 검색 버튼을 누르십시오.
  • 프린터 찾기 도움말 원하는 네트워크 프린터가 나열되지 않았거나 USB 또는 병렬 케이블을 사용하여 컴퓨터에 직접 연결할 프린터를 설치하려면 이 옵션을 선택합니다.  네트워크 프린터의 경우 이 옵션은 IP 주소로 프린터 검색과 같은 추가 검색 옵션을 제공합니다. 

주: 로컬 네트워크에 없는 프린터를 설정하려면 대형 IP 네트워크에 프린터 추가를 참조하십시오.

목록에 있는 프린터는 다음 필드로 식별됩니다. 하드웨어 주소는 프린터를 식별하는 가장 쉬운 방법일 수 있습니다.

모델 프린터 모델을 식별합니다(예: HP LaserJet 5000 시리즈). 알 수 없음은 모델을 식별할 수 없거나, 프린터가 꺼져 있거나 프린터가 연결되어 있지 않은 경우 표시됩니다. HP Jetdirect 외부 인쇄 서버에 대해 프린터가 연결되어 있지만 식별할 수 없는 경우 다음 항목이 표시됩니다.
  • 병렬: Centronics 프린터/플로터
  • USB(단방향): 단방향 주변 기기
  • USB(양방향): 양방향 주변 기기
하드웨어 주소 (스테이션 주소 또는 MAC 주소라고도 함) 로컬 네트워크에서 장치를 고유하게 식별합니다. 여러 개의 프린터 포트가 있는 HP 프린터 및 HP Jetdirect 외부 인쇄 서버의 경우 포트 번호(포트 1, 포트 2, 포트 3)가 추가될 수 있습니다. 이 주소는 인쇄 서버의 구성(테스트) 페이지에 나열됩니다.
IP 주소 (인터넷 프로토콜) TCP/IP 네트워크에서 장치를 고유하게 식별하고 네트워크 관리자가 지정합니다. IPv4 및 IPv6 주소는 모두 이 목록에 표시됩니다.  프린터에 여러 개의 주소가 있는 경우에는 사용할 인터페이스를 선택할 수 있도록 각 주소에 대해 한 번만 표시합니다. 

주: HP Jetdirect 인쇄 서버에 전원이 켜져 있고 2분 이내에 유효한 IP 주소로 구성되지 않은 경우 자동 IP 주소(169.254.x.x) 또는 기본 IP 주소(192.0.0.192)가 자동으로 지정됩니다. 기본 IP 주소는 변경하는 것이 좋습니다. 기본 주소가 같은 프린터가 여러 개 있는 경우 올바른 프린터를 식별하려면 하드웨어 주소를 확인하십시오.
IP 호스트 이름 IP 호스트 이름은 프린터의 IP 주소 대신 사용할 수 있는 친숙한(또는 식별하기 쉬운) 이름으로, 네트워크에서 이름 서버가 해당 IP 주소에 호스트 이름을 매핑하도록 설정된 경우 사용할 수 있습니다. 호스트 이름이 프린터(예: printer1.marketing.com)에 지정 및 구성된 경우 프린터를 식별하도록 호스트 이름을 지정할 수 있습니다.

주: 필드 항목이 잘린 상태에서 전체 필드를 보려면 필드 너비의 크기를 조정합니다. 크기를 조정하려면 커서를 필드 머리글의 경계선에 놓고 마우스로 누른 채 끄십시오.

네트워크 구성 페이지 인쇄에 대한 자세한 내용은 프린터와 함께 제공되는 설명서를 참조하십시오.

선택한 프린터가 SNMP 커뮤니티 이름 또는 Jetdirect 암호로 보호된 경우 커뮤니티 이름 또는 암호를 입력하라는 메시지가 표시됩니다.
