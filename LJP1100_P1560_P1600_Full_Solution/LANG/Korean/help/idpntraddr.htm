프린터 설정

프린터 설정 화면에서는 네트워크에 없는 프린터의 설정을 지정할 수 있습니다.

프린터에 대해 하나 이상의 주소를 입력하고 다음을 누릅니다.  여러 개의 주소를 입력하면 지정한 프린터에 작성한 포트가 연결될 가능성이 높아집니다.

  • IP 호스트 이름. IP 호스트 이름은 프린터의 IP 주소 대신 사용할 수 있는 사용이 간편한(또는 식별하기 쉬운) 이름입니다.
  • 하드웨어 주소. LAN 하드웨어 주소(스테이션 주소 또는 MAC 주소라고도 함)는 장치를 고유하게 식별합니다.
  • IP 주소(IPv4). IP(인터넷 프로토콜) 주소는 TCP/IP 네트워크상에서 각 장치를 식별하며 일반적으로 네트워크 관리자에 의해 할당됩니다. 장치에서 IPv4 설정을 수동으로 구성하는 경우에만 이 유틸리티를 사용할 수 있습니다.

주: 로컬 네트워크에 없는 프린터를 설정하려면 대형 IP 네트워크에 프린터 추가를 참조하십시오.
