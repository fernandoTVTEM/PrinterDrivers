프린터 연결 확인

프린터를 무선으로 연결할 경우 프린터의 무선 표시등이 켜져 있는지 확인합니다. 자세한 내용은 프린터와 함께 제공된 설명서에서 ‘프린터의 무선 표시등 확인’에 관한 설명을 참조하십시오. 그런 다음 화면에서 다음과 같은 옵션을 선택합니다.

네트워크 케이블로 연결하는 경우에는 프린터 뒷면에 있는 네트워크 포트 표시등을 확인합니다.

연결 표시등 상태수행 작업

녹색으로 켜짐

 화면에서 다음과 같은 옵션 중 하나를 선택합니다. 프린터의 하드웨어 주소(MAC 주소), IP 주소 또는 장치 이름(IP 호스트 이름)은 네트워크 구성 페이지를 인쇄하여 확인할 수 있습니다. 네트워크 구성 페이지를 인쇄하는 방법은 프린터와 함께 제공된 설명서를 참조하십시오.

꺼짐
  • 네트워크 케이블이 프린터와 WAP(무선 액세스 포인트) 또는 컴퓨터에 모두 제대로 연결되어 있는지 확인합니다.
  • WAP가 켜져 있는지 확인합니다.
  • 프린터 전원이 켜져 있는지 확인합니다.
  • 컴퓨터가 네트워크에 연결되어 있는지 확인합니다.

위와 같은 사항을 확인했으면 프린터를 다시 검색할 방법을 선택합니다.

  • 다시 검색을 선택하면 설치 마법사가 검색을 다시 한 번 수행합니다.
  • 주소로 프린터 지정을 선택하면 프린터의 하드웨어 주소(MAC 주소), IP 주소 또는 장치 이름(IP 호스트 이름)을 수동으로 입력할 수 있습니다.
  • 네트워크에 연결되지 않은 프린터의 포트를 구성할 경우에는 네트워크에 없는 프린터의 설정을 입력합니다를 선택하십시오. 이 옵션에 대한 자세한 내용은 프린터와 함께 제공된 설명서를 참조하십시오.
