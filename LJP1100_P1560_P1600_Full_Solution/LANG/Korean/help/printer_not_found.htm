프린터를 찾을 수 없음

설치 마법사가 프린터와 통신할 수 없습니다

사용자의 PC에 방화벽이 사용 중이면 설치 프로그램이 프린터를 찾지 못할 수 있습니다. 설치를 계속하기 전에 잠시 방화벽을 해제하십시오.

방화벽(ZoneAlarm, BlackICE, Sygate Security Agent 및 Norton Personal Firewall 등)을 사용하고 있을 경우 방화벽 공급업체가 제공한 설명서에서 방화벽을 해제하는 방법을 참조하십시오.

소프트웨어 방화벽이 해제된 컴퓨터에는 항상 다음과 같은 제품이 설치되어 있어야 합니다.

  • 인터넷과 컴퓨터 사이를 방화벽으로 보호해 주는 하드웨어 장치(예: 게이트웨이)
  • 최신 바이러스 탐지 프로그램

통신 문제 해결에 대한 전반적인 내용은 프린터와 함께 제공된 설명서와 지원 웹 사이트(http://www.hp.com/support)를 참조하십시오.

방화벽은 통신 문제를 일으키는 경우가 많습니다. 네트워크 컴퓨터에서 방화벽을 사용할 경우 다음과 같은 문제가 발생할 수 있습니다.

    소프트웨어 설치 중:

  • HP 프린터 설치 프로그램이 네트워크 상의 HP 프린터를 찾지 못합니다.  설치 프로그램이 네트워크에서 프린터를 찾지 못했다는 오류 메시지가 표시됩니다.  이 문제를 방지하려면 HP 프린터 소프트웨어를 설치하는 동안 방화벽을 잠시 해제하십시오.  방화벽 해제 방법은 해당 방화벽의 설명서를 참조하십시오.

    소프트웨어 설치 후:

  • HP 프린터의 IP 주소가 변경될 경우 컴퓨터가 HP 프린터와 통신할 수 없게 됩니다.  인쇄, 스캔, PC 팩스, 이미지 전송 작업에 모두 오류가 발생하며 제품 상태에 HP 프린터의 연결이 해제되었다고 표시됩니다.  가정 네트워크 환경에서는 IP 주소가 가끔 변경되는 것이 정상이며, 소프트웨어 방화벽이 사용되지 않을 경우에는 새 주소가 컴퓨터로 전송되므로 컴퓨터가 HP 프린터와 계속 통신할 수 있습니다.  따라서 통신 불능 문제를 방지하려면 방화벽을 해제하십시오.  네트워킹에 익숙한 고급 사용자의 경우에는 고정 IP 주소를 사용하는 방법도 있습니다.  고정 IP 주소로 프린터를 구성하는 방법은 HP 웹 사이트(http://www.hp.com/support)를 참조하십시오.
  • HP 복합기의 경우 해당 장치에 있는 메모리 카드 슬롯을 통해 컴퓨터로 파일을 전송할 수 없습니다.  사진 카드의 파일은 매핑된 네트워크 드라이브나 이미지 전송 소프트웨어에서 액세스할 수 없기 때문입니다.

방화벽에 관한 자세한 정보는 HP 웹사이트를 참조하십시오.

Windows Vista에서 Windows 방화벽 해제 방법:

  1. 시작 > 네트워크를 누릅니다.
  2. 네트워크를 마우스 오른쪽 단추로 누르고 속성을 선택합니다.
  3. 왼쪽 하단 코너에서 Windows 방화벽을 누릅니다.
  4. 'Windows 방화벽 사용/해제'를 선택합니다.
  5. 사용자 액세스 제어 대화 상자에서 '계속'을 선택합니다.
  6. '해제' 옵션을 선택하여 방화벽을 해제한 후 '확인'을 누릅니다.  Windows 방화벽이 해제됩니다.

Windows XP에서 인터넷 연결 방화벽 해제 방법:

  1. 화면 하단의 작업 표시줄에서 시작, 제어판을 차례로 누릅니다.
  2. 네트워크 및 인터넷 연결을 누릅니다._네트워크 및 인터넷 연결이 표시되지 않는 경우, 제어판 창 왼쪽 하단에 있는 제어판 아래에서 종류별 보기로 전환을 누르십시오.
  3. 네트워크 연결을 누릅니다.
  4. 인터넷 연결을 위해 전화 걸기, LAN 또는 고속 인터넷 중 하나를 마우스 오른쪽 단추로 누른 후 컨텍스트 메뉴에서 등록 정보를 누릅니다.
  5. 고급 탭에서 인터넷 연결 방화벽 아래의 내 컴퓨터 또는 네트워크 보호를 선택 해제한 다음 확인을 누릅니다. Windows XP 방화벽이 해제되었습니다.
  6. 설치 마법사에서 다음을 눌러 프린터 설치를 계속합니다.
  7. 프린터를 설치한 후 인터넷 연결 방화벽을 다시 사용하려면 내 컴퓨터 또는 네트워크 보호 옵션을 선택하기 전까지의 위의 단계를 반복합니다. 그런 다음 내 컴퓨터 또는 네트워크 보호 옵션을 다시 선택합니다.

주: 설치가 완료되면 방화벽을 다시 사용하도록 합니다.
