포트 이름

주: 포트 이름 대화 상자는 Windows NT, Windows 2000 및 Windows XP에서만 나타납니다. 이 대화 상자는 네트워크 통신 모드 설정 대화 상자에서 기본 포트 이름 사용이 선택되지 않았거나 IP 통신 모드가 선택된 경우에만 나타납니다.

설치할 프린터에 대해 작성하려는 네트워크 포트 이름을 지정할 수 있습니다. 포트 이름은 프린터 등록 정보에서 포트를 식별하기 위해 사용됩니다.

포트 이름은 최대 63자까지 지정할 수 있으며 모든 문자를 사용할 수 있습니다. 포트 이름은 고유해야 하며, 요약 화면과 테스트 페이지에 표시됩니다.
