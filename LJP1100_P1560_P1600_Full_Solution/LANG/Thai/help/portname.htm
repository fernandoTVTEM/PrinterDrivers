ชื่อพอร์ท

หมายเหตุ: กรอบโต้ตอบ ชื่อพอร์ท จะปรากฏขึ้นเฉพาะใน Windows NT, Windows 2000 และ Windows XP เท่านั้น กรอบโต้ตอบนี้จะปรากฏขึ้นเฉพาะเมื่อไม่ได้เลือก ใช้ชื่อพอร์ทเริ่มต้น ในกรอบโต้ตอบ ตั้งค่าโหมดการสื่อสารเครือข่าย และถ้าเลือกโหมดการสื่อสารเป็น IP

ท่านสามารถระบุชื่อพอร์ทเครือข่ายที่จะสร้างขึ้นสำหรับเครื่องพิมพ์ที่กำลังติดตั้ง ชื่อพอร์ทจะใช้เพื่อระบุพอร์ทในคุณสมบัติเครื่องพิมพ์

ชื่อพอร์ทที่ถูกต้องต้องมีความยาวไม่เกิน 63 ตัวอักษร และประกอบด้วยอักขระใดก็ได้ ชื่อพอร์ทจะต้องไม่ซ้ำกันชื่อพอร์ทจะแสดงในหน้าจอสรุปและหน้าทดสอบ
