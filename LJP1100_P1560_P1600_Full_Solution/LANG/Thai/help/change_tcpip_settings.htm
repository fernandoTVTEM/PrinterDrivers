เปลี่ยนการตั้งค่า TCP/IP

ท่านอาจระบุการตั้งค่า TCP/IP ใหม่ (IPv4) สำหรับเครื่องพิมพ์นี้ ท่านควรปรึกษาผู้บริหารเครือข่ายก่อนที่จะเปลี่ยนแปลงการตั้งค่าใดๆ

หมายเหตุ: ยูทิลิตี้การติดตั้งจะตรวจพบการตั้งค่าระบบของท่าน และจะพยายามแนะนำการตั้งค่า TCP/IP (ทั้ง IPv4 และ IPv6) แต่ท่านสามารถเปลี่ยนแปลงการตั้งค่า IPv4 ได้ตามต้องการ (ยูทิลิตี้นี้จะสามารถใช้ในการตั้งค่า IPv4 ด้วยตนเองได้เท่านั้น) ค่าพารามิเตอร์ที่ท่านกำหนดจะไม่นำไปกำหนดให้กับเครื่องพิมพ์จนกว่าท่านจะทำการติดตั้งเสร็จสิ้น

ท หากเครื่องพิมพ์ของท่านไม่มีการตั้งค่า TCP/IP ที่ถูกต้อง:
ให้ป้อนแอดเดรส IP ที่ถูกต้อง (IPv4) สำหรับเครื่องพิมพ์ ซับเนตมาสค์ และแอดเดรส IP ที่ถูกต้องสำหรับเกทเวย์ในฟีลด์ที่กำหนดให้ ควรปรึกษาผู้บริหารเครือข่ายของท่าน ก่อนกำหนดค่าเหล่านี้

คลิก แนะนำการตั้งค่า เพื่อให้ยูทิลิตี้ติดตั้งทำการป้อนค่าพารามิเตอร์ TCP/IP โดยอัตโนมัติ

ใช้กล่องตัวเลือกพอร์ทเริ่มต้น (Windows NT, 2000 และ XP) ทำเครื่องหมายที่กล่องตัวเลือกนี้เพื่อตั้งค่าพอร์ทเริ่มต้นสำหรับเครื่องพิมพ์ของท่าน หากช่องนี้ไม่มีเครื่องหมาย กรอบโต้ตอบ ชื่อพอร์ท จะเป็นหน้าจอถัดไปของโปรแกรมติดตั้ง

คำเตือน: หากจะต่อเชื่อมเครื่องพิมพ์ของท่านกับเครือข่ายอื่น ค่าของ TCP/IP ที่แนะนำโดยยูทิลิตี้ อาจไม่ใช่ค่าเฉพาะที่ใช้ได้ทั่วไป และอาจขัดแย้งกับอุปกรณ์อื่น ให้ใช้ความระมัดระวังเมื่อต้องต่อเชื่อมกับเครือข่าย TCP/IP อื่น

ท หากเครื่องพิมพ์มีการตั้งค่า TCP/IP ที่ถูกต้องอยู่แล้ว:
ค่าของ TCP/IP ปัจจุบันจะแสดงขึ้น

คำเตือน: การเปลี่ยนการตั้งค่า TCP/IP ที่ถูกต้องอยู่แล้ว อาจทำให้ผู้ใช้อื่นเรียกใช้งานไม่ได้ ควรปรึกษาผู้บริหารเครือข่ายของท่านก่อนเปลี่ยนการตั้งค่าที่ถูกต้องเหล่านี้

หมายเหตุ: หากพารามิเตอร์ TCP/IP ถูกป้องกันไว้ด้วยรหัสผ่านโดยซอฟท์แวร์บริหารเครือข่าย เช่น HP Web Jetadmin ช่องข้อมูลจะแสดงเป็นสีเทา

ใช้กล่องตัวเลือกพอร์ทเริ่มต้น (Windows NT, 2000 และ XP เท่านั้น) ทำเครื่องหมายที่กล่องตัวเลือกนี้เพื่อตั้งค่าพอร์ทเริ่มต้นสำหรับเครื่องพิมพ์ของท่าน หากช่องนี้ไม่มีเครื่องหมาย กรอบโต้ตอบ ชื่อพอร์ท จะเป็นหน้าจอถัดไปของโปรแกรมติดตั้ง
