-A-

ARP
(Address Resolution Protocol - โปรโตคอลการแปลค่าแอดเดรส)ARP คือโปรโตคอล TCP/IP ที่ใช้ในการแปลงแอดเดรส IP เป็นแอดเดรสทางกายภาพ (เรียกว่า แอดเดรส DLC) เช่น แอดเดรสของอีเทอร์เน็ต แม่ข่ายที่ต้องการได้รับแอดเดรสทางกายภาพจะส่งข้อมูลคำขอ ARP ไปยังเครือข่าย TCP/IP แม่ข่ายที่อยู่บนเครือข่าย ซึ่งมีแอดเดรส IP ตามคำขอจะตอบแอดเดรสฮาร์ดแวร์ทางกายภาพกลับไปให้ทราบ

BootP
โปรโตคอลบูทสแตรปที่ใช้ดาวน์โหลดแอดเดรส IP และพารามิเตอร์ TCP/IP อื่นๆ โดยอัตโนมัติจากเซิร์ฟเวอร์เครือข่าย เมื่อเปิดเครื่องพิมพ์ (หรืออุปกรณ์ All-in-One) เซิร์ฟเวอร์พิมพ์ HP Jetdirect จะส่งข้อมูลขอบูท หากเซิร์ฟเวอร์ BootP ติดตั้งไว้อย่างถูกต้อง ก็จะส่งพารามิเตอร์ไปยังเซิร์ฟเวอร์พิมพ์ Jetdirect โดยผ่านทางเครือข่าย โปรดดูที่ DHCP

DHCP
(โปรโตคอลการกำหนดค่าแม่ข่ายแบบไดนามิค) คล้ายกับ BootP เนื่องจากมีการดาวน์โหลดแอดเดรส IP จากเซิร์ฟเวอร์ของเครือข่าย อย่างไรก็ตาม เซิร์ฟเวอร์ DHCP จะเก็บแอดเดรส IP จำนวนหนึ่งที่กำหนดให้กับอุปกรณ์ที่ขอแบบไดนามิค และจะให้เซิร์ฟเวอร์พิมพ์ยืมแอดเดรส IP เท่านั้น เมื่อมีการปิดเครื่องพิมพ์ (หรืออุปกรณ์ All-in-One) แล้วเปิดใหม่ จะไม่รับประกันว่าเซิร์ฟเวอร์พิมพ์ Jetdirect จะได้รับการกำหนดแอดเดรส IP เดิม แม้ว่าเซิร์ฟเวอร์ DHCP จะพยายามดำเนินการนั้นแล้วก็ตาม

-I-

IPv4
IPv4 เป็นเวอร์ชัน 4 ของโปรโตคอลอินเตอร์เน็ต แอดเดรส IPv4 ประกอบด้วย 32 บิตที่ระบุในรูปแบบตัวเลขที่คั่นด้วยจุดทศนิยม เช่น 192.168.100.250 หากเซิร์ฟเวอร์เครือข่ายไม่ได้กำหนดแอดเดรส IPv4 ที่ถูกต้อง เซิร์ฟเวอร์พิมพ์ Jetdirect ก็จะกำหนดแอดเดรส link-local/Auto-IP (169.254.x.x) หรือแอดเดรสเริ่มต้นแบบเก่า (192.0.0.192) ทั้งนี้ขึ้นอยู่กับเครือข่าย

IPv6
IPv6 เป็นเวอร์ชันปรับปรุงของโปรโตคอลอินเทอร์เน็ต ซึ่งในที่สุดแล้วอาจจะถูกใช้แทนที่ IPv4 แต่ก็สามารถใช้งานควบคู่ไปกับ IPv4 ได้เช่นกัน แอดเดรส IPv6 มีความยาว 128 บิต และมีการระบุค่าในรูปแบบอักขระฐานสิบหกที่คั่นด้วยเครื่องหมายจุดคู่ (:) เช่น 2001:0DB8:0000:0000:0000:0000:1428:57AB  ดังนั้นแอดเดรส IPv6 จึงสามารถรองรับการเพิ่มขึ้นของอุปกรณ์ที่เชื่อมต่อกับเครือข่ายได้ระยะหนึ่ง และสิ่งที่แตกต่างจาก IPv4 ก็คือ สามารถกำหนดแอดเดรส IPv6 หลายๆ แอดเดรสให้กับเซิร์ฟเวอร์พิมพ์ ในขณะที่แอดเดรส link-local IPv6 เริ่มต้นถูกกำหนดโดยอัตโนมัติ แต่จะสามารถกำหนดแอดเดรส IPv6 ตามเราเตอร์, ตามเซิร์ฟเวอร์ และด้วยตนเองได้เช่นกัน

Packet Internet Groper PING เป็นยูทิลิตี้ที่ใช้ดูว่าสามารถใช้งานแอดเดรส IP ใดได้หรือไม่ ซึ่งจะทำงานโดยการส่งชุดข้อมูลไปยังแอดเดรสที่ระบุไว้และรอการตอบกลับ PING ใช้ในการแก้ไขปัญหาการเชื่อมต่ออินเตอร์เน็ตเป็นหลัก ยูทิลิตี้ PING มีทั้งในแบบฟรีแวร์และแชร์แวร์ให้เลือกสำหรับคอมพิวเตอร์ส่วนบุคคล

WEP
มาตรฐานการป้องกันแบบดั้งเดิมที่ใช้ในเครือข่าย Wi-Fi เพื่อเข้ารหัสการสื่อสารในเครือข่ายไร้สาย WEP จะสามารถป้องกันข้อมูลได้โดยการเข้ารหัสข้อมูลที่ส่งผ่านทางคลื่นวิทยุจากอุปกรณ์ไร้สายหนึ่งไปยังอุปกรณ์ไร้สายอื่นๆ ซึ่งจะทำให้ผู้ที่ลักลอบใช้งานไม่สามารถเข้าใจข้อมูลได้ เฉพาะอุปกรณ์ที่มีการตั้งค่า WEP เหมือนกับเครื่องพิมพ์เท่านั้น ที่สามารถสื่อสารกับเครื่องพิมพ์ได้

การตรวจสอบสิทธิ์
การตรวจสอบสิทธิ์ คือวิธีการในการป้องกันข้อมูลของเครือข่ายไร้สายในเครือข่ายที่มีการตรวจสอบสิทธิ์ อุปกรณ์ต่างๆ จะใช้คีย์ร่วมเป็นรหัสผ่าน และจะสื่อสารเฉพาะกับอุปกรณ์ที่มีคีย์นั้น การตรวจสอบสิทธิ์นั้นต่างจาก WEP เนื่องจากจะไม่มีการเข้ารหัสข้อมูลที่ส่งระหว่างอุปกรณ์ไร้สาย แต่ท่านสามารถใช้การตรวจสอบสิทธิ์ร่วมกับ WEP ได้ คีย์การตรวจสอบสิทธิ์และคีย์ WEP อาจเหมือนกันก็ได้

การตั้งค่าเริ่มต้นจากโรงงาน สำหรับ HP Jetdirect
ดูที่ การรีเซตแบบเย็น

การตั้งชื่อข้อตกลง เครื่องพิมพ์
เมื่อตั้งชื่อเครื่องพิมพ์ (หรืออุปกรณ์ All-in-One) ให้ปฏิบัติตามคำแนะนำดังนี้

  • ความยาวไม่เกิน 31 ตัวอักษร
  • ตัวอักษร A - Z ตัวพิมพ์ใหญ่
  • ตัวอักษร a - z ตัวพิมพ์ตัวเล็ก
  • ตัวเลข 0 ถึง 9
  • หลีกเลี่ยงอักขระซึ่งมีค่า ASCII น้อยกว่า 32 หรือมากกว่า 127
  • ไม่ใช้เครื่องหมายจุลภาค (,)

การตั้งชื่อข้อตกลง ชื่อที่ใช้ร่วมกัน
ตามคำแนะนำของ Microsoft ชื่อจะจำกัดที่อักขระซึ่งมีค่า ASCII 7 บิต (127 ตัวแรก) ไม่อนุญาตให้มีช่องว่าง ปฏิบัติตามแนวทางต่อไปนี้:

  • ตัวอักษร A - Z ตัวพิมพ์ใหญ่
  • ตัวอักษร a - z ตัวพิมพ์ตัวเล็ก
  • ตัวเลข 0 ถึง 9
  • อักขระพิเศษที่จำกัด:$ % ' - _ @ ~ ` ! ( ) { } . ^ # & + , ; = [ ]

การแบ่งใช้งานเครื่องพิมพ์ร่วมกัน
การแบ่งใช้งานเครื่องพิมพ์ร่วมกันช่วยให้ผู้ใช้อื่นๆ ในเครือข่ายสามารถพิมพ์มายังเครื่องพิมพ์นี้ผ่านคอมพิวเตอร์เครื่องนี้ ยูทิลิตี้ในการติดตั้งนี้สนับสนุนการใช้เครื่องพิมพ์ร่วมกันใน Windows 2000 และ Windows NT 4.0 เท่านั้นจะต้องเปิดใช้บริการเซิร์ฟเวอร์ในระบบสำหรับการใช้เครื่องพิมพ์ร่วมกัน ท่านต้องเลือกตัวเลือกใดตัวเลือกหนึ่งต่อไปนี้

  • ไม่แบ่งใช้: เลือกตัวเลือกนี้เพื่อกำหนดให้คอมพิวเตอร์นี้ พิมพ์แบบ peer-to-peer หากท่านไม่ ต้องการให้ผู้ใช้รายอื่นในเครือข่ายเข้าใช้เครื่องพิมพ์ผ่านคอมพิวเตอร์นี้
  • ใช้งานร่วมกันในชื่อ: เลือกตัวเลือกนี้เพื่อกำหนดคอมพิวเตอร์นี้ให้เป็นเซิร์ฟเวอร์สำหรับ การพิมพ์ไคลเอนท์-เซิร์ฟเวอร์ เครื่องไคลเอนท์เครือข่ายจะยอมให้ส่งงานพิมพ์ไปยังเครื่องพิมพ์ผ่านทางคอมพิวเตอร์นี้

การพิมพ์ไคลเอนท์-เซิร์ฟเวอร์
คอมพิวเตอร์ไคลเอนท์จะส่งงานพิมพ์ไปยังคอมพิวเตอร์เซิร์ฟเวอร์ที่กำหนด ซึ่งทำหน้าที่ควบคุมการใช้เครื่องพิมพ์แต่ละเครื่อง (หรืออุปกรณ์ All-in-One)

การพิมพ์หน้าทดสอบ HP Jetdirect
ในการพิมพ์หน้าทดสอบ HP Jetdirect โปรดดูคำแนะนำการติดตั้งฮาร์ดแวร์ HP Jetdirect หรือคำแนะนำจากเอกสารประกอบเครื่องพิมพ์ของท่าน สำหรับเซิร์ฟเวอร์พิมพ์ HP Jetdirect แบบภายนอก ให้กดและปล่อยปุ่ม Test ที่เซิร์ฟเวอร์พิมพ์

การรีเซตแบบเย็น
ก่อนที่จะรีเซตแบบเย็น ขอแนะนำให้ท่านพิมพ์หน้าทดสอบสำหรับเครื่องพิมพ์และเซิร์ฟเวอร์พิมพ์ Jetdirect

ในการรีเซตเซิร์ฟเวอร์พิมพ์ภายนอก HP Jetdirect เป็นค่าเริ่มต้นที่มาจากโรงงาน ให้ถอดสายไฟ จากนั้นกดปุ่ม Test ที่เซิร์ฟเวอร์พิมพ์ในขณะที่ต่อสายไฟกลับเข้าที่ รอให้เซิร์ฟเวอร์พิมพ์เริ่มต้นทำงาน

ในการรีเซตเซิร์ฟเวอร์พิมพ์ HP Jetdirect ภายใน ให้รีเซตเครื่องพิมพ์ (หรืออุปกรณ์ All-in-One) และรอให้เซิร์ฟเวอร์พิมพ์เริ่มต้นทำงาน (สำหรับเครื่องพิมพ์ HP LaserJet รุ่นต่างๆ โดยส่วนใหญ่ ให้กดปุ่ม Go บนแผงควบคุมของเครื่องพิมพ์ขณะเปิดเครื่อง สำหรับเครื่องพิมพ์อื่นๆ โปรดดูจากคู่มือการใช้เครื่องพิมพ์ของท่าน)

ข้อควรระวัง: การรีเซตเครื่องพิมพ์อาจลบค่าที่ตั้งไว้ในเครื่องพิมพ์ทั้งหมด

เกทเวย์
เกทเวย์คือเราเตอร์หรือคอมพิวเตอร์ที่เชื่อมต่อเครือข่ายสองเครือข่ายที่ต่างกันเข้าด้วยกัน (ตัวอย่างเช่น มีความต่างกันในโปรโตคอลการสื่อสาร รูปแบบข้อมูล โครงสร้าง หรือสถาปัตยกรรม) เมื่อเครือข่ายถูกแบ่งอออกเป็นซับเนต เกทเวย์จะถูกใช้เพื่อแยกซับเนตหนึ่งออกจากซับเนตอื่น เพื่อควบคุมการสื่อสารในเครือข่าย
บนเครือข่าย TCP/IP อุปกรณ์เครือข่ายแต่ละชุดมักจะได้รับการกำหนดค่าด้วยแอดเดรส IP ของเกทเวย์ที่ใกล้ที่สุด หากไม่มีเกทเวย์ในเครือข่าย โดยทั่วไปแล้วระบบจะใช้แอดเดรส IP ของคอมพิวเตอร์
โปรดดูข้อมูลเพิ่มเติมที่ การเพิ่มเครื่องพิมพ์เข้าในเครือข่าย IP ขนาดใหญ่

คีย์ WEP
คีย์ WEP หรือ "คีย์การเข้ารหัส" คือชุดอักขระที่เป็นตัวอักษรและตัวเลข หรือเป็นตัวเลขฐานสิบหกที่เรียงกันเป็นลำดับ หลังจากที่สร้างคีย์ WEP แล้ว ท่านจะต้องจดจำไว้ หรือเก็บไว้ในที่ปลอดภัย ท่านอาจไม่สามารถเรียกคีย์ WEP กลับคืน หากทำสูญหาย คีย์ WEP นั้นอาจเป็นแบบ 64 บิตหรือ 128 บิตก็ได้ โดยที่ 24 บิตแรกของคีย์จะถูกสร้างขึ้นโดยอัตโนมัติ เมื่อสร้างคีย์ WEP บุคคลที่สร้างคีย์จะกำหนดบิตส่วนที่เหลือ (40 บิตในกรณีของคีย์แบบ 64 บิต หรือ 104 บิตสำหรับแบบ 128 บิต)
ในการสร้างคีย์ WEP ให้ป้อนค่าต่อไปนี้ ว่าคีย์ WEP นั้นจะเป็นตัวอักษรและตัวเลขหรือเป็นตัวเลขฐานสิบหก และจะมีกี่บิต:

ประเภทบิตป้อนสตริง...
ตัวอักษรและตัวเลข64ตัวอักษรและตัวเลข 5 ตัว
128ตัวอักษรและตัวเลข 13 ตัว
เลขฐานสิบหก64ตัวเลขฐานสิบหก 10 หลัก
128ตัวเลขฐานสิบหก 20 หลัก

เครือข่ายเฉพาะกิจ (เครือข่ายแบบเชื่อมต่ออุปกรณ์เข้ากับอุปกรณ์)
เครือข่ายแบบเชื่อมต่ออุปกรณ์เข้ากับอุปกรณ์คืออุปกรณ์ไร้สาย 2 อย่างขึ้นไปที่เชื่อมต่อกันโดยไม่ใช้สถานีฐานส่วนกลางหรือจุดเชื่อมต่อแบบไร้สาย (WAP)

เครือข่ายประเภทนี้จะมีประสิทธิภาพลดลง หากมีอุปกรณ์อยู่ในเครือข่ายมากกว่าหกเครื่อง

เครือข่ายท้องถิ่น
เครือข่ายท้องถิ่นคือเครือข่ายที่ไม่ขยายเกินกว่าเราเตอร์ หรือ เกทเวย์ หากคอมพิวเตอร์และเครื่องพิมพ์ (หรืออุปกรณ์ All-in-One) ของท่าน เชื่อมต่อเข้ากับเครือข่ายท้องถิ่น อุปกรณ์เหล่านี้จะเชื่อมต่อกับเซกเมนต์เครือข่ายเดียวกัน

เครือข่ายแบบ Peer-to-peer
ดู เครือข่ายแบบเชื่อมต่ออุปกรณ์เข้ากับอุปกรณ์

เครือข่ายแบบเชื่อมต่ออุปกรณ์เข้ากับอุปกรณ์ (เครือข่ายเฉพาะกิจ)
เครือข่ายแบบเชื่อมต่ออุปกรณ์เข้ากับอุปกรณ์คืออุปกรณ์ไร้สาย 2 อย่างขึ้นไปที่เชื่อมต่อกันโดยไม่ใช้สถานีฐานส่วนกลางหรือจุดเชื่อมต่อแบบไร้สาย (WAP)

เครือข่ายไร้สาย (โครงสร้างภายใน)
เครือข่ายไร้สายที่ประกอบด้วยอุปกรณ์ที่เชื่อมต่อโดยใช้สถานีฐานส่วนกลางหรือจุดเชื่อมต่อแบบไร้สาย (WAP) เพื่อใช้ในเชิงพานิชย์หรือเพื่อการให้บริการสาธารณะ

ชื่อ SNMP Community
ชื่อ SNMP Set Community เป็นรหัสผ่านที่จะต้องส่งให้เมื่อใช้ SNMP เพื่อเปลี่ยนแปลงการกำหนดค่าเครื่องพิมพ์ หากเครื่องพิมพ์ (หรืออุปกรณ์ All-in-One) ที่เลือกมีการป้องกันโดยชื่อ SNMP Community หรือรหัสผ่าน Jetdirect ท่านจะได้รับแจ้งให้ใส่ชื่อ community ชื่อ community ที่ถูกต้องมีความยาวได้มากถึง 255 อักขระ
หมายเหตุ: ท่านจะต้องป้อนค่าก็ต่อเมื่อมีการกำหนดชื่อ SNMP Set Community ที่ไม่ใช่ค่าเริ่มต้นเท่านั้น ชื่อ SNMP Set Community สามารถกำหนดโดยผู้บริหารเครือข่ายโดยใช้โปรแกรมประยุกต์บริหารเครือข่าย (เช่น HP Web Jetadmin) หรือเครื่องมืออื่นๆ

ชื่อเครือข่าย
ชื่อเครือข่ายคือสตริงของอักขระที่เป็นตัวอักษรและตัวเลข ซึ่งเป็นการควบคุมการเข้าใช้ขั้นพื้นฐานของเครือข่ายไร้สายชื่อเครือข่ายนี้เรียกอีกอย่างว่า ตัวระบุชุดบริการ หรือ SSID

ชื่อเครื่องพิมพ์
ป้อนชื่อเครื่องพิมพ์ของท่านโดยใช้ ข้อตกลงการตั้งชื่อเครื่องพิมพ์ ที่เหมาะสม ชื่อนี้ใช้เพื่อระบุไอคอนเครื่องพิมพ์สำหรับเครื่องพิมพ์นี้ ในโฟลเดอร์ Printers ของระบบของท่าน ในการแสดงโฟลเดอร์ Printers คลิกที่ Start บนเดสก์ทอปของ Windows จากนั้นเลือก Settings และคลิก Printers

ชื่อพอร์ท
กรอบโต้ตอบชื่อพอร์ทจะปรากฏขึ้นใน Windows NT และ Windows 2000 เท่านั้น กรอบโต้ตอบนี้จะปรากฏขึ้นเฉพาะกรณีที่ไม่ทำเครื่องหมาย "ใช้ชื่อพอร์ทเริ่มต้น" ในกรอบโต้ตอบตั้งค่าโหมดการสื่อสารเครือข่าย และเลือกโหมดการสื่อสาร IP

ท่านสามารถระบุชื่อพอร์ทเครือข่ายที่จะสร้างขึ้นสำหรับเครื่องพิมพ์ที่กำลังติดตั้ง ชื่อพอร์ทจะใช้เพื่อระบุพอร์ทในคุณสมบัติเครื่องพิมพ์

ชื่อพอร์ทที่ถูกต้องต้องมีความยาวไม่เกิน 63 ตัวอักษร และประกอบด้วยอักขระใดก็ได้ ชื่อพอร์ทจะต้องเป็นชื่อที่กำหนดเฉพาะบนเครื่องพีซีของท่าน ชื่อพอร์ทจะแสดงในหน้าจอสรุปและหน้าทดสอบ

ชื่อแม่ข่าย IP
ชื่อแม่ข่าย IP เป็นชื่อที่คุ้นเคย (หรือระบุได้ง่าย) ซึ่งสามารถนำมาใช้แทนแอดเดรส IP ของเครื่องพิมพ์ได้ โดยสามารถนำมาใช้เมื่อ เซิร์ฟเวอร์ชื่อ บนเครือข่ายได้รับการตั้งค่าให้แทนข้อมูลชื่อแม่ข่ายตามแอดเดรส IP ที่เกี่ยวข้อง หากมีการกำหนดและตั้งค่าชื่อแม่ข่ายบนเครื่องพิมพ์ (ตัวอย่างเช่น printer1.marketing.com) ท่านสามารถระบุชื่อแม่ข่ายเพื่อกำหนดเครื่องพิมพ์ได้

ซับเนต
ส่วนของเครือข่ายที่แบ่งใช้ส่วนประกอบแอดเดรสสามัญร่วมกัน ในเครือข่าย TCP/IP ซับเนตถูกกำหนดให้เป็นอุปกรณ์ทั้งหมดที่มีหมายเลขข้างหน้าของแอดเดรส IP เป็นหมายเลขเดียวกัน ตัวอย่างเช่น อุปกรณ์ทั้งหมดที่มีแอดเดรส IP ที่เริ่มต้นด้วย 100.100.100. จะเป็นส่วนหนึ่งของซับเนตเดียวกัน การแบ่งเครือข่ายออกเป็นซับเนตมีประโยชน์ทั้งในด้านความปลอดภัยและประสิทธิภาพ เครือข่าย IP จะถูกแบ่งออก โดยใช้ซับเนต มาสก์

ซับเนต มาสก์
ซับเนต มาสก์คือกลไกที่ใช้แบ่งเครือข่าย TCP/IP ออกเป็นซับเนต (หากมีการใช้ซับเนต) การใช้ซับเนตจะทำให้การใช้เครือข่ายขนาดใหญ่มีประสิทธิภาพมากขึ้น รูปแบบของซับเนต มาสก์จะคล้ายกับแอดเดรส IP: n.n.n.n โดย n เป็นตัวเลขตั้งแต่ 0 ถึง 255 หากไม่กำหนดซับเนต มาสก์ จะใช้ซับเนต มาสก์ที่กำหนดให้กับคอมพิวเตอร์ของท่าน
เมื่อกำหนดใช้กับแอดเดรส IP ซับเนต มาสก์จะระบุส่วนของแอดเดรส IP ที่ใช้เพื่อระบุเครือข่าย/ซับเนต และส่วนที่ใช้เพื่อระบุโหนดแม่ข่ายแต่ละโหนด

ไดรเวอร์เครื่องพิมพ์
โปรแกรมที่ควบคุมเครื่องพิมพ์เมื่อใดที่สั่งพิมพ์เอกสาร ไดรเวอร์เครื่องพิมพ์จะเข้ามาทำหน้าที่แทน โดยส่งข้อมูลไปยังเครื่องพิมพ์ด้วยคำสั่งควบคุมที่ถูกต้อง ระบบปฏิบัติการสมัยใหม่ส่วนใหญ่จะมาพร้อมไดรเวอร์เครื่องพิมพ์ของเครื่องพิมพ์สามัญทั่วไป แต่ท่านต้องติดตั้งไดรเวอร์ก่อนจึงจะสามารถใช้เครื่องพิมพ์ได้ ท่านยังสามารถดาวน์โหลดไดรเวอร์ที่ได้รับการปรับปรุงล่าสุดจากเว็บไซต์ของผู้ผลิตเครื่องพิมพ์ได้ด้วย ดูที่ PCL หรือ PostScript

ไดรเวอร์เครื่องพิมพ์ PCL
เหมาะสำหรับการพิมพ์ทั่วไปและเหมาะที่สุดสำหรับงานเวิร์ดโปรเซสซิ่งส่วนใหญ่ กระดาษทำการ และโปรแกรมพรีเซนเทชั่น (ตัวอย่างเช่น Word, Excel และ PowerPoint)

ไดรเวอร์เครื่องพิมพ์ PostScript
เหมาะสำหรับการพิมพ์เฉพาะเหมาะที่สุดสำหรับเดสก์ท็อปพับลิชชิ่ง และโปรแกรมกราฟิกขั้นสูง (เช่น Photoshop, PageMaker และ FrameMaker)

แอดเดรส IP
แอดเดรสของโปรโตคอลอินเตอร์เน็ต แอดเดรส IP จะระบุหมายเลขเฉพาะของแต่ละอุปกรณ์บนเครือข่าย TCP/IP ซึ่งกำหนดโดยผู้บริหารเครือข่าย เซิร์ฟเวอร์พิมพ์ HP Jetdirect รุ่นใหม่สนับสนุนทั้งแอดเดรส IPv4 และ IPv6
เมื่อเปิดเครื่องเซิร์ฟเวอร์พิมพ์ HP Jetdirect เครื่องจะพยายามกำหนดแอดเดรส IP ที่ถูกต้องจากเซิร์ฟเวอร์เครือข่าย หากไม่สามารถกำหนดแอดเดรส IP ที่ถูกต้องได้ ก็จะกำหนดแอดเดรสเริ่มต้นโดยอัตโนมัติ หากมีอุปกรณ์หลายตัวที่ใช้แอดเดรสเริ่มต้นเดียวกัน ให้ตรวจสอบแอดเดรสฮาร์ดแวร์ LAN (MAC) เพื่อระบุอุปกรณ์ที่ถูกต้อง

ดูที่ ฮาร์ดแวร์ LAN, MAC หรือแอดเดรสสถานี

ฮับของเครือข่ายไร้สาย
ดู จุดเชื่อมต่อไร้สาย (WAP)

ฮาร์ดแวร์ LAN, MAC หรือแอดเดรสสถานี
แอดเดรส ฮาร์ดแวร์ ซึ่งเรียกอีกอย่างว่าแอดเดรส MAC (Media Access Control) หรือ แอดเดรสสถานี เป็นแอดเดรสระดับการเชื่อมโยงที่กำหนดหมายเลขเฉพาะของแต่ละโหนดหรืออุปกรณ์แต่ละอุปกรณ์บนเครือข่าย ตามการกำหนดโดยมาตรฐานเครือข่าย IEEE 802.3 แอดเดรสจะประกอบด้วยเลขฐานสิบหก 12 หลัก (ตัวอย่างเช่น 001083123ABC) แอดเดรสฮาร์ดแวร์จะกำหนดโดยผู้ขายอุปกรณ์
HP จะกำหนดเซิร์ฟเวอร์พิมพ์ HP Jetdirect แต่ละเครื่องจากแอดเดรสฮาร์ดแวร์เฉพาะ ท่านสามารถกำหนดแอดเดรสฮาร์ดแวร์ของเซิร์ฟเวอร์พิมพ์ HP Jetdirect ของท่านได้โดยการสั่งพิมพ์หน้าทดสอบ HP Jetdirect สำหรับเซิร์ฟเวอร์พิมพ์ HP Jetdirect แบบภายนอก จะมีแอดเดรสอยู่บนฉลากซึ่งติดอยู่ที่เซิร์ฟเวอร์พิมพ์ด้วยเช่นกัน
