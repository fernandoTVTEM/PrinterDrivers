พบเครื่องพิมพ์

หน้าจอ พบเครื่องพิมพ์ ให้ท่านเลือกเครื่องพิมพ์ที่พบบนเครือข่ายท้องถิ่น หรือเลือกตัวเลือกการตั้งค่าเพิ่มเติมสำหรับการติดตั้งเครื่องพิมพ์บนเครือข่าย เครื่องพิมพ์ที่เชื่อมต่อด้วย USB หรือเครื่องพิมพ์ที่เชื่อมต่อผ่านพอร์ตขนาน

  • ติดตั้งเครื่องพิมพ์ที่พบบนเครือข่าย เลือกเครื่องพิมพ์ที่ต้องการจากรายการเครื่องพิมพ์ทั้งหมด (หรืออุปกรณ์มัลติฟังก์ชัน) ที่พบบนเครือข่ายท้องถิ่น และคลิก ถัดไป คลิกปุ่ม ค้นหาอีกครั้ง เพื่อทำการค้นหาเครื่องพิมพ์บนเครือข่ายอีกครั้ง
  • ช่วยค้นหาเครื่องพิมพ์ของฉันบนเครือข่าย เลือกตัวเลือกนี้หากเครื่องพิมพ์เครือข่ายที่ท่านต้องการไม่แสดงอยู่ในรายการ หรือท่านต้องการติดตั้งเครื่องพิมพ์ที่จะเชื่อมต่อโดยตรงกับคอมพิวเตอร์ของท่านโดยใช้สายเคเบิล USB หรือ แบบขนาน  สำหรับเครื่องพิมพ์เครือข่าย ตัวเลือกนี้จะให้ตัวเลือกการค้นหาเพิ่มเติม เช่น การค้นหาเครื่องพิมพ์ตามแอดเดรส IP 

หมายเหตุ: เมื่อต้องการติดตั้งเครื่องพิมพ์ที่ไม่ได้อยู่บนเครือข่ายท้องถิ่น โปรดดู การติดตั้งเครื่องพิมพ์บนเครือข่าย IP ขนาดใหญ่

ระบุเครื่องพิมพ์ในรายการด้วยฟิลด์ต่อไปนี้ แอดเดรสฮาร์ดแวร์อาจเป็นวิธีที่ง่ายที่สุดในการระบุเครื่องพิมพ์:

รุ่น ระบุรุ่นของเครื่องพิมพ์ (เช่น HP LaserJet 5000 Series) ไม่รู้จัก จะปรากฏขึ้นหากไม่สามารถระบุรุ่นของเครื่องพิมพ์ เครื่องพิมพ์ถูกปิด หรือหากไม่ได้เชื่อมต่อเครื่องพิมพ์ใดๆ สำหรับเซิร์ฟเวอร์การพิมพ์ภายนอก HP Jetdirect ข้อความต่อไปนี้จะปรากฏขึ้นหากเครื่องพิมพ์ถูกเชื่อมต่อ แต่ไม่สามารถระบุได้
  • แบบขนาน: เครื่องพิมพ์/พล็อตเตอร์ Centronics
  • USB (ทิศทางเดียว): อุปกรณ์ต่อพ่วงแบบทิศทางเดียว
  • USB (สองทิศทาง): อุปกรณ์ต่อพ่วงแบบสองทิศทาง
แอดเดรสฮาร์ดแวร์ (หรืออาจเรียกว่า แอดเดรสสถานี หรือแอดเดรส MAC) ระบุอุปกรณ์บนเครือข่ายท้องถิ่นอย่างเฉพาะเจาะจง สำหรับเครื่องพิมพ์ของ HP และเซิร์ฟเวอร์การพิมพ์ภายนอก HP Jetdirect ที่มีพอร์ตเครื่องพิมพ์หลายพอร์ต อาจมีการผนวกหมายเลขพอร์ต (พอร์ต 1, พอร์ต 2, พอร์ต 3) แอดเดรสนี้แสดงอยู่บนหน้า (ทดสอบ) การกำหนดค่าของเซิร์ฟเวอร์การพิมพ์
แอดเดรส IP (Internet Protocol) ระบุอุปกรณ์บนเครือข่าย TCP/IP อย่างเฉพาะเจาะจง และถูกกำหนดโดยผู้ดูแลระบบเครือข่าย ทั้งแอดเดรส IPv4 และ IPv6 จะแสดงในรายการนี้  หากเครื่องพิมพ์มีหลายแอดเดรส เครื่องพิมพ์จะปรากฏเพียงครั้งเดียวสำหรับแต่ละแอดเดรส ซึ่งจะทำให้ท่านสามารถเลือกอินเทอร์เฟซที่จะใช้ 

หมายเหตุ: เมื่อเซิร์ฟเวอร์การพิมพ์ HP Jetdirect ถูกเปิดเครื่อง และไม่ได้รับการกำหนดค่าด้วยแอดเดรส IP ที่ถูกต้องภายในสองวินาที แอดเดรส IP อัตโนมัติ (169.254.x.x) หรือแอดเดรส IP เริ่มต้น (192.0.0.192) จะถูกกำหนดโดยอัตโนมัติ  HP ขอแนะนำให้เปลี่ยนแปลงแอดเดรส IP เริ่มต้น  หากมีเครื่องพิมพ์หลายเครื่องที่ใช้แอดเดรสเริ่มต้นเดียวกัน ให้ตรวจสอบแอดเดรสฮาร์ดแวร์เพื่อระบุเครื่องพิมพ์ที่ถูกต้อง
ชื่อแม่ข่ายของ IP ชื่อแม่ข่ายของ IP เป็นชื่อที่คุ้นเคย (หรือระบุได้ง่าย) ซึ่งสามารถใช้แทนที่แอดเดรส IP ของเครื่องพิมพ์ โดยสามารถใช้เมื่อเซิร์ฟเวอร์ชื่อบนเครือข่ายถูกตั้งค่าให้แม็ปชื่อแม่ข่ายกับแอดเดรส IP ที่สัมพันธ์กัน หากชื่อแม่ข่ายถูกกำหนดและตั้งค่าบนเครื่องพิมพ์ (เช่น printer1.marketing.com) ท่านจะสามารถกำหนดชื่อแม่ข่ายเพื่อระบุเครื่องพิมพ์

หมายเหตุ: หากค่าในฟิลด์ถูกตัดทอน และท่านต้องการดูฟิลด์ทั้งหมด ให้ปรับขนาดความกว้างของฟิลด์ เมื่อต้องการปรับขนาด ให้วางเคอร์เซอร์ไว้ที่ส่วนขอบของหัวข้อฟิลด์ จากนั้นให้คลิกและลาก

สำหรับคำแนะนำเกี่ยวกับการพิมพ์หน้าการกำหนดค่าเครือข่าย โปรดดูเอกสารที่ให้มาพร้อมกับเครื่องพิมพ์

หากเครื่องพิมพ์ที่เลือกถูกคุ้มครองด้วยชื่อชุมชน SNMP หรือรหัสผ่าน Jetdirect เครื่องจะแจ้งให้ท่านป้อนชื่อชุมชนหรือรหัสผ่าน
