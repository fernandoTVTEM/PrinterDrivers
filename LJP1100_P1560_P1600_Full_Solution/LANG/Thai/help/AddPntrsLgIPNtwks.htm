การเพิ่มเครื่องพิมพ์เข้าในเครือข่าย IP ขนาดใหญ่

หากท่านต้องการเพิ่มเครื่องพิมพ์ (หรืออุปกรณ์หลายฟังก์ชัน) ผ่านทางเราเตอร์ หรือเกทเวย์ (ไม่ใช่ทางเครือข่ายท้องถิ่น) การระบุเครื่องพิมพ์ผ่านโปรแกรมยูทิลิตี้นี้อาจทำได้ยาก

  • เราเตอร์อาจไม่ส่งผ่านแอดเดรสฮาร์ดแวร์ แอดเดรสของ MAC หรือแอดเดรสสเตชั่นของเครื่องพิมพ์
  • หากมีเครื่องพิมพ์ใหม่หลายเครื่องในเครือข่ายท้องถิ่น การใช้แอดเดรส IP เริ่มต้น (192.0.0.192) ในการระบุเครื่องพิมพ์จะไม่รับรองว่าเป็นการเลือกเครื่องพิมพ์ที่ถูกต้อง ตรวจสอบ แอดเดรสฮาร์ดแวร์ เพื่อระบุเครื่องพิมพ์อย่างถูกต้อง

ในการใช้ยูทิลิตี้นี้ติดตั้งเครื่องพิมพ์ ให้ปฏิบัติดังนี้:

  1. ปฏิบัติอย่างใดอย่างหนึ่งต่อไปนี้:
    • ย้ายเครื่องพิมพ์ไปยังเครือข่ายท้องถิ่นของคอมพิวเตอร์ของท่าน หรือ
    • ใช้ยูทิลิตี้นี้ในคอมพิวเตอร์ซึ่งต่อเชื่อมเข้ากับเครือข่ายท้องถิ่นที่เครื่องพิมพ์ของท่านต่อเชื่อมอยู่
  2. ระบุเครื่องพิมพ์ผ่านยูทิลิตี้นี้ และกำหนดค่าพารามิเตอร์ IP ที่ตรงกับตำแหน่งทางเครือข่ายของเครื่องพิมพ์ที่ต้องการ หากต้องการค้นหาแอดเดรสของ MAC ท่านสามารถค้นหาได้โดยการพิมพ์หน้า (ทดสอบ) การกำหนดค่า (ทั้งจากทางเครื่องพิมพ์หรือจากเซิร์ฟเวอร์พิมพ์ภายนอก) เพื่อให้มั่นใจว่าท่านเลือกแอดเดรสฮาร์ดแวร์หรือแอดเดรสของ MAC ที่ถูกต้อง
  3. ถ้าจำเป็น ให้นำเครื่องพิมพ์ (หรือเซิร์ฟเวอร์พิมพ์ HP Jetdirect ภายนอก) กลับไปยังตำแหน่งเครือข่ายที่ต้องการ
  4. ในการติดตั้งเครื่องพิมพ์นี้ในคอมพิวเตอร์เครื่องอื่นๆ ให้ปฏิบัติอย่างใดอย่างหนึ่งต่อไปนี้:
    • สำหรับคอมพิวเตอร์เครื่องอื่นหรือเซิร์ฟเวอร์ที่จะพิมพ์ไปยังเครื่องพิมพ์นี้โดยตรง ให้ใช้ยูทิลิตี้นี้เพื่อกำหนดค่าให้กับคอมพิวเตอร์หรือเซิร์ฟเวอร์แต่ละเครื่อง หรือ
    • สำหรับเครื่องไคลเอนท์ที่ใช้ Windows ซึ่งจะพิมพ์ด้วยเครื่องพิมพ์นี้ผ่านทางระบบการใช้งานร่วมกัน ให้ใช้โปรแกรมยูทิลิตี้ Add Printer ของระบบไคลเอนท์ ซึ่งอยู่ในโฟลเดอร์ Printers หรือใช้โปรแกรมการติดตั้งเครื่องพิมพ์แบบใช้งานร่วมกัน ที่สร้างขึ้นสำหรับเครื่องพิมพ์นี้
