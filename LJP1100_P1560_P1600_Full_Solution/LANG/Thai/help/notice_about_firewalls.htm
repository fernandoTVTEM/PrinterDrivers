ตรวจพบไฟร์วอลล์

เพื่อสื่อสารผ่าน Windows Firewall โปรแกรมติดตั้งจะเพิ่มตัวเองลงในรายการข้อยกเว้นของโปรแกรมประยุกต์ไฟร์วอลล์ตามที่ปรากฏในข้อกำหนดรายละเอียดของ Microsoft วิธีนี้จะช่วยให้โปรแกรมติดตั้งสื่อสารกับเครื่องพิมพ์บนเครือข่ายสำหรับการติดตั้งได้

คลิกที่ 'ถัดไป' เพื่อค้นหาเครื่องพิมพ์บนเครือข่ายของคุณและทำการติดตั้งต่อไป หรือคลิกที่ 'ยกเลิก' เพื่อออกจากโปรแกรมติดตั้ง

หากคุณเปิดการทำงานของไฟร์วอลล์ไว้ (เช่น ZoneAlarm©, BlackICE©, Sygate Security Agent© และ Norton Personal Firewall© เป็นต้น) โปรดดูที่เอกสารประกอบของผู้จำหน่ายไฟร์วอลล์เพื่อดูวิธีปิดการทำงานของไฟร์วอลล์

เมื่อปิดการทำงานของไฟร์วอลล์สำหรับซอฟต์แวร์แล้ว HP ขอแนะนำให้ติดตั้งรายการต่างๆ ต่อไปนี้บนคอมพิวเตอร์ของคุณ:

  • อุปกรณ์ฮาร์ดแวร์ที่เป็นเครื่องป้องกันไฟร์วอลล์ระหว่างอินเทอร์เน็ตและคอมพิวเตอร์ของคุณ เช่น เกตเวย์
  • โปรแกรมตรวจจับไวรัสล่าสุด

สำหรับข้อมูลทั่วไปเกี่ยวกับการแก้ปัญหาด้านการสื่อสาร โปรดดูที่เอกสารประกอบที่มาพร้อมกับเครื่องพิมพ์ และคลิกเข้ามาดูเว็บไซต์สนับสนุนได้ที่ http://www.hp.com/support

ไฟร์วอลล์มักก่อให้เกิดปัญหาด้านการสื่อสาร หากเปิดใช้งานไฟร์วอลล์บนคอมพิวเตอร์เครือข่ายอยู่ ปัญหาต่อไปนี้อาจเกิดขึ้น:

    ในระหว่างการติดตั้งซอฟต์แวร์:

  • โปรแกรมติดตั้งเครื่องพิมพ์ HP จะไม่สามารถระบุตำแหน่งของเครื่องพิมพ์ HP บนเครือข่าย  ข้อความแสดงข้อผิดพลาดจะแจ้งให้ทราบว่า โปรแกรมติดตั้งไม่สามารถตรวจหาเครื่องพิมพ์ใดบนเครือข่าย  แต่คุณสามารถป้องกันปัญหานี้ได้ด้วยการปิดการทำงานของไฟร์วอลล์ชั่วคราวในระหว่างการติดตั้งซอฟต์แวร์เครื่องพิมพ์ของ HP ดูรายละเอียดวิธีการปิดการทำงานของไฟร์วอลล์ได้จากเอกสารประกอบของผู้จำหน่ายไฟร์วอลล์

    หลังจากการติดตั้งซอฟต์แวร์:

  • หากเปลี่ยนแปลงที่อยู่ IP ของเครื่องพิมพ์ HP ผลที่เกิดขึ้นคือ คอมพิวเตอร์ของคุณจะไม่สามารถสื่อสารกับเครื่องพิมพ์ของ HP  คุณจะไม่สามารถทำการพิมพ์ สแกน โทรสารผ่านคอมพิวเตอร์ และโอนภาพได้ และสถานะของเครื่องพิมพ์จะแจ้งให้ทราบว่า เครื่องพิมพ์ HP ถูกตัดการเชื่อมต่อ  ในสภาพแวดล้อมของเครือข่ายโฮม ถือเป็นเรื่องปกติที่ที่อยู่ IP จะเปลี่ยนไปจากเดิม  หากไม่ได้เปิดใช้งานไฟร์วอลล์ของซอฟต์แวร์ ที่อยู่ใหม่จะถูกส่งไปที่คอมพิวเตอร์ เพื่อให้ทั้งคอมพิวเตอร์และเครื่องพิมพ์ของ HP สื่อสารกันได้อย่างต่อเนื่อง  คุณป้องกันปัญหานี้ได้ด้วยการปิดการทำงานของไฟร์วอลล์ หรือหากคุณเป็นผู้ใช้ที่มีความชำนาญและคุ้นเคยกับระบบเครือข่าย คุณอาจเลือกใช้ที่อยู่ IP แบบตายตัวแทน   สำหรับคำแนะนำวิธีกำหนดค่าเครื่องพิมพ์ด้วยที่อยู่ IP แบบตายตัว โปรดดูที่เว็บไซต์ของ HP ที่ http://www.hp.com/support
  • หากคุณมีเครื่องออลล์-อิน-วันของ HP คุณจะไม่สามารรถใช้สล็อตของการ์ดหน่วยความจำบนเครื่องออลล์-อิน-วันของ HP เพื่อโอนไฟล์ไปไว้ที่คอมพิวเตอร์  เนื่องจากไดรฟ์เครือข่ายที่ถูกแมปหรือซอฟต์แวร์การโอนภาพจะไม่เข้าถึงไฟล์ที่อยู่บนการ์ดภาพถ่าย

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับไฟร์วอลล์ คุณสามารถดูที่เว็บไซต์ของ HP

วิธีปิดการทำงานของ Windows Firewall บน Windows Vista:

  1. คลิก เริ่ม > เครือข่าย
  2. คลิกขวาที่ เครือข่าย และเลือก คุณสมบัติ
  3. คลิก Windows Firewall ที่มุมล่างซ้าย
  4. เลือก 'เปิดหรือปิด Windows Firewall'
  5. เลือก 'ดำเนินการต่อ' บนกล่องโต้ตอบ User Access Control
  6. เลือกตัวเลือก 'ปิด' เพื่อปิดการทำงานของไฟร์วอลล์ และคลิก 'ตกลง'  ตอนนี้ไฟร์วอลล์ของ Windows จะไม่ทำงาน

วิธีปิดการทำงานของ Internet Connection Firewall (ICF) บน Windows XP:

  1. บนแถบงานที่ด้านล่างหน้าจอ คลิก เริ่ม และคลิก แผงควบคุม
  2. คลิก การเชื่อมต่อเครือข่ายและอินเทอร์เน็ต (หากมองไม่เห็น การเชื่อมต่อเครือข่ายและอินเทอร์เน็ต ให้คลิกที่ เปลี่ยนเป็นมุมมอง ประเภท ใต้ แผงควบคุม ทางด้านซ้ายของหน้าต่างแผงควบคุม)
  3. คลิก การเชื่อมต่อเครือข่าย
  4. คลิกขวาที่การเชื่อมต่อแบบ การเรียกผ่านสายโทรศัพท์, LAN หรือ อินเทอร์เน็ตความเร็วสูง ที่คุณใช้เชื่อมต่อกับอินเทอร์เน็ต และคลิก คุณสมบัติ จากเมนูเนื้อหา
  5. บนแท็บ ขั้นสูง ใต้ Internet Connection Firewall ยกเลิกที่เลือกไว้หรือปิดการทำงานของ ป้องกันคอมพิวเตอร์หรือเครือข่าย และคลิก ตกลง ตอนนี้ ไฟร์วอลล์ของ Windows XP จะไม่ทำงาน
  6. คลิก ถัดไป บนตัวช่วยติดตั้งเพื่อทำการติดตั้งเครื่องพิมพ์ต่อ
  7. ในการเปิดการทำงานของ Internet Connection Firewall เมื่อติดตั้งเครื่องพิมพ์เสร็จแล้ว ให้ทำตามขั้นตอนข้างต้นจนถึงตัวเลือก ป้องกันคอมพิวเตอร์หรือเครือข่าย แล้วให้เลือกตัวเลือก ป้องกันคอมพิวเตอร์หรือเครือข่าย อีกครั้ง

หมายเหตุ: ต้องเปิดการทำงานของไฟร์วอลล์สำหรับ Windows อีกครั้งเมื่อติดตั้งเสร็จเรียบร้อยแล้ว
