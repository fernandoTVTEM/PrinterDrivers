การตั้งค่าเครื่องพิมพ์

ท่านสามารถใช้หน้าจอ การตั้งค่าเครื่องพิมพ์ เพื่อระบุการตั้งค่าสำหรับเครื่องพิมพ์ที่ไม่ได้อยู่บนเครือข่าย

ท่านอาจป้อนหนึ่งที่อยู่ต่อไปนี้หนึ่งที่อยู่หรือหลายที่อยู่สำหรับเครื่องพิมพ์ แล้วคลิก Next  การป้อนที่อยู่มากกว่าหนึ่งที่อยู่จะช่วยให้แน่ใจได้ว่าพอร์ตที่สร้างจะถูกเชื่อมต่อไปยังเครื่องพิมพ์ที่ระบุ

  • ชื่อโฮสต์ IP ชื่อโฮสต์ IP เป็นชื่อที่จดจำได้ง่าย (หรือง่ายต่อการระบุ) ที่สามารถนำมาใช้แทนที่อยู่ IP ของเครื่องพิมพ์
  • ที่อยู่ฮาร์ดแวร์ ที่อยู่ฮาร์ดแวร์ LAN (หรือเรียกอีกชื่อหนึ่งว่าที่อยู่สถานีหรือที่อยู่ MAC) ระบุเฉพาะถึงอุปกรณ์
  • ที่อยู่ IP (IPv4) ที่อยู่ IP (อินเทอร์เน็ตโปรโตคอล) ทำหน้าที่ระบุถึงลักษณะเฉพาะของแต่ละอุปกรณ์ที่อยู่บนเครือข่าย TCP/IP ซึ่งโดยปกติผู้ดูแลระบบเครือข่ายจะเป็นผู้กำหนดยูทิลิตีนี้สามารถนำใช้เพื่อกำหนดค่าการตั้งค่า IPv4 ด้วยตัวเองบนอุปกรณ์เท่านั้น

หมายเหตุ: ในการตั้งค่าเครื่องพิมพ์ที่ไม่ได้อยู่บนเครือข่ายท้องถิ่น ให้ดูที่ การเพิ่มเครื่องพิมพ์บนเครือข่าย IP ขนาดใหญ่
