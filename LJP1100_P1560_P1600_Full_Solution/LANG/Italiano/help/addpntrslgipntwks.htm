Aggiunta di stampanti su reti IP di grandi dimensioni

Se le stampanti (o le periferiche multifunzione) vengono aggiunte tramite un router o un gateway (ovvero al di fuori della rete locale), l'identificazione delle stampanti tramite questa utility potrebbe risultare difficoltosa:

  • È possibile che il router non comunichi l'indirizzo hardware, MAC o di stazione della stampante.
  • Se ci sono diverse stampanti nuove sulla rete locale, l'identificazione tramite l'indirizzo IP predefinito (192.0.0.192) non garantisce la selezione della stampante corretta. Per identificare la stampante corretta, verificare l'indirizzo hardware.

Per utilizzare questa utility durante l'installazione di una stampante, seguire le istruzioni riportate di seguito.

  1. Scegliere uno dei metodi riportati di seguito.
    • Spostare la stampante sulla rete locale del computer utilizzato.
    • Utilizzare questa utility su un computer connesso alla rete locale della stampante.
  2. Identificare la stampante mediante questa utility e configurare i parametri IP in modo che corrispondano alla posizione finale della stampante sulla rete. Se è necessario individuare l'indirizzo MAC, stampare una pagina di configurazione (prova) dalla stampante o dal server di stampa esterno per accertarsi che venga selezionato l'indirizzo MAC o hardware corretto.
  3. Se necessario, ricollocare la stampante (o il server di stampa esterno HP Jetdirect) nella posizione finale all'interno della rete.
  4. Per installare questa stampante su altri computer, scegliere uno dei metodi seguenti:
    • Per altri computer o server che eseguono la stampa direttamente su questa stampante, utilizzare questa utility per configurare ciascun computer o server.
    • Per i client Windows che eseguono la stampa passando attraverso un sistema condiviso, utilizzare l'utility del sistema client Aggiungi stampante che si trova nella cartella Stampanti oppure utilizzare un programma per l'installazione di stampanti condivise creato appositamente per questa stampante.
