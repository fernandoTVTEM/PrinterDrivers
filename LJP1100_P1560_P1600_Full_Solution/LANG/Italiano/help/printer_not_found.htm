Stampante non trovata

Durante l'installazione guidata non è possibile comunicare con la stampante.

Se sul PC è stato abilitato un firewall, è possibile che durante l’installazione non venga rilevata la stampante. Prima di procedere con l’installazione, disabilitare temporaneamente il firewall.

Se è presente un firewall abilitato, ad esempio ZoneAlarm©, BlackICE©, Sygate Security Agent© e Norton Personal Firewall©, consultare le istruzioni per disabilitare il firewall nella documentazione fornita dal produttore.

Se il firewall software è disabilitato, HP consiglia che nel computer siano installati i componenti seguenti:

  • Una periferica hardware che fornisca protezione agendo come firewall tra Internet e il computer, ad esempio un gateway.
  • Un programma antivirus aggiornato.

Per informazioni generali sulla risoluzione di problemi di comunicazione, fare riferimento alla documentazione fornita con la stampante e visitare il sito Web del supporto all’indirizzo http://www.hp.com/support.

I firewall sono spesso causa di problemi di comunicazione. Se su un computer di rete è abilitato un firewall, è possibile che si verifichino i problemi seguenti:

   Durante l’installazione del software:

  • Viene visualizzato un messaggio di errore che indica che durante l’installazione non è stata rilevata alcuna stampante HP sulla rete. Per evitare questo problema, disabilitare temporaneamente il firewall durante l’installazione del software della stampante HP. Consultare le istruzioni per disabilitare il firewall nella documentazione fornita dal produttore.

   Dopo l’installazione del software:

  • Se l’indirizzo IP della stampante cambia, non sarà possibile stabilire una comunicazione tra il computer e la stampante HP. Tutti i processi di stampa, scansione, fax su PC e trasferimento di immagini non verranno portati a termine e lo stato del prodotto indicherà che la stampante HP è scollegata. In un ambiente di rete domestico è normale che ogni tanto l’indirizzo IP cambi. In tal caso, se il firewall software non è abilitato, il nuovo indirizzo viene inviato al computer, consentendo la continuazione della comunicazione tra il computer e la stampante HP. Per evitare questo problema, disabilitare il firewall. In alternativa, se si è un utente avanzato con conoscenze dell’ambiente di rete, è possibile assegnare alla stampante un indirizzo IP statico. Per suggerimenti sulla configurazione di un indirizzo IP statico per la stampante, vedere il sito Web HP all’indirizzo http://www.hp.com/support.
  • Non è possibile utilizzare gli alloggiamenti delle schede di memoria sulle unità HP All-in-one per trasferire file su computer in quanto i file sulle schede foto non risultano accessibili tramite l’unità di rete connessa o il software di trasferimento immagini.

Per ulteriori informazioni sui firewall, visitare il sito Web HP.

Disabilitare Windows Firewall in Windows Vista:

  1. Fare clic su Start > Rete.
  2. Fare clic con il pulsante destro del mouse su Rete, quindi scegliere Proprietà.
  3. Fare clic su Windows Firewall nell’angolo inferiore sinistro.
  4. Selezionare "Attiva/Disattiva Windows Firewall".
  5. Selezionare "Continua" nella finestra di dialogo del controllo di accesso utente.
  6. Per disabilitare il firewall, selezionare l'opzione "Disattivato", quindi fare clic su "OK".  Windows Firewall è ora disabilitato.

Disabilitare Firewall connessione Internet (ICF) in Windows XP:

  1. Fare clic su Start sulla barra delle applicazioni nella parte inferiore dello schermo, quindi scegliere Pannello di controllo.
  2. Scegliere la categoria Rete e connessioni Internet. Se Rete e connessioni Internet non viene visualizzata, fare clic su Passa alla visualizzazione per categorie sul lato sinistro della finestra del Pannello di controllo.
  3. Fare clic su Connessioni di rete.
  4. Fare clic con il pulsante destro del mouse su Connessione remota, LAN o Internet ad alta velocità, ossia il tipo di connessione utilizzato per connettersi a Internet, quindi scegliere Proprietà dal menu di scelta rapida.
  5. Nella sezione Firewall connessione Internet della scheda Avanzate, deselezionare o disabilitare l'opzione Proteggi il computer e la rete, quindi fare clic su OK. Il firewall di Windows XP è ora disabilitato.
  6. Fare clic su Avanti nell'installazione guidata per procedere con l'installazione della stampante.
  7. Per abilitare Firewall connessione Internet al termine dell'installazione della stampante, seguire la procedura sopra descritta per raggiungere l'opzione Proteggi il computer e la rete, quindi selezionarla nuovamente.

Nota: al termine dell'installazione, assicurarsi di riabilitare Windows Firewall.
