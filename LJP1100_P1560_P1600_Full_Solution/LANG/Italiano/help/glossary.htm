-A-

ARP
Acronimo di Address Resolution Protocol. Protocollo TCP/IP utilizzato per convertire un indirizzo IP in un indirizzo fisico (denominato indirizzo DLC), ad esempio un indirizzo Ethernet. Per richiedere un indirizzo fisico, l'host invia una richiesta ARP sulla rete TCP/IP. L'host in rete con l'indirizzo IP nella richiesta risponde con l'indirizzo hardware fisico.

Autenticazione
L'autenticazione è una strategia di protezione per le reti senza fili. Su una rete con autenticazione le periferiche utilizzano una chiave condivisa come password e comunicano solo con le periferiche che conoscono la chiave. A differenza della crittografia WEP, l'autenticazione non crittografa i dati inviati tra periferiche senza fili. Tuttavia, può essere utilizzata in combinazione con la crittografia WEP. Le chiavi di autenticazione e le chiavi WEP possono essere identiche.

BootP
Protocollo di bootstrap utilizzato per scaricare automaticamente un indirizzo IP e altri parametri TCP/IP da un server di rete. Quando la stampante (o periferica All-in-One) è accesa, il server di stampa HP Jetdirect invia una richiesta di avvio. Se è stato configurato correttamente un server BootP, questo trasmette i parametri al server di stampa Jetdirect attraverso la rete. Vedere anche DHCP.

Cavo USB
Un cavo USB (Universal Serial Bus) è un cavo a collegamento diretto comune.

Chiave WEP
Una chiave WEP, o "chiave di crittografia", è una sequenza di caratteri alfanumerici o cifre esadecimali. Dopo aver creato una chiave WEP, è necessario ricordarla o memorizzarla in una posizione sicura. È possibile che non si riesca a recuperare una chiave WEP persa. Una chiave WEP ha una lunghezza di 64 o di 128 bit. I primi 24 bit della chiave vengono forniti automaticamente. I rimanenti bit (40 per le chiavi a 64 bit o 104 per le chiavi a 128 bit) vengono forniti dalla persona che crea la chiave.
Per creare una chiave WEP, immettere i dati riportati di seguito in base al tipo di chiave WEP (alfanumerica o esadecimale) e al numero di bit che contiene.

Condivisione di stampanti
La condivisione delle stampanti consente ad altri utenti della rete di stampare sulla stampante tramite il computer dell'utente. Questa utility di installazione supporta la condivisione delle stampanti solo in ambiente Windows 2000 e Windows NT 4.0. Per consentire la condivisione delle stampanti, è necessario che nel sistema sia abilitato il servizio server. Selezionare una delle opzioni riportate di seguito.

  • Non condivisa: selezionare questa opzione per configurare il computer per la stampa peer-to-peer se non si desidera che altri utenti della rete accedano alla stampante tramite questo computer.
  • Condividi con nome: selezionare questa opzione per configurare il computer come server per la stampa client-server. I client di rete potranno inviare i processi di stampa alla stampante tramite questo computer.

Convenzioni di denominazione, nome di condivisione
In base alle raccomandazioni Microsoft, i nomi sono limitati a caratteri ASCII a 7 bit (i primi 127 caratteri). Non sono ammessi spazi. Attenersi alle seguenti istruzioni:

  • Sono consentite le lettere maiuscole dalla A alla Z.
  • Sono consentite le lettere minuscole dalla a alla z.
  • Sono consentiti i numeri da 0 a 9.
  • Sono consentiti i seguenti caratteri speciali: $ % ' - _ @ ~ ` ! ( ) { } . ^ # & + , ; = [ ]

Convenzioni di denominazione, stampante
Per assegnare un nome a una stampante (o periferica All-in-One), attenersi alle seguenti istruzioni:

  • Non sono consentiti nomi con una lunghezza superiore ai 31 caratteri.
  • Sono consentite le lettere maiuscole dalla A alla Z.
  • Sono consentite le lettere minuscole dalla a alla z.
  • Sono consentiti i numeri da 0 a 9.
  • Evitare di utilizzare i caratteri con valori ASCII inferiori a 32 o superiori a 127.
  • Non utilizzare le virgole (,).

-D-

DHCP
Acronimo di Dynamic Host Configuration Protocol. Protocollo che, in modo simile a BootP, scarica automaticamente un indirizzo IP da un server di rete. Tuttavia, un server DHCP gestisce un pool di indirizzi IP che assegna dinamicamente alle periferiche che li richiedono, assegnando solo un indirizzo IP temporaneo al server di stampa. Quando la stampante (o periferica All-in-One) viene spenta e riaccesa, non esistono garanzie che al server di stampa Jetdirect venga assegnato sempre lo stesso indirizzo IP, anche se il server DHCP cerca di ripetere l'assegnazione.

Driver
Vedere Driver della stampante, PCL o PostScript.

Driver della stampante
Un programma che controlla una stampante. Quando si stampa un documento, il driver della stampante invia i dati alla stampante tramite i comandi di controllo corretti. La maggior parte dei sistemi operativi attuali è dotata di driver per i tipi di stampante più comuni, che devono comunque essere installati per potere utilizzare la stampante. È inoltre possibile scaricare driver aggiornati dal sito Web del produttore della stampante. Vedere PCL o PostScript.

Driver della stampante PCL
Adatto alla stampa di tipo generico. È adatto in particolare per applicazioni di elaborazione testi, fogli di calcolo e presentazioni (ad esempio Word, Excel e PowerPoint).

Driver della stampante PostScript
Adatto alla stampa di tipo specialistico. È adatto in particolare per il desktop publishing e per applicazioni che prevedono un ampio utilizzo della grafica (ad esempio, Photoshop, PageMaker e FrameMaker).

Gateway
Un gateway è un router o un computer che connette due reti eterogenee (ad esempio, con diversi protocolli di comunicazione, formattazione dei dati, struttura o architettura). Quando le reti sono suddivise in sottoreti, i gateway vengono spesso utilizzati per isolare una sottorete da un'altra e controllare così il traffico di rete.
Nelle reti TCP/IP ogni periferica di rete è spesso configurata con l'indirizzo IP del gateway più vicino. Se sulla rete non esistono gateway, in genere viene utilizzato l'indirizzo IP del computer in uso.
Per ulteriori informazioni, vedere Aggiunta di stampanti su reti IP di grandi dimensioni.
Per informazioni sulle reti senza fili, vedere WAP (Wireless Access Point, punto di accesso senza fili).

Indirizzo hardware LAN, MAC o di stazione
Un indirizzo hardware, noto anche come indirizzo MAC (Media Access Control) o di stazione, è un indirizzo a livello di collegamento che identifica in modo univoco ogni nodo o periferica di una rete. In base agli standard di rete IEEE 802.3, questo indirizzo è costituito da un numero esadecimale a 12 cifre (ad esempio, 001083123ABC). L'indirizzo hardware è assegnato dal fornitore della periferica.
A ciascun server di stampa HP Jetdirect viene assegnato un indirizzo hardware univoco. È possibile determinare l'indirizzo hardware del server di stampa HP Jetdirect stampando una pagina di prova di HP Jetdirect. Nei server di stampa HP Jetdirect esterni l'indirizzo è riportato anche su un'etichetta applicata al server di stampa.

Indirizzo IP
Indirizzo del protocollo Internet. Un indirizzo IP identifica in modo univoco ciascuna periferica all'interno di una rete TCP/IP e viene gestito di solito da un amministratore di rete. I server di stampa HP Jetdirect più recenti supportano sia gli indirizzi IPv4 che gli indirizzi IPv6.
Se un server di stampa Jetdirect viene acceso, in genere tenta di autoconfigurarsi con un indirizzo IP valido da un server di rete. Se non è possibile assegnare un indirizzo IP valido, viene assegnato automaticamente un indirizzo predefinito. Se più periferiche vengono configurate con lo stesso indirizzo predefinito, per identificare quella corretta è necessario controllare l'indirizzo hardware LAN (MAC).

IPv4
Con l'acronimo IPv4 si definisce la versione 4 del protocollo Internet. Gli indirizzi IPv4 sono composti da 32 bit specificati in notazione decimale puntata, ad esempio 192.168.100.250. Se non viene assegnato un indirizzo IPv4 valido da un server di rete, il server di stampa Jetdirect assegnerà a se stesso un indirizzo di collegamento locale/Auto-IP (169.254.x.x) o un indirizzo predefinito (192.0.0.192), a seconda della rete.

IPv6
IPv6 è una versione migliorata del protocollo Internet, in grado di sostituire, ma anche di coesistere con IPv4. Gli indirizzi IPv6 presentano una lunghezza di 128 bit e i relativi valori vengono specificati in notazione esadecimale a due punti, ad esempio 2001:0DB8:0000:0000:0000:0000:1428:57AB. Pertanto, tali indirizzi potranno sostenere l'aumento del numero di periferiche che si collegano alle reti per un certo tempo. A differenza di quanto accade con gli indirizzi IPv4, è possibile assegnare più indirizzi IPv6 al server di stampa. Sebbene un indirizzo IPv6 di collegamento locale predefinito venga assegnato automaticamente, è anche possibile assegnare indirizzi IPv6 basati su router, basati su server e manuali.

Maschera di sottorete
La maschera di sottorete è il meccanismo utilizzato per suddividere una rete TCP/IP in sottoreti, qualora venga utilizzata questa suddivisione. La suddivisione in sottoreti consente di utilizzare in modo più efficiente le reti di grandi dimensioni. Il formato di una maschera di sottorete è simile a quello di un indirizzo IP, ossia n.n.n.n, dove n è un numero compreso tra 0 e 255. Se non è stata assegnata una maschera di sottorete, di solito viene utilizzata quella assegnata al computer in uso.
Quando viene applicata a un indirizzo IP, la maschera di sottorete specifica la parte dell'indirizzo IP utilizzata per identificare la rete/sottorete e la parte utilizzata per identificare i singoli nodi host.

Modalità di comunicazione
La modalità di comunicazione descrive il tipo di rete e il protocollo (TCP/IP) utilizzati per eseguire la stampa su una determinata stampante.

-N-

NDS
Acronimo di Novell Directory Services. NDS è un database relazionale che organizza in modo logico le risorse di rete come oggetti in una struttura ad albero gerarchica. Se si dispone dei diritti appropriati, è possibile accedere alle risorse di rete indipendentemente dalla loro posizione fisica. NDS è stato progettato per sostituire il bindery Novell.

Nome di comunità SNMP
Un nome di comunità SNMP è una password che deve essere fornita quando si utilizza SNMP per apportare una modifica alla configurazione della stampante. Se la stampante selezionata (o periferica All-in-One) è stata protetta da un nome di comunità SNMP o da una password Jetdirect, viene richiesto di immettere il nome di comunità. I nomi di comunità validi possono avere una lunghezza massima di 255 caratteri.
Nota: la richiesta viene effettuata solo se è stato assegnato un nome di comunità SNMP non predefinito. Il nome di comunità SNMP può essere assegnato da un amministratore di rete utilizzando un'applicazione di gestione (ad esempio, HP Web Jetadmin) o altri strumenti.

Nome di rete
Un nome di rete è una stringa di caratteri alfanumerici con distinzione tra maiuscole e minuscole che fornisce il controllo dell'accesso di base a una rete senza fili. Un nome di rete è noto anche come "identificatore del set di servizi" o "SSID".

Nome host IP
Un nome host IP è un nome di facile identificazione che può essere utilizzato al posto dell'indirizzo IP della stampante. Può essere utilizzato nei casi in cui è stato impostato sulla rete un server dei nomi per associare i nomi host ai corrispondenti indirizzi IP. Se è stato assegnato e configurato sulla stampante un nome host (ad esempio stampante1.marketing.com), questo nome può essere specificato per identificare la stampante.

Nome porta
La finestra di dialogo Nome porta viene visualizzata solo in Windows NT e Windows 2000, se non è stata selezionata l'opzione "Usa nome porta predefinito" nella finestra di dialogo Imposta modalità di comunicazione in rete e se è selezionata la modalità di comunicazione IP.

È possibile specificare il nome della porta di rete da creare per la stampante che si sta installando. Il nome della porta è utilizzato per identificare la porta nelle proprietà della stampante.

I nomi di porta validi sono costituiti da un massimo di 63 caratteri e possono contenere qualsiasi carattere. Il nome della porta deve essere univoco nel PC. Tale nome viene visualizzato nella schermata di riepilogo e nella pagina di prova.

Nome stampante
Immettere un nome per la stampante utilizzando le convenzioni di denominazione delle stampanti appropriate. Questo nome verrà utilizzato per identificare licona della stampante visualizzata nella cartella Stampanti del sistema. Per visualizzare la cartella Stampanti, fare clic su Avvio/Start sul desktop di Windows, selezionare Impostazioni, quindi fare clic su Stampanti.

-P-

Password Jetdirect
Una password Jetdirect è assegnata dall'amministratore di rete per limitare l'accesso ai parametri di configurazione sul server di stampa HP Jetdirect. Le password Jetdirect vengono assegnate utilizzando un'applicazione per la gestione della rete (ad esempio, HP Web Jetadmin) o altri strumenti.

PING
Acronimo di Packet Internet Groper. PING è un'utility che consente di determinare se un indirizzo IP specifico è accessibile. Questa utility invia un pacchetto all'indirizzo specificato e attende la risposta. L'utility PING viene utilizzata principalmente per risolvere i problemi di collegamento a Internet. Sono disponibili molte utility PING freeware e shareware per personal computer.

Profilo senza fili
Insieme di impostazioni di rete senza fili valide per una particolare rete senza fili. Ad esempio, una scheda LAN senza fili può avere un profilo per una rete domestica e un altro profilo per una rete aziendale. Quando si installa una periferica su una rete, assicurarsi di selezionare il profilo appropriato.

Rete Adhoc (rete periferica-periferica)
Una rete di tipo periferica-periferica è costituita da due o più periferiche senza fili collegate senza l'uso di una stazione di base centralizzata o di un punto di accesso senza fili (WAP).

Le prestazioni su questi tipi di rete possono peggiorare se la rete comprende più di sei periferiche.

Rete locale
Una rete locale è una rete che non si estende oltre i router o i gateway. Se il computer e la stampante (o periferica All-in-One) sono collegati a una rete locale, sono collegati allo stesso segmento di cavo di rete.

Rete peer-to-peer
Vedere Rete periferica-periferica.

Rete periferica-periferica (Adhoc)
Una rete di tipo periferica-periferica è costituita da due o più periferiche senza fili collegate senza l'uso di una stazione di base centralizzata o di un punto di accesso senza fili (WAP).

Rete senza fili (infrastruttura)
Rete costituita da periferiche che utilizzano una stazione di base centralizzata o un punto di accesso senza fili (WAP) per l'uso di servizi commerciali o pubblici.

Riavvio hardware
Prima di eseguire un riavvio hardware, si consiglia di stampare una pagina di prova per la stampante e il server di stampa Jetdirect.

Per ripristinare le impostazioni predefinite dei server di stampa esterni HP Jetdirect, scollegare il cavo di alimentazione, quindi premere il pulsante di prova sul server di stampa mentre si ricollega il relativo cavo di alimentazione. Attendere che il server di stampa venga reinizializzato.

Per ripristinare i server di stampa interni di HP Jetdirect, eseguire un riavvio hardware della stampante (o periferica All-in-One) e attendere la reinizializzazione del server di stampa. Per la maggior parte dei modelli di stampante HP LaserJet, premere il pulsante di continuazione sul pannello di controllo della stampante durante la procedura di accensione. Per le altre stampanti, consultare i rispettivi manuali.

ATTENZIONE: il ripristino della stampante può causare la cancellazione di tutte le impostazioni.

Server dei nomi
Un server dei nomi è un computer utilizzato sulla rete per associare (o risolvere) i nomi host (nomi di computer o server di stampa) a indirizzi IP. Esempi di server dei nomi sono DNS (Domain Name System) e WINS (Windows Internet Name Service).

Server di stampa
Vedere Server di stampa HP Jetdirect.

Server di stampa HP Jetdirect
I server di stampa HP Jetdirect sono accessori di rete Hewlett-Packard che consentono di connettere le stampanti direttamente a una rete. Le connessioni di rete dirette garantiscono flessibilità di posizione e prestazioni ottimali della stampante (o periferica All-in-One).
Le schede per server di stampa interni HP Jetdirect sono utilizzate con stampanti HP dotate di alloggiamenti di input/output. Alcuni server di stampa esterni HP Jetdirect possono essere utilizzati con quasi tutte le stampanti dotate di una porta parallela, mentre altri server di stampa HP Jetdirect possono essere utilizzati con quasi tutte le stampanti dotate di una porta USB. Entrambi i tipi di server di stampa supportano tutte le topologie e i protocolli di rete più diffusi.

Server di stampa Jetdirect
Vedere Server di stampa HP Jetdirect.

Sottorete
Una parte di rete che utilizza un componente di indirizzo comune. Nelle reti TCP/IP le sottoreti sono definite come tutte le periferiche i cui indirizzi IP utilizzano lo stesso prefisso. Ad esempio, tutte le periferiche con indirizzi IP che iniziano con 100.100.100. fanno parte della stessa sottorete. La divisione di una rete in sottoreti presenta dei vantaggi sia in termini di protezione che di prestazioni. Le reti IP vengono suddivise tramite una maschera di sottorete.

Stampa client-server
I computer client inviano i processi di stampa a server dedicati che controllano l'utilizzo di ciascuna stampante (o periferica All-in-One).

-W-

WAP (Wireless Access Point, punto di accesso senza fili)
Un WAP è una periferica attraverso la quale comunicano le periferiche (ad esempio, computer e stampanti) di una rete senza fili. Un WAP è denominato anche "stazione di base".

WEP
Standard di sicurezza originale utilizzato nelle reti Wi-Fi per crittografare il traffico della rete senza fili. WEP fornisce la sicurezza mediante la crittografia dei dati inviati sulle onde radio da una periferica senza fili a un'altra rendendo i dati non intellegibili per gli intrusi. Solo le periferiche che condividono le impostazioni WEP della stampante possono comunicare con la stampante.

WPA personale (accesso protetto Wi-Fi)
Standard di sicurezza migliorato per le reti Wi-Fi che fornisce un'elevata protezione dei dati e il controllo dell'accesso alla rete.
