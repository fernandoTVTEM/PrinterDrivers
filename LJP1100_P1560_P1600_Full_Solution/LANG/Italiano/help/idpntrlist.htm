Stampanti trovate

La schermata Stampanti trovate consente di selezionare una stampante individuata all'interno della rete locale o di poter usufruire di ulteriori opzioni di configurazione per l'installazione di stampanti di rete o dotate di connessione USB o parallela.

  • Installazione di una stampante di rete rilevata. Selezionare la stampante desiderata dall'elenco di tutte le stampanti (o periferiche multifunzione) rilevate all'interno della rete locale e fare clic su Avanti. Fare clic sul pulsante Cerca nuovamente per eseguire un'altra ricerca di stampanti all'interno della rete locale.
  • Guida per la ricerca della stampante. Selezionare questa opzione se la stampante desiderata non è presente nell'elenco o se si desidera installare una stampante e collegarla direttamente al computer in uso tramite un cavo USB o parallelo.  Nel caso di stampanti di rete, sono disponibili opzioni di ricerca aggiuntive, ad esempio per la ricerca di una stampante in base al relativo indirizzo IP. 

Nota: per installare stampanti non presenti nella rete locale, consultare la sezione Aggiunta di stampanti su reti IP di grandi dimensioni.

Le stampanti presenti nell'elenco vengono identificate attraverso i campi seguenti: il modo migliore per identificare una stampante è probabilmente attraverso il relativo indirizzo hardware.

Modello Identifica il modello della stampante (ad esempio Serie HP LaserJet 5000). Viene visualizzato il messaggio Sconosciuto se non è possibile identificare il modello della stampante o se non vengono rilevate stampanti collegate. Nel caso dei server di stampa esterni HP Jetdirect, viene visualizzato il seguente messaggio in caso non sia possibile identificare la stampante sebbene questa sia collegata.
  • Parallelo: Stampante/plotter Centronics
  • USB (unidirezionale): Periferica unidirezionale
  • USB (bidirezionale): Periferica bidirezionale
Indirizzo hardware (chiamato anche indirizzo della stazione o indirizzo MAC). Identifica in modo univoco una periferica presente nella rete locale. Nel caso di stampanti HP e di server di stampa esterni HP Jetdirect dotati di più porte stampante, è possibile che venga aggiunto un numero di porta (Porta 1, Porta 2, Porta 3). Questo indirizzo è presente nella pagina (di prova) della configurazione del server di stampa.
Indirizzo IP (Internet Protocol). Identifica in modo univoco una periferica presente su una rete TCP/IP e viene assegnato dall'amministratore di rete. Gli indirizzi IPv4 e IPv6 non vengono visualizzati in questo elenco.  Se una stampante dispone di più indirizzi IP, verrà visualizzata una volta per ciascuno di essi, consentendo all'utente di selezionare l'interfaccia desiderata. 

Nota: Se un server di stampa HP Jetdirect non viene configurato con un indirizzo IP valido entro due minuti dall'accensione, gli viene assegnato un indirizzo IP automatico  (169.254.x.x) o un indirizzo IP predefinito (192.0.0.192). HP consiglia di modificare l'indirizzo IP  predefinito. Se più stampanti hanno lo stesso indirizzo predefinito, controllare l'indirizzo hardware per identificare la stampante corretta.
Nome host IP Un nome host IP è un nome di facile identificazione che può essere utilizzato al posto dell'indirizzo IP della stamapante. Tale nome può essere utilizzato quando un server dei nomi presente sulla rete è stato configurato per associare i nomi host agli indirizzi IP corrispondenti. Se un nome host è stato assegnato e configurato per una determianta stampante (ad esempio, printer1.marketing.com), è possibile specificare il nome host per l'identificazione della stampante.

Nota: se il valore immesso nel campo viene troncato e si desidera visualizzare l'intero contenuto del campo, modificare la larghezza del campo. Per eseguire questa operazione, posizionare il cursorse al limite estremo dell'intestazione del campo e trascinarlo fino a raggiungere la dimensione desiderata.

Per istruzioni sulla stampa di una pagina di configurazione di rete, consultare la documentazione fornita con la stampante.

Se la stampante selezionata è protetta mediante un nome community SNMP o una password Jetdirect, verrà richiesto di specficare questi dati.
