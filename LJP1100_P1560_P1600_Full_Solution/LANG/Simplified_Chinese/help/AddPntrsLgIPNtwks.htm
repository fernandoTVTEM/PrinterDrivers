在大型 IP 网络上添加打印机

如果是跨路由器或网关（不在本地网络上）添加打印机（或多功能设备），则要通过此实用程序来标识打印机可能比较困难：

  • 路由器可能不经由打印机的硬件地址、MAC 地址或站地址。
  • 如果在本地网络上有多台新打印机，则使用默认 IP 地址 (192.0.0.192) 来标识打印机将不能保证选择的是正确打印机。检查硬件地址，确保标识的是正确的打印机。

要使用本实用程序来安装打印机，请按以下说明进行：

  1. 执行下列事项之一：
    • 将打印机实际移到所用计算机的本地网络，或
    • 在与打印机的本地网络连接的计算机上使用此实用程序。
  2. 通过此实用程序标识打印机，并配置与打印机的最终网络位置相匹配的 IP 参数。如果需要查明 MAC 地址，可以（从该打印机或从外置式打印服务器）打印配置（测试）页，然后从中找到它，以确保选择了正确的 MAC 地址或硬件地址。
  3. 如有必要，可将该打印机（或 HP Jetdirect 外置式打印服务器）恢复到其最终网络位置。
  4. 要在其它计算机上安装该打印机，请执行以下操作之一：
    • 对于直接打印到该打印机的其它计算机或服务器，请使用本实用程序来配置每一台计算机或服务器，或
    • 对于通过共享系统打印到该打印机的 Windows 客户机，请使用客户机系统的“打印机”文件夹中的“添加打印机”实用程序，或使用为该打印机创建的共享打印机安装程序。
