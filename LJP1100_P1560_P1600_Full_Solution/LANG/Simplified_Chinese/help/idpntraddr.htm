打印机设置

打印机设置屏幕允许您为不在网络上的打印机指定设置。

您可以为打印机输入以下一个或多个地址，然后单击下一步。  输入多个地址有助于确保所创建的端口连接到指定的打印机。

  • IP 主机名。 IP 主机名是用于代替打印机 IP 地址的友好（或易于标识的）名称。
  • 硬件地址。 LAN 硬件地址（也称作“站地址”或 MAC 地址）对设备进行唯一标识。
  • IP 地址 (IPv4)。 IP（Internet 协议）地址对 TCP/IP 网络上的各个设备进行唯一标识。它通常由网络管理员指定。此实用程序仅用于在设备上手动配置 IPv4 设置。

注： 要安装不处于本地网络的打印机，请参阅在大型 IP 网络上添加打印机。
