已找到打印机

在 Printers Found（已找到打印机）屏幕上，您可以选择在本地网络上找到的打印机，或者找到安装网络、USB 或并连打印机的其它安装选项。

  • 安装已找到的网络打印机。 从本地网络上找到的所有打印机（或多功能设备）列表中选择想要的打印机，然后单击下一步。 单击 Search Again（再次搜索）按钮，在网络上再次搜索打印机。
  • 帮助我查找打印机。 如果您想要的网络打印机没有列出，或者希望通过 USB 或并行电缆安装一台直接连接到计算机的打印机，请选择该选项。  对于网络打印机，该选项将提供其它搜索选项，例如通过 IP 地址来搜索打印机。 

注： 要安装不在本地网络上的打印机，请参阅在大型 IP 网络上添加打印机。

列表中的打印机通过以下字段加以标识。 标识打印机最容易的方法可能是使用硬件地址：

型号 标识打印机型号（如 HP LaserJet 5000 系列）。 如果不能标识打印机型号、打印机已关闭或尚未连接打印机，则将显示为 Unknown（未知）。 对于 HP Jetdirect 外置式打印服务器，如果打印机已连接但无法对其进行标识，将显示以下内容：
  • 并行： Centronics 打印机/绘图仪
  • USB（单向）： 单向外围设备
  • USB（双向）： 双向外围设备
硬件地址 （也称作“工作站地址”或“MAC 地址”）。 唯一标识本地网络上的一个设备。 对于有多个打印机端口的 HP 打印机和 HP Jetdirect 外置式打印服务器，可能会在其地址后附加一个端口号（端口 1、端口 2、端口 3）。 在打印服务器的配置（测试）页，列出了该地址。
IP 地址 （Internet 协议）唯一标识 TCP/IP 网络上的一个设备，并由网络管理员进行分配。 IPv4 和 IPv6 地址都将在该列表中显示。 如果一台打印机有多个地址，则每个地址的打印机都将显示一次，让您可以选择要使用的接口。 

注： 如果 HP Jetdirect 打印服务器已打开电源，并且在两分钟内没有为其配置一个有效的 IP 地址，则将自动分配一个自动 IP 地址 (169.254.x.x) 或一个默认 IP 地址 (192.0.0.192)。HP 建议应更改默认 IP 地址。 如果同一个默认地址存在多台打印机，请检查“硬件地址”以标识正确的打印机。
IP 主机名 IP 主机名称是一个很友好（易于标识）的名称，可以用于替代打印机 IP 地址。 如果已经将网络上的名称服务器设置为把主机名称映射到相对应的 IP 地址，则可以使用 IP 主机名称。 如果打印机上已经分配并配置了主机名称（如 printer1.marketing.com），您可以指定该主机名称来标识此打印机。

注： 如果一个字段条目被截断，而您想查看整个字段，请调整该字段的宽度。 要调整字段宽度，将光标置于该字段标题的边缘，然后单击并拖动。

有关打印网络配置页的说明，请查看该打印机附带的文档。

如果所选打印机受 SNMP 团体名称或 Jetdirect 密码保护，将提示您输入团体名称或密码。
