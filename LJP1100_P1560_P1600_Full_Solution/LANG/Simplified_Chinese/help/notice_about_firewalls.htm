发现防火墙

检测到防火墙

为了通过 Windows 防火墙进行通信，安装程序按照 Microsoft 规范将其自己添加到防火墙应用程序例外列表中。这样，安装程序就能在网络上与打印机通信，以进行安装。

单击下一步以在网络上搜索打印机，并继续安装。否则，请单击取消退出安装程序。

如果您已启用了某个防火墙（例如 ZoneAlarm©、BlackICE©、Sygate Security Agent© 和 Norton Personal Firewall© 等），请参阅防火墙厂商提供的文档，了解有关如何禁用防火墙的说明。

在禁用软件防火墙之后，HP 极力建议您在计算机上安装以下硬件和软件：

  • 在 Internet 和计算机之间提供防火墙保护的硬件设备（如网关）。
  • 最新的病毒检测程序。

有关排除通信故障的一般信息，请参阅随打印机提供的文档，并访问支持网站 http://www.hp.com/support。

防火墙经常会造成通信问题。如果在联网计算机上启用了防火墙，可能会出现以下问题：

    在安装软件时：

  • HP 打印机安装程序将无法在网络上找到该 HP 打印机。 此时会出现一条错误消息，指出安装程序无法在网络上检测到任何打印机。 为了避免出现此问题，在安装 HP 打印机软件时，应暂时禁用防火墙。有关如何禁用防火墙的说明，请参阅防火墙厂商提供的文档。

    在安装软件后：

  • 如果 HP 打印机的 IP 地址发生更改，您的计算机将无法与该 HP 打印机通信。 所有打印、扫描、PC 传真和图像传输作业都将失败，产品状态表明 HP 打印机已断开。在家庭网络环境中，有时改变 IP 地址属于正常情况。 如果未启用软件防火墙，新地址将会发送给计算机，使计算机和 HP 打印机可以继续通信。为了避免出现此问题，应禁用防火墙，如果您是熟悉联网技术的高级用户，可以选择使用静态 IP 地址。 有关如何为打印机配置静态 IP 地址的建议，请访问 HP 网站 http://www.hp.com/support。
  • 如果您使用的是 HP 一体机，将无法使用 HP 一体机上的内存卡插槽将文件传输到计算机上。这是因为照片卡上的文件无法从映射网络驱动器或图像传输软件中访问。

有关防火墙的详细信息，请访问 HP 网站。

如何在 Windows Vista 上禁用 Windows 防火墙：

  1. 单击 Start（开始）> Network（网络）。
  2. 右键单击 Network（网络），并选择 Properties（属性）。
  3. 单击左下角的 Windows Firewall（Windows 防火墙）。
  4. 选择 Turn Windows Firewall on/off（打开/关闭 Windows 防火墙）。
  5. 在 User Access Control（用户访问控制）对话框中选择 Continue（继续）。
  6. 选择 Off（关闭）选项以禁用防火墙，然后单击 OK（确定）。 Windows 防火墙现在已禁用。

如何在 Windows XP 上禁用 Internet 连接防火墙 (ICF)：

  1. 在屏幕底部的任务栏上，单击 Start（开始），然后单击 Control Panel（控制面板）。
  2. 单击 Network and Internet Connections（网络和 Internet 连接）类别（如果未显示 Network and Internet Connections（网络和 Internet 连接），单击 Control Panel（控制面板）窗口左侧 Control Panel（控制面板）下的 Switch to Category View（切换到分类视图））。
  3. 单击 Network Connections（网络连接）。
  4. 右键单击您连接 Internet 所使用的 Dial-up（拨号）、LAN 或 High-Speed Internet（高速 Internet）连接，然后从上下文菜单中单击 Properties（属性）。
  5. 在 Advanced（高级）选项卡的 Internet Connection Firewall（Internet 连接防火墙）下，取消选中或禁用 Protect my computer or network（保护我的计算机或网络），然后单击 OK（确定）。Windows XP 防火墙现在已禁用。
  6. 在安装向导上单击 Next（下一步），继续安装打印机。
  7. 要在安装打印机之后启用 Internet Connection Firewall（Internet 连接防火墙），按照上述步骤转到 Protect my computer or network（保护我的计算机或网络）选项。然后重新选中 Protect my computer or network（保护我的计算机或网络）选项。

注：安装完成后，请务必重新启用 Windows 防火墙。
