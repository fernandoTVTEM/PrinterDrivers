更改 TCP/IP 设置

可以为此打印机指定新的 TCP/IP 设置 (IPv4)。在更改任何设置之前，应先咨询网络管理员。

注：该安装实用程序将检测您的系统设置，并提供推荐的 TCP/IP 设置（包括 IPv4 和 IPv6），但是如果需要，可以修改 IPv4 设置（此实用程序只能用于手动配置 IPv4 设置）。您指定的参数值只有在完成了安装之后才会在打印机上配置。

・         如果打印机尚无有效的 TCP/IP 设置：
请在提供的字段中，输入打印机的有效 IP 地址 (IPv4)、子网掩码以及网关 IP 地址。在指定这些值前应当向网络管理员咨询。

单击建议设置由安装实用程序自动输入 TCP/IP 参数。

使用默认端口名称复选框（Windows NT、2000 和 XP）。选中该框以使用打印机的默认端口设置。如果不选中该框，下一个向导屏幕将是端口名称对话框。

注意：如果要将打印机连接到其它网络，则该实用程序建议的 TCP/IP 设置可能不是全局中唯一的，也就可能会与其它设备冲突。因此，在连接到其它 TCP/IP 网络时务必小心。

・         如果打印机已经有有效的 TCP/IP 设置：
则会显示当前 TCP/IP 设置。

注意：更改有效的 TCP/IP 设置会使其他用户无法访问。因此，在更改有效设置以前，请咨询网络管理员。

注：如果 TCP/IP 参数已由管理软件（如 HP Web Jetadmin）用密码保护起来，则条目字段为灰显。

“使用默认端口名称”复选框（仅 Windows NT、2000 和 XP）。选中该框以使用打印机的默认端口设置。如果不选中该框，下一个向导屏幕将是端口名称对话框。
