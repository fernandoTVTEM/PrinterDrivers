-A-

ARP
地址解析协议。ARP 是一种用来将 IP 地址转换为实际地址（称作 DLC 地址），例如以太网地址，的 TCP/IP 协议。希望获取实际地址的主机会将 ARP 请求广播到 TCP/IP 网络上。然后，该网络上有所需的 IP 地址的主机会用回复其实际的硬件地址。

BootP
用于从网络服务器自动下载 IP 地址和其它 TCP/IP 参数的引导协议。当打印机（或多功能设备）接通电源后，HP Jetdirect 打印服务器会广播引导请求。如果 BootP 服务器已妥当设置，它将在网络上把参数传输给 Jetdirect 打印服务器。另请参见 DHCP。

本地网络
本地网络是不超越路由器或网关的网络。如果计算机和打印机（或多功能设备）连接到本地网络，则它们连接在同一网络电缆段。

DHCP
动态主机配置协议。它与 BootP 相似，自动从网络服务器下载 IP 地址。但是，DHCP 服务器维护其动态分配给发出请求的设备的 IP 地址池，并只将 IP 地址借给打印服务器。任何时候只要打印机（或多功能设备）关闭后再打开，就不能保证 DHCP 服务器以后会为 Jetdirect 打印服务器指定同一个 IP 地址，尽管它会试图这样做。

打印 HP Jetdirect 测试页
要打印 HP Jetdirect 测试页，请参见 HP Jetdirect 硬件安装指南或打印机说明文档。对于 HP Jetdirect 外置式打印服务器，按一下打印服务器上的“测试”按钮。

打印服务器
请参见 HP Jetdirect 打印服务器。

打印机共享
打印机共享允许其他网络用户通过此台计算机打印到该打印机。该安装实用程序只支持 Windows 2000 和 Windows NT 4.0 上的打印机共享。必须在系统上启用服务器服务才能进行打印机共享。必须选择以下选项之一：

  • 不共享：如果您不想让其他网络用户通过此计算机访问打印机，则选择此选项，将计算机设置为用于对等打印。
  • 共享为：选择此选项，可将此计算机设置为客户-服务器打印的服务器。这样，网络客户机就能通过此计算机给打印机发送打印作业。

打印机名称
采用合适的打印机命名约定输入打印机的名称。此名称将用于在系统的“打印机”文件夹中标识该打印机的打印机图标。要显示“打印机”文件夹，从 Windows 桌面上单击开始，然后依次选择设置、打印机。

打印机驱动程序
控制打印机的程序。只要您打印文档，打印机驱动程序就会接管打印任务，使用正确的控制命令将数据传送给打印机。大多数常用操作系统都带有最通用的打印机的打印机驱动程序，但您必须先安装它们，才能使用打印机。您还可以从打印机生产商的网站下载更新的驱动程序。请参见 PCL 或 PostScript。

端口名称
“端口名称”对话框仅出现在 Windows NT 和 Windows 2000 中。仅在没有选中“设置网络通信模式”对话框中的“使用默认端口名称”并且选择了 IP 通信模式时，才出现该对话框。

您可以给安装的打印机指定要创建的网络端口名称。该端口名称用于标识打印机属性中的端口。

有效的端口名称可长达 63 个字符，并可由任何字符组成。该端口名称在您的 PC 上必须是唯一的。端口名称显示在摘要屏幕和测试页上。

-H-

HP Jetdirect 打印服务器
HP Jetdirect 打印服务器是将打印机直接连接到网络的 Hewlett-Packard 网络附件。采用直接网络连接，可以使设备的位置更加灵活并能获得最大的打印机（或多功能设备）性能。
HP Jetdirect 内置式打印机服务器卡用于具有输入/输出插槽的 HP 打印机。某些 HP Jetdirect 外置式打印服务器几乎可以用于任何具有并行端口的打印机，而其它 HP Jetdirect 打印服务器几乎可以用于任何带有 USB 端口的打印机。这两种类型的打印服务器支持所有通用的网络拓扑结构和协议。

-I-

INF 文件
INF 文件具有 .inf 扩展名，并且存储有关设备的信息。它可以让安装程序了解设备的名称、需要复制哪些文件以及从哪里复制。

IP 地址
Internet 协议地址。IP 地址唯一标识 TCP/IP 网络上的每个设备，通常由网络管理员管理。最近的 HP Jetdirect 打印机服务器同时支持 IPv4 和 IPv6 地址。
当 Jetdirect 打印服务器接通电源时，其通常会尝试使用来自网络服务器的有效 IP 地址对自身进行配置。如果无法分配有效 IP 地址，则将会自动分配默认地址。如果多个设备配置了相同的默认地址，则请选中 LAN 硬件 (MAC) 地址以标识正确的设备。

IP 主机名
IP 主机名是可用于替代打印机 IP 地址的友好（或易于识别的）名称。当网络上的名称服务器已设置为将主机名映射到其相应的 IP 地址时，可使用该选项。如果在打印机上已经指定并配置了主机名（例如，printer1.marketing.com），则可指定该主机名来标识打印机。

IPv4
IPv4 是 Internet 协议的第 4 版。IPv4 地址包含32 位，使用以点分隔的十进制的数字符号进行指定，例如 192.168.100.250。如果网络服务器未指定有效的 IPv4 地址，则 Jetdirect 打印服务器将根据网络使用链路本地 (link-local)/自动 IP 地址 (169.254.x.x) 或默认旧地址 (192.0.0.192)。

IPv6
IPv6 是增强型版本的 Internet 协议，其可能最终会取代 IPv4，但是也可能与 IPv4 共存。IPv6 长度为128 位，其值是以冒号分隔的十六进制符号指定的，例如 2001:0DB8:0000:0000:0000:0000:1428:57AB。因而，IPv6 地址将能够维持一段时间内连接到网络的设备数量的增长。与 IPv4 不同的是，可以分配多个 IPv6 地址给打印服务器。在自动分配默认链路本地 IPv6 地址的同时，也可以分配基于路由器、基于服务器和手动的 IPv6 地址。

LAN 硬件、MAC 或站地址
硬件地址，也称作介质访问控制 (MAC) 或站地址，是唯一地标识网络上每个节点或设备的链接层地址。按 IEEE 802.3 网络标准定义，它是一个 12 位的十六进制数字（例如，001083123ABC）。硬件地址由设备供应商指定。
HP 为每一个 HP Jetdirect 打印服务器分配一个唯一的硬件地址。通过打印 HP Jetdirect 测试页，可确定您的 HP Jetdirect 打印服务器的硬件地址。对于 HP Jetdirect 外置式打印服务器，地址也包括在打印服务器的标签上。

冷重置
在冷重置之前，建议打印一份打印机和 Jetdirect 打印服务器的测试页。

要将 HP Jetdirect 外置式打印服务器重置为出厂时的默认设置，请拔下电源线，然后重新插上电源线并按打印服务器上的“测试”按钮。等待打印服务器重新初始化。

要重置 HP Jetdirect 内置式打印服务器，请执行打印机（或多功能设备）的冷重置操作，并等待打印服务器重新初始化。（对于大多数 HP LaserJet 打印机型号，请在接通电源时，按打印机控制面板上的“执行”按钮。对于其它打印机，参阅相应的打印机手册。）

警告：重置打印机可能会清除打印机的所有设置。

名称服务器
名称服务器是网络上的计算机，用于映射（或解析）有 IP 地址的主机名（计算机或打印服务器名）。“域名系统”(DNS) 服务器和“Windows 因特网名称服务”(WINS) 服务器就是名称服务器。

命名约定，打印机
给打印机（或多功能设备）命名时，请遵守下列准则：

  • 长度不超过 31 个字符
  • 字母 A 至 Z，大写
  • 字母 a 至 z，小写
  • 数字 0 至 9
  • 避免使用 ASCII 值小于 32 或大于 127 的字符
  • 不能使用逗号 (,)

命名约定，共享名称
根据 Microsoft 的建议，名称应限制在 7 位 ASCII（前 127 个）字符。不允许有空格。请遵循这些准则：

  • 字母 A 至 Z，大写
  • 字母 a 至 z，小写
  • 数字 0 至 9
  • 有限的特殊字符： $ % ' - _ @ ~ ` ! ( ) { } . ^ # & + , ; = [ ]

SNMP 团体名称

身份验证
身份验证是一种保障无线网络安全的策略。在采用身份验证的网络上，各台设备均使用一个共享密钥作为密码，且它们只与知道此密钥的设备进行通信。与 WEP 不同的是，身份验证不对在无线设备之间发送的数据进行加密。但是，身份验证可以与 WEP 结合使用。身份认证密钥和 WEP 密钥可以完全相同。

-W-

WEP
Wi-Fi 网络中用来加密无线网络通信的原始安全标准。 WEP 提供安全保护的方法是，对通过无线电波从一台无线设备发送到另一台无线设备的数据进行加密，使窃听者无法识别数据。只有那些与打印机共享相同的 WEP 设置的设备才能与该打印机通信。

WEP 密钥
WEP 密钥（或“加密密钥”）是一组字母数字字符或十六进制数字。创建了 WEP 密钥后，您必须记住该密钥或将其保存在一个安全的地方。如果丢失了该密钥，您将无法获得 WEP 密钥。WEP 密钥的长度为 64 位或 128 位。该密钥的前 24 位是自动提供的。创建 WEP 密钥时，创建者应提供剩下的位数（64 位的密钥还需提供 40 位，128 位的密钥则还需提供 104 位）。
要创建 WEP 密钥，请根据 WEP 密钥是字母数字还是十六进制数字以及该密钥包含的位数输入下列信息：

类型位数输入的字符串包括...
字母数字64五个字母数字字符
128十三个字母数字字符
十六进制64十位十六进制数
128二十六位十六进制数

WPA 个人（Wi-Fi 受保护访问）
一种经过改进的 Wi-Fi 网络安全标准，可以提供强大的数据保护和网络访问控制。

网关
网关是连接两个不同的网络（例如，不同的通信协议、数据格式、结构或体系结构）的路由器或计算机。当网络被划成多个子网络时，经常使用网关来将一个子网与另一个子网隔离，以便控制网络通信。
在 TCP/IP 网络上，经常用最接近网关的 IP 地址配置各网络设备。如果网络上没有任何网关，则通常使用您的计算机的 IP 地址。
有关详细信息，请参见在大型 IP 网络上添加打印机。
有关无线网络的信息，请参见无线接入点 (WAP)。

网络名称
网络名称是区分大小写的字母数字字符串，它基本控制对无线网络的访问。 网络名称也称为“服务集标识符”或“SSID”。

无线接入点 (WAP)
无线接入点 (WAP) 是一台设备，无线网络上的各台设备（如计算机和打印机）通过它相互通信。WAP 也称作“基站”。

无线配置文件
无线配置文件是应用于特定无线网络的一组无线网络设置。例如，无线 LAN 卡可以有一个配置文件用于家庭网络，另一个用于办公网络。在网络上安装设备时，确保选择适当的配置文件。

无线网络（基础结构）
一种无线网络，其设备通过中心基站或无线接入点 (WAP) 连接来获取商业或公共服务。

-Z-

子网
共享公用地址部件的网络的一部分。在 TCP/IP 网络上，子网指那些其 IP 地址有相同前缀的所有设备。例如，其 IP 地址以 100.100.100. 开头的所有设备应属于同一个子网。将网络划分为多个子网，有助于提高安全和性能。IP 网络使用子网掩码进行划分。

子网掩码
子网掩码是用于将 TCP/IP 网络划分为多个子网的机制（如果使用子网化）。子网化可以更有效地利用大型网络。子网掩码的形式与 IP 地址相似：n.n.n.n（其中 n 是从 0 至 255 之间的数字）。如果尚未指定子网掩码，则通常使用指定给所用计算机的子网掩码。
在应用于 IP 地址时，子网掩码指定用于标识网络/子网的 IP 地址部分和指定用于标识各个主机节点的部分。
