Принтер Баптаулары

Принтер Баптаулары экран көрінісі Сізге желіден тыс принтердің баптауларын белгілеу мүмкіндігін береді.

Сіз принтердің келесі адрестерінің біреуін немесе одан көбін енгізіп, одан кейін Келесі.  батырмасын баса аласыз. Біреуден көп адресті енгізу құрылған порттың белгіленген принтерге қосылғандығына көз жеткізу мүмкіндігін береді.

  • IP Хост атауы. IP хост атауы принтердің ІР адресінің орнына пайдаланыла алатын ыңғайлы (немесе идентификациялау-үшін-қолайлы) атау болып табылады.
  • Құрылғы Адресі. Жергілікті желі аспаптық құрал адресі (Орналасу Адресі немесе MAC адресі деп те аталынатын) құрылғыны бірегей анықтайды.
  • IP адресі (IPv4). IP (Интернет Протокол) адресі TCP/IP желісіндегі әр құрылғыны анықтайды. Әдетте ол желі администраторымен беріледі. Бұл утилита құрылғының IPv4 баптауларын қолмен конфигурациялау үшін ғана қолданылады.

Ескерту: Жергілікті желіде орналаспаған принтерлерді орнату үшін, Кең IP Желілілеріне Принтер Қосу

қараңыз.
