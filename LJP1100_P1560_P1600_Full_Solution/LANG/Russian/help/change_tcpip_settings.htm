Изменение параметров TCP/IP

Вы можете указать новые параметры TCP/IP (IPv4) для этого принтера. Перед изменением каких-либо параметров необходимо проконсультироваться с администратором сети.

Примечание. Программа установки определит параметры системы и попытается установить рекомендуемые параметры TCP/IP (как IPv4, так и IPv6), однако параметры IPv4 при необходимости можно изменить (данная утилита позволяет изменить вручную только параметры IPv4). Значения параметров, назначенные вручную, не будут установлены в принтере до окончания установки.

·         Если в принтере установлены недействительные параметры TCP/IP
В отображенных полях укажите действительный адрес IP принтера (IPv4), допустимую маску подсети, а также действительный адрес IP шлюза. Перед установкой значений необходимо проконсультироваться с администратором сети.

Чтобы позволить программе установки автоматически назначить параметры TCP/IP, щелкните на Предложить параметры.

Используйте флажок порта по умолчанию (Windows NT, 2000 и XP). Для использования параметров порта по умолчанию установите этот флажок. Если этот флажок не установлен, мастер далее отобразит диалоговое окно Имя порта.

ВНИМАНИЕ! Если принтер будет подключаться к другим сетям, предложенные утилитой установки параметры TCP/IP могут конфликтовать с другими устройствами. Будьте внимательны при подключении к другим сетям TCP/IP.

·         Если в принтере уже установлены действительные параметры TCP/IP
Будут отображены только текущие параметры TCP/IP.

ВНИМАНИЕ! Изменение действительных параметров TCP/IP может препятствовать доступу других пользователей. Прежде чем изменять эти параметры, проконсультируйтесь с администратором сети.

Примечание. Если параметры TCP/IP были защищены паролем с помощью управляющей программы, например, HP Web Jetadmin, эти поля будут заблокированы.

Используйте флажок для установки имени порта по умолчанию (только для Windows NT, 2000 и XP). Для использования параметров порта по умолчанию установите этот флажок. Если этот флажок не установлен, мастер далее отобразит диалоговое окно Имя порта.
