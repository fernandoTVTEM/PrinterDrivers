Проверка соединения с принтером

Если для работы с принтером используется беспроводное соединение, проверьте, чтобы соответствующий индикатор принтера горел (для определения индикатора беспроводного соединения см. документацию, прилагаемую к принтеру). Затем выберите один из описанных ниже параметров.

Если принтер подключен с помощью сетевого кабеля, проверьте индикаторы сетевого порта сзади принтера.

Если индикатор... Тогда...

Горит зеленым цветом

 Затем выберите один из описанных ниже параметров. Для определения аппаратного адреса принтера (адреса MAC), адреса IP или имени устройства (имени хоста IP) распечатайте страницу конфигурации сети. Для получения инструкций по печати страницы конфигурации сети см. документацию, поставляемую вместе с принтером.

Не горит
  • Проверьте, надежно ли закреплен сетевой кабель, соединяющий принтер с беспроводной точкой доступа (WAP) или с компьютером.
  • Проверьте, включено ли устройство WAP.
  • Проверьте, включен ли принтер.
  • Проверьте, соединен ли компьютер с сетью.

После проверки перечисленных выше элементов выберите способ повторного поиска принтера:

  • Выберите Искать снова, чтобы мастер установки выполнил повторный поиск.
  • Выберите Указать принтер по адресу, чтобы вручную указать аппаратный адрес (адрес MAC) принтера, его адрес IP или имя устройства (имя хоста IP).
  • Выберите Ввести параметры несетевого принтера, если выполняется настройка порта принтера, не подключенного к сети. Для получения дополнительной информации об этом параметре см. документацию, прилагаемую к принтеру.
