Установка принтеров в больших IP-сетях

Если принтеры (или многофункциональные устройства) устанавливаются на участках сети за маршрутизатором или шлюзом (за пределами локальной сети), идентификация принтеров с помощью этой утилиты может вызывать затруднения по следующим причинам:

  • Маршрутизаторы могут не передавать аппаратный адрес, адрес MAC или адрес станции.
  • Если в локальной сети имеется несколько новых принтеров, использующих адрес IP по умолчанию (192.0.0.192), выбор правильного принтера не гарантируется. Для определения правильного принтера проверьте параметр Аппаратный адрес.

Для настройки принтера с помощью этой утилиты необходимо:

  1. Выполните одно из следующих действий:
    • Физически переместите принтер в локальную сеть, к которой подключен компьютер, или
    • Запустите эту утилиту на компьютере, который подключен к локальной сети настраиваемого принтера.
  2. С помощью этой утилиты определите принтер и настройте параметры IP, чтобы они соответствовали параметрам сети, в которой этот принтер будет использоваться. Если необходимо определить адрес MAC (или аппаратный адрес), его можно найти на странице конфигурации (пробной странице), которую можно распечатать либо с принтера, либо с внешнего сервера печати.
  3. Если необходимо, подключите принтер (или внешний сервер печати HP Jetdirect) к сети, в которой он будет использоваться.
  4. Для установки принтера на других компьютерах выполните одно из следующих действий:
    • С помощью этой утилиты настройте каждый компьютер или сервер, с которых этот принтер будет непосредственно использоваться, или
    • Для рабочих станций Windows, которые будут использовать этот принтер через систему совместного доступа, запустите из папки "Принтеры" утилиту "Установка принтера" или воспользуйтесь программой установки совместно используемого принтера, созданной для этого принтера.
