Обнаруженные принтеры

На экране Обнаруженные принтеры можно выбрать принтер, обнаруженный в локальной сети, или получить дополнительные параметры настройки для установки принтеров, подключенных к сети, USB или параллельным портам.

  • Установить обнаруженный сетевой принтер. Выберите нужный принтер из списка всех принтеров (или многофункциональных устройств), обнаруженных в локальной сети, и щелкните Далее. Нажмите кнопку Искать снова, чтобы еще раз выполнить поиск принтеров в сети.
  • Помочь найти принтер. Выберите этот вариант, если нужный сетевой принтер отстутствует в списке или необходимо установить принтер, подключенный непосредственно к компьютеру через USB или с помощью параллельного кабеля.  При выборе этого варианта для сетевых принтеров предоставляются дополнительные возможности поиска, например поиск принтера по адресу IP. 

Примечание. Чтобы настроить принтеры, расположенные не в локальной сети, см. раздел Установка принтеров в больших IP-сетях.

Принтеры в списке идентифицируются следующими полями. Аппаратный адрес является самым простым способом идентификации принтера.

Модель Определяет модель принтера (например HP LaserJet серии 5000). Значение Неизвестно отображается, если модель принтера не удается определить, принтер выключен или не подключен. Если принтер подключен, но его не удается идентифицировать, для внешних серверов печати HP Jetdirect отображается следующее.
  • Параллельный: принтер/плоттер Centronics
  • USB (однонаправленный): однонаправленное периферийное устройство
  • USB (двунаправленный): двунаправленное периферийное устройство
Аппаратный адрес (также называется адресом станции или адресом MAC). Уникально идентифицирует устройство в локальной сети. Для принтеров HP и внешних серверов печати HP Jetdirect, имеющих несколько портов, может быть добавлен номер порта (порт 1, порт 2, порт 3). Этот адрес отображается на странице конфигурации (пробной) сервера печати.
Адрес IP (Internet Protocol) Однозначно идентифицирует устройство в сети TCP/IP; этот адрес назначается администратором сети. В этом списке отображается и адрес IPv4, и IPv6.  При наличии у принтера нескольких адресов он появится в списке с каждым из адресов, что позволит выбрать нужный интерфейс. 

Примечание. Если в течение двух минут после включения питания сервера печати HP Jetdirect для него не будет назначен допустимый адрес IP, адрес будет назначен либо с помощью функции "Авто IP" (169.254.x.x), либо будет назначен адрес IP по умолчанию (192.0.0.192). Адрес IP по умолчанию рекомендуется изменять. При наличии нескольких принтеров с одинаковым адресом по умолчанию для определения нужного принтера проверьте аппаратный адрес.
Имя хоста IP Имя хоста IP – это удобное (легко распознаваемое) имя, которое можно использовать вместо адреса IP принтера. Его можно использовать, если с помощью сервера имен в сети установлено соответствие между именами хостов и их адресами IP. Если для принтера назначено и настроено имя хоста (например printer1.marketing.com), принтер можно идентифицировать по этому имени.

Примечание. Если приведенное в поле значение усечено, а вы хотите увидеть значение полностью, измените ширину поля. Для этого поместите курсор на границу заголовка поля, щелкните ее и перетащите в нужное положение.

Для получения инструкций по печати страницы конфигурации сети см. документацию, поставляемую вместе с принтером.

Если выбранный принтер защищен именем сообщества SNMP или паролем Jetdirect, потребуется указать имя сообщества или пароль.
