Адрес IP
Адрес протокола Internet Protocol. Адрес IP однозначно идентифицирует устройство в сети TCP/IP и обычно азначается администратором сети. Новые серверы печати HP Jetdirect поддерживают как адреса IPv4, так и IPv6. При включении питания сервера печати Jetdirect обычно он пытается выполнить самонастройку и получить действительный адрес IP у сетевого сервера. Если действительный адрес IP получить не удается, автоматически назначается адрес по умолчанию. Если имеется несколько устройств с одинаковым адресом по умолчанию, для идентификации правильного устройства укажите аппаратный адрес (MAC).

Аппаратный адрес
См. Аппаратный адрес в локальной сети, адрес MAC или адрес станции

Аппаратный адрес в локальной сети, адрес MAC или адрес станции
Аппаратный адрес, известный также как адрес MAC (Media Access Control) или адрес станции, представляет собой адрес канального уровня для уникальной идентификации каждого узла или устройства в сети. В соответствии со стандартом IEEE 802.3 этот адрес представляется шестнадцатеричным числом, состоящим из 12 цифр (например, 001083123ABC). Аппаратный адрес назначается производителем устройства.
HP назначает каждому серверу печати HP Jetdirect уникальный аппаратный адрес. Аппаратный адрес сервера печати HP Jetdirect можно определить, распечатав пробную страницу HP Jetdirect. На внешних серверах печати HP Jetdirect адрес также указывается на наклейке, прикрепленной к серверу печати.

Аутентификация
Аутентификация - это метод защиты в беспроводной сети. В сети, где применяется аутентификация, устройства в качестве пароля используют общий ключ и взаимодействуют только с теми устройствами, у которых имеется такой же ключ. В отличие от WEP при использовании аутентификации данные, передаваемые между беспроводными устройствами, не шифруются. Однако аутентификация может использоваться совместно с WEP. Ключи аутентификации и ключи WEP могут быть идентичными.

Драйвер принтера
Программа управления принтером. При печати документа драйвер принтера обеспечивает передачу данных на печать с правильными командами управления. Большинство современных операционных систем поставляются с набором драйверов для самых распространенных принтеров, но их необходимо устанавливать перед использованием принтера. С Web-серверов производителей принтеров можно также выгрузить обновленные версии драйверов. См. PCL или PostScript.

Драйвер принтера Postscript
Хорошо подходит для специализированной печати. Наилучшим образом подходит для настольных публикаций или приложений с расширенными возможностями использования графики (например, Photoshop, PageMaker и FrameMaker).

Драйвер принтера PCL
Хорошо подходит для обычной печати. Наилучшим образом подходит для большинства приложений, предназначенных для обработки текста, работы с электронными таблицами и подготовки презентаций (например, Word, Excel и PowerPoint).

Имя группы SNMP
Имя группы SNMP представляет собой пароль, который необходимо предоставлять при изменении параметров конфигурации принтера с помощью SNMP. Если выбранный принтер (или устройство All-in-One) защищен с помощью имени группы SNMP или пароля Jetdirect, необходимо указать имя группы. Имя группы может быть длиной до 255 символов.
Примечание. Запрос на ввод имени группы выдается только в том случае, если назначено имя группы SNMP, отличное от имени по умолчанию. Имя группы SNMP может быть назначено администратором сети с помощью приложения управления сетью (например, HP Web Jetadmin) или других подобных утилит.

Имя порта
Диалоговое окно "Имя порта" появляется только в системах Windows NT и Windows 2000. Данное диалоговое окно появляется только в том случае, если в окне установки режима работы по сети не выбран параметр "Использовать имя порта по умолчанию", но выбран режим взаимодействия по протоколу IP.

Здесь можно указать имя сетевого порта, который будет создан для устанавливаемого принтера. Имя порта используется для идентификации порта в свойствах принтера.

Имя порта может быть длиной до 63 символов и может включать любые символы. Имя порта должно быть уникальным для отдельного компьютера. Имя порта отображается на экране резюме и на пробной странице.

Имя принтера
Введите имя принтера, используя соответствующие соглашения об именовании принтера. Это имя будет использоваться для идентификации значка принтера в системной папке "Принтеры". Для отображения папки "Принтеры" нажмите кнопку Пуск в панели задач Windows, выберите Настройка, а затем Принтеры.

Имя сети
Имя сети является буквенно-цифровой строкой, символы в которой зависят от регистра. С помощью этого имени осуществляется управление доступом к беспроводной сети. Имя сети также известно под названием "идентификатор набора услуг" или "SSID".

Имя хоста IP
Имя хоста IP является удобным (или легко определяемым) именем, которое может использоваться вместо адреса IP принтера. Это имя можно использовать, если в сети установлен сервер имен, который выполняет преобразование имен хостов в их соответствующие адреса IP. Если для принтера назначено имя хоста (например, printer1.marketing.com), этот принтер можно идентифицировать по имени хоста.

Кабель USB
Кабель универсальной последовательной шины (USB) является наиболее распространенным методом прямого подключения.

Ключ WEP
Ключ WEP или "ключ шифрования" является буквенно-цифровой последовательностью или последовательностью шестнадцатеричных цифр. После создания ключа WEP необходимо его запомнить или сохранить в недоступном месте. Если ключ WEP потерян, его невозможно будет восстановить. Ключ WEP имеет длину либо 64 разряда, либо 128 разрядов. Первые 24 разряда ключа устанавливаются автоматически. При создании ключа WEP необходимо заполнить оставшиеся разряды (40 разрядов в случае 64-разрядного ключа или 104 разряда в случае 128-разрядного ключа).
В зависимости от типа (буквенно-цифровой или шестнадцатеричный), а также от длины ключа WEP, при его создании необходимо руководствоваться следующими правилами:

Маска подсети
Маска подсети используется в качестве механизма деления сети TCP/IP на подсети (в случае использования подсетей). Деление на подсети обеспечивает более эффективное использование больших сетей. Формат маски подсети подобен адресу IP: n.n.n.n, где n представляет собой число от 0 до 255. Если маска подсети не назначена, обычно используется маска подсети, назначенная компьютеру.
Маска подсети, наложенная на адрес IP, определяет одну часть адреса IP для идентификации сети/подсети, а другую часть для идентификации отдельных хост-узлов.

-П-

Пароль Jetdirect
Пароль Jetdirect назначается администратором сети для ограничения доступа к параметрам конфигурации на сервере печати HP Jetdirect. Пароли Jetdirect назначаются с помощью приложения управления сетью (например, с помощью HP Web Jetadmin) или других подобных средств.

Печать в среде "клиент-сервер"
Рабочие станции (клиенты) передают задания печати на специализированные компьютеры (серверы), которые управляют отдельными принтерами (или устройствами All-in-One).

Печать пробной страницы HP Jetdirect
Для печати пробной страницы HP Jetdirect см. инструкции в руководстве по установке оборудования HP Jetdirect или в документации к принтеру. Для внешних серверов печати HP Jetdirect нажмите и отпустите кнопку тестирования на самом сервере печати.

Подсеть
Часть сети, в которой используется общий адресный компонент. В сетях TCP/IP в подсеть входят все устройства, чьи адреса IP имеют один и тот же префикс. Например, все устройства с адресами IP, начинающимися с 100.100.100., являются частью одной подсети. Деление сети на подсети полезно как с точки зрения защиты, так и с точки зрения производительности. Деление сети IP осуществляется с помощью масок подсети.

Профиль беспроводной сети
Профилем беспроводной сети является набор параметров беспроводной сети, которые настраиваются и применяются для конкретной беспроводной сети. Например, плата беспроводной локальной сети может иметь один профиль для домашней сети и другой профиль для офисной сети. При установке устройства в сети необходимо убедиться в выборе соответствующего профиля.

Сервер имен
Сервером имен является компьютер в сети, который используется для преобразования (разрешения) имен хостов (имен компьютеров или серверов печати) в адреса IP. Сервер системы имен доменов (Domain Name System - DNS) и сервер службы имен Интернет (Windows Internet Name Service -WINS) являются примерами таких серверов.

Сервер печати

Сервер печати HP Jetdirect
Серверы печати HP Jetdirect являются сетевым оборудованием Hewlett-Packard, с помощью которого принтеры можно подключить напрямую к сети. Прямые соединения с сетью обеспечивают более гибкий подход к размещению оборудования и максимально повышают производительность принтера (или устройства All-in-One).
С принтерами HP, имеющими слоты ввода/вывода, используются платы внутреннего сервера печати HP Jetdirect. Некоторые внешние серверы печати HP Jetdirect могут использоваться практически с любым принтером, имеющим параллельный порт, в то время, как другие серверы печати HP Jetdirect могут использоваться практически с любым принтером, имеющим порт USB. Оба типа серверов печати поддерживают все наиболее распространенные сетевые топологии и протоколы.

Сеть "точка-точка"
См. Одноранговая сеть.

Сеть Adhoc (одноранговая сеть)
Одноранговая сеть состоит из 2 или более беспроводных устройств, подключенных к сети без использования централизованной базовой станции или беспроводной точки доступа (WAP). 

Если к такой сети подключить более шести устройств, ее производительность может резко упасть.

Совместное использование принтера
Совместное использование принтера позволяет другим пользователям сети печатать на данном принтере, подключенном к данному компьютеру. Утилита установки позволяет настроить совместное использование принтера только в среде Windows 2000 и Windows NT 4.0. Для совместного использования принтера в системе должна быть активизирована служба сервера. Кроме того, необходимо выбрать один из следующих параметров:

  • Нет общего доступа. Этот параметр используется для одноранговой печати, если доступ других пользователей к принтеру с помощью данного компьютера нежелателен.
  • Общий доступ. Этот параметр используется на данном компьютере для печати в режиме клиент-сервер. Сетевые рабочие станции (клиенты) получают возможность передавать задания печати на принтер, подключенный к данному компьютеру.

Соглашение об именовании, общее имя
В соответствии с рекомендациями Microsoft, в именах могут использоваться только 7-разрядные символы ASCII (первые 127 символов в таблице). Не допускаются пробелы. При составлении имен действуют следующие правила:

  • буквы от A до Z, прописные;
  • буквы от a до z, строчные;
  • цифры от 0 до 9;
  • отдельные специальные символы: $ % ' - _ @ ~ ` ! ( ) { } . ^ # & + , ; = [ ].

Соглашение об именовании, принтер
При указании имени принтера (или устройства All-in-One) придерживайтесь следующих рекомендаций:

  • длина не более 31 символа;
  • буквы от A до Z, прописные;
  • буквы от a до z, строчные;
  • цифры от 0 до 9;
  • не допускаются символы, коды ASCII которых меньше 32 и больше 127;
  • не допускаются запятые (,).

Требования, связанные с использованием утилиты

  • Система
    • Операционная система: Windows 95/98, Windows Me, Windows NT 4.0, Windows 2000 или Windows XP

    • Принтеры должны быть подключены к сети с помощью поддерживаемого сервера печати HP Jetdirect. Текущий список поддерживаемых серверов печати HP Jetdirect и других решений см. на Web-сервере HP (http://www.hp.com/go/support).

  • Совместное использование
    • Данное программное обеспечение поддерживает совместное использование принтера только в системах Windows XP, Windows NT 4.0 или Windows 2000.

Холодный сброс
Перед выполнением холодного сброса рекомендуется распечатать пробную страницу для принтера и для сервера печати Jetdirect.

Чтобы восстановить заводские параметры по умолчанию для внешних серверов печати HP Jetdirect, отключите шнур питания, нажмите кнопку тестирования и, удерживая ее нажатой, снова подключите шнур питания. Дождитесь выполнения инициализации сервера печати.

Чтобы выполнить сброс внутренних серверов печати HP, выполните холодный сброс принтера (или устройства All-in-One) и дождитесь инициализации сервера печати. (Для большинства моделей принтеров HP LaserJet нужно нажать кнопку "Пуск" на панели управления в момент включения питания. Для других принтеров обратитесь к соответствующим руководствам.)

ВНИМАНИЕ! При выполнении сброса принтера значения всех его параметров могут быть удалены.

Шлюз
Шлюзом является маршрутизатор или компьютер, соединяющий две разнородные сети (отличающиеся, например, протоколами передачи данных, форматом данных, структурой или архитектурой). Если сети разделяются на подсети, шлюзы часто используются для изоляции одной подсети от другой с целью более эффективного управления трафиком.
В сетях TCP/IP сетевые устройства часто настраиваются на адрес IP ближайшего шлюза. Если шлюзов нет в сети, обычно используется адрес IP компьютера.
Для получения дополнительной информации см. Установка принтеров в больших IP-сетях.
Для получения информации о беспроводной сети см. Устройство беспроводной точки доступа (WAP).

Adhoc (одноранговая сеть)
Одноранговая сеть состоит из 2 или более беспроводных устройств, подключенных к сети без использования централизованной станции или беспроводной точки доступа (WAP). 

ARP
Протокол определения адресов. ARP (Address Resolution Protocol) является протоколом TCP/IP, который используется для преобразования адреса IP в физический адрес (называемый адресом DLC), например, в адрес Ethernet. Узел, которому необходимо получить физический адрес, передает широковещательный запрос ARP в сеть TCP/IP. Узел в сети, который имеет адрес IP, указанный в запросе, передает в ответ свой физический аппаратный адрес.

BootP
Протокол начальной загрузки, который используется для автоматической выгрузки адреса IP и других параметров TCP/IP с сетевого сервера. При включении принтера (или устройства All-in-One) сервер печати HP Jetdirect рассылает широковещательный запрос на загрузку. Если сервер BootP был настроен правильно, он передает параметры серверу печати Jetdirect по сети. См. также DHCP.

DHCP
Протокол динамической конфигурации хоста (Dynamic Host Configuration Protocol). Этот протокол похож на протокол BootP в той степени, что с его помощью адрес IP автоматически выгружается с сетевого сервера. Однако сервер DHCP поддерживает пул адресов IP, которые он динамически назначает запрашивающим устройствам и предоставляет адрес IP серверу печати только в аренду. Каждый раз, когда принтер (или устройство All-in-One) выключается и через некоторое время снова включается, нет никакой гарантии, что серверу печати Jetdirect будет назначен тот же адрес IP, хотя сервер DHCP попытается это сделать.

IPv4
IPv4 это протокол Internet Protocol версии 4. Адреса IPv4 состоят из 32 битов, указываемых в десятичном формате, например 192.168.100.250. Если действительный адрес IPv4 не назначается сетевым сервером, в зависимости от типа сети сервер печати назначает его сам, указывая либо локальный/автоматический адрес IP (169.254.x.x), либо старый адрес по умолчанию (192.0.0.192).

IPv6
IPv6 – это расширенная версия протокола Internet Protocol, которая может заменить IPv4, но также может сосуществовать с IPv4. Адреса IPv6 состоят из 128 битов, а их значения указываются в шестнадцатеричном формате с двоеточиями, например 2001:0DB8:0000:0000:0000:0000:1428:57AB. Таким образом, адреса IPv6 позволят подключать к сетям множество дополнительных устройств. В отличие от IPv4 серверу печати можно назначить несколько адресов IPv6. Кроме того, что локальный адрес IPv6 можно назначать автоматически, адреса IPv6 могут назначаться маршрутизатором, сервером, а также вручную.

NDS
Сервис Каталога Novell (Novell Directory Services). NDS - это реляционная база данных, которая логически организует сетевые ресурсы, представленные объектами, в иерархической древовидной структуре. При наличии соответствующих прав можно получить доступ к сетевым ресурсам независимо от их физического расположения. NDS разработан взамен прежней структуры Novell Bindery.

PING
Утилита поиска Интернет-пакетов (Packet Internet Groper). Утилита PING предназначена для определения, доступен ли определенный адрес IP. Принцип ее работы заключается в передаче пакета на определенный адрес и ожидании ответа. Утилита PING используется в основном при устранении проблем с Интернет-соединениями. Существует множество бесплатных и условно бесплатных утилит PING для персональных компьютеров.

WEP
Стандарт базовой защиты, используемый в сетях Wi-Fi для шифрования передаваемых по беспроводной сети данных. Стандарт WEP обеспечивает защиту данных, передаваемых по радиоволнам от одного беспроводного устройства другому, путем их шифрования, что обеспечивает их нечитаемость при перехвате. Только те устройства, параметры WEP которых имеют те же значения, что и принтер, могут взаимодействовать с принтером.
