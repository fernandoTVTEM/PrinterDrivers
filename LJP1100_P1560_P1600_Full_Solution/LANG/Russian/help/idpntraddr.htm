Параметры принтера

На экране Параметры принтера можно указать параметры принтера, не подключенного к сети.

Для принтера можно ввести один из описанных ниже адресов, а затем нажать на Далее.  При вводе нескольких адресов созданный порт будет гарантированно подключен к указанному принтеру.

  • Имя хоста IP. Именем хоста IP является удобочитаемое (или легко распознаваемое) имя, которое можно использовать вместо адреса IP принтера.
  • Аппаратный адрес. Аппаратный адрес (также называемый адресом станции или адресом MAC) однозначно идентифицирует устройство в локальной сети.
  • Адрес IP (IPv4). Адрес IP (Internet Protocol) однозначно идентифицирует каждое устройство в сети TCP/IP. Обычно он назначается администратором сети. Данную утилиту можно использовать только для настройки параметров IPv4 устройства вручную.

Примечание. Для получения информации о настройке принтеров, которые находятся не в локальной сети, см. Установка принтеров в больших IP-сетях.
