Принтер не найден

Мастер установки не может установить соединение с принтером.

Если на данном ПК установлен брандмауэр, его необходимо отключить, т.к. он может воспрепятствовать программе установки обнаружить принтер. Перед продолжением установки временно отключите брандмауэр.

Если на компьютере работает брандмауэр (например ZoneAlarm©, BlackICE©, Sygate Security Agent©, Norton Personal Firewall© и т.д.), для получения инструкций по его отключению см. документацию поставщика.

Так как программный брандмауэр может периодически отключаться, специалисты компании HP настоятельно рекомендуют установить следующие дополнительные компоненты:

  • Аппаратное устройство, обеспечивающее функции защиты брандмауэра между Интернетом и компьютером, например шлюз.
  • Постоянно обновляемое антивирусное программное обеспечение.

Для получения общей информации о решении проблем взаимодействия обратитесь к документации, прилагаемой к принтеру, и посетите Web-узел поддержки http://www.hp.com/support.

Брандмауэры часто становятся причиной коммуникационных проблем. Если брандмауэр установлен на подключенном к сети компьютере, могут возникнуть следующие проблемы:

    Во время установки программного обеспечения:

  • Программа установки принтера HP не сможет обнаружить принтер HP в сети. Появится сообщение о невозможности обнаружить какой-либо принтер в сети. Во избежание этой проблемы во время установки принтера HP временно отключите брандмауэр. За инструкциями по отключению обратитесь к документации поставщика брандмауэра.

    После установки программного обеспечения:

  • Если адрес IP принтера HP изменился, компьютер не сможет взаимодействовать с принтером HP. Все задания печати, сканирования, отправки факса и передачи изображений не будут выполняться, а в состоянии принтера HP будет указано, что он не подключен. В домашних сетях периодическое изменение адреса IP является обычным явлением. Если программный брандмауэр отключен, новый адрес передается компьютеру, чтобы он мог продолжать взаимодействовать с принтером HP. Во избежание этой проблемы отключите брандмауэр или, если Вы являетесь опытным пользователем и знакомы с сетевыми технологиями, можно использовать статический адрес IP. Предложения по настройке статического адреса IP для принтера HP см. на Web-узле http://www.hp.com/support.
  • При использовании устройства HP All-in-One невозможно использовать карту памяти для передачи файлов на компьютер. Причиной является то, что при назначении сетевого диска или с помощью программы переноса изображений файлы на фотокарте будут недоступны.

Для получения дополнительной информации о брандмауэрах можно посетить Web-узел HP.

Отключение брандмауэра Windows в системе Windows Vista:

  1. Выберите Пуск > Сеть.
  2. Щелкните правой кнопкой мыши на Сеть и выберите Свойства.
  3. В нижнем левом углу выберите Брандмауэр Windows.
  4. Выберите Включить или выключить брандмауэр Windows.
  5. В диалоговом окне управления доступом пользователя выберите Продолжить.
  6. Для отключения брандмауэра выберите Выключить и нажмите на OK. При этом брандмауэр Windows будет отключен.

Отключение брандмауэра подключения к Интернету (ICF) в Windows XP

  1. На панели задач внизу экрана нажмите на Пуск, а затем на Панель управления.
  2. Выберите категорию Сеть и подключения к Интернету. Если категория Сеть и подключения к Интернету отсутствует, в левой стороне окна панели управления под заголовком Панель управления щелкните на Переключение к виду по категориям.
  3. Нажмите на Сетевые подключения.
  4. Щелкните правой кнопкой мыши на используемом типе соединения (Удаленный доступ, ЛВС или Высокоскоростной Интернет), а затем из контекстного меню выберите Свойства.
  5. На вкладке Дополнительно под заголовком Брандмауэр подключения к Интернету отключите параметр Защитить мое подключение к Интернету, а затем нажмите на OK. При этом брандмауэр Windows XP будет отключен.
  6. Для продолжения установки принтера с помощью мастера установки нажмите на Далее.
  7. Чтобы включить брандмауэр подключения к Интернету после установки принтера, следуйте приведенным выше инструкциям для доступа к параметру Защитить мое подключение к Интернету. Затем снова установите параметр Защитить мое подключение к Интернету.

Примечание. Обязательно включите брандмауэр Windows после окончания установки.
