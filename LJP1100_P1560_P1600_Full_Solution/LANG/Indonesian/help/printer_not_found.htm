Printer Tidak Ditemukan

Wizard konfigurasi tidak dapat berkomunikasi dengan printer.

Jika firewall telah diaktifkan pada PC, hal ini mungkin menghambat penginstal untuk mendeteksi printer. Nonaktifkan sementara firewall pada PC sebelum melanjutkan penginstalan.

Jika firewall diaktifkan (misalnya ZoneAlarm©, BlackICE©, Sygate Security Agent©, and Norton Personal Firewall©, dll.), lihat dokumentasi vendor firewall Anda untuk memperoleh petunjuk tentang cara menonaktifkan firewall tersebut.

Setiap kali firewall perangkat lunak dinonaktifkan, HP sangat menganjurkan untuk menginstal berikut ini pada komputer Anda:

  • Perangkat keras yang memberikan perlindungan firewall antara Internet dan komputer Anda, misalnya gateway.
  • Program deteksi virus terkini.

Untuk informasi umum tentang mengatasi masalah komunikasi, lihat dokumentasi yang diberikan bersama printer dan kunjungi situs Web dukungan di http://www.hp.com/support.

Firewall sering menyebabkan masalah komunikasi. Jika firewall diaktifkan pada komputer jaringan, masalah berikut ini dapat terjadi:

    Selama menginstal perangkat lunak:

  • Penginstal printer HP tidak akan dapat menemukan printer HP pada jaringan.  Pesan kesalahan akan memberitahukan bahwa konfigurasi tidak dapat mendeteksi printer apapun pada jaringan.  Untuk mencegah masalah ini, nonaktifkan sementara firewall Anda selama menginstal perangkat lunak printer HP. Lihat dokumentasi vendor firewall Anda untuk memperoleh petunjuk tentang cara menonaktifkannya. 

    Setelah menginstal perangkat lunak:

  • Jika alamat IP printer HP berubah, maka komputer Anda tidak akan dapat berkomunikasi dengan printer HP.  Semua pekerjaan cetak, pindai, faks PC, dan transfer gambar akan gagal dan status produk akan menunjukkan bahwa printer HP terputus.  Di lingkungan jaringan asal, alamat IP yang terkadang berubah merupakan hal normal.  Jika firewall perangkat lunak tidak diaktifkan, alamat baru akan dikirim ke komputer, yang memungkinkan komputer dan printer HP untuk terus berkomunikasi.  Untuk mencegah masalah ini, nonaktifkan firewall atau jika Anda pengguna mahir yang menguasai jaringan, gunakan alamat IP statis.  Untuk saran tentang cara mengkonfigurasi printer dengan alamat IP statis, kunjungi situs Web HP di http://www.hp.com/support.
  • Jika memiliki HP all-in-one, Anda mungkin tidak akan dapat menggunakan slot kartu memori pada HP all-in-one untuk mentransfer file ke komputer Anda.  Hal ini disebabkan karena file pada kartu foto tidak akan dapat diakses dari drive jaringan yang dipetakan atau perangkat lunak transfer gambar.

Untuk informasi lebih lanjut tentang firewall, kunjungi situs web HP.

Cara menonaktifkan Firewall Windows pada Windows Vista:

  1. Klik Start > Network.
  2. Klik kanan Network lalu pilih Properties.
  3. Klik Windows Firewall di sudut kiri bawah.
  4. Pilih 'Turn Windows Firewall on/off.'
  5. Pilih 'Continue' pada dialog User Access Control.
  6. Tentukan pilihan 'Off' untuk menonaktifkan firewall, lalu klik 'OK.'  Firewall Windows kini dinonaktifkan.

Cara Menonaktifkan Firewall Koneksi Internet (ICF) pada Windows XP:

  1. Pada baris tugas di bagian bawah layar Anda, klik Start, lalu klik Control Panel.
  2. Klik kategori Network and Internet Connections. (Jika Network and Internet Connections tidak terlihat, klik Switch to Category View pada Control Panel di sebelah kiri jendela Control Panel.)
  3. Klik Network Connections.
  4. Klik kanan koneksi Dial-up, LAN, atau High-Speed Internet yang digunakan untuk terhubung ke Internet, lalu klik Properties dari menu konteks.
  5. Pada tab Advanced, di bawah Internet Connection Firewall, batalkan pilihan atau nonaktifkan Protect my computer or network, lalu klik OK. Firewall Windows XP kini dinonaktifkan.
  6. Klik Next pada wizard konfigurasi untuk melanjutkan penginstalan printer.
  7. Untuk mengaktifkan Internet Connection Firewall setelah printer diinstal, ikuti langkah-langkah di atas agar dapat membuka pilihan Protect my computer or network. Setelah itu, tentukan kembali pilihan Protect my computer or network.

Catatan: Pastikan untuk mengaktifkan kembali Firewall Windows setelah penginstalan selesai.
