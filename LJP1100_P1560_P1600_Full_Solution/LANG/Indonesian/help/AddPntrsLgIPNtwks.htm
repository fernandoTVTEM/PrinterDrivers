Menambahkan Printer pada Jaringan IP Besar

Jika Anda menambahkan printer atau perangkat multifungsi melalui router atau gateway (bukan pada jaringan lokal Anda), mungkin sulit untuk mengenali printer dengan utilitas ini:

  • Router mungkin tidak melalui Alamat Perangkat Keras, MAC, atau Stasiun printer.
  • Jika Anda memiliki beberapa printer pada jaringan lokal, menggunakan alamat IP default (192.0.0.192) untuk mengenali printer tidak menjamin bahwa printer yang benar akan dipilih. Periksa Alamat Perangkat Keras untuk mengenali printer yang benar.

Untuk menggunakan utilitas ini dalam mengatur printer, lanjutkan sebagai berikut:

  1. Lakukan salah satu tindakan berikut ini:
    • Pindahkan printer secara fisik ke jaringan lokal komputer Anda, atau
    • Gunakan utilitas ini pada komputer yang terhubung ke jaringan lokal printer Anda.
  2. Identifikasi printer menggunakan utilitas ini lalu konfigurasikan parameter IP yang cocok dengan lokasi jaringan akhir printer. Jika Anda perlu mengetahui alamat MAC, Anda dapat menemukannya dengan mencetak halaman konfigurasi (pengujian) (dari printer atau dari server cetak eksternal) untuk memastikan Anda telah memilih alamat MAC atau perangkat keras yang benar.
  3. Bila perlu, kembalikan printer (atau server cetak eksternal HP Jetdirect) ke lokasi jaringan akhirnya.
  4. Untuk menginstal printer ini pada komputer lain, lakukan salah satu tindakan berikut ini:
    • Untuk komputer atau server lain yang akan mencetak langsung ke printer ini, gunakan utilitas ini untuk mengkonfigurasi setiap komputer atau server, atau
    • Untuk klien Windows yang akan mencetak ke printer ini melalui sistem bersama, gunakan utilitas Tambah Printer sistem klien dalam folder Printer, atau gunakan program penginstalan printer bersama yang dibuat untuk printer ini.
