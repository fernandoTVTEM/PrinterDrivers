-A-

Adhoc, jaringan (Jaringan Perangkat ke Perangkat)
Jaringan Perangkat ke Perangkat adalah 2 atau lebih perangkat nirkabel yang terhubung tanpa menggunakan stasiun basis terpusat atau Titik Akses Nirkabel (WAP). 

Performa jaringan jenis ini dapat menurun jika ada lebih dari enam perangkat pada jaringan.

Alamat IP
Alamat Protokol Internet. Alamat IP mengidentifikasi secara unik setiap perangkat pada jaringan TCP/IP dan umumnya dikelola oleh administrator jaringan. Server cetak HP Jetdirect terbaru mendukung alamat IPv4 dan IPv6.
Bila dihidupkan, umumnya server cetak Jetdirect akan mencoba mengkonfigurasikan dirinya sendiri dengan alamat IP yang berlaku dari server jaringan. Jika alamat IP yang berlaku tidak dapat ditetapkan, maka alamat default akan ditetapkan secara otomatis. Jika beberapa perangkat dikonfigurasikan dengan alamat standar yang sama, periksa Alamat Perangkat Keras (MAC) untuk mengidentifikasi perangkat yang benar.

Alamat Perangkat Keras LAN, MAC, atau Stasiun
Alamat perangkat keras, juga disebut alamat Media Access Control (MAC) atau stasiun, adalah alamat tingkat link yang mengidentifikasi secara unik setiap node atau perangkat pada jaringan. Sesuai dengan standar jaringan IEEE 802.3, alamat ini terdiri dari 12-digit angka heksadesimal (misalnya, 001083123ABC). Alamat perangkat keras ditetapkan oleh vendor perangkat.
HP menetapkan alamat perangkat keras yang unik untuk setiap server cetak HP Jetdirect. Anda dapat menetapkan alamat perangkat keras dari server cetak HP Jetdirect Anda dengan mencetak halaman pengujian HP Jetdirect. Untuk server cetak eksternal HP Jetdirect, alamat ini juga tercantum pada label yang terpasang pada server cetak.

ARP
Protokol Resolusi Alamat. ARP adalah protokol TCP/IP yang digunakan untuk mengkonversi alamat IP menjadi alamat fisik (disebut alamat DLC), misalnya alamat Ethernet. Host yang ingin mendapatkan alamat fisik menyiarkan permintaan ARP ke jaringan TCP/IP. Host pada jaringan yang memiliki alamat IP yang diminta akan menjawab dengan alamat fisik perangkat keras.

BootP
Protokol bootstrap digunakan untuk men-download alamat IP dan parameter TCP/IP lainnya secara otomatis dari server jaringan. Bila printer (atau perangkat All-in-One) diaktifkan, server cetak HP Jetdirect akan menyiarkan permintaan boot. Jika server BootP telah diatur dengan benar, server cetak Jetdirect akan mengirim parameter tersebut kepada server cetak Jetdirect melalui jaringan. Lihat juga DHCP.

Cold reset (Atur ulang dingin)
Sebelum mengatur ulang dingin (cold reset), Anda dianjurkan untuk mencetak halaman pengujian untuk printer Anda dan server cetak Jetdirect.

Untuk mengatur ulang server cetak eksternal HP Jetdirect ke pengaturan standar pabrik, lepaskan kabel daya, lalu tekan tombol Test (Pengujian) pada server cetak ketika menyambungkan kembali kabel daya. Tunggu server cetak diinisialisasi ulang.

Untuk mengatur ulang server cetak internal HP Jetdirect, lakukan atur ulang dingin printer Anda (atau perangkat All-in-One), lalu tunggu server cetak diinisialisasi ulang. (Untuk hampir semua model printer HP LaserJet, tekan tombol Go [Jalankan] pada panel kontrol printer ketika Anda menghidupkan daya. Untuk printer lain, lihat buku petunjuk printer Anda.)

PERHATIAN: Mengatur ulang printer Anda mungkin menghapus semua pengaturan printer.

Mode komunikasi menentukan jenis jaringan dan protokol (TCP/IP) yang akan digunakan untuk mencetak ke printer ini.

DHCP
Protokol Konfigurasi Host Dinamis. Serupa dengan BootP yang men-download alamat IP secara otomatis dari server jaringan. Namun, server DHCP menyimpan serangkaian alamat IP yang ditetapkan secara dinamis ke perangkat yang meminta dan hanya memberikan alamat IP ke server cetak. Setiap kali printer (atau perangkat All-in-One) dimatikan dan dihidupkan kembali, tidak ada jaminan bahwa server cetak Jetdirect akan selalu ditetapkan ke alamat IP yang sama, meskipun server DHCP akan berusaha untuk melakukannya.

Driver Printer
Program yang mengontrol printer. Setiap kali Anda mencetak dokumen, driver printer mengambil alih, mengirim data ke printer dengan perintah kontrol yang benar. Hampir semua sistem operasi modern dilengkapi dengan driver printer untuk jenis printer yang paling umum, tetapi Anda harus menginstalnya sebelum dapat menggunakan printer tersebut. Anda juga dapat men-download driver yang diperbarui dari situs Web pabrikan printer. Lihat PCL atau PostScript.

Driver Printer PCL
Bagus untuk pencetakan secara umum. Terbaik untuk hampir semua aplikasi pengolah kata, spreadsheet, dan presentasi, seperti Word, Excel, dan PowerPoint.

Gateway
Gateway adalah router atau komputer yang menghubungkan dua jaringan yang tidak sama (misalnya dengan protokol komunikasi, format data, struktur, atau arsitektur yang berbeda). Bila jaringan terbagi dalam beberapa subnet, gateway sering digunakan untuk memisahkan satu subnet dari subnet yang lain untuk mengontrol lalu-lintas jaringan.
Pada jaringan TCP/IP, setiap perangkat jaringan sering dikonfigurasikan dengan alamat IP dari gateway yang terdekat. Jika tidak ada gateway pada jaringan, umumnya alamat IP komputer Anda akan digunakan.
Untuk informasi lebih lanjut lihat Menambahkan Printer pada Jaringan IP Besar.
Untuk jaringan nirkabel, lihat Titik Akses Nirkabel (WAP).

INF, file
File INF memiliki ekstensi inf dan berisi informasi tentang perangkat. File ini memberitahukan program penginstalan dari nama perangkat tersebut, dan file mana yang perlu disalin beserta lokasinya.

IPv4
IPv4 adalah Protokol Internet versi 4. Alamat IPv4 terdiri dari 32 bit yang ditetapkan dalam notasi titik desimal, misalnya 192.168.100.250. Jika alamat IPv4 yang berlaku tidak ditetapkan oleh server jaringan, maka server cetak Jetdirect akan menetapkannya sendiri dengan alamat link-lokal/IP-Otomatis (169.254.x.x) atau alamat default sebelumnya (192.0.0.192), tergantung pada jaringan.

IPv6
IPv6 adalah versi Protokol Internet yang disempurnakan dan pada akhirnya mungkin menggantikan IPv4, tetapi dapat juga digunakan bersama IPv4. Alamat IPv6 terdiri dari 128 bit dan nilainya ditetapkan dalam notasi heksadesimal yang dipisahkan dengan titik dua, misalnya 2001:0DB8:0000:0000:0000:0000:1428:57AB. Oleh karena itu, alamat IPv6 akan dapat mengikuti pertambahan perangkat yang terhubung ke jaringan dalam waktu lama. Tidak seperti IPv4, beberapa alamat IPv6 dapat ditetapkan ke server cetak. Bila alamat IPv6 link lokal default ditetapkan secara otomatis, alamat IPv6 berbasis router, berbasis server dan manual juga dapat ditetapkan.

Nama host IP
Nama host IP adalah nama sederhana (atau mudah dikenali)) yang dapat digunakan untuk menggantikan alamat IP printer. Nama ini dapat digunakan bila server nama pada jaringan telah diatur untuk memetakan nama host ke alamat IP yang terkait. Jika nama host telah ditetapkan dan dikonfigurasikan pada printer (misalnya, printer1.marketing.com), Anda dapat menetapkan nama host untuk mengidentifikasi printer.

Nama Jaringan
Nama jaringan adalah string karakter alfanumerik, tergantung huruf besar/kecil, yang memberikan kontrol akses dasar ke jaringan nirkabel. Nama jaringan juga disebut “service set identifier,” atau “SSID.”

Nama printer
Masukkan nama untuk printer dengan mengikuti pedoman penamaan printer. Nama ini akan digunakan untuk mengidentifikasi ikon printer ini dalam folder Printers sistem Anda. Untuk menampilkan folder Printers, klik Start pada Desktop Windows Anda, pilih Settings, lalu klik Printers.

NDS
Layanan direktori Novell. NDS adalah database relasional yang mengatur sumber daya jaringan berdasarkan logika dalam struktur hierarki bertingkat. Bila Anda memiliki hak yang sesuai, Anda dapat mengakses sumber ini, di manapun lokasinya. NDS dirancang untuk menggantikan Novell bindery (ikatan Novell).

-O-

Otentikasi
Otentikasi adalah strategi pengamanan jaringan nirkabel. Pada jaringan dengan otentikasi, perangkat menggunakan kunci bersama sebagai sandi dan hanya berkomunikasi dengan perangkat yang mengetahui kunci tersebut. Tidak seperti WEP, otentikasi tidak menyandikan data yang dikirim di antara perangkat nirkabel. Namun, otentikasi dapat digunakan bersama dengan WEP. Kunci otentikasi dan kunci WEP mungkin sama.

Password Jetdirect
Password Jetdirect ditetapkan oleh administrator jaringan untuk membatasi akses ke parameter konfigurasi pada server cetak HP Jetdirect. Sandi Jetdirect ditetapkan menggunakan aplikasi manajemen jaringan (seperti HP Web Jetadmin) atau perangkat lainnya.

Pedoman Penamaan, Printer
Ikuti petunjuk ini ketika memberi nama printer (atau perangkat All-in-One):

  • Panjangnya tidak lebih dari 31 karakter
  • Huruf A sampai Z, huruf besar
  • Huruf a sampai z, huruf kecil
  • Angka dari 0 hingga 9
  • Hindari karakter dengan nilai ASCII kurang dari 32 atau lebih dari 127
  • Jangan gunakan koma (,)

Pedoman Penamaan, Nama Bersama
Berdasarkan rekomendasi dari Microsoft, nama dibatasi hingga 7-bit ASCII (127 pertama) karakter. Spasi tidak diperbolehkan. Ikuti pedoman ini:

  • Huruf A sampai Z, huruf besar
  • Huruf a sampai z, huruf kecil
  • Angka dari 0 hingga 9
  • Karakter khusus tertentu: $ % ' - _ @ ~ ` ! ( ) { } . ^ # & + , ; = [ ]

PING
Packet Internet Groper. PING adalah utilitas untuk menentukan apakah alamat IP tertentu dapat diakses. Utilitas ini bekerja dengan mengirim paket ke alamat tertentu dan menunggu jawaban. PING terutama digunakan untuk mengatasi masalah koneksi Internet. Ada banyak utilitas PING gratis dan shareware yang tersedia untuk komputer PC.

Port Name (Nama Port)
Dialog Port Name (Nama Port) hanya ditampilkan pada Windows NT dan Windows 2000. Dialog ini hanya ditampilkan jika "Use default port name (Gunakan nama port default)" tidak ditandai pada dialog Set Network Communication Mode (Atur Mode Komunikasi Jaringan), dan mode komunikasi IP dipilih.

Anda dapat menetapkan nama port jaringan yang akan dibuat untuk printer yang sedang diinstal. Nama port ini akan digunakan untuk mengenali port dalam properti printer.

Nama port yang valid maksimal 63 karakter dan dapat terdiri dari karakter apapun. Nama port ini harus unik pada PC Anda. Nama port ditampilkan pada layar ringkasan dan pada halaman pengujian.

PostScript Printer Driver
Bagus untuk pencetakan khusus. Terbaik untuk penerbitan desktop dan aplikasi grafis penuh, misalnya Photoshop, PageMaker, dan FrameMaker.

Printer Sharing (Berbagi Printer)
Dengan berbagi printer, pengguna jaringan lainnya dapat mencetak ke printer melalui komputer ini. Utilitas penginstalan ini hanya mendukung berbagi printer pada Windows 2000 dan Windows NT 4.0. Layanan server harus diaktifkan untuk berbagi printer pada sistem. Anda harus menetapkan salah satu pilihan berikut:

  • Not Shared (Tidak Dibagi): Tentukan pilihan ini untuk mengatur komputer bagi pencetakan peer-to-peer jika Anda tidak ingin pengguna jaringan lain mengakses printer melalui komputer ini.
  • Share as (Bagi sbg):: Tentukan pilihan ini untuk mengatur komputer sebagai server bagi pencetakan klien-server. Klien jaringan akan dapat mengirim pekerjaan cetak ke printer melalui komputer ini.

Profil nirkabel
Profil nirkabel adalah kumpulan pengaturan jaringan nirkabel yang berlaku untuk jaringan nirkabel tertentu. Sebagai contoh, kartu LAN nirkabel dapat dilengkapi satu profil untuk jaringan rumah dan profil lain untuk jaringan kantor. Ketika menginstal perangkat pada jaringan, pastikan untuk memilih profil yang sesuai.

Server Cetak HP Jetdirect
Server cetak HP Jetdirect adalah aksesori jaringan Hewlett-Packard yang menghubungkan printer langsung ke jaringan. Koneksi jaringan langsung juga memungkinkan fleksibilitas lokasi dan performa printer (atau perangkat All-in-One) yang maksimal.
Kartu server printer internal HP Jetdirect digunakan dengan printer HP yang dilengkapi slot masukan/keluaran. Beberapa server cetak eksternal HP Jetdirect dapat digunakan dengan hampir semua printer yang dilengkapi port paralel, sementara beberapa server cetak HP Jetdirect lain dapat digunakan dengan hampir semua printer dengan port USB. Kedua jenis server cetak tersebut mendukung semua topologi dan protokol jaringan yang populer.

Server Nama
Server nama adalah komputer pada jaringan yang digunakan untuk memetakan (atau menyelesaikan) nama host (nama komputer atau server cetak) dengan alamat IP. Server Domain Name System (DNS) dan server Windows Internet Name Service (WINS) adalah contoh server nama.

SNMP Community Name (Nama Komunitas SNMP)
Nama Komunitas SNMP adalah sandi yang harus diberikan bila SNMP digunakan untuk mengubah konfigurasi printer.

Subnet (Subjaringan)
Bagian dari jaringan yang menggunakan komponen alamat yang sama. Pada jaringan TCP/IP, subnet ditetapkan sebagai semua jaringan dengan awalan alamat IP yang sama. Sebagai contoh, semua perangkat dengan alamat IP yang dimulai dengan 100.100.100. akan menjadi bagian dari subnet yang sama. Membagi jaringan ke dalam beberapa subnet berguna untuk meningkatkan keamanan dan performa. Jaringan IP dibagi menggunakan subnet mask.

Subnet Mask
Subnet mask adalah mekanisme yang digunakan untuk membagi jaringan TCP/IP menjadi beberapa subnet (jika subnetting digunakan). Subnetting memungkinkan jaringan besar untuk digunakan secara lebih efisien. Format subnet mask serupa dengan alamat IP: n.n.n.n dengan n adalah angka dari 0 hingga 255. Jika subnet mask belum ditetapkan, umumnya digunakan subnet mask yang ditetapkan untuk komputer Anda.
Bila diterapkan untuk alamat IP, subnet mask menetapkan bagian dari alamat IP yang digunakan untuk mengenali jaringan/subnet, dan bagian yang digunakan untuk mengenali masing-masing node host.

WEP
Standar keamanan asli yang digunakan dalam jaringan Wi-Fi untuk menyandikan lalu lintas jaringan nirkabel.  WEP memberikan pengamanan dengan menyandikan data yang dikirim melalui gelombang radio dari satu perangkat nirkabel ke perangkat nirkabel lain yang membuat data tidak terbaca oleh pihak yang tidak berhak. Hanya perangkat yang berbagi pengaturan WEP yang sama dengan printer yang akan dapat berkomunikasi dengan printer.

WEP key (kunci WEP)
Kunci WEP, atau “kunci enkripsi”, terdiri dari serangkaian karakter alfanumerik atau digit heksadesimal. Setelah membuat kunci WEP, Anda harus mengingatnya atau mencatatnya di lokasi yang aman. Jika hilang, Anda mungkin tidak dapat mengembalikan kunci WEP. Kunci WEP panjangnya 64 atau 128 bit. 24 bit pertama dari kunci tersebut ditetapkan secara otomatis. Ketika membuat kunci WEP, pembuat kunci menambahkan bit yang tersisa (40 bit pada kunci 64-bit, atau 104 bit untuk kunci 128-bit).
