Nama Port

Catatan: Dialog Nama Port hanya muncul di Windows NT, Windows 2000, dan Windows XP. Dialog ini hanya muncul jika Gunakan nama port default tidak dicentang pada dialog Tetapkan Mode Komunikasi Jaringan dan jika mode komunikasi IP dipilih.

Anda dapat menetapkan pembuatan nama port jaringan untuk printer yang sedang diinstal. Nama port digunakan untuk mengidentifikasi port dalam properti printer.

Panjang nama port yang valid maksimal 63 karakter dan mencakup karakter apapun. Nama port harus unik. Nama port ditampilkan pada layar ringkasan dan halaman pengujian.
