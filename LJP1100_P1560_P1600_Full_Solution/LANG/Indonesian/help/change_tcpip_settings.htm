Ubah Pengaturan TCP/IP

Anda dapat menetapkan pengaturan TCP/IP baru (IPv4) untuk printer ini. Anda harus menghubungi Administrator Jaringan sebelum mengubah pengaturan.

Catatan: Utilitas penginstalan akan mendeteksi pengaturan sistem dan memberikan pengaturan TCP/IP yang disarankan (IPv4 dan IPv6), namun Anda dapat memodifikasi pengaturan IPv4 jika dikehendaki (utilitas ini hanya dapat digunakan untuk mengkonfigurasi pengaturan IPv4 secara manual). Nilai parameter yang Anda tetapkan tidak dikonfigurasi pada printer hingga penginstalan selesai.

•         Jika printer tidak memiliki pengaturan TCP/IP yang valid:
Masukkan alamat IP (IPv4) untuk printer, subnet mask, dan alamat IP yang valid untuk gateway ke bidang yang tersedia. Hubungi Administrator Jaringan sebelum menetapkan nilai.

Klik Sarankan Pengaturan untuk memasukkan parameter TCP/IP secara otomatis melalui utilitas penginstalan.

Gunakan kotak centang nama port default (Windows NT, 2000, dan XP). Centang kotak ini untuk menggunakan pengaturan port default printer. Jika kotak ini tidak dicentang, maka dialog Nama Port adalah layar wizard berikutnya.

PERHATIAN: Jika printer akan dihubungkan ke jaringan lain, pengaturan TCP/IP yang disarankan oleh utilitas mungkin tidak unik secara global dan dapat bertentangan dengan perangkat lain. Hati-hati bila terhubung ke jaringan TCP/IP lainnya.

•         Jika printer telah memiliki pengaturan TCP/IP yang valid:
Pengaturan TCP/IP saat ini akan ditampilkan.

PERHATIAN: Mengubah pengaturan TCP/IP yang valid dapat mencegah akses oleh pengguna lainnya. Hubungi Administrator Jaringan sebelum mengubah pengaturan yang valid.

Catatan: Jika parameter TCP/IP merupakan sandi yang dilindungi oleh perangkat lunak manajemen, seperti HP Web Jetadmin, bidang entri akan berwarna abu-abu.

Gunakan kotak centang nama port default (hanya Windows NT, 2000, dan XP). Centang kotak ini untuk menggunakan pengaturan port default printer. Jika kotak ini tidak dicentang, maka dialog Nama Port adalah layar wizard berikutnya.
