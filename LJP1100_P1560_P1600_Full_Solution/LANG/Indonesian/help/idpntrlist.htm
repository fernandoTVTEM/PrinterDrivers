Printer Ditemukan

Layar Beberapa Printer Ditemukan memungkinkan Anda memilih printer yang ditemukan di jaringan lokal maupun memperoleh pilihan konfigurasi tambahan untuk menginstal jaringan, USB, atau printer yang tersambung secara paralel.

  • Instal printer jaringan yang ditemukan. Pilih printer yang dikehendaki dari daftar semua printer (atau perangkat multifungsi) yang ditemukan di jaringan lokal, kemudian klik Lanjut. Klik tombol Cari Kembali untuk mencari printer di jaringan.
  • Bantu saya mencari printer. Tetapkan pilihan ini jika printer jaringan yang dikehendaki tidak tercantum atau Anda ingin menginstal printer yang akan disambungkan langsung ke komputer menggunakan kabel USB maupun paralel.  Untuk printer jaringan, pilihan ini akan memberikan pilihan pencarian tambahan, seperti mencari printer melalui alamat IP. 

Catatan: Untuk mengkonfigurasi printer yang tidak terdapat di jaringan lokal, lihat Menambahkan Printer pada Jaringan IP Besar.

Printer dalam daftar diidentifikasi oleh bidang sebagai berikut. Alamat Perangkat Keras mungkin adalah cara termudah untuk mengidentifikasi printer:

Model Mengidentifikasi model printer (misalnya, HP LaserJet 5000 Series). Tidak Diketahui akan muncul di layar jika model tidak teridentifikasi, printer dimatikan, atau jika tidak ada printer yang tersambung. Untuk server cetak eksternal HP Jetdirect, berikut adalah item yang akan muncul di layar jika printer tersambung, namun tidak terdeteksi:
  • Paralel: Centronics Printer/Plotter
  • USB (satu arah): Periferal Satu Arah
  • USB (dua arah): Periferal Dua Arah
Alamat Perangkat Keras (disebut juga Alamat Stasiun atau alamat MAC). Mengidentifikasi perangkat secara unik di jaringan lokal. Untuk printer HP dan server cetak eksternal HP Jetdirect yang memiliki beberapa port printer, nomor port (Port 1, Port 2, Port 3) dapat ditambahkan. Alamat ini tercantum pada halaman (pengujian) konfigurasi server cetak.
Alamat IP (Internet Protocol) Mengidentifikasi perangkat secara unik di jaringan TCP/IP dan ditetapkan oleh administrator jaringan. Alamat IPv4 dan IPv6 akan ditampilkan dalam daftar.  Jika printer memiliki beberapa alamat, maka alamat tersebut akan ditampilkan secara berurutan, sehingga Anda dapat memilih antarmuka yang akan digunakan. 

Catatan: Bila server cetak HP Jetdirect dihidupkan dan tidak dikonfigurasi dengan alamat IP yang valid dalam waktu dua menit, alamat IP Otomatis (169.254.x.x) atau alamat IP default (192.0.0.192) akan secara otomatis ditetapkan. HP menyarankan agar alamat IP default diubah. Jika terdapat beberapa printer dengan alamat default yang sama, periksa Alamat Perangkat Keras untuk mengidentifikasi printer yang benar.
Nama Host IP Nama host IP adalah nama yang praktis (atau mudah dikenali) dan dapat digunakan sebagai pengganti alamat IP printer. Nama tersebut dapat digunakan bila server nama di jaringan telah dikonfigurasi untuk memetakan nama host ke alamat IP yang sesuai. Jika nama host telah ditetapkan dan dikonfigurasikan pada printer (misalnya, printer1.pemasaran.com), Anda dapat menetapkan nama host untuk mengidentifikasi printer.

Catatan: Jika entri bidang terpotong dan Anda ingin melihat seluruh bidang, ubah ukuran lebar bidang. Untuk mengubah ukuran, tempatkan kursor di sekitar kepala bidang, kemudian klik dan tarik.

Untuk petunjuk tentang cara mencetak halaman konfigurasi jaringan, lihat dokumen yang diberikan bersama printer.

Jika printer yang dipilih telah dilindungi oleh Nama Komunitas SNMP maupun sandi Jetdirect, Anda akan diminta untuk memasukkan Nama Komunitas atau sandi.
