Firewall Terdeteksi

Agar dapat berkomunikasi melalui Windows Firewall, penginstal akan secara otomatis menambahkan daftar pengecualian aplikasi firewall untuk setiap spesifikasi Microsoft. Hal ini memungkinkan penginstal berkomunikasi dengan printer pada jaringan untuk penginstalan.

Klik Lanjut untuk mencari printer pada jaringan dan melanjutkan penginstalan. Atau klik Batal untuk keluar dari penginstal.

Jika firewall diaktifkan (misalnya ZoneAlarm©, BlackICE©, Sygate Security Agent©, dan Norton Personal Firewall©, dll.), lihat dokumentasi vendor firewall Anda untuk memperoleh petunjuk tentang cara menonaktifkan firewall tersebut.

Setiap kali firewall perangkat lunak dinonaktifkan, HP sangat menganjurkan untuk menginstal berikut ini pada komputer Anda:

  • Perangkat keras yang memberikan perlindungan firewall di antara Internet dan komputer Anda, misalnya gateway.
  • Program deteksi virus terkini.

Untuk informasi umum tentang mengatasi masalah komunikasi, lihat dokumentasi yang disertakan dengan printer Anda dan kunjungi situs Web dukungan di http://www.hp.com/support.

Firewall sering menyebabkan masalah komunikasi. Jika firewall diaktifkan pada komputer jaringan, masalah berikut ini dapat terjadi:

    Selama menginstal perangkat lunak:

  • Penginstal printer HP tidak akan dapat menemukan printer HP pada jaringan.  Pesan kesalahan akan memberitahukan bahwa konfigurasi tidak dapat mendeteksi printer apapun pada jaringan.  Untuk mencegah masalah ini, nonaktifkan sementara firewall Anda selama menginstal perangkat lunak printer HP. Lihat dokumentasi vendor firewall Anda untuk memperoleh petunjuk tentang cara menonaktifkan firewall tersebut. 

    Setelah menginstal perangkat lunak:

  • Jika alamat IP printer HP berubah, maka komputer Anda tidak akan dapat berkomunikasi dengan printer HP.  Semua tugas cetak, pindai, faks PC, dan transfer gambar akan gagal dan status produk akan menunjukkan bahwa printer HP terputus.  Di lingkungan jaringan rumah, adalah normal bila alamat IP kadang-kadang berubah.  Jika firewall perangkat lunak tidak diaktifkan, alamat baru akan dikirim ke komputer, yang memungkinkan komputer dan printer HP untuk terus berkomunikasi.  Untuk mencegah masalah ini, nonaktifkan firewall Anda atau jika Anda pengguna mahir yang menguasai jaringan maka Anda dapat memilih untuk menggunakan alamat IP statis.  Untuk saran mengenai cara mengkonfigurasi printer Anda dengan alamat IP statis, kunjungi situs Web HP di http://www.hp.com/support.
  • Jika Anda memiliki HP all-in-one, Anda mungkin tidak akan dapat menggunakan slot kartu memori pada HP all-in-one untuk mentransfer file ke komputer Anda.  Hal ini disebabkan karena file pada kartu foto tidak akan dapat diakses dari drive jaringan yang dipetakan atau perangkat lunak transfer gambar.

Untuk informasi lebih lanjut tentang firewall Anda dapat mengunjungi situs Web HP.

Cara menonaktifkan Firewall Windows pada Windows Vista:

  1. Klik Start > Network.
  2. Klik kanan Network lalu pilih Properties.
  3. Klik Windows Firewall di sudut kiri bawah.
  4. Pilih 'Turn Windows Firewall on/off.'
  5. Pilih 'Continue' pada dialog User Access Control.
  6. Tentukan pilihan 'Off' untuk menonaktifkan firewall, lalu klik 'OK.'  Firewall Windows sekarang dinonaktifkan.

Cara Menonaktifkan Firewall Koneksi Internet (ICF) pada Windows XP:

  1. Pada baris tugas di bagian bawah layar Anda, klik Start, lalu klik Control Panel.
  2. Klik kategori Network and Internet Connections. (Jika Network and Internet Connections tidak terlihat, klik Switch to Category View pada Control Panel di sebelah kiri jendela Control Panel.)
  3. Klik Network Connections.
  4. Klik kanan koneksi Dial-up, LAN, atau High-Speed Internet yang digunakan untuk terhubung ke Internet, lalu klik Properties dari menu konteks.
  5. Pada tab Advanced, di bawah Internet Connection Firewall, hapus pilihan atau nonaktifkan Protect my computer or network, lalu klik OK. Firewall Windows XP sekarang dinonaktifkan.
  6. Klik Next pada wizard konfigurasi untuk melanjutkan penginstalan printer.
  7. Untuk mengaktifkan Internet Connection Firewall setelah printer diinstal, ikuti langkah-langkah di atas untuk membuka pilihan Protect my computer or network. Kemudian tentukan kembali pilihan Protect my computer or network.

Catatan: Pastikan untuk mengaktifkan kembali Firewall Windows setelah penginstalan selesai.
