Προσθήκη εκτυπωτών σε μεγάλα δίκτυα IP

Εάν η προσθήκη εκτυπωτών (ή πολυλειτουργικών συσκευών) γίνεται μέσω δρομολογητή ή πύλης (που δεν βρίσκεται στο τοπικό δίκτυό σας), η αναγνώριση των εκτυπωτών μέσω αυτού του βοηθητικού προγράμματος μπορεί να αποδειχθεί δύσκολη:

  • Οι δρομολογητές ίσως να μην μεταδίδουν τη διεύθυνση υλικού του εκτυπωτή, τη διεύθυνση MAC ή τη διεύθυνση σταθμού.
  • Εάν έχετε πολλούς καινούργιους εκτυπωτές στο τοπικό δίκτυο, η χρήση της προεπιλεγμένης διεύθυνσης IP (192.0.0.192) για την αναγνώριση ενός εκτυπωτή δεν αποτελεί εγγύηση για το ότι θα επιλεγεί ο σωστός εκτυπωτής. Ελέγξτε τη Διεύθυνση υλικού για να κάνετε αναγνώριση του σωστού εκτυπωτή.

Εάν θέλετε να χρησιμοποιήσετε το συγκεκριμένο βοηθητικό πρόγραμμα για να ρυθμίσετε έναν εκτυπωτή, εφαρμόστε τις ακόλουθες οδηγίες:

  1. Κάντε ένα από τα παρακάτω:
    • Μεταφέρετε φυσικά τον εκτυπωτή στο τοπικό δίκτυο του υπολογιστή σας, ή
    • Χρησιμοποιήστε αυτό το βοηθητικό πρόγραμμα σε έναν υπολογιστή που είναι συνδεδεμένος με το τοπικό δίκτυο του εκτυπωτή σας.
  2. Κάντε αναγνώριση του εκτυπωτή μέσω αυτού του βοηθητικού προγράμματος και διαμορφώστε τις παραμέτρους IP έτσι ώστε να συμφωνούν με την τελική θέση του εκτυπωτή στο δίκτυο. Εάν θέλετε να μάθετε τη διεύθυνση MAC, μπορείτε να εκτυπώσετε τη σελίδα διαμόρφωσης (δοκιμαστική σελίδα) (είτε μέσω του εκτυπωτή είτε μέσω του εξωτερικού διακομιστή εκτύπωσης) για να βεβαιωθείτε ότι η διεύθυνση MAC ή η διεύθυνση υλικού που θα επιλέξετε είναι η σωστή.
  3. Εάν είναι απαραίτητο, επαναφέρετε τον εκτυπωτή (ή τον εξωτερικό διακομιστή εκτύπωσης Jetdirect της HP) στην τελική θέση του στο δίκτυο.
  4. Εάν θέλετε να εγκαταστήσετε το συγκεκριμένο εκτυπωτή και σε άλλους υπολογιστές, κάντε ένα από τα παρακάτω:
    • Όσον αφορά άλλους υπολογιστές ή διακομιστές που θα εκτυπώνουν απευθείας στο συγκεκριμένο εκτυπωτή, χρησιμοποιήστε αυτό το βοηθητικό πρόγραμμα για να διαμορφώσετε κάθε υπολογιστή ή διακομιστή, ή
    • Όσον αφορά τα προγράμματα-πελάτες των Windows που θα εκτυπώνουν σε αυτόν τον εκτυπωτή μέσω κοινόχρηστου συστήματος, χρησιμοποιήστε το βοηθητικό πρόγραμμα "Προσθήκη εκτυπωτή" του συστήματος-πελάτη που βρίσκεται στο φάκελο "Εκτυπωτές" ή χρησιμοποιήστε το πρόγραμμα εγκατάστασης κοινόχρηστου εκτυπωτή που έχει δημιουργηθεί για το συγκεκριμένο εκτυπωτή.
