Εκτυπωτές που βρέθηκαν

Η οθόνη Εκτυπωτές που βρέθηκαν σας επιτρέπει να επιλέξετε έναν εκτυπωτή που εντοπίστηκε στο τοπικό δίκτυο ή να αποκτήσετε πρόσθετες επιλογές εγκατάστασης για την εγκατάσταση εκτυπωτών μέσω δικτύου, USB ή παράλληλης σύνδεσης.

  • Εγκαταστήστε έναν εκτυπωτή δικτύου που εντοπίστηκε. Επιλέξτε τον εκτυπωτή που θέλετε από τη λίστα όλων των εκτυπωτών (ή των συσκευών πολλαπλών λειτουργιών) που εντοπίστηκαν στο τοπικό δίκτυο και κάντε κλικ στο κουμπί Επόμενο. Κάντε κλικ στο κουμπί Επανάληψη αναζήτησης για να εκτελέσετε νεά αναζήτηση για εκτυπωτές στο δίκτυο.
  • Βοηθήστε με να βρω τον εκτυπωτή μου. Ενεργοποιήστε αυτήν την επιλογή, αν ο υπολογιστής δικτύου που θέλετε δεν βρίσκεται στη λίστα ή αν θέλετε να εγκαταστήσετε έναν εκτυπωτή που θα συνδέεται αυτόματα στον υπολογιστή σας μέσω καλωδίου USB ή παράλληλου καλωδίου.  Για τους εκτυπωτές δικτύου, αυτή η επιλογή προσφέρει πρόσθετες επιλογές αναζήτησης, όπως η αναζήτηση εκτυπωτή με βάση τη διεύθυνση ΙΡ. 

Σημείωση: Για να εγκαταστήσετε εκτυπωτές που δεν βρίσκονται στο τοπικό δίκτυο, ανατρέξτε στην ενότητα Προσθήκη εκτυπωτών σε μεγάλα δίκτυα ΙΡ.

Οι εκτυπωτές που βρίσκονται στη λίστα προσδιορίζονται με βάση τα παρακάτω πεδία. Ενδεχομένως, η Διεύθυνση υλικού είναι ο ευκολότερος τρόπος προσδιορισμού ενός εκτυπωτή:

Μοντέλο Προσδιορίζει το μοντέλο του εκτυπωτή (για παράδειγμα, HP LaserJet 5000 Series). Η ένδειξη ’γνωστο εμφανίζεται αν δεν είναι δυνατός ο προσδιορισμός του μοντέλου, αν ο εκτυπωτής είναι απενεργοποιημένος ή αν δεν έχει συνδεθεί κανένας εκτυπωτής. Όσον αφορά τους εξωτερικούς διακομιστές εκτυπώσεων HP Jetdirect, τα παρακάτω εμφανίζονται όταν ο υπολογιστής έχει συνδεθεί, αλλά δεν είναι δυνατός ο εντοπισμός του:
  • Παράλληλα: Εκτυπωτής/Σχεδιογράφος Centronics
  • USB (μίας κατεύθυνσης): Περιφερειακή συσκευή μίας κατεύθυνσης
  • USB (δύο κατευθύνσεων): Περιφερειακή συσκευή δύο κατευθύνσεων
Διεύθυνση υλικού (γνωστή και ως Διεύθυνση σταθμού ή Διεύθυνση MAC). Εντοπίζει μοναδικά μια συσκευή στο τοπικό δίκτυο. Όσον αφορά τους εκτυπωτές HP και τους εξωτερικούς διακομιστές εκτυπώσεων HP Jetdirect, μπορεί να προστεθεί ένας αριθμός θύρας (Θύρα 1, Θύρα 2, Θύρα 3). Αυτή η διεύθυνση καταχωρίζεται στη (δοκιμαστική) σελίδα διαμόρφωσης του διακομιστή εκτυπώσεων.
Διεύθυνση IP (Πρωτόκολλο Internet) Προσδιορίζει με μοναδικό τρόπο μια συσκευή σε ένα δίκτυο TCP/IP και εκχωρείται από έναν διαχειριστή δικτύου. Οι διευθύνσεις IPv4 και IPv6 θα εμφανιστούν σε αυτήν τη λίστα.  Αν σε έναν εκτυπωτή αντιστοιχούν πολλές διευθύνσεις, θα εμφανίζεται μία φορά για κάθε διεύθυνση, επιτρέποντάς σας να επιλέξετε ποιο περιβάλλον εργασίας θέλετε να χρησιμοποιήσετε. 

Σημείωση: Όταν ένας διακομιστής εκτυπώσεων HP Jetdirect είναι ενεργοποιημένος και δεν έχει διαμορφωθεί με έγκυρη διεύθυνση ΙΡ σε διάστημα δύο λεπτών, εκχωρείται αυτόματα είτε μια αυτόματη διεύθυνση ΙΡ  (169.254.x.x) είτε μια προεπιλεγμένη διεύθυνση ΙΡ (192.0.0.192). H ΗP συνιστά την αλλαγής της προεπιλεγμένης διεύθυνσης IP . Αν υπάρχουν πολλοί εκτυπωτές με την ίδια προεπιλεγμένη διεύθυνση, ελέγξτε τη Διεύθυνση υλικού για να προσδιορίσετε τον κατάλληλο εκτυπωτή.
Όνομα κεντρικού υπολογιστή IP Το όνομα κεντρικού υπολογιστή IP είναι ένα εύκολο (ή εύκολα αναγνωρίσιμο) όνομα που μπορεί να χρησιμοποιηθεί στη θέση της διεύθυνσης ΙΡ του εκτυπωτή. Μπορεί να χρησιμοποιηθεί όταν ένας διακομιστής ονομάτων στο δίκτυο έχει ρυθμιστεί έτσι, ώστε να αντιστοιχίζει ονόματα κεντρικών υπολογιστών στις αντίστοιχες διευθύνσεις ΙΡ. Αν ένα όνομα κεντρικού υπολογιστή έχει εκχωρηθεί και διαμορφωθεί στον εκτυπωτή (για παράδειγμα, printer1.marketing.com), μπορείτε να ορίσετε το όνομα κεντρικού υπολογιστή, ώστε να προσδιορίζει τον εκτυπωτή.

Σημείωση: Αν μια καταχώρηση πεδίου έχει περικοπεί και θέλετε να προβληθεί ολόκληρο το πεδίο, αλλάξτε το πλάτος του πεδίου. Για να αλλάξετε το μέγεθος, τοποθετήστε το δείκτη του ποντικιού στο όριο της κεφαλίδας του πεδίου, και στη συνέχεια κάντε κλικ για μεταφορά.

Για οδηγίες σχετικά με την εκτύπωση μιας σελίδας διαμόρφωσης δικτύου, ανατρέξτε στην τεκμηρίωση που συνόδευε τον εκτυπωτή.

Αν έχετε επιλέξει έναν εκτυπωτή που προστατεύεται με όνομα κοινότητας SNMP ή με κωδικό πρόσβασης Jetdirect, θα σας ζητηθεί το όνομα κοινότητας ή ο κωδικός πρόσβασης.
