Αλλαγή των ρυθμίσεων TCP/IP

Μπορείτε να ορίσετε νέες ρυθμίσεις TCP/IP (IPv4) για αυτόν τον εκτυπωτή. Θα πρέπει να συμβουλευθείτε το διαχειριστή του δικτύου πριν να αλλάξετε οποιαδήποτε ρύθμιση.

Σημείωση: Το βοηθητικό πρόγραμμα εγκατάστασης θα ανιχνεύσει τις ρυθμίσεις του συστήματός σας και θα ορίσει τις συνιστώμενες ρυθμίσεις TCP/IP (IPv4 και IPv6), αλλά μπορείτε, εάν θέλετε, να τροποποιήσετε τις ρυθμίσεις IPv4 (αυτό το βοηθητικό πρόγραμμα μπορεί να χρησιμοποιηθεί μόνο για μη αυτόματη διαμόρφωση των ρυθμίσεων IPv4). Οι τιμές παραμέτρων που εκχωρείτε δεν διαμορφώνονται στον εκτυπωτή μέχρι να ολοκληρωθεί η εγκατάσταση.

·         Εάν δεν έχουν οριστεί έγκυρες ρυθμίσεις TCP/IP για τον εκτυπωτή:
Εισαγάγετε στα προσφερόμενα πεδία μια έγκυρη διεύθυνση IP (IPv4) για τον εκτυπωτή, μια έγκυρη μάσκα subnet και μια έγκυρη διεύθυνση IP για την πύλη. Πριν να εκχωρήσετε τιμές, θα πρέπει να συμβουλευθείτε το διαχειριστή του δικτύου.

Κάντε κλικ στην επιλογή Πρόταση ρυθμίσεων για να γίνει αυτομάτως η εισαγωγή των παραμέτρων TCP/IP από το βοηθητικό πρόγραμμα εγκατάστασης.

Χρησιμοποιήστε το πλαίσιο ελέγχου προεπιλεγμένου ονόματος θύρας (Windows NT, 2000 και XP). Επιλέξτε το πλαίσιο αυτό, εάν θέλετε να χρησιμοποιηθούν οι προεπιλεγμένες ρυθμίσεις θύρας για τον εκτυπωτή σας. Εάν δεν επιλεγεί το συγκεκριμένο πλαίσιο, το παράθυρο διαλόγου Όνομα θύρας θα είναι η επόμενη οθόνη του οδηγού.

ΠΡΟΣΟΧΗ: Εάν συνδέσετε τον εκτυπωτή σας και με άλλα δίκτυα, οι ρυθμίσεις TCP/IP που προτάθηκαν από το βοηθητικό πρόγραμμα μπορεί να μην είναι καθολικά μοναδικές με αποτέλεσμα να προκληθούν διενέξεις με άλλες συσκευές. Πρέπει να είστε προσεκτικοί, εάν συνδέσετε τον εκτυπωτή και με άλλα δίκτυα TCP/IP.

·         Εάν οι ρυθμίσεις TCP/IP του εκτυπωτή είναι έγκυρες:
Θα εμφανιστούν οι τρέχουσες ρυθμίσεις TCP/IP.

ΠΡΟΣΟΧΗ: Η αλλαγή των έγκυρων ρυθμίσεων TCP/IP μπορεί να αποτρέψει την πρόσβαση άλλων χρηστών σε αυτές. Πριν να αλλάξετε τις έγκυρες ρυθμίσεις, συμβουλευθείτε το διαχειριστή του δικτύου.

Σημείωση: Εάν οι παράμετροι TCP/IP προστατεύονται με κωδικό πρόσβασης από κάποιο λογισμικό διαχείρισης, όπως το HP Web Jetadmin, τα πεδία καταχώρησης θα είναι απενεργοποιημένα με γκρίζο χρώμα.

Χρησιμοποιήστε το πλαίσιο ελέγχου προεπιλεγμένου ονόματος θύρας (μόνο για Windows NT, 2000 και XP). Επιλέξτε το πλαίσιο αυτό, εάν θέλετε να χρησιμοποιηθούν οι προεπιλεγμένες ρυθμίσεις θύρας για τον εκτυπωτή σας. Εάν δεν επιλεγεί το συγκεκριμένο πλαίσιο, το παράθυρο διαλόγου Όνομα θύρας θα είναι η επόμενη οθόνη του οδηγού.
