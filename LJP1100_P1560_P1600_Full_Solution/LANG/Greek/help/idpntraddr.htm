Ρυθμίσεις εκτυπωτή

Η οθόνη Ρυθμίσεις εκτυπωτή σας δίνει τη δυνατότητα να καθορίσετε ρυθμίσεις για έναν εκτυπωτή που δεν είναι στο δίκτυο.

Μπορείτε να εισάγετε μία ή περισσότερες από τις ακόλουθες διευθύνσεις για τον εκτυπωτή, και στη συνέχεια να κάνετε κλικ στο κουμπί Επόμενο.  Εισάγοντας παραπάνω από μία διευθύνσεις, διασφαλίζετε ότι η θύρα που δημιουργείται θα συνδεθεί με τον συγκεκριμένο εκτυπωτή.

  • Όνομα κεντρικού υπολογιστή IP. Ένα όνομα κεντρικού υπολογιστή ΙΡ είναι ένα φιλικό (ή εύκολα αναγνωρίσιμο) όνομα που μπορεί να χρησιμοποιηθεί στη θέση της διεύθυνσης ΙΡ του εκτυπωτή.
  • Διεύθυνση υλικού. Μια διεύθυνση υλικού LAN (ονομάζεται επίσης και διεύθυνση σταθμού ή διεύθυνση MAC) αναγνωρίζει με μοναδικό τρόπο μια συσκευή.
  • Διεύθυνση ΙP (IPv4). Μια διεύθυνση IP (Πρωτόκολλο Internet) αναγνωρίζει με μοναδικό τρόπο κάθε συσκευή σε ένα δίκτυο TCP/IP. Συνήθως εκχωρείται από τον διαχειριστή δικτύου. Το βοηθητικό αυτό πρόγραμμα μπορεί να χρησιμοποιηθεί μόνο για την διαμόρφωση των ρυθμίσεων IPv4 στη συσκευή.

Σημείωση: Για να εγκαταστήσετε εκτυπωτές που δεν βρίσκονται στο τοπικό δίκτυο, δείτε Προσθήκη εκτυπωτών σε μεγάλα δίκτυα IP.
