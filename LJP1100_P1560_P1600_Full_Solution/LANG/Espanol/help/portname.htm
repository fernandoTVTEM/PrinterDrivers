Nombre de puerto

Nota: El cuadro de diálogo Nombre de puerto sólo aparece en Windows NT, Windows 2000 y Windows XP. Este cuadro de diálogo sólo aparece si no se ha marcado la opción Utilizar nombre de puerto predeterminado en el cuadro de diálogo Establecer modo de comunicación de la red y se ha seleccionado el modo de comunicación IP.

Puede especificar el nombre del puerto de red que se va a crear para la impresora que se está instalando. El nombre del puerto se utiliza para identificar el puerto en las propiedades de la impresora.

Los nombres de puerto válidos tienen una longitud máxima de 63 caracteres y pueden utilizar todo tipo de caracteres. El nombre del puerto debe ser exclusivo. El nombre del puerto aparece en la pantalla de resumen y en la página de prueba.
