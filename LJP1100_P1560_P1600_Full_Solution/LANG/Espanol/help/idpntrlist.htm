Impresoras encontradas

La pantalla Impresoras encontradas permite seleccionar una de las impresoras encontradas en la red local, así como obtener opciones de configuración adicionales para instalar impresoras conectadas a través de la red, USB o con conexión paralela.

  • Instalación de una impresora descubierta en la red. Seleccione la impresora deseada de la lista de impresoras (o dispositivos multifunción) detectadas en la red local y haga clic en Siguiente. Haga clic en el botón Buscar de nuevo para volver a buscar las impresoras de la red.
  • Ayuda para encontrar mi impresora. Seleccione esta opción si la impresora de red que desea no aparece o si desea instalar una impresora que se conectará directamente a su equipo mediante cable USB o paralelo.  En el caso de las impresoras de red, esta opción ofrecerá otras opciones de búsqueda, por ejemplo, la búsqueda de impresoras a través de la Dirección IP. 

Nota: Para configurar impresoras que no se encuentran en la red local, consulte Añadir impresoras en redes IP grandes.

Las impresoras de la lista se identifican por los siguientes campos. El modo más sencillo de identificar la impresora es mediante la dirección de hardware.

Modelo Permite identificar el modelo de la impresora (por ejemplo, HP LaserJet serie 5000). Si no es posible identificar el modelo, si se apaga la impresora o si no hay ninguna impresora conectada, se mostrará el mensaje Desconocido. En el caso de los servidores de impresión externos HP Jetdirect, se mostrará el siguiente mensaje si se conecta una impresora, pero no es posible identificarla.
  • Paralelo: plotter/impresora Centronics
  • USB (unidireccional): periférico unidireccional
  • USB (bidireccional): periférico bidireccional
Dirección de hardware (también llamada Dirección de estación o Dirección MAC). Únicamente identifica un dispositivo en la red local. Para las impresoras HP y los servidores de impresión externos HP Jetdirect que disponen de múltiples puertos de impresora se adjuntará un número de puerto (Puerto 1, Puerto 2, Puerto 3). Esta dirección se enumerará en la página (de prueba) de configuración del servidor de impresión.
Dirección IP (Internet Protocol) Lo asigna un administrador de la red y únicamente identifica un dispositivo en una red TCP/IP. En la lista se mostrarán tanto las direcciones IPv4 como IPv6.  Si una impresora tiene varias direcciones, aparecerá una vez para cada una de las direcciones; de este modo, podrá seleccionar la interfaz que desee utilizar. 

Nota: Cuando se enciende un servidor de impresión HP Jetdirect y no se configura con una dirección IP válida en un intervalo de tiempo de 2 minutos, se asignará una dirección IP automática (169.254.x.x) o una dirección IP predeterminada (192.0.0.192). HP recomienda cambiar la dirección IP predeterminada. Si varias impresoras disponen de la misma dirección predeterminada, verifique la Dirección de hardware para identificar la impresora correcta.
Nombre de host IP Un nombre de host IP es un nombre fácil de identificar que se puede utilizar en lugar de la dirección IP de la impresora. Se puede utilizar cuando se ha configurado un servidor de nombres en la red para asignar nombres de host a sus correspondientes direcciones IP. Si se ha asignado y configurado un nombre de host para la impresora (por ejemplo, printer1.marketing.com), podrá especificar un nombre de host para identificarla.

Nota: si se ha truncado una entrada de campo y desea visualizar el campo completo, ajuste su anchura. Para ello, coloque el cursor en el borde del campo, haga clic y arrástrelo.

Para obtener instrucciones sobre cómo imprimir en una página de configuración de red, consulte la documentación suministrada con la impresora.

Si se ha protegido una impresora seleccionada mediante un Nombre de la comunidad SNMP o contraseña Jetdirect, el sistema solicitará dicho nombre o contraseña.
