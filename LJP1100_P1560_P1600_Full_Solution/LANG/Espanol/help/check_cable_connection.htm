Comprobar la conexión de la impresora

Si la conexión con la impresora es inalámbrica, compruebe que el indicador de conexión inalámbrica de la impresora esté encendido (consulte la documentación suministrada con la impresora para obtener ayuda sobre cómo identificar dicho indicador). A continuación, seleccione una de las opciones que aparecen en la pantalla, descritas más abajo.

Si la conexión es a través de un cable de red, compruebe las luces del puerto de red en la parte posterior de la impresora:

Encendida y es de color verde

 A continuación, seleccione una de las opciones que aparecen en la pantalla, descritas más abajo. Para encontrar la dirección de hardware (dirección MAC), la dirección IP o el nombre de dispositivo (nombre de Host IP) de la impresora, imprima una página de configuración de red. Si desea obtener instrucciones sobre cómo imprimir una página de configuración de red, consulte la documentación suministrada con la impresora.

Apagada
  • Compruebe que el cable de red esté correctamente conectado a la impresora y al punto de acceso inalámbrico (WAP) o al equipo.
  • Compruebe que el punto de acceso inalámbrico esté encendido.
  • Compruebe que la impresora esté encendida.
  • Compruebe que el equipo esté conectado a la red.

Una vez verificados los elementos de la lista anterior, elija un nuevo método para buscar la impresora:

  • Seleccione Buscar de nuevo para que el asistente de instalación realice otra búsqueda.
  • Seleccione Especificar una impresora mediante la dirección para introducir manualmente la dirección de hardware (dirección MAC), la dirección IP o el nombre de dispositivo (nombre de Host IP) de la impresora.
  • Seleccione Introducir la configuración de una impresora que no se encuentra en la red si desea configurar un puerto para una impresora que no esté conectada a la red. Para obtener más información sobre esta opción, consulte la documentación suministrada con la impresora.
