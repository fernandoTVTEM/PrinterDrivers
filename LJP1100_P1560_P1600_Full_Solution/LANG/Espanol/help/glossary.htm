archivo CAB
Archivo contenedor. Un archivo CAB tiene la extensión .cab y es un archivo único creado para incluir varios archivos comprimidos.

Archivo INF
Este tipo de archivo tiene la extensión .inf y contiene información acerca de un dispositivo. Informa al programa de instalación cuál es el nombre del dispositivo, cuáles son los archivos que deben copiarse y dónde.

ARP
Protocolo de resolución de dirección. El ARP es un protocolo TCP/IP que se utiliza para convertir una dirección IP en una dirección física (denominada dirección DLC), como una dirección de Ethernet. Cuando un dispositivo Host quiere obtener una dirección física envía una solicitud ARP a la red TCP/IP. Luego, el dispositivo Host de la red que tenga la dirección IP solicitada responde enviando su dirección de hardware física.

Autenticación
La autenticación es una estrategia de seguridad para redes inalámbricas. En una red con autenticación, los dispositivos utilizan una clave compartida como contraseña y se comunican únicamente con los dispositivos que conocen dicha clave. A diferencia del cifrado WEP, la autenticación no cifra los datos que se envían entre dispositivos inalámbricos. No obstante, la autenticación puede utilizarse en combinación con el cifrado WEP. Las claves de autenticación y las claves WEP pueden ser idénticas.

BootP
Protocolo bootstrap que se utiliza para obtener automáticamente una dirección IP y otros parámetros TCP/IP de un servidor de red. Cuando se enciende la impresora (o dispositivo multifuncional), el servidor de impresión HP Jetdirect envía una solicitud de arranque (boot). Si hay un servidor BootP configurado, éste transmitirá los parámetros al servidor de impresión Jetdirect a través de la red. Consulte también DHCP.

Clave WEP
Una clave WEP, o “clave de cifrado”, es una secuencia de caracteres alfanuméricos o dígitos hexadecimales. Cuando cree una clave WEP, deberá recordarla de memoria o guardarla en un lugar seguro. Es posible que no pueda recuperar la clave WEP si la pierde. Una clave WEP tiene una longitud de 64 o 128 bits. Los primeros 24 bits de la clave se proporcionan automáticamente. La persona que crea la clave WEP debe proporcionar el resto de los bits (40 bits en el caso de las claves de 64 bits o 104 bits en el caso de las de 128 bits).
Para crear una clave WEP, introduzca la siguiente información, según si la clave WEP es alfanumérica o hexadecimal y según la cantidad de bits que contiene:

Alfanumérico64Cinco caracteres alfanuméricos
128Trece caracteres alfanuméricos
Hexadecimal64Diez dígitos hexadecimales
128Veintiséis dígitos hexadecimales

Controlador de impresora
Programa que controla una impresora. En la impresión de un documento, el controlador de la impresora envía los datos a la impresora mediante los comandos de control correspondientes. La mayoría de los sistemas operativos se suministran con controladores de impresora que sirven para los tipos de impresoras más habituales; sólo hay que instalarlos antes de utilizar la impresora. En el sitio Web del fabricante de la impresora encontrará los controladores más recientes. Consulte PCL o PostScript.

Convenciones de asignación de nombres: Impresora
Para asignar un nombre a una impresora (o dispositivo multifuncional), siga estas reglas:

  • El nombre puede incluir un máximo de 31 caracteres.
  • Letras de la "A" a la "Z", mayúsculas, excluida la Ñ.
  • Letras de la "a" a la "z", minúsculas, excluida la ñ.
  • Números del 0 al 9.
  • Evite caracteres con valores ASCII inferiores a 32 y superiores a 127.
  • No utilice comas (,).

Convenciones de asignación de nombres: Nombre de recurso compartido
Según recomendaciones de Microsoft, los nombres están limitados a caracteres ASCII de 7 bits (los primeros 127). No se permiten espacios. Siga estas reglas:

  • Letras de la "A" a la "Z", mayúsculas, excluida la Ñ.
  • Letras de la "a" a la "z", minúsculas, excluida la ñ.
  • Números del 0 al 9.
  • Determinados caracteres especiales: $ % ' - _ @ ~ ` ! ( ) { } . ^ # & + , ; = [ ].

DHCP
Protocolo de configuración dinámica de host. Es parecido a BootP, dado que también obtiene automáticamente la dirección IP por medio de un servidor de red. No obstante, un servidor DHCP mantiene un conjunto de direcciones IP que se asignan en forma dinámica a los dispositivos que las solicitan y presta sólo una dirección IP al servidor de impresión. Cuando la impresora (o dispositivo multifuncional) se apaga y se vuelve a encender, no existe garantía alguna de que el servidor de impresión Jetdirect tenga asignada la misma dirección IP, aunque el servidor DHCP lo intentará.

Dirección de hardware
Consulte Dirección de hardware de una red LAN, dirección MAC o dirección de estación.

Dirección de hardware en la LAN, dirección MAC o dirección de estación
Una dirección de hardware, también denominada dirección MAC (Control de acceso a medios) o de estación, es una dirección a nivel de enlace que identifica de forma unívoca cada nodo o dispositivo de la red. Tal como lo definen las normas de red IEEE 802.3, esta dirección consiste en un número hexadecimal de 12 dígitos (por ejemplo, 001083123ABC). El proveedor del dispositivo es quien asigna la dirección de hardware.
HP asigna a cada servidor de impresión HP Jetdirect una dirección de hardware exclusiva. Puede averiguar la dirección de hardware de su servidor de impresión HP Jetdirect imprimiendo una página de prueba de HP Jetdirect. En el caso de los servidores de impresión externos HP Jetdirect, la dirección también figura en una etiqueta adherida al servidor de impresión.

Dirección IP
Dirección del Protocolo de Internet. La dirección IP, que es asignada por el administrador de la red, identifica de forma unívoca a cada uno de los dispositivos de una red TCP/IP. Consiste en un número de cuatro bytes (32 bits) con el formato n.n.n.n, donde n es un número entre 0 y 255, normalmente asignado por el administrador de red.
Al encender un servidor de impresión HP Jetdirect, si éste no obtiene una dirección IP válida en los dos primeros minutos, se le asignará automáticamente una dirección IP predeterminada (192.0.0.192). Esta dirección IP predeterminada podría no ser válida para una red determinada, aunque puede utilizarse para identificar una nueva impresora (o dispositivo multifuncional). Sin embargo, para que el dispositivo funcione correctamente en una red TCP/IP es preciso asignarle una dirección IP válida. Si hay varias impresoras que poseen la dirección predeterminada, fíjese en el campo Dirección de hardware para identificar la impresora correcta.

Máscara de subred
La máscara de subred es el mecanismo que se utiliza para dividir una red TCP/IP en subredes (si se utiliza esta función). La división en subredes permite mejorar la eficacia de una red grande. La forma de una máscara de subred es similar a una dirección IP: n.n.n.n, donde n es un número entre 0 y 255. Normalmente, si no se ha asignado ninguna máscara de subred, se utilizará la que se haya asignado a su equipo.
Cuando se aplica a una dirección IP, la máscara de subred especifica la parte de la dirección IP que sirve para identificar la red o subred, así como la parte que sirve para identificar los diferentes nodos de Host.

Modo de comunicación
El modo de comunicación describe el tipo de red y el protocolo (TCP/IP) que se utilizará con la impresora.

NDS
Servicios de directorio Novell. NDS es una base de datos relacional que organiza los recursos de la red en forma lógica como objetos de una estructura de árbol con diferentes jerarquías. Si dispone de los derechos correspondientes, podrá acceder a estos recursos de la red independientemente de cuál sea su ubicación física. NDS fue concebido para sustituir las bases de datos de seguridad Novell.

Nombre de comunidad SNMP
Un nombre de comunidad SNMP establecido es una contraseña que se debe proporcionar cuando se utiliza SNMP para realizar un cambio en la configuración de la impresora. Si una impresora seleccionada (o dispositivo multifuncional) está protegida por un Nombre de comunidad SNMP o una contraseña de Jetdirect, el sistema solicita dicho nombre de comunidad. Los nombres de comunidad válidos pueden tener una longitud máxima de 255 caracteres.
Nota: El sistema sólo pregunta el nombre de comunidad si se ha asignado un Nombre de comunidad SNMP establecido no predeterminado. El administrador de la red puede asignar el Nombre de comunidad SNMP establecido, utilizando un programa de administración (como HP Web Jetadmin) u otras herramientas.

Nombre de Host IP
Se trata de un nombre fácil de identificar que puede utilizarse en lugar de la dirección IP de la impresora. Puede emplearse cuando el servidor de nombres de la red se ha configurado para establecer una correlación entre los nombres de Host y sus correspondientes direcciones IP. Una vez que se ha asignado y configurado un nombre de Host en la impresora (por ejemplo, printer1.marketing.com), se puede utilizar dicho nombre de Host para identificarla.

Nombre de la impresora
Introduzca un nombre para la impresora siguiendo las convenciones de asignación de nombres que correspondan. Este nombre se utilizará para identificar el icono de la impresora en la carpeta Impresoras del sistema. Para abrir la carpeta Impresoras, haga clic en Inicio en el escritorio de Windows, seleccione Configuración y, a continuación, Impresoras.

Nombre de puerto
El cuadro de diálogo Nombre de puerto sólo aparece en Windows NT y Windows 2000, y únicamente si no está marcada la opción “Utilizar nombre de puerto predeterminado” en el cuadro de diálogo Establecer modo de comunicación de red y si está seleccionado el modo de comunicación IP.

Puede especificar el nombre del puerto de red que se va a crear para la impresora que se está instalando. El nombre del puerto se utiliza para identificar el puerto en las propiedades de la impresora.

Los nombres de puerto válidos tienen una longitud máxima de 63 caracteres y pueden utilizar todo tipo de caracteres. El nombre del puerto debe ser exclusivo en cada equipo. El nombre del puerto aparece en la pantalla de resumen y en la página de prueba.

  • No compartida: Seleccione esta opción para configurar el equipo en modo de impresión de igual a igual si no desea que los demás usuarios de la red puedan acceder a la impresora a través de este equipo.
  • Compartida como: Seleccione esta opción para configurar este equipo como servidor para la impresión cliente-servidor. Los clientes de la red podrán enviar sus trabajos a la impresora a través de este equipo.

Nombre de red (SSID)
Un nombre de red es una cadena de caracteres alfanuméricos, con distinción de mayúsculas y minúsculas, que proporciona control de acceso básico a una red inalámbrica. El nombre de red se denomina también “identificador de conjunto de servicios” o “SSID”.

Perfil inalámbrico
Un perfil inalámbrico es una recopilación de configuraciones de red inalámbrica que se aplica a una red inalámbrica en particular. Por ejemplo, una tarjeta de LAN inalámbrica puede tener un perfil para una red doméstica y otro perfil para una red de pequeña oficina. Al instalar un dispositivo en una red, debe asegurarse de que selecciona el perfil adecuado.

PING
Packet Internet Groper. PING es una utilidad empleada para determinar si una dirección IP determinada es accesible. Lo que hace es enviar un paquete a la dirección especificada y esperar una respuesta. Esta utilidad se utiliza principalmente para resolver problemas con las conexiones a Internet. Existe un gran número de utilidades PING de freeware y shareware para equipos personales.

Puerta de enlace
Se trata de un direccionador o un equipo que funciona como enlace entre dos redes de características diferentes (por ejemplo, diferentes protocolos de comunicación, formatos de datos, estructuras o arquitecturas). Cuando las redes se encuentran divididas en subredes, las puertas de enlace se suelen utilizar para aislar las subredes con el objeto de controlar el tráfico de la red.
En las redes TCP/IP, cada dispositivo de la red está configurado con la dirección IP de la puerta de enlace más próxima. Si en la red no hay puertas de enlace, normalmente se utiliza la dirección IP del equipo.
Si desea más información, consulte Añadir impresoras en redes IP grandes.
Para una red inalámbrica, consulte Punto de acceso inalámbrico (WAP).

Punto de acceso inalámbrico (WAP)

Red Adhoc (Red de comunicación Dispositivo a Dispositivo)
Red de comunicación directa entre dos o más dispositivos conectados entre sí sin una estación base centralizada o un punto de acceso inalámbrico (WAP).

El rendimiento de estos tipos de redes puede disminuir si hay más de seis dispositivos en la red.

Red inalámbrica (Infraestructura)
Una red inalámbrica está compuesta por varios dispositivos conectados entre sí mediante una estación base centralizada o un punto de acceso inalámbrico (WAP) para uso comercial o público.

Red local
Una red local es una red que no se extiende más allá de los direccionadores o de las puertas de enlace. Si el equipo y la impresora (o dispositivo multifuncional) están conectados a una red local, los dos estarán conectados al mismo segmento de cable de la red.

Reinicio en frío
Antes de reiniciar el equipo en frío, se recomienda imprimir una página de prueba en la impresora y el servidor de impresión Jetdirect.

Para restablecer la configuración predeterminada de fábrica de los servidores de impresión externos HP Jetdirect, desconecte el cable de alimentación y, luego, pulse el botón de prueba del servidor de impresión mientras vuelve a conectar el cable de alimentación. Espere a que el servidor de impresión se reinicie.

Para reiniciar servidores de impresión internos HP Jetdirect, lleve a cabo un reinicio en frío de la impresora (o dispositivo multifunción) y espere a que el servidor de impresión vuelva a inicializarse. (En la mayoría de las impresoras HP LaserJet, debe presionar el botón “Go” (Continuar) en el panel de control de la impresora mientras la enciende. En el caso de las impresoras de otros fabricantes, consulte el manual correspondiente.)

PRECAUCIÓN: Si reinicia la impresora, se podría borrar la configuración.

Requisitos de soporte

  • Sistema

      En el caso de redes de Microsoft: deben ejecutarse el protocolo Microsoft IP y los servicios de cliente.
  • Pantalla
    • Resolución de la pantalla: VGA (640 x 480) mínimo

    Nota: Mientras se estén detectando las impresoras, pueden producirse importantes variaciones en el rendimiento del sistema, según la velocidad del sistema y el tamaño de la red.

    • Las impresoras deben estar conectadas a la red a través de un servidor de impresión HP Jetdirect compatible. Para obtener una lista de las soluciones y los servidores de impresión HP Jetdirect compatibles, consulte el sitio Web de HP (http://www.hp.com/go/support).
    • La utilidad de instalación puede emplearse con impresoras HP y de otros fabricantes, siempre que los controladores de impresora sean compatibles con el software que esta utilidad emplea para comprobar los puertos y admitan el formato de archivo de instalación de impresoras (.INF) de Microsoft.
  • Compartir
    • Este asistente sólo permite compartir impresoras en Windows XP, Windows NT 4.0 y Windows 2000.
    • Para compartir impresoras, el servicio de servidores debe estar habilitado.

Servidor de nombres
Un servidor de nombres es un equipo de la red que se utiliza para correlacionar (o resolver) nombres de Host (nombres de equipo o servidores de impresión) con direcciones IP. Los servidores DNS (Sistema de nombre de dominio) y los servidores WINS (Servicio de Windows para nombres de Internet) son ejemplos de servidores de nombres.

Servidores de impresión HP Jetdirect
Los servidores de impresión HP Jetdirect son accesorios de red de Hewlett-Packard que sirven para conectar las impresoras directamente a una red. La conexión directa a una red permite que la impresora (o dispositivo multifuncional) pueda colocarse en diferentes lugares de la red y también ayuda a optimizar su rendimiento.
Las tarjetas de servidor de impresión HP Jetdirect interno se utilizan con las impresoras HP que tienen ranuras de entrada/salida. Algunos servidores de impresión HP Jetdirect externos pueden utilizarse con prácticamente cualquier impresora que disponga de un puerto paralelo, mientras que otros servidores de impresión HP Jetdirect pueden utilizarse con prácticamente cualquier impresora que disponga de un puerto USB. Ambos tipos de servidor de impresión son compatibles con todas las topologías y protocolos de red más comunes.

Subred
Parte de la red que comparte un componente de dirección común. En las redes TCP/IP, subredes son todos los dispositivos cuyas direcciones IP tienen el mismo prefijo. Por ejemplo, todos los dispositivos con direcciones IP que empiezan con 100.100.100. forman parte de la misma subred. Dividir una red en subredes resulta útil tanto por motivos de seguridad como de rendimiento. Las redes IP se dividen mediante una Máscara de subred.

El estándar WEP proporciona seguridad mediante el cifrado de los datos que se envían a través de las ondas de radio de un dispositivo inalámbrico a otro para que los curiosos no puedan leerlos. Sólo podrán comunicarse con la impresora los dispositivos que tengan la misma configuración WEP que la impresora.
