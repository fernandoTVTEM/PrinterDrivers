Añadir impresoras en redes IP grandes

Si desea añadir impresoras (o dispositivos multifunción) a través de un direccionador o una puerta de enlace (no en la red de área local), podría resultar difícil identificar las impresoras mediante esta utilidad:

  • Es posible que los direccionadores no transmitan la dirección del hardware, la dirección MAC o la dirección de la estación.
  • Si tiene varias impresoras nuevas en la red local, el hecho de utilizar la dirección IP predeterminada (192.0.0.192) para identificar una impresora no garantiza que se seleccione la impresora correcta. Compruebe la dirección de hardware para identificar la impresora correcta.

Si desea usar esta utilidad para configurar una impresora:

  1. Realice una de las siguientes acciones:
    • mueva físicamente la impresora a la red local del equipo, o bien,
    • utilice esta utilidad en un equipo que se encuentre conectado a la misma red local que la impresora.
  2. Identifique la impresora mediante esta utilidad y configure los parámetros IP para que coincidan con la ubicación final que tendrá la impresora en la red. Puede averiguar la dirección MAC imprimiendo una página (de prueba) de configuración (desde la impresora o el servidor de impresión externo) para verificar que haya seleccionado la dirección MAC o de hardware correcta.
  3. Si es necesario, vuelva a instalar la impresora (o el servidor de impresión HP Jetdirect externo) en su ubicación definitiva en la red.
  4. Para instalar la impresora en otros equipos:
    • en el caso de otros equipos o servidores que imprimirán directamente en esta impresora, puede emplear esta utilidad para configurarlos, o bien,
    • en el caso de los clientes Windows que imprimirán en esta impresora a través de un sistema compartido, puede emplear la utilidad Añadir impresora, que se encuentra en la carpeta Impresoras del sistema cliente, o puede utilizar el programa para la instalación de impresoras compartidas creado para esta impresora.
