Cambiar la configuración TCP/IP

Especifique la nueva configuración TCP/IP (IPv4) de esta impresora. Consulte al administrador del sistema antes de modificar la configuración.

Nota: La utilidad de instalación detectará la configuración de su sistema y ofrecerá una configuración TCP/IP recomendada (tanto de IPv4 como de IPv6), pero puede modificar la configuración de IPv4 si lo desea (esta utilidad sólo puede usarse para configurar manualmente los parámetros de IPv4). Los valores de parámetros asignados no se configuran en la impresora hasta finalizar la instalación.

·         Si la impresora no tiene una configuración TCP/IP válida:
Introduzca, en los campos correspondientes, una dirección IP (IPv4) para la impresora, una máscara de subred y una dirección IP para la puerta de enlace que sean válidas. Antes de asignar algún valor, póngase en contacto con el administrador de la red.

Haga clic en Configuración recomendada para que la utilidad de instalación introduzca automáticamente los parámetros TCP/IP.

Marque la casilla del nombre de puerto predeterminado (sólo para Windows NT, 2000 y XP). Marque esta casilla para utilizar la configuración del puerto predeterminado de su impresora. Si no se marca esta casilla el cuadro de diálogo Nombre de puerto será la siguiente pantalla del asistente.

PRECAUCIÓN: Si desea conectar la impresora a otras redes, puede ser que la configuración TCP/IP recomendada por la utilidad no sea única para todas las redes y entre en conflicto con otros dispositivos. Proceda con cuidado al conectar con otras redes TCP/IP.

·         Si la impresora ya tiene una configuración TCP/IP válida:
Se mostrará la configuración TCP/IP actual.

PRECAUCIÓN: Cambiar una configuración TCP/IP válida puede impedir el acceso por parte de otros usuarios. Consulte con el administrador de la red antes de cambiar una configuración válida.

Nota: Si los parámetros TCP/IP se han protegido con contraseña por parte de un software de administración, como HP Web Jetadmin, los campos de entrada aparecerán inhabilitados.
