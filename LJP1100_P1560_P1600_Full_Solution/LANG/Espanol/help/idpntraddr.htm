Configuración de la impresora

La pantalla Configuración de la impresora le permite especificar la configuración de una impresora que no está en la red.

Puede escribir una o más de las siguientes direcciones para la impresora y entonces hacer clic en Siguiente.  El ingreso de más de una dirección ayudará a garantizar que el puerto creado se conecte a la impresora especificada.

  • Nombre de host IP. Un nombre de host IP es un nombre fácil de identificar que puede usarse en lugar de la dirección IP de la impresora.
  • Dirección de hardware (MAC). Una dirección de hardware de LAN (también llamada dirección de estación o dirección MAC) identifica de forma exclusiva a un dispositivo.
  • Dirección IP (IPv4). Una dirección IP (Protocolo de Internet) identifica un dispositivo de forma exclusiva en una red TCP/IP. Habitualmente es asignado por un administrador de red. Esta utilidad sólo puede usarse para configurar manualmente ajustes IPv4 en el dispositivo.

Nota: Para configurar impresoras que no están localizadas en la red local, consulte Añadir impresoras en redes IP grandes.
