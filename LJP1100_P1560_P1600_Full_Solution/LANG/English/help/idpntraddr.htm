Printer Settings

The Printer Settings screen allows you to specify settings for a printer that is not on the network.

You may enter one or more of the following addresses for the printer, and then click Next.  Entering more than one address will help ensure that the port created will be connected to the specified printer.

  • IP Hostname. An IP hostname is a friendly (or easy-to-identify) name that can be used in place of the printer's IP address.
  • Hardware Address. A LAN hardware address (also called a Station Address or MAC address) uniquely identifies a device.
  • IP Address (IPv4). An IP (Internet Protocol) address uniquely identifies each device on a TCP/IP network. It is usually assigned by a network administrator. This utility can only be used to manually configure IPv4 settings on the device.

Note: To set up printers that are not located on the local network, see Adding Printers on Large IP Networks.
