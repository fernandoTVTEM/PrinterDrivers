Check Printer Connection

If you are connecting to the printer wirelessly, verify that the printer's wireless indicator is on (see the documentation that came with your printer for help in identifying the printer's wireless indicator).  Then select one of the options on the screen described below.

If you are connecting with a network cable, check the network port lights on the back of the printer:

If the link light is... Then...

On and green

 Then select one of the options on the screen described below.  To find the printer’s Hardware address (MAC address), IP address, or device name (IP Hostname), print a network configuration page. For instructions on printing a network configuration page, see the documentation that came with the printer.

Off
  • Verify that the network cable is securely connected to both the printer and the Wireless Access Point (WAP) or computer.
  • Verify that the WAP is turned on.
  • Verify that the printer is turned on.
  • Verify that the computer is connected to the network.

Once you have verified the items listed above, choose a method to search for your printer again:

  • Select Search again to have the setup wizard perform another search.
  • Select Specify a printer by address to manually enter the printer’s Hardware address (MAC address), IP address, or device name (IP Hostname).
  • Select Enter settings for a printer not on the network if you are configuring a port for a printer that is not connected to the network.  For more information on this option, see the documentation that came with your printer.
