Port Name

Note: The Port Name dialog appears only in Windows NT, Windows 2000 and Windows XP. This dialog appears only if Use default port name is not checked on the Set Network Communication Mode dialog and if IP communication mode is selected.

You can specify the name of the network port to be created for the printer being installed. The port name is used to identify the port in the printer properties.

Valid port names are up to 63 characters long and consist of any characters. The port name must be unique. The port name is displayed on the summary screen and on the test page.
