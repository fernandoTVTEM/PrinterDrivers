Drucker nicht gefunden

Der Setup-Assistent kann nicht mit dem Drucker kommunizieren.

Auf diesem PC wurde eine Firewall aktiviert. Möglicherweise kann das Installationsprogramm den Drucker deshalb nicht erkennen. Deaktivieren Sie die Firewall vorübergehend auf diesem PC, bevor Sie mit der Installation fortfahren.

Falls auf Ihrem Computer eine Firewall aktiviert ist (z. B. ZoneAlarm©, BlackICE©, Sygate Security Agent©, Norton Personal Firewall© usw.), finden Sie in der Dokumentation des Herstellers eine Anleitung zum Deaktivieren der Firewall.

Wenn die Softwarefirewall deaktiviert ist, sollte nach der dringenden Empfehlung von HP Folgendes auf Ihrem Computer installiert sein:

  • Ein Hardwaregerät, das Firewallschutz zwischen dem Internet und Ihrem Computer bietet, wie ein Gateway.
  • Ein aktuelles Virenerkennungsprogramm.

Allgemeine Informationen zur Kommunikationsfehlerbehebung finden Sie in der mit Ihrem Drucker gelieferten Dokumentation sowie auf der Support-Website http://www.hp.com/support.

Firewalls verursachen häufig Kommunikationsprobleme. Wenn eine Firewall auf einem vernetzten Computer aktiviert ist, können folgende Schwierigkeiten auftreten:

    Während der Softwareinstallation:

  • Das HP Druckerinstallationsprogramm kann den HP Drucker im Netzwerk nicht finden. In einer Fehlermeldung wird mitgeteilt, dass das Setup-Programm keine Drucker im Netzwerk erkennen konnte. Zur Vermeidung dieses Problems sollten Sie während der HP Druckersoftwareinstallation vorübergehend Ihre Firewall deaktivieren. Anweisungen zur Deaktivierung der Firewall finden Sie in der Dokumentation des jeweiligen Herstellers.

    Nach der Softwareinstallation:

  • Wenn sich die IP-Adresse des HP Druckers ändert, ist Ihr Computer nicht in der Lage, mit dem HP Drucker zu kommunizieren.  Alle Druck-, Scan-, PC-Fax- und Bildübertragungsaufträge schlagen fehl, und für den Produktstatus wird angegeben, dass der HP Drucker nicht angeschlossen ist.  In einer privaten Netzwerkumgebung ist es normal, dass sich die IP-Adresse gelegentlich ändert. Wenn die Softwarefirewall nicht aktiviert ist, wird die neue Adresse an den Computer gesendet, so dass Computer und HP Drucker weiterhin miteinander kommunizieren können. Deaktivieren Sie zur Vermeidung dieses Problems Ihre Firewall. Als erfahrener Benutzer, der mit Netzwerkbetrieb vertraut ist, können Sie stattdessen auch festlegen, dass eine statische IP-Adresse verwendet wird. Vorschläge zur Konfiguration Ihres Druckers mit einer statischen IP-Adresse finden Sie auf der HP Website unter http://www.hp.com/support.
  • Bei einem HP All-in-One-Gerät können Sie die Speicherkartensteckplätze des HP All-in-One-Geräts nicht zur Übertragung von Dateien auf Ihren Computer verwenden. Der Grund dafür liegt darin, dass über das zugeordnete Netzlaufwerk oder die Bildübertragungssoftware nicht auf die Dateien der Fotokarte zugegriffen werden kann.

Weitere Informationen zu Firewalls finden Sie auf der HP Website.

So deaktivieren Sie die Windows-Firewall in Windows Vista:

  1. Klicken Sie auf Start > Netzwerk.
  2. Klicken Sie mit der rechten Maustaste auf Netzwerk, und wählen Sie Eigenschaften.
  3. Klicken Sie unten links auf Windows-Firewall.
  4. Wählen Sie die Option Windows-Firewall ein- oder ausschalten.
  5. Wählen Sie im Dialogfeld Benutzerzugriffssteuerung die Option Weiter.
  6. Wählen Sie die Option Inaktiv, um die Firewall zu deaktivieren, und klicken Sie anschließend auf OK. Die Windows-Firewall ist jetzt deaktiviert.

So deaktivieren Sie die Windows-Firewall unter Windows XP:

  1. Klicken Sie auf der Taskleiste am unteren Bildschirmrand auf Start und anschließend auf Systemsteuerung.
  2. Klicken Sie auf die Kategorie Netzwerk- und Internetverbindungen. (Wenn Netzwerk- und Internetverbindungen nicht angezeigt wird, klicken Sie auf die Option Zur Kategorieansicht wechseln auf der linken Seite des Fensters "Systemsteuerung" unter Systemsteuerung).
  3. Klicken Sie auf Netzwerkverbindungen.
  4. Klicken Sie mit der rechten Maustaste auf DFÜ-, LAN- oder Hochgeschwindigkeits-Internetverbindung, mit der Sie eine Verbindung zum Internet herstellen. Klicken Sie anschließend im Kontextmenü auf Eigenschaften.
  5. Deaktivieren Sie auf der Registerkarte Erweitert unter Windows-Firewall über die Schaltfläche Einstellungen die Windows XP-Firewall.
  6. Klicken Sie im Setup-Assistenten auf Weiter, um mit der Druckerinstallation fortzufahren.
  7. Um die Windows-Firewall nach der Druckerinstallation wieder zu aktivieren, folgen Sie den oben genannten Anweisungen zum Aufrufen des Dialogfelds für die Windows-Firewall. Aktivieren Sie die Windows-Firewall über die Schaltfläche Einstellungen anschließend erneut.

Hinweis: Vergessen Sie nicht, die Windows-Firewall nach Abschluss der Installation wieder zu aktivieren.
