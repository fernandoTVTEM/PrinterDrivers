Druckereinstellungen

Im Dialogfeld Druckereinstellungen können Sie die Einstellungen für einen Drucker angeben, der nicht an das Netzwerk angeschlossen ist.

Sie können eine oder mehrere der folgenden Adressen für den Drucker eingeben und anschließend auf Weiter klicken.  Das Eingeben von mehr als einer Adresse erhöht die Sicherheit, dass der erstellte Anschluss mit dem gewünschten Drucker verbunden wird.

  • IP-Hostname. Ein IP-Hostname ist ein leicht zu identifizierender Name, der anstelle der IP-Adresse verwendet werden kann.
  • Hardwareadresse. Durch die LAN-Hardwareadresse (auch physikalische oder MAC-Adresse genannt) wird ein Gerät eindeutig identifiziert.
  • IP Addresse (IPv4). Die IP (Internet Protocol)-Adresse dient zur eindeutigen Identifizierung eines Geräts in einem TCP/IP-Netzwerk. Sie wird normalerweise vom Netzwerkadministrator vergeben. Dieses Dienstprogramm dient nur zur manuellen Konfiguration von IPv4-Einstellungen auf dem Gerät.

Hinweis: Informationen zum Einrichten von Druckern außerhalb des lokalen Netzwerks erhalten Sie unter Drucker in großen IP-Netzwerken hinzufügen.
