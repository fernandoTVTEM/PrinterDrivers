Drucker in großen IP-Netzwerken hinzufügen

Beim Hinzufügen von Druckern (oder Multifunktionsgeräten) über einen Router oder ein Gateway (also von Geräten, die sich nicht in Ihrem lokalen Netzwerk befinden) kann dieses Dienstprogramm Probleme bei der Druckererkennung haben:

  • Die Hardware-, MAC- oder physikalische Adresse des Druckers wird von den Routern oftmals nicht weitergegeben.
  • Wenn im lokalen Netzwerk mehrere neue Drucker vorhanden sind, ist nicht gewährleistet, dass bei Verwendung der IP-Standardadresse (192.0.0.192) zur Identifizierung des Geräts der richtige Drucker gewählt wird. Überprüfen Sie die Hardwareadresse, um den richtigen Drucker zu ermitteln.

Um einen Drucker mit diesem Dienstprogramm richtig zu konfigurieren, gehen Sie wie folgt vor:

  1. Führen Sie einen der folgenden Schritte aus:
    • Entfernen Sie den Drucker physikalisch aus dem lokalen Netzwerk Ihres Computers.
    • Führen Sie dieses Dienstprogramm auf einem Computer aus, der an das lokale Netzwerk des Druckers angeschlossen ist.
  2. Lassen Sie den Drucker vom Dienstprogramm ermitteln und konfigurieren Sie die IP-Parameter so, dass sie dem endgültigen Platz des Druckers im Netzwerk entsprechen. Sie können bei Bedarf die MAC-Adresse herausfinden, indem Sie eine Konfigurationsseite bzw. Testseite (entweder vom Drucker oder vom externen Druckserver aus) drucken, um sicherzustellen, dass Sie die korrekte MAC- oder Hardwareadresse gewählt haben.
  3. Stellen Sie den Drucker (oder den externen HP Jetdirect Druckserver) ggf. wieder an seinem endgültigen Standort im Netzwerk auf.
  4. Um diesen Drucker auf anderen Computern zu installieren, führen Sie einen der folgenden Schritte aus:
    • Andere Computer oder Server, die den Drucker direkt benutzen sollen, können Sie mit Hilfe dieses Dienstprogramms konfigurieren.
    • Für Windows-Clients, die den Drucker über ein freigegebenes System nutzen, verwenden Sie das Dienstprogramm "Neuer Drucker" im Ordner "Drucker" auf dem Client oder ein für den Drucker erstelltes Installationsprogramm für freigegebene Drucker.
