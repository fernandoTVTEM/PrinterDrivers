Drucker gefunden

Auf der Bildschirmanzeige Drucker gefunden können Sie einen im lokalen Netzwerk erkannten Drucker auswählen oder zusätzliche Setup-Optionen für die Installation von Netzwerk-, USB- oder parallel angeschlossenen Druckern nutzen.

  • Erkannten Netzwerkdrucker installieren. Wählen Sie den gewünschten Drucker aus der Liste aller Drucker (oder Multifunktionsgeräte) aus, die im lokalen Netzwerk erkannt wurden, und klicken Sie auf Weiter. Klicken Sie auf die Schaltfläche Erneut suchen, um erneut nach Druckern im Netzwerk zu suchen.
  • Hilfe zum Suchen des Druckers. Wählen Sie diese Option, wenn der gewünschte Netzwerkdrucker nicht aufgeführt ist oder Sie einen Drucker installieren möchten, der über ein USB- oder Parallel-Kabel direkt an Ihren Computer angeschlossen ist. Für Netzwerkdrucker beinhaltet diese Option zusätzliche Suchoptionen wie z. B. die Suche eines Druckers über seine IP-Adresse. 

Hinweis: Hinweise zum Einrichten eines Druckers, der sich nicht im lokalen Netzwerk befindet, finden Sie unter: Drucker in großen IP-Netzwerken hinzufügen.

Die Drucker in der Liste werden durch die folgenden Felder bestimmt. Die Hardware-Adresse ist die einfachste Option für die Bestimmung des Druckers:

Modell Bestimmt das Druckermodell (z. B. HP LaserJet 5000 Serie). Unbekannt wird angezeigt, wenn das Modell nicht bestimmt werden konnte, der Drucker ausgeschaltet ist oder wenn kein Drucker angeschlossen ist. Bei externen HP Jetdirect Druckservern wird Folgendes angezeigt, wenn ein Drucker angeschlossen ist, jedoch nicht identifiziert werden konnte:
  • Parallel: Centronics Drucker/Plotter
  • USB (unidirektional): Unidirektionales Peripheriegerät
  • USB (bidirektional): Bidirektionales Peripheriegerät
Hardware-Adresse (Auch Stationsadresse oder MAC-Adresse). Identifiziert ein Gerät eindeutig im lokalen Netzwerk. Bei Hewlett-Packard Druckern und externen Hewlett-Packard Druckservern, die über mehrere Druckeranschlüsse verfügen, kann eine Anschlussnummer (Port 1, Port 2, Port 3) angehängt sein. Diese Adresse ist auf der Konfigurations(test)seite des Druckservers aufgeführt.
IP-Adresse (Internet Protocol) Identifiziert ein Gerät eindeutig in einem TCP/IP-Netzwerk und wird durch einen Netzwerkadministrator zugewiesen. Sowohl IPv4- als auch IPv6-Adressen werden in dieser Liste angezeigt.  Wenn ein Drucker verschiedene Adressen hat, wird er für jede Adresse einmal angezeigt, so dass Sie die gewünschte Schnittstelle auswählen können. 

Hinweis: Wenn ein HP Jetdirect Druckserver eingeschaltet wird und innerhalb von zwei Minuten nicht über eine konfigurierte gültige IP-Adresse verfügt, wird automatisch entweder eine Auto-IP-Adresse (169.254.x.x) oder eine Standard-IP-Adresse (192.0.0.192) zugewiesen. Hewlett-Packard empfiehlt, die Standard-IP-Adresse zu ändern. Gibt es mehrere Drucker mit der gleichen Standardadresse, überprüfen Sie die Hardware-Adresse, um den richtigen Drucker zu ermitteln.
IP-Hostname Ein IP-Hostname ist ein bekannter (oder einfach identifizierbarer) Name, der anstelle der IP-Adresse des Druckers verwendet werden kann. Er kann dann genutzt werden, wenn ein Namensserver im Netzwerk die Hostnamen mit den entsprechenden IP-Adressen verknüpft. Wenn auf dem Drucker ein Hostname zugewiesen und konfiguriert wurde (z. B.drucker1.marketing.com), können Sie den Hostnamen zur Identifizierung festlegen.

Hinweis: Wenn ein Feldeintrag verkürzt dargestellt wird und Sie das gesamte Feld sehen wollen, erweitern Sie die Größe des Feldes. Platzieren Sie dafür den Cursor auf dem Rand des Feldkopfes, klicken Sie darauf, und ziehen Sie das Feld breiter.

Informationen zum Drucken einer Netzwerkkonfigurationsseite finden Sie in der Dokumentation des Druckers.

Wenn ein ausgewählter Drucker durch einen SNMP-Community-Namen oder ein Jetdirect Kennwort geschützt ist, werden Sie zur Eingabe des Community-Namens bzw. des Kennworts aufgefordert.
