Druckerverbindung überprüfen

Wenn Sie eine drahtlose Verbindung zum Drucker herstellen, prüfen Sie, ob die Drahtlos-Anzeige des Druckers leuchtet. (Hilfe zur Drahtlos-Anzeige des Druckers finden Sie in der Dokumentation zum Drucker.) Wählen Sie anschließend eine Option aus dem nachfolgend beschriebenen Bildschirm aus.

Wenn Sie die Verbindung über ein Netzwerkkabel herstellen, überprüfen Sie die Lämpchen für die Netzwerkanschlüsse auf der Rückseite des Druckers:

Status Verbindungs-LED Bedeutung

Eingeschaltet, leuchtet grün

 Wählen Sie anschließend eine Option aus dem nachfolgend beschriebenen Bildschirm aus. Um die Hardware-Adresse (MAC-Adresse) des Druckers, die IP-Adresse oder den Gerätenamen (IP-Hostnamen) zu finden, drucken Sie eine Netzwerkkonfigurationsseite. Anweisungen zum Ausdruck einer Netzwerkkonfigurationsseite finden Sie in der mitgelieferten Dokumentation Ihres Druckers.

Aus
  • Vergewissern Sie sich, dass das Netzwerkkabel richtig an den Drucker, den Wireless Access Point (WAP) oder den Computer angeschlossen ist.
  • Vergewissern Sie sich, dass der WAP aktiviert ist.
  • Vergewissern Sie sich, dass der Drucker eingeschaltet ist.
  • Vergewissern Sie sich, dass der Computer an das Netzwerk angeschlossen ist.

Nachdem Sie die oben aufgeführten Elemente überprüft haben, wählen Sie eine Methode für die erneute Suche des Druckers aus:

  • Wählen Sie Erneut suchen aus, damit der Setup-Assistent eine neue Suche durchführt.
  • Wählen Sie Drucker über Adresse angeben aus, um manuell die Hardware-Adresse (MAC-Adresse) des Druckers, die IP-Adresse oder den Gerätenamen (IP-Hostnamen) einzugeben.
  • Wählen Sie Einstellungen für einen Drucker außerhalb des Netzwerks eingeben, wenn Sie einen Anschluss für einen Drucker konfigurieren, der nicht an das Netzwerk angeschlossen ist. Weitere Informationen zu dieser Option finden Sie in der Dokumentation zu Ihrem Drucker.
