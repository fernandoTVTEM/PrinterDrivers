Drucker bestimmen

Wenn das Setup-Programm Ihren Drucker im Netzwerk nicht finden konnte, überprüfen Sie bitte Folgendes:

  • Vergewissern Sie sich, dass der Drucker tatsächlich korrekt angeschlossen und eingeschaltet ist.
  • Überprüfen Sie, ob auf Ihrem Computer eine Firewall aktiv ist.  Wenn auf Ihrem Computer eine Firewall aktiviert ist, könnte dies verhindern, dass das Installationsprogramm den Drucker erkennt. Deaktivieren Sie die Firewall auf Ihrem Computer vorübergehend, bevor Sie mit der Installation fortfahren.  Wenn eine Firewall aktiviert ist (wie z. B. ZoneAlarm©, BlackICE©,etc.), lesen Sie bitte in der Dokumentation des Herstellers nach, wie die Firewall deaktiviert werden kann.  HP empfiehlt dringend, Folgendes auf dem Computer zu installieren: (1) Ein Hardwaregerät, das Firewallschutz zwischen dem Internet und Ihrem Computer bietet, wie z. B. ein Gateway, und (2) ein aktuelles Virenerkennungsprogramm.

    • Weitere Informationen zu Firewalls

Wenn Sie die oben aufgeführten Punkte überprüft haben, wählen Sie eine der folgenden Vorgehensweisen, um nach Ihrem Drucker zu suchen:

  • Aus der Liste der ermittelten Drucker auswählen. Sie können ein Gerät aus der Liste der verfügbaren Drucker, die im Netz gefunden wurden, auswählen. Wenn Sie auf Weiter klicken, sucht das Dienstprogramm nach Netzwerkgeräten und zeigt diese in einer Liste an. Sie können den gewünschten Drucker dann aus der Liste auswählen.

  • Drucker über Adresse angeben. Sie können einen bestimmten Drucker anhand seiner Adresse suchen. Wenn Sie auf Weiter klicken, werden Sie aufgefordert, den Drucker anhand seiner LAN-Hardwareadresse, der IP-Adresse oder des IP-Hostnamens zu suchen.

  • Einstellungen für einen Drucker außerhalb des Netzwerks eingeben.  Wählen Sie diese Option, wenn Sie einen Anschluss für ein Gerät konfigurieren, das nicht an das Netzwerk angeschlossen ist.  Weitere Informationen zu dieser Option finden Sie in der Dokumentation zu Ihrem Drucker.
