Nom de port

Nota: El quadre de diàleg Nom de port es troba disponible només al Windows NT, Windows 2000 i Windows XP. Aquest quadre de diàleg es visualitza únicament si el quadre de verificació Utilitza el nom de port per defecte no està activat al quadre de diàleg Establir mode de comunicació de xarxa, i si el mode de comunicació IP està seleccionat.

Podeu especificar el nom del port de xarxa que cal crear per a la impressora que voleu instal·lar. El nom de port s'utilitza per identificar el port a les propietats de la impressora.

Els noms de port vàlids poden tenir fins a 63 caràcters de qualsevol tipus. El nom de port ha de ser exclusiu. El nom de port s'indica a la pantalla de resum i a la pàgina de prova.
