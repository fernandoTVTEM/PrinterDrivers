Printera iestatîjumi

Ekrânâ Printera iestatîjumi var norâdît ârpus tîkla esoða printera iestatîjumus.

Ievadiet vienu vai vairâkas no ðîm printera adresçm un pçc tam noklikðíiniet uz Next (Tâlâk).  Ievadot vairâkas adreses, nodroðinâsiet, ka izveidotais ports tiks savienots ar norâdîto printeri.

  • IP resursdatora nosaukums. IP resursdators ir viegli identificçjams nosaukums, ko var izmantot printera IP adreses vietâ.
  • Aparatûras adrese. LAN aparatûras adrese (tiek saukta arî par stacijas adresi vai MAC adresi) unikâli identificç ierîci.
  • IP adrese (IPv4). IP (interneta protokola) adrese unikâli identificç katru TCP/IP tîkla ierîci. To parasti pieðíir tîkla administrators. Ðo utilîtu var izmantot tikai, lai manuâli konfigurçtu ierîces IPv4 iestatîjumus.

Piezîme. Lai uzstâdîtu printerus, kas ir ârpus lokâlâ tîkla, skatiet Printeru pievienoðana lielos IP tîklos.
