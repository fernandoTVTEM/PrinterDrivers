Printeru pievienoðana lielos IP tîklos

Ja pievienojat printerus (vai daudzfunkciju ierîces), izmantojot marðrutçtâju vai vârteju (ne lokâlajam tîklam), printeru identificçðana, izmantojot ðo utilîtu, var bût apgrûtinâta:

  • Marðrutçtâji var nenodot printera aparatûras, MAC vai stacijas adresi.
  • Ja lokâlajam tîklam ir pievienoti vairâki printeri, noklusçjuma IP adreses (192.0.0.192) izmantoðana printera identificçðanai nevar garantçt, ka tiks atlasîts pareizais printeris. Lai identificçtu pareizo printeri, atzîmçjiet aparatûras adresi.

Lai uzstâdîtu printeri, izmantojot ðo utilîtu, rîkojieties ðâdi:

  1. Veiciet kâdu no ðîm darbîbâm:
    • fiziski pârvietojiet printeri uz datora lokâlo tîklu vai
    • lietojiet ðo utilîtu datorâ, kas ir pievienots lokâlajam printera tîklam.
  2. Identificçjiet printeri, izmantojot ðo utilîtu, un konfigurçjiet IP parametrus, lai tie atbilstu printera galîgajai atraðanâs vietai tîklâ. Lai pârliecinâtos, vai ir atlasîta pareizâ MAC vai aparatûras adrese, izdrukâjot konfigurâcijas (testa) lappusi (no printera vai no ârçjâ drukas servera), var atrast MAC adresi.
  3. Ja nepiecieðams, atgrieziet printeri (vai HP Jetdirect ârçjo drukas serveri) tâ galîgajâ atraðanâs vietâ tîklâ.
  4. Lai instalçtu ðo printeri citiem datoriem, veiciet kâdu no ðîm darbîbâm:
    • Izmantojiet ðo utilîtu, lai atseviðíi konfigurçtu katru datoru un serveri, kas drukâs tieði ar ðo printeri, vai
    • Windows klientiem, kuri drukâs ar ðo printeri, izmantojot koplietotu sistçmu, izmantojiet mapç Printeri atrodamo klientu sistçmas Printera pievienoðanas utilîtu vai izmantojiet koplietotajam printerim paredzçto instalçðanas programmu.
