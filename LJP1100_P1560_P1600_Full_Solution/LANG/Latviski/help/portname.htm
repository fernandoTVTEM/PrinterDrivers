Porta nosaukums

Piezîme. Dialoglodziòð Port Name (Porta nosaukums) tiek parâdîts tikai operçtâjsistçmâ Windows NT, Windows 2000 un Windows XP. Ðis dialoglodziòð tiek parâdîts tikai tad, ja dialoglodziòâ Set Network Communication Mode (Iestatît tîkla saziòas reþîmu) nav atzîmçta izvçles rûtiòa Use default port name (Izmantot noklusçjuma porta nosaukumu) un ja ir atlasîts IP saziòas reþîms.

Varat norâdît tîkla porta nosaukumu, kas jâizveido instalçjamam printerim. Porta nosaukums tiek izmantots porta noteikðanai printera rekvizîtos.

Derîgi portu nosaukumi var bût ne vairâk kâ 63 rakstzîmes gari, un tie var sastâvçt no jebkâdâm rakstzîmçm. Porta nosaukumam jâbût unikâlam. Porta nosaukums ir redzams kopsavilkuma ekrânâ un testa lappusç.
