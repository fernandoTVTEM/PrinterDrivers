Printeris nav atrasts

Uzstâdîðanas vednis nevar sazinâties ar printeri.

Ja ugunsmûris ir iespçjots, tas, iespçjams, neïauj instalçðanas programmai noteikt printeri. Pirms turpinât instalçðanu, îslaicîgi atspçjojiet ugunsmûri.

Ja ir iespçjots ugunsmûris (piemçram, ZoneAlarm©, BlackICE©, Sygate Security Agent©, Norton Personal Firewall© u.c.), ugunsmûra atspçjoðanas norâdîjumus skatiet ugunsmûra piegâdâtâja dokumentâcijâ.

Ja atspçjojat programmatûras ugunsmûri, HP iesaka datorâ instalçt:

  • aparatûras ierîci, kas nodroðina ugunsmûra aizsardzîbu starp internetu un datoru, piemçram, vârteju;
  • atjauninâtu vîrusu noteikðanas programmu.

Lai iegûtu vispârîgu informâciju par saziòas traucçjummeklçðanu, lûdzu, skatiet printera dokumentâciju un apmeklçjiet atbalsta Web vietu http://www.hp.com/support.

Ugunsmûri bieþi izraisa saziòas problçmas. Ja ugunsmûris ir iespçjots datorâ, kas ir pievienots tîklam, var rasties ðâdas problçmas:

    Programmatûras instalçðanas laikâ:

  • HP printera instalçðanas programma nevar atrast HP printeri tîklâ.  Tiks parâdîts kïûdas paziòojums, ka uzstâdîðanas programma tîklâ nevar atrast printerus.  Lai novçrstu ðo problçmu, îslaicîgi atspçjojiet ugunsmûri, lîdz tiek instalçta HP printera programmatûra. Ugunsmûra atspçjoðanas norâdîjumus skatiet ugunsmûra piegâdâtâja dokumentâcijâ.

    Pçc programmatûras instalçðanas:

  • Ja HP printera IP adrese tiek mainîta, dators nevarçs sazinâties ar HP printeri.  Neviens drukâðanas, skençðanas, datora faksa sûtîðanas un attçlu pârsûtîðanas darbs netiks izpildîts, un izstrâdâjuma statusâ tiks norâdîts, ka HP printeris nav pievienots.  Mâjas tîkla vidç IP adrese reizçm var mainîties.  Ja programmatûras ugunsmûris nav iespçjots, jaunâ adrese tiek sûtîta uz datoru, kas ïauj datoram un HP printerim turpinât sazinâties.  Lai novçrstu ðo problçmu, atspçjojiet ugunsmûri vai, ja esat pieredzçjis lietotâjs un pârzinât tîkloðanu, varat tâ vietâ izvçlieties izmantot statisku IP adresi.  Lai skatîtu ieteikumus par to, kâ konfigurçt printeri ar statisku IP adresi, apmeklçjiet HP Web vietu http://www.hp.com/support.
  • Nevar izmantot daudzfunkciju ierîces atmiòas kartes slotus, lai pârsûtîtu failus uz datoru.  Tas ir tâdçï, ka foto kartç esoðie dati nav pieejami no kartçta tîkla diska vai attçla pârsûtîðanas programmatûras.

Lai iegûtu papildinformâciju par ugunsmûriem, apmeklçjiet HP Web vietu.

Kâ atspçjot Windows ugunsmûri sistçmâ Windows Vista:

  1. Noklikðíiniet uz Start (Sâkt) > Network (Tîkls).
  2. Ar peles labo pogu noklikðíiniet uz Network (Tîkls) un atlasiet Properties (Rekvizîti).
  3. Kreisajâ apakðçjâ stûrî noklikðíiniet uz Windows Firewall (Windows ugunsmûris).
  4. Atlasiet Turn Windows Firewall on/off (Ieslçgt/izslçgt Windows ugunsmûri).
  5. Lietotâja piekïuves vadîbas dialogâ atlasiet Continue (Turpinât).
  6. Atlasiet opciju Off (Izslçgts), lai atspçjotu ugunsmûri, un pçc tam noklikðíiniet uz OK (Labi).  Windows ugunsmûris ir atspçjots.

Kâ atspçjot interneta savienojuma ugunsmûri (Internet Connection Firewall — ICF) sistçmâ Windows XP:

  1. Uzdevumjoslâ ekrâna apakðdaïâ, noklikðíiniet uz Start (Sâkt) un pçc tam uz Control Panel (Vadîbas panelis).
  2. Noklikðíiniet uz kategorijas Network and Internet Connections (Tîkla un interneta savienojumi). (Ja Network and Internet Connections (Tîkla un interneta savienojumi) nav redzams, vadîbas paneïa loga kreisâs puses sadaïâ Control Panel (Vadîbas panelis) noklikðíiniet uz Switch to Category View (Pârslçgties uz kategoriju skatu).)
  3. Noklikðíiniet uz Network Connections (Tîkla savienojumi).
  4. Ar peles labo pogu noklikðíiniet uz savienojuma, ko izmantot savienojumam ar internetu — Dial-up (Iezvane), LAN (LAN) vai High-Speed Internet (Âtras darbîbas internets) —, un pçc tam kontekstizvçlnç noklikðíiniet uz Properties (Rekvizîti).
  5. Cilnes Advanced (Papildu) sadaïâ Internet Connection Firewall (Interneta savienojuma ugunsmûris) notîriet atlasi vai atspçjojiet Protect my computer or network (Aizsargât datoru un tîklu) un pçc tam noklikðíiniet uz OK (Labi). Windows XP ugunsmûris ir atspçjots.
  6. Uzstâdîðanas vednî noklikðíiniet uz Next (Tâlâk), lai turpinâtu printera instalçðanu.
  7. Lai iespçjotu Internet Connection Firewall (Interneta savienojuma ugunsmûris), kad printeris ir instalçts, izpildiet iepriekð minçtâs darbîbas lîdz opcijai Protect my computer or network (Aizsargât datoru un tîklu). Pçc tam atkârtoti atlasiet opciju Protect my computer or network (Aizsargât datoru un tîklu).

Piezîme. Noteikti iespçjojiet Windows ugunsmûri, kad instalâcija ir pabeigta.
