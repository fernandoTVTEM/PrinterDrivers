ARP
Address Resolution Protocol‏. ARP זהו פרוטוקול TCP/IP המשמש להמרת כתובת IP לכתובת פיזית (הנקראת כתובת DLC), כגון כתובת Ethernet. מארח המעוניין לקבל כתובת פיזית, משדר בקשת ARP לרשת ה- TCP/IP. המארח ברשת שברשותו כתובת ה- IP שבבקשה, עונה עם כתובת החומרה הפיזית שלו.

BootP
פרוטוקול אתחול (bootstrap) המשמש להורדה אוטומטית של כתובת IP ופרמטרים אחרים של TCP/IP משרת רשת. עם הפעלת המדפסת (או מכשיר משולב), שרת ההדפסה HP Jetdirect משדר בקשת אתחול. אם שרת BootP הותקן כהלכה, הוא משדר את הפרמטרים לשרת ההדפסה Jetdirect דרך הרשת. ראה גם DHCP.

DHCP
Dynamic Host Configuration Protocol (פרוטוקול להגדרת תצורה דינמית של מארח). דומה ל- BootP בכך שהוא מוריד אוטומטית כתובת IP משרת רשת. עם זאת, שרת DHCP מחזיק מאגר של כתובות IP שהוא מקצה בצורה דינמית להתקנים המבקשים אותן ורק משאיל כתובת IP לשרת ההדפסה. בכל כיבוי והפעלה מחדש של המדפסת (או מכשיר משולב), לשרת הדפסה Jetdirect לא תוקצה בהכרח כתובת IP זהה, למרות ששרת ה- DHCP ינסה לעשות זאת.

-I-

IPv4

IPv4 היא גרסה 4 של פרוטוקול האינטרנט. כתובות IPv4 מורכבות מ-‏32 סיביות שמצוינות בייצוג עשרוני מנוקד. למשל, 192.168.100.250. אם לא הוקצתה כתובת IPv4 חוקית על-ידי שרת רשת, שרת ההדפסה Jetdirect יקצה את עצמו באמצעות כתובת link-local‏/Auto-IP‏ (‎169.254.x.x‏) או כתובת legacy המשמשת כברירת מחדל (192.0.0.192), בהתאם לרשת.

IPv6

IPv6 היא גרסה משופרת של פרוטוקול האינטרנט, העשויה להחליף בסופו של דבר את IPv4, אך מסוגלת להתקיים יחד עם IPv4. כתובות IPv6 הן באורך 128 סיביות והערכים מצויינים בייצוג הקסדצימלי בנקודתיים, לדוגמה, ‏‎2001:0DB8:0000:0000:0000:0000. כך, כתובות IPv6 יכולות להכיל גדילה של התקנים המחוברים לרשתות לזמן מה. בשונה מ-IPv4, ניתן להקצות מספר כתובות IPv6 לשרת ההדפסה. בעוד שכתובת link-local IPv6 המשמשת כברירת מחדל מוקצית באופן אוטומטי, ניתן להקצות גם כתובות מבוססות נתב, מבוססות שרת וכתובות IPv6 ידניות.

SNMP Community Name
SNMP Set Community Name היא סיסמה שיש לספק כאשר SNMP משמש לביצוע שינויים בתצורת המדפסת. אם מדפסת שנבחרה (או מכשיר All-in-One) מוגנת באמצעות SNMP Community Name או סיסמת Jetdirect, תתבקש לספק את ה- Community Name. שמות community תקפים הם באורך של 255 תווים לכל היותר.
הערה: תתבקש לספק סיסמה רק כאשר הוקצה SNMP Set Community Name שאינו ברירת המחדל. מנהל רשת יכול להקצות את ה- SNMP Set Community name באמצעות יישום ניהול רשת (כגון HP Web Jetadmin) או כלים אחרים.

WEP
סטנדרט האבטחה המקורי המשמש ברשתות Wi-Fi להצפנת תעבורת רשת אלחוטית. WEP מספק אבטחה על-ידי הצפנת נתונים הנשלחים בגלי רדיו מהתקן אלחוטי אחד להתקן אלחוטי אחר, פעולה ההופכת את הנתונים לבלתי מובנים למי שמצותת להם. רק התקנים עם הגדרות WEP זהות למדפסת יוכלו ליצור קשר עם המדפסת.

אימות
אימות משמש כאסטרטגיית אבטחה ברשת אלחוטית. ברשת עם אימות, ההתקנים משתמשים במפתח משותף כסיסמה ויוצרים תקשורת רק עם התקנים שיודעים מפתח זה. שלא כמו WEP, אימות אינו מצפין את הנתונים שנשלחים בין התקנים אלחוטיים. עם זאת, ניתן להשתמש באימות יחד עם WEP. מפתחות אימות ומפתחות WEP יכולים להיות זהים.

איפוס קר
לפני איפוס קר, מומלץ להדפיס עמוד ניסיון של המדפסת ושרת ההדפסה Jetdirect.

כדי לאפס שרתי הדפסה חיצוניים מסוג HP Jetdirect להגדרות ברירת המחדל של היצרן, נתק את כבל החשמל ולאחר מכן לחץ על הלחצן Test (בדיקה) שעל שרת ההדפסה ובו-זמנית חבר חזרה את כבל החשמל. המתן עד ששרת ההדפסה יאתחל מחדש.

כדי לאתחל שרתי הדפסה פנימיים מסוג HP Jetdirect, בצע איפוס קר של המדפסת (או מכשיר All-in-One) והמתן עד ששרת ההדפסה יאתחל מחדש. (ברוב דגמי מדפסות HP LaserJet, לחץ על הלחצן Go שבלוח הבקרה של המדפסת תוך כדי הפעלת המדפסת. למדפסות אחרות, עיין במדריך המדפסת).

כתובת IP
כתובת פרוטוקול אינטרנט (Internet Protocol‏). כתובת IP מזהה באופן ייחודי כל התקן ברשת TCP/IP ומנוהלת בדרך-כלל על-ידי מנהל רשת. שרתי הדפסה  HP Jetdirect עדכניים תומכים בכתובות IPv4 ו- IPv6.
כאשר מופעל שרת הדפסה HP Jetdirect, הוא מנסה בדרך-כלל להגדיר את עצמו עם כתובת IP חוקית משרת רשת. אם לא ניתן להקצות כתובת IP חוקית, תוקצה באופן אוטומטי כתובת שתשמש כברירת מחדל. אם מוגדרים מספר התקנים לאותה כתובת ברירת מחדל, בדוק את כתובת LAN Hardware‏ (MAC‏) כדי לזהות את ההתקן הנכון.

כתובת חומרה
ראה חומרת LAN, MAC או כתובת תחנה

כתובת חומרת LAN, MAC או תחנה
כתובת חומרה הידועה גם בשם כתובת בקרת גישה למדיה (Media Access Control - MAC) או כתובת תחנה, היא כתובת ברמת קישור המזהה בצורה ייחודית כל צומת או התקן ברשת. כפי שמוגדר על-ידי תקני הרשת IEEE 802.3, היא מורכבת ממספר הקסדצימלי בן 12 ספרות (לדוגמה, 001083123ABC). כתובת החומרה מוקצית על-ידי יצרן ההתקן.
HP מקצה לכל שרת הדפסה HP Jetdirect כתובת חומרה ייחודית. ניתן לקבוע את כתובת החומרה של שרת ההדפסה HP Jetdirect שברשותך על-ידי הדפסת עמוד ניסיון של HP Jetdirect. בשרתי הדפסה HP Jetdirect חיצוניים, הכתובת כלולה גם בתווית המצורפת לשרת ההדפסה.

מנהל התקן מדפסת
תוכנה השולטת במדפסת. בכל הדפסת מסמך, מנהל התקן המדפסת "לוקח פיקוד" ומזין נתונים למדפסת באמצעות פקודות הבקרה הנכונות. רוב מערכות ההפעלה החדישות מגיעות עם מנהלי התקנים למדפסת עבור רוב הסוגים הנפוצים של המדפסות, אולם יש להתקין אותם לפני שניתן יהיה להשתמש במדפסת. ניתן גם להוריד מנהלי התקנים מעודכנים מאתר האינטרנט של יצרן המדפסת. ראה PCL או PostScript.

מנהל התקן מדפסת PCL
מתאים למטרות הדפסה כללית. מתאים במיוחד ליישומי עיבוד תמלילים, גיליונות אלקטרוניים ומצגות, כגון Word, Excel ו- PowerPoint.

מנהל התקן מדפסת PostScript
מתאים למטרות הדפסה מיוחדת. הטוב ביותר להוצאה לאור שולחנית ויישומים עשירים בגרפיקה, כגון Photoshop, PageMaker ו- FrameMaker.

מסכת רשת-משנה
מסכת רשת משנה היא המנגנון המשמש לחלוקת רשת TCP/IP לרשתות-משנה (אם נעשה שימוש ברשתות משנה). חלוקה לרשתות-משנה מאפשרת שימוש יעיל יותר ברשתות גדולות. צורת מסכת רשת המשנה זהה לזו של כתובת IP: n.n.n.n כאשר n הוא מספר מ- 0 עד 255. אם לא הוקצתה מסכת רשת-משנה, בדרך כלל יעשה שימוש במסכת רשת המשנה שמוקצית למחשב.
כאשר מחילים אותה על כתובת IP, מסכת רשת המשנה מציינת את חלק כתובת ה- IP שמשמש לזיהוי הרשת/רשת-משנה ואת החלק שמשמש לזיהוי צמתים של מארח אינדיבידואלי.

מפתח WEP
מפתח WEP, או "מפתח הצפנה", הוא רצף של תווים אלפא-נומריים או ספרות הקסדצימליות. לאחר יצירת מפתח WEP, עליך לזכור אותו או לאחסנו במקום בטוח. אם תאבד את מפתח ה- WEP, לא תוכל לשחזרו. מפתח WEP הוא באורך 64 או 128 סיביות. 24 הסיביות הראשונות של המפתח מסופקות באופן אוטומטי. בעת יצירת מפתח WEP, המשתמש שיוצר את המפתח מספק את הביטים הנותרים (40 ביטים במקרה של מפתח 64 ביט, או 104 ביטים במקרה של מפתח 128 ביט).
כדי ליצור מפתח WEP, הזן את הנתונים הבאים בהתאם לסוג מפתח ה- WEP, אלפא-נומרי או הקסדצימלי, ולמספר הביטים שהוא מכיל:

פרופיל אלחוטי

רכזת רשת אלחוטית
ראה נקודת גישה אלחוטית (WAP).

רשת אלחוטית (תשתית)
רשת אלחוטית כוללת התקנים המחוברים באמצעות תחנת בסיס מרוכזת או נקודת גישה אלחוטית (WAP) כדי להשתמש בשירותים מסחריים או ציבוריים.

רשת התקן-להתקן (אד-הוק)
רשת התקן-להתקן היא שני התקנים אלחוטיים או יותר, המחוברים ללא שימוש בתחנת בסיס מרוכזת או נקודת גישה אלחוטית (WAP).

רשתות אד-הוק (רשת התקן-להתקן)
רשת התקן-להתקן היא שני התקנים אלחוטיים או יותר, המחוברים ללא שימוש בתחנת בסיס מרוכזת או נקודת גישה אלחוטית (WAP).

הביצועים בסוגי רשתות אלה עלולים להיפגע אם מחברים לרשת יותר משישה התקנים.

רשת מקומית
רשת מקומית היא רשת שאינה משתרעת מעבר לנתבים או שערים (gateways). אם המחשב והמדפסת (או מכשיר All-in-One) שברשותך מחוברים לרשת מקומית, פירושו שהם מחוברים לאותו מקטע של כבל רשת.

רשת משנה
חלק מרשת שיש לו רכיב כתובת משותף. ברשתות TCP/IP, רשתות משנה מוגדרות כמו כל ההתקנים שלכתובת ה- IP שלהם יש קידומת זהה. לדוגמה, כל ההתקנים בעלי כתובת IP המתחילה ב- 100.100.100 יהיו חלק מאותה רשת משנה. חלוקת רשת לרשתות משנה שימושית מטעמי אבטחה וביצועים. רשתות IP מחולקות באמצעות מסכת רשת-משנה.

שיתוף מדפסת
שיתוף מדפסת מאפשר למשתמשי רשת אחרים להדפיס למדפסת זו דרך מחשב זה. כלי שירות זה להתקנה תומך בשיתוף מדפסת רק ב- Windows 2000 וב- Windows NT 4.0. לשיתוף מדפסת, שירות השרת חייב להיות זמין במערכת. עליך לבחור אחת מהאפשרויות הבאות:

·         לא משותפת (Not Shared): בחר באפשרות זו עבור הדפסת peer-to-peer אם אינך רוצה שמשתמשי רשת אחרים ייגשו למדפסת דרך מחשב זה.

·         משותפת כ: (:Share as): בחר באפשרות זו כדי להגדיר מחשב זה כשרת עבור הדפסת לקוח-שרת. לקוחות רשת יורשו לשלוח עבודות הדפסה למדפסת דרך מחשב זה.

שם יציאה
תיבת הדו-שיח שם יציאה מופיעה רק ב-Windows NT ו-Windows 2000. תיבת דו-שיח זו מופיעה רק אם האפשרות "השתמש בשם יציאת ברירת המחדל" אינה מסומנת בתיבת הדו-שיח קביעת מצב תקשורת רשת ואם נבחר מצב תקשורת IP.

באפשרותך לציין את שם יציאת הרשת שיש ליצור של המדפסת שמותקנת. שם היציאה ישמש לזיהוי היציאה במאפייני המדפסת.

שמות יציאה חוקיים הם באורך 63 תווים לכל היותר ומורכבים מכל תו. שם היציאה חייב להיות ייחודי במחשב. שם היציאה מוצג במסך הסיכום ובעמוד הניסיון.

שם מארח IP
שם מארח IP הוא שם ידידותי (או קל לזיהוי) שניתן להשתמש בו במקום בכתובת ה- IP של המדפסת. ניתן להשתמש בו כאשר שרת שמות ברשת הוגדר למפות שמות מארחים אל כתובות ה- IP המתאימות. אם הוקצה שם מארח ותצורתו הוגדרה במדפסת (לדוגמה, printer1.marketing.com), ניתן לציין את שם המארח כדי לזהות את המדפסת.

שם מדפסת
יש להזין שם של המדפסת בהתאם לקווים המנחים המתאימים שמפורטים בנושא מוסכמות למתן שמות מדפסת. שם זה ישמש לזיהוי סמל המדפסת של מדפסת זו בתיקייה מדפסות במערכת. כדי להציג את התיקייה מדפסות, בשולחן העבודה של Windows לחץ על התחל, בחר הגדרות ולאחר מכן לחץ על מדפסות.

שם רשת
שם רשת הוא מחרוזת תווים אלפא-נומריים, תלוי-רישיות המספק בקרת גישה בסיסית לרשת אלחוטית. שם רשת מוכר גם כ"מזהה קבוצת שירות", או "SSID".

שער (Gateway)
שער הוא נתב או מחשב המחבר שתי רשתות שונות (לדוגמה, פרוטוקולי תקשורת, תבנית נתונים, מבנים, או ארכיטקטורות שונים). כשרשתות מחולקות לרשתות-משנה, לעיתים קרובות משתמשים בשערים לבידוד רשת-משנה אחת מאחרת כדי לשלוט בתעבורת הרשת.
ברשתות TCP/IP, התצורה של כל התקן ברשת מוגדרת לעיתים קרובות עם כתובת ה- IP של השער הקרוב ביותר. אם אין שערים ברשת, נעשה בדרך כלל שימוש בכתובת ה-IP של המחשב שלך.
לקבלת מידע נוסף ראה הוספת מדפסות ברשתות IP גדולות.
עבור רשת אלחוטית, ראה נקודת גישה אלחוטית (WAP).

שער ברירת מחדל
ראה שער.

שרת הדפסה HP Jetdirect
שרתי הדפסה HP Jetdirec הם אביזרי רשת של Hewlett-Packard המחברים מדפסות ישירות לרשת. חיבורים ישירים לרשת מאפשרים גמישות במיקום וביצועים מיטביים של המדפסת (או מכשיר All-in-One).
במדפסות HP המצוידות בחריצי קלט/פלט, נעשה שימוש בכרטיסי שרת הדפסה HP Jetdirect פנימיים. בחלק משרתי הדפסה HP Jetdirect חיצוניים ניתן להשתמש עם כמעט כל מדפסת המצוידת ביציאה מקבילית, בעוד שבשרתי הדפסה HP Jetdirect אחרים ניתן להשתמש עם כמעט כל מדפסת המצוידת ביציאת USB. שני סוגי שרתי ההדפסה תומכים בכל הטופולוגיות והפרוטוקולים הנפוצים של רשתות.

שרת שמות
שרת שמות הוא מחשב ברשת המשמש למיפוי (או פיענוח) שמות מארחים (שמות שרת הדפסה או מחשב) עם כתובות IP. שרת DNS (Domain Name System) ושרת WINS (Windows Internet Name Service) הם דוגמאות של שרתי שמות.
