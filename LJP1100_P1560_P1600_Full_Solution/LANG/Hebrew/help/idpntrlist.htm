מדפסות שנמצאו

מסך מדפסות שנמצאו מאפשר לבחור מדפסת שזוהתה ברשת המקומית או לקבל אפשרויות התקנה נוספות להתקנת מדפסות רשת, מדפסות USB או מדפסות המחוברות באופן מקבילי.

  • התקנת מדפסת רשת שזוהתה. בחר את המדפסת הרצויה מהרשימה של כל המדפסות (או ההתקנים הרב-תכליתיים) שזוהתה ברשת המקומית, ולחץ על הבא. לחץ על לחצן חיפוש חוזר כדי לבצע חיפוש אחר מדפסות ברשת.
  • עזור לי לחפש את המדפסת שלי. בחר באפשרות זו אם מדפסת הרשת הרצויה שלך אינה מופיעה או שברצונך להתקין מדפסת שתתחבר ישירות למחשב שלך באמצעות כבל USB או כבל מקבילי.  עבור מדפסות רשת, אפשרות זו תציע אפשרויות חיפוש נוספות, כגון חיפוש אחר מדפסת לפי כתובת IP. 

הערה: כדי להגדיר מדפסות שאינן נמצאות ברשת המקומית, ראה הוספת מדפסות ברשתות IP גדולות.

מדפסות ברשימה מזוהות באמצעות השדות הבאים. כתובת החומרה יכולה להיות הדרך הקלה ביותר לזיהוי המדפסת:

דגם מזהה את דגם המדפסת (לדוגמה, HP LaserJet 5000 Series). האפשרות לא מוכר מוצגת אם לא הייתה אפשרות לזהות את המודל, אם המדפסת כבויה או אם אין מדפסת מחוברת. בשרתי הדפסה חיצוניים של HP Jetdirect, הפרטים הבאים מוצגים כאשר מחוברת מדפסת אך לא ניתן לזהותה:
  • מקבילי: מדפסת Centronics/תוויין
  • USB (חד-כיווני): ציוד היקפי חד-כיווני
  • USB (דו-כיווני): ציוד היקפי דו-כיווני
כתובת חומרה (נקראת גם כתובת תחנה או כתובת MAC). מזהה באופן ייחודי התקן ברשת מקומית. עבור מדפסות HP ושרתי הדפסה חיצוניים של HP Jetdirect הכוללים יציאות מדפסת מרובות, ייתכן שיצורף מספר יציאה (יציאה 1, יציאה 2, יציאה 3). כתובת זו מפורטת בדף התצורה (ניסיון) של שרת ההדפסה.
כתובת IP (פרוטוקול אינטרנט) מזהה באופן ייחודי התקן ברשת TCP/IP ומוקצית על-ידי מנהל רשת. הן כתובת IPv4 והן כתובת IPv6 יוצגו ברשימה זו.  אם למדפסת יש כתובות מרובות, היא תופיע פעם אחת עבור כל כתובת, ותאפשר לך לבחור באיזה ממשק להשתמש. 

הערה: כאשר שרת ההדפסה של HP Jetdirect דולק ולא מוגדרת עבורו כתובת IP חוקית בתוך שתי דקות, כתובת IP אוטומטית (169.254.x.x) או כתובת IP המוגדרת כברירת מחדל (192.0.0.192) מוקצית באופן אוטומטי. HP ממליצה לשנות את כתובת ה-IP המשמשת כברירת מחדל. אם ישנן מדפסות מרובות עם אותה כתובת ברירת מחדל, בדוק את כתובת החומרה כדי לזהות את המדפסת הנכונה.
שם מארח של IP שם מארח של IP הוא שם ידידותי (או קל לזיהוי) שבו ניתן להשתמש במקום כתובת ה-IP של המדפסת. ניתן להשתמש בשם זה כאשר שם שרת ברשת הוגדר כדי למפות את שמות המארחים לכתובות ה-IP המתאימות. אם שם מארח הוקצה והוגדר במדפסת (לדוגמה, printer1.marketing.com), באפשרותך לציין את שם המארח כדי לזהות את המדפסת.

הערה: אם ערך של שדה נחתך וברצונך להציג את השדה כולו, שנה את רוחב השדה. כדי לשנות את רוחב השדה, מקם את הסמן בגבול של כותרת השדה ולאחר מכן לחץ וגרור.

לקבלת הוראות להדפסת דף תצורת רשת, עיין בתיעוד המצורף למדפסת.

אם מדפסת נבחרת מוגנת על-ידי שם קהילה של SNMP או על-ידי סיסמה של Jetdirect, תתבקש להזין את שם הקהילה או הסיסמה.
