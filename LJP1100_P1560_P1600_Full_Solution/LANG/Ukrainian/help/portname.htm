Ім'я порту

Примітка. Діалогове вікноІм'я порту відображається лише для операційних систем Windows NT, Windows 2000 і Windows XP. Це діалогове вікно з'являється лише тоді, якщо в діалоговому вікні Установити режим мережного зв'язку не встановлено прапорець у пункті Використовувати ім'я порту за умовчанням і, якщо вибрано режим IP-комунікації.

Ви можете вказати ім'я мережного порту, який потрібно створити під час установки принтера. Ім'я порту необхідно для його ідентифікації в параметрах принтера.

Довжина дійсніих імен порту становить до 63 символів різного характеру. Ім'я порту повинно бути унікальним. Ім'я порту відображено у вікні "Підсумок" і на пробній сторінці.
