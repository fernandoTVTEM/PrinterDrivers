Име на порт

Забележка: Диалоговият прозорец Port Name (Име на порт) се показва само в Windows NT, Windows 2000 и Windows XP. Този диалогов прозорец се показва само ако квадратчето Use default port name (Използвай името по подразбиране на порта) не е отметнато в диалоговия прозорец Set Network Communication Mode (Задаване на режима за комуникиране на мрежата) и ако е избран IP режимът за комуникиране.

Можете да укажете името на мрежовия порт да се създаде за инсталирания принтер. Името на порта се използва за идентифициране на порта в прозореца със свойства на принтера.

Валидните имена на портове са до 63 знака и съдържат всякакви знаци. Името на порта трябва да е уникално. Името на порта се показва в екрана с обобщение и в тестовата страница.
