I. INTRODUZIONE

SCOPO DI QUESTO DOCUMENTO

Questo documento fornisce istruzioni di installazione e altre informazioni specifiche che è opportuno conoscere prima di installare e utilizzare il sistema di stampa. Questo software della stampante HP LaserJet funziona con i seguenti sistemi operativi:

·         Windows XP (32 bit e 64 bit)

·         Windows Server 2003 (32 bit e 64 bit)

·         Windows Vista (32 bit e 64 bit)

·         Windows Server 2008 (32 bit e 64 bit)

 

Nota: Questo software HP LaserJet non supporta Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000 e Windows Server 2000.

Per istruzioni sull'installazione di altri sistemi, vedere le relative note di installazione, i file leggimi o altre informazioni sui prodotti per il sistema specifico.

 

Questo documento fornisce istruzioni di installazione, compatibilità e risoluzione dei problemi. Per ulteriori informazioni sulle funzioni della periferica e del driver, vedere la guida per l'utente della periferica e la guida in linea fornite in dotazione con ciascun componente software.

II. PROBLEMI NOTI E SOLUZIONI

Generali

 

·         Alcune schede USB 2.0 ad alta velocità vengono fornite con driver per controller host privi di firma. Se è stata acquistata una scheda aggiuntiva USB 2.0 e si dispone di un driver privo di firma, potrebbero verificarsi problemi di installazione del driver o problemi di comunicazione tra la periferica e il software HP. Si consiglia di utilizzare sempre un driver controller host certificato Microsoft WHQL con l'hardware USB.

·         I pacchetti di driver per la stampa "plug and play" e la scansione sono disponibili all'indirizzo:

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

·         Le funzionalità sui sistemi che soddisfano solo i requisiti minimi potrebbero essere limitate. L'uso di un numero elevato di immagini o video richiede che il sistema soddisfi i requisiti consigliati, se si desidera ottenere prestazioni e affidabilità accettabili.

·         Risoluzione dei problemi relativi all'impostazione e all'installazione

Se la periferica non viene riconosciuta:

  • Verificare che il cavo sia compatibile con USB 2.0 ad alta velocità.
  • Accertarsi che il cavo sia collegato in modo corretto. Rimuovere il cavo USB, controllare l'estremità e quindi verificarne la corrispondenza con lo schema della porta USB. Assicurarsi che non sia capovolto. Inserire un'estremità del cavo USB fino in fondo nella porta USB della periferica HP e collegare l'altra estremità al retro del computer.
  • Verificare che la periferica sia accesa. Le periferiche non vengono riconosciute se sono spente.
  • Verificare che la lunghezza del cavo USB non superi i 2 metri.
  • Assicurarsi che la porta USB sia abilitata. Alcuni PC vengono forniti con le porte USB disabilitate.

Aggiornamento di Windows

·         Si consiglia di disinstallare le periferiche HP prima di effettuare l'aggiornamento a una versione più recente di Windows.  Ad esempio, quando si passa da Windows XP a Windows Vista, disinstallare la periferica HP utilizzando il programma di disinstallazione, quindi effettuare l'aggiornamento al nuovo sistema operativo. Al termine dell'aggiornamento del sistema operativo, reinstallare la periferica HP utilizzando il software più recente disponibile all'indirizzo:

http://www.hp.com/support/LJP1100series 

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

III. ISTRUZIONI DI INSTALLAZIONE

In ambiente Windows, il software per l'impostazione o l'installazione presente nel CD-ROM rileva la versione e la lingua del sistema operativo in esecuzione.

INSTALLAZIONE DEL SOFTWARE IN WINDOWS con HP Smart install - Metodo di installazione CONSIGLIATO

NOTA: sono richiesti privilegi di amministratore per installare il sistema di stampa su un sistema operativo Windows XP o Vista.

1.       Inserire il cavo USB nel computer e nella stampante.

2.       Il programma HP Smart Install viene eseguito automaticamente. Se il programma HP Smart Install non si avvia entro trenta secondi, eseguire le operazioni da 3 a 5.

3.       Selezionare Esegui dal menu Start sulla barra delle applicazioni.

4.       Fare clic su Sfoglia. Sarà disponibile un nuovo CD-ROM denominato HP Smart Install. Passare a questa unità.

5.       Selezionare il file SISetup.EXE, fare clic su Apri, quindi su OK.

6.       Seguire le istruzioni a video.

 

Installazione del SOFTWARE con HP Smart install sulla rete

1.       Da un browser Web, immettere l'indirizzo IP del dispositivo nella barra dell'indirizzo.

2.       Fare clic sulla scheda HP Smart Install.

3.       Fare clic sul pulsante Download.

4.       Seguire le istruzioni a video.

 

INSTALLAZIONE DI SOFTWARE IN WINDOWS con CD-ROM

NOTA: sono richiesti privilegi di amministratore per installare il sistema di stampa su un sistema operativo Windows XP o Vista.

Installazione del sistema di stampa da un CD-ROM

1.       Non inserire il cavo USB/parallelo nel computer.

2.       Chiudere tutti i programmi.

3.       Inserire il CD-ROM.

4.       Se il programma di installazione del sistema di stampa HP non si avvia entro trenta secondi, eseguire le operazioni da 5 a 7.

5.       Selezionare Esegui dal menu Start sulla barra delle applicazioni.

6.       Selezionare la directory principale nell'unità CD-ROM.

7.       Selezionare il file Setup.EXE, fare clic su Apri, quindi su OK.

8.       Seguire le istruzioni a video.

 

Ulteriori istruzioni per l'installazione della stampante serie HP LaserJet P1100w.

La stampante serie HP LaserJet Professional P1100w supporta Wi-Fi 802.11 b/g.  Prima di iniziare l'installazione della stampante, assicurarsi che il router senza fili o il punto di accesso sia configurato correttamente.  Se necessario, tenere a disposizione le informazioni di configurazione della rete senza fili.

 

Disabilitazione di HP Smart Install per utilizzare altri driver oltre a quelli basati su host

Per utilizzare un driver su USB diverso dal driver standard basato su host incluso con HP Smart Install e il CD (ad esempio, PCL5E o XPS), è prima necessario disabilitare HP Smart Install sul computer o sulla stampante.

1.       Inserire il cavo USB nel computer e nella stampante.

2.       Il programma HP Smart Install viene eseguito automaticamente. Chiudere il programma senza installare il driver basato su Host.

3.       Inserire il CD-ROM.

4.       Il programma di installazione viene eseguito automaticamente. Chiudere il programma senza installare il driver basato su Host. 

5.       Selezionare la directory UTIL nell'unità CD-ROM.

6.       Selezionare il file HPSmartInstallConfig.EXE, fare clic su Apri, quindi su OK.

7.       Nell'utility sono disponibili due opzioni. La prima disattiva HP Smart Install sul computer, che viene sempre eseguito se la stampante è collegata ad altri computer. L'altra opzione disattiva HP Smart Install sulla stampante. Se si seleziona questa opzione, la stampante non eseguirà HP Smart Install su nessun computer a meno che non venga riabilitato tramite questa utility.

 

 

Disabilitazione di HP Smart Install sui prodotti di rete

HP Smart Install può essere disabilitato sulle stampanti di rete se richiesto dall'amministratore.

1.       Da un browser Web, immettere l'indirizzo IP del dispositivo nella barra dell'indirizzo.

2.       Selezionare la scheda Impostazioni.

3.       Fare clic sul menu Impostazioni di stampa.

4.       Per disabilitare HP Smart Install su USB, impostare l'opzione Abilita HP Smart Install USB su No.

5.       Per disabilitare HP Smart Install sulla rete, impostare l'opzione Mostra opzioni di download di HP Smart Install su No.

Installazione mediante Aggiungi stampante

La stampante HP LaserJet può essere installata anche mediante la funzione di Microsoft Aggiungi stampante dalla cartella Stampanti. Si tratta del programma di installazione guidata della stampante. Le istruzioni possono variare leggermente da una versione di Windows all'altra. Per installare tramite APW, è prima necessario disabilitare HP Smart Install utilizzando le istruzioni sopra riportate.

 

Installazione di WinXP APW 

1.         Verificare che la stampante serie HP LaserJet P1100/P1560/P1600 non sia collegata alla presa e che non sia connessa al computer.

2.         Dal menu Start, scegliere Pannello di controllo e fare clic su Stampanti e fax.

3.         Fare clic sull'opzione Aggiungi stampante, quindi fare clic su Avanti nella finestra "Installazione guidata stampante".

4.         Seguire le indicazioni sullo schermo.

5.         Quando viene richiesto il tipo di stampante (locale o di rete), selezionare Stampante locale (opzione predefinita) Deselezionare la casella di controllo Rileva e installa Plug and Play automaticamente.  Fare clic su Avanti.

6a.         Sel'installazione viene eseguita attraverso l'USB senza aver prima installato la stampante serie HP LaserJet P1100/P1560/P1600, la porta USB non sarà ancora installata. Si tratta di una situazione normale. Selezionare LPT1 e fare clic su Avanti.

6b.         Sel'installazione viene eseguita attraverso USB e la stampante serie HP LaserJet P1100/P1560/P1600 è stata precedentemente installata e collegata, la porta USB sarà installata. Selezionare la porta USB e fare clic su Avanti

Nota:    Verificare che la stampante serie HP LaserJet P1100/P1560/P1600 sia collegata alla stessa porta USB alla quale era stata collegata in precedenza.

7.         Selezionare "Disco driver" e quindi specificare la posizione del driver della serie HP LaserJet P1100/P1560/P1600 da installare.

8.         Se sono presenti diverse opzioni, selezionare un modello di stampante e fare clic su Avanti.

9.         Seguire le indicazioni sullo schermo.

10.        Quando viene richiesto il collegamento della periferica, inserire la spina e collegare la stampante serie HP LaserJet P1100/P1560/P1600 al computer.

 

Nota:    Quando viene richiesto il collegamento della periferica, inserire la spina e collegare la stampante serie HP LaserJet P1100/P1560/P1600 al computer.

 

11.        Seguire le istruzioni visualizzate sullo schermo fino al completamento dell'installazione.

 

Al termine dell'installazione, possono essere state create due copie del driver della stampante. In questo caso. eliminare la prima copia creata che solitamente avrà lo stesso nome del modello della stampante. Per verificare quanti driver sono stati creati ed eliminarne il primo, attenersi alla seguente procedura

 

1.         Dal menu Start, scegliere Impostazioni e fare clic su Stampanti.

2.         Se sono presenti sia il modello serie " HP LaserJet P1100/P1560/P1600" che il modello serie "HP LaserJet P1100/P1560/P1600 (Copia 2)", non eliminare il driver con la dicitura "Copia 2" nel nome. Eliminare il primo driver creato con il nome serie "HP LaserJet P1100/P1560/P1600"

 

A questo punto è possibile eseguire la stampa con il driver della serie "HP LaserJet P1100/P1560/P1600 (Copia 2)".

PROGRAMMA DI DISINSTALLAZIONE

Dopo avere effettuato l'installazione del sistema di stampa, utilizzare il programma di disinstallazione per rimuovere tutti i componenti del sistema di stampa HP.

Avvio del programma di disinstallazione per la stampante serie HP LaserJet Professional P1100

1.       Dal menu Start, nel gruppo dei programmi della serie HP LaserJet Professional P1100, fare clic sull'icona del programma di disinstallazione.

2.       Verrà visualizzata la procedura guidata per la rimozione di tutti i componenti del sistema di stampa.

Avvio del programma di disinstallazione per la stampante serie HP LaserJet Professional P1560/P1600

1.       Dal menu Start, nel gruppo dei programmi della serie HP LaserJet Professional P1560-P1600, fare clic sull'icona del programma di disinstallazione.

2.       Verrà visualizzata la procedura guidata per la rimozione di tutti i componenti del sistema di stampa.
