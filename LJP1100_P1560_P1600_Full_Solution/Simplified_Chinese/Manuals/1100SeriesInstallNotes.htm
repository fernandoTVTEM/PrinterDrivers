I. 简介

本文档的用途

本文档提供安装和使用打印系统前应了解的特定安装说明和其它信息。 此 HP LaserJet 软件可用于以下操作系统：

・         Windows XP（32 位和 64 位）

・         Windows Server 2003（32 位和 64 位）

・         Windows Vista（32 位和 64 位）

・         Windows Server 2008（32 位和 64 位）

 

注： 此 HP LaserJet 软件不支持 Windows 95、Windows 98、Windows ME、Windows NT、Windows 2000 和 Windows Server 2000。

有关安装其它系统的说明，请参阅该特定系统的相关安装说明、自述文件或其它产品信息。

 

此文件提供了详细的安装、兼容性和故障排除信息。 有关设备功能和驱动程序功能的其它信息，请参阅每个软件组件中附带的设备用户指南和联机帮助。

此软件的适用人群

所有用户都需要安装此打印系统软件才能使用 HP LaserJet Professional P1100/P1560/P1600 打印机系列。

II. 已知问题和解决方法

常规

 

・         某些高速 USB 2.0 卡附带提供了未签名的主机控制器驱动程序。 如果您购买了 USB 2.0 插件卡并带有未签名的驱动程序，则可能碰到设备和 HP 软件之间的驱动程序安装问题或通信问题。 建议您始终对 USB 硬件使用 Microsoft WHQL 认证的主机控制器驱动程序。

・         以下网址中提供了即插即用打印驱动程序包：

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

・         在仅满足最低系统要求的系统上，功能可能会非常有限。 要使用很多图像和视频，系统必须满足建议的系统需求以获得可接受的性能和可靠性。

・         设置和安装故障排除

如果无法识别您的设备：

  • 请检查电缆是否符合 USB 2.0 高速的标准。
  • 确保电缆连接正确。 拔出 USB 电缆，查看电缆末端，然后将其插入 USB 端口上匹配的形状中。 确保电缆没有上下颠倒。 将 USB 电缆的一端完全插入到 HP 设备上的 USB 端口，另一端插入计算机背面。
  • 检查外围设备电源是否打开。 如果电源关闭，则无法识别外围设备。
  • 确保 USB 电缆长度不超过 2 米或 6.5 英尺。
  • 检查是否启用了 USB 端口。 某些 PC 在出厂时禁用了 USB 端口。

Windows 升级

・         HP 建议您先卸载 HP 设备，然后再升级到较新的 Windows 支持版本。 例如，从 Windows XP 升级到 Windows Vista 时，先使用卸载程序或“添加/删除程序”卸载 HP 设备，然后再升级到新操作系统。 在完成操作系统升级后，请使用以下网址中提供的最新软件重新安装 HP 设备：

III. 安装说明

在 Windows 环境中运行时，光盘设置或安装软件将检测正在运行的操作系统的版本和语言。

使用 HP 智能安装在 Windows 上安装软件 C 建议的安装方法

注： 要在 Windows XP 或 Windows Vista 操作系统上安装打印系统，您必须具有管理员权限。

1.       将 USB 电缆插入计算机和打印机中。

2.       将会自动运行 HP 智能安装程序。 如果 HP 智能安装程序在 30 秒后还没有启动，请完成步骤 3 至 5。

3.       从任务栏的开始菜单中选择运行。

4.       单击“浏览”。 您可以找到一个名为 HP 智能安装的新光驱。 浏览至该驱动器。

5.       选择 SISetup.EXE 文件，单击打开，然后单击确定。

6.       按照提示进行操作。

 

使用 HP 智能安装通过网络安装软件

1.       在 Web 浏览器的地址栏中，键入设备的 IP 地址。

2.       单击 HP 智能安装标签。

3.       单击下载按钮。

4.       按照提示进行操作。

 

通过光盘在 Windows 上安装软件

注： 要在 Windows XP 或 Windows Vista 操作系统上安装打印系统，您必须具有管理员权限。

通过光盘安装打印系统

1.       请勿将 USB/并行电缆插入计算机系统中。

2.       关闭所有程序。

3.       插入光盘。

4.       如果 HP 打印系统安装程序在 30 秒后还没有启动，请完成步骤 5 至 7。

5.       从任务栏的开始菜单中选择运行。

6.       浏览至光驱的根目录。

7.       选择 Setup.EXE 文件，单击打开，然后单击确定。

8.       按照提示进行操作。

 

安装 HP Laserjet P1100w 打印机系列的附加说明

HP LaserJet Professional P1100w 系列打印机支持 Wi-Fi 802.11 b/g。在开始安装打印机之前，请确保成功设置了无线路由器或接入点。请准备好无线网络配置信息以备不时之需。

 

禁用 HP 智能安装以使用非基于主机的驱动程序

如果要通过 USB 使用的驱动程序不是 HP 智能安装和光盘中包含的基于主机的标准驱动程序（如 PCL5E 或 XPS）， 您需要先在计算机或打印机上禁用 HP 智能安装。

1.       将 USB 电缆插入计算机和打印机中。

2.       将会自动运行 HP 智能安装程序。 退出该程序，而不安装基于主机的驱动程序。

3.       插入光盘。

4.       将会自动运行安装程序。 退出该程序，而不安装基于主机的驱动程序。 

5.       浏览至光盘上的 UTIL 目录。

6.       选择 HPSmartInstallConfig.EXE 文件，单击打开，然后单击确定。

7.       此实用程序中有两个选项。 一个选项是在该计算机上禁用 HP 智能安装；如果打印机连接到其它计算机上，仍会运行该安装。 另一个选项是在打印机上禁用 HP 智能安装。 如果选择此选项，除非使用此实用程序重新启用 HP 智能安装，否则，打印机不会在任何计算机上运行该安装。

 

 

在网络产品上禁用 HP 智能安装

如果需要，管理员可以在网络打印机上禁用 HP 智能安装。

1.       在 Web 浏览器的地址栏中，键入设备的 IP 地址。

2.       单击“设置”标签。

3.       单击“打印设置”菜单。

4.       要禁用通过 USB 的 HP 智能安装，请将启用 USB HP 智能安装选项设置为否。

5.       要禁用通过网络的 HP 智能安装，请将显示 HP 智能安装下载选项设置为否。

通过“添加打印机”进行安装

还可以通过 Microsoft 的“打印机”文件夹中的“添加打印机”功能来安装 HP LaserJet 打印机。 这是通过“添加打印机向导”（即 APW）来完成安装的。 APW 说明因 Windows 版本不同而略有差异。 要通过 APW 进行安装，您需要先按照上面的说明禁用 HP 智能安装。

 

WinXP APW 安装 

1.         确保 HP LaserJet P1100/P1560/P1600 打印机系列尚未插入电源线，并且未连接到计算机上。

2.         在“开始”菜单中，指向“控制面板”，然后单击“打印机和传真”。

3.         单击“添加打印机”，然后在“欢迎使用添加打印机向导”屏幕中单击“下一步”。

4.         按照屏幕上的说明进行操作。

5.         当系统询问产品是本地打印机还是网络打印机时，请选择“本地打印机”（这是默认选择）。 取消选中“自动检测并安装即插即用打印机”复选框。  然后按“下一步”。

6a.         如果通过 USB 进行安装，并且以前没有安装过 HP LaserJet P1100/P1560/P1600 打印机系列，则不存在已安装的 USB 端口。 这是正常情况。 请选择“LPT1:”， 然后单击“下一步”。

6b.         如果通过 USB 进行安装，并且以前安装并连接了 HP LaserJet P1100/P1560/P1600 打印机系列，则存在已安装的 USB 端口。 选择此 USB 端口，然后单击“下一步”

注：    确保将 HP LaserJet P1100/P1560/P1600 打印机系列连接到以前所连接的同一物理 USB 端口。

7.         选择“从磁盘安装”，然后指定要安装的 HP LaserJet P1100/P1560/P1600 系列驱动程序的位置。

8.         如果提供多个选项，请选择您的打印机型号，然后单击“下一步”。

9.         按照屏幕上的说明进行操作。

10.        如果提示您连接打印机，请插入 HP LaserJet P1100/P1560/P1600 打印机系列电源线并将其连接到计算机上。

 

注：    如果此时未提示您连接 HP LaserJet P1100/P1560/P1600 打印机系列，应在安装完成时插入打印机电源线并将其连接到计算机上。

 

11.        按照屏幕上的说明进行操作，直到安装完成为止。

 

安装完成时，可能会创建两个打印机驱动程序。 如果出现这种情况，应删除所创建的第一个副本。 通常，其名称与打印机的型号相同。 要检查所创建的驱动程序数目并删除第一个副本，请按照以下步骤操作

 

1.         在“开始”菜单中，指向“设置”，然后单击“打印机”。

2.         如果同时包含“HP LaserJet P1100/P1560/P1600 系列”和“HP LaserJet P1100/P1560/P1600 系列（副本 2）”，请勿删除名称包含“副本 2”的打印机， 而是删除创建的第一个名为“HP LaserJet P1100/P1560/P1600 系列”的驱动程序。

 

此时，您已经准备就绪，可以使用“HP LaserJet P1100/P1560/P1600 系列（副本 2）”驱动程序进行打印了。

使用 PCL5e 进行打印

HP LaserJet Professional P1600 打印机系列上仅支持 PCL5e。

以下网址中提供了 PCL5e 驱动程序：

http://www.hp.com/support/LJP1600series

 

使用 XPS 进行打印

XPS 驱动程序位于随打印机提供的软件光盘上，也可以从以下网址中获得该驱动程序：

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

使用“添加打印机”向导安装 XPS 驱动程序。 

 

卸载程序

在安装打印系统后，要删除所有 HP 打印系统组件，请使用卸载程序。

启动 HP LaserJet Professional P1100 打印机系列的卸载程序

1.       在开始菜单中，单击 HP LaserJet Professional P1100 系列程序组中的卸载图标。

2.       打印系统将指导您完成删除所有打印系统组件的过程。

启动 HP LaserJet Professional P1560/P1600 打印机系列的卸载程序

1.       在开始菜单中，单击 HP LaserJet Professional P1560-P1600 系列程序组中的卸载图标。

2.       打印系统将指导您完成删除所有打印系统组件的过程。
