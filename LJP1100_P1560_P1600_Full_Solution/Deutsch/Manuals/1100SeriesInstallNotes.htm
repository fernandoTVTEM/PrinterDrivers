II. BEKANNTE PROBLEME UND LÖSUNGEN

Allgemein

 

·         Einige Highspeed-USB-2.0-Karten werden mit unsignierten Host-Controller-Treibern ausgeliefert. Wenn Sie eine USB 2.0 Add-in-Karte erworben haben und einen nicht signierten Treiber verwenden, treten unter Umständen Probleme bei der Treiberinstallation oder bei der Kommunikation zwischen dem Gerät und der HP Software auf. Es wird empfohlen, mit der USB-Hardware immer einen Microsoft WHQL Certified-Hostcontrollertreiber zu verwenden.

·         Plug-and-Play Druckertreiber stehen im Internet unter folgender URL zur Verfügung:

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

·         Systeme, die lediglich die Mindestsystemanforderungen erfüllen, sind möglicherweise nur begrenzt funktionstüchtig. Bei der Verwendung vieler Bilder oder Videos sollten die empfohlenen Systemanforderungen erfüllt sein, um eine zufrieden stellende Leistung und Zuverlässigkeit des Systems zu gewährleisten.

·         Fehlerbehebung bei der Konfiguration und Installation

Das Gerät wird nicht erkannt:

  • Stellen Sie sicher, dass das Kabel mit USB 2.0 High-Speed-Anschlüssen kompatibel ist.
  • Vergewissern Sie sich, dass das Kabel ordnungsgemäß angeschlossen ist. Ziehen Sie das USB-Kabel aus der Buchse, sehen Sie sich an, wie das Kabelende in den USB-Anschluss passt, und stecken Sie es entsprechend an. Vergewissern Sie sich, dass nicht die falsche Seite nach oben zeigt. Stecken Sie ein Ende des USB-Kabels fest in den USB-Anschluss des HP Geräts und das andere Ende in den entsprechenden Anschluss an der Rückseite des Computers.
  • Prüfen Sie, ob das Peripheriegerät eingeschaltet ist. Ausgeschaltete Peripheriegeräte werden nicht erkannt.
  • Das USB-Kabel darf maximal 2 Meter lang sein.
  • Überprüfen Sie, ob der USB-Anschluss aktiviert ist. Manche Computer werden mit deaktivierten USB-Anschlüssen ausgeliefert.

III. INSTALLATIONSANLEITUNGEN

In einer Windows Umgebung erkennt die CD-ROM-Setup- oder Installationssoftware die Version und die Sprache des verwendeten Betriebssystems.

INSTALLIEREN VON SOFTWARE AUF WINDOWS MIT HP SMART INSTALL – EMPFOHLENE INSTALLATION

HINWEIS: Zur Installation des Drucksystems unter Windows XP oder Windows Vista sind Administratorrechte erforderlich.

1.       Verbinden Sie den Computer und den Drucker mit einem USB-Kabel.

2.       HP Smart Install sollte jetzt automatisch ausgeführt werden. Wenn HP Smart Install nach 30 Sekunden nicht gestartet ist, führen Sie die Schritte 3 bis 5 durch.

3.       Wählen Sie in der Taskleiste im Menü Start die Option Ausführen.

4.       Klicken Sie auf Durchsuchen. Navigieren Sie zum neuen CD-ROM-Laufwerk HP Smart Install.

5.       Wählen Sie die Datei SISetup.EXE aus, klicken Sie auf Öffnen, und klicken Sie dann auf OK.

6.       Folgen Sie den Anleitungen.

 

Netzwerkgestütztes Installieren von Software mit HP Smart Install

1.       Geben Sie in die Adresszeile eines Web-Browsers die IP-Adresse des Geräts ein.

2.       Klicken Sie auf die Registerkarte HP Smart Install.

3.       Klicken Sie auf die Schaltfläche Download (Herunterladen).

4.       Folgen Sie den Anleitungen.

 

INSTALLIEREN VON WINDOWS SOFTWARE VON CD-ROM

HINWEIS: Zur Installation des Drucksystems unter Windows XP oder Windows Vista sind Administratorrechte erforderlich.

Installation des Drucksystems von einer CD-ROM

1.       Schließen Sie das USB-Kabel/das Kabel für den Parallelanschluss nicht an den Computer an.

2.       Schließen Sie alle Programme.

3.       Legen Sie die CD-ROM ein.

4.       Wenn das HP Drucksystem-Installationsprogramm nach 30 Sekunden nicht gestartet ist, führen Sie die Schritte 5 bis 7 durch.

5.       Wählen Sie in der Taskleiste im Menü Start die Option Ausführen.

6.       Durchsuchen Sie das Root-Verzeichnis auf dem CD-ROM-Laufwerk.

7.       Wählen Sie die Datei Setup.EXE aus, klicken Sie auf Öffnen, und klicken Sie dann auf OK.

8.       Folgen Sie den Anleitungen.

 

Zusätzliche Anleitungen für die Installation der HP Laserjet P1100w Serie

Die HP LaserJet Professional P1100w Serie unterstützt die Funknetzstandards 802.11b und 802.11g. Stellen Sie sicher, dass Ihr Wireless-Router bzw. Wireless-Zugriffspunkt richtig eingerichtet ist. Beginnen Sie erst dann mit der Installation des Druckers. Bewahren Sie die Konfigurationsangaben Ihres Wireless-Netzwerks griffbereit auf.

 

HP Smart Install deaktivieren, um anderen Treiber als Host-basierten Treiber zu verwenden

In HP Smart Install und auf der mitgelieferten CD befinden sich standardmäßige Host-basierte Treiber (z. B. PCL5E oder XPS). Wenn Sie über USB einen anderen Treiber verwenden möchten, müssen Sie HP Smart Install auf dem Computer bzw. Drucker deaktivieren.

1.       Verbinden Sie den Computer und den Drucker mit dem USB-Kabel.

2.       HP Smart Install sollte jetzt automatisch ausgeführt werden. Installieren Sie den Host-basierten Treiber nicht, und beenden Sie das Programm.

3.       Legen Sie die CD-ROM ein.

4.       Das Installationsprogramm sollte jetzt automatisch ausgeführt werden. Installieren Sie den Host-basierten Treiber nicht, und beenden Sie das Programm. 

5.       Navigieren Sie zum Verzeichnis UTIL auf der CD.

6.       Wählen Sie die Datei HPSmartInstallConfig.EXE aus, klicken Sie auf Öffnen, und klicken Sie dann auf OK.

7.       In diesem Dienstprogramm stehen zwei Optionen zur Auswahl. Mit der einen deaktivieren Sie HP Smart Install auf dem Computer. Das Programm wird weiterhin ausgeführt, wenn der Drucker an andere Computer angeschlossen ist. Mit der anderen deaktivieren Sie HP Smart Install auf dem Drucker. Wenn Sie diese Option auswählen, führt der Drucker HP Smart Install auf keinem Computer aus. Diese Einstellung kann nur durch Wiederaktivierung mit diesem Dienstprogramm aufgehoben werden.

 

 

HP Smart Install für Netzwerkprodukte deaktivieren

Der Administrator kann HP Smart Install auf ausgesuchten Netzwerkdruckern deaktivieren.

1.       Geben Sie in die Adresszeile eines Web-Browsers die IP-Adresse des Geräts ein.

2.       Klicken Sie auf die Registerkarte Einstellungen.

3.       Klicken Sie auf das Menü Druckeinstellungen.

4.       Um die USB-gestützte Ausführung von HP Smart Install zu deaktivieren, setzen Sie die Option Enable USB HP Smart Install (HP Smart Install über USB aktivieren) auf Nein.

5.       Um die netzwerkgestützte Ausführung von HP Smart Install zu deaktivieren, setzen Sie die Option Show HP Smart Install download (Herunterladen von HP Smart Install zeigen) auf Nein.

Installieren über „Drucker hinzufügen“

Der HP LaserJet kann auch über die Funktion "Drucker hinzufügen" des Ordners "Drucker" von Microsoft installiert werden. Die Funktion wird auch Druckerinstallationsassistent genannt. Die Anleitungen des Druckerinstallationsassistenten unterscheiden sich je nach Windows Version. Um den Drucker mit Hilfe des Druckerinstallationsassistenten installieren zu können, müssen Sie zuerst HP Smart Install deaktivieren (siehe Anleitung oben).

 

Installation mit dem Druckerinstallationsassistenten unter WinXP 

1.         Der Drucker der HP LaserJet P1100/P1560/P1600 Serie darf weder an die Steckdose noch an den Computer angeschlossen sein.

2.         Zeigen Sie im Menü "Start" auf "Systemsteuerung", und klicken Sie auf "Drucker und Faxgeräte".

3.         Klicken Sie auf "Drucker hinzufügen", und klicken Sie anschließend auf der Seite "Willkommen" des Druckerinstallationsassistenten auf "Weiter".

4.         Folgen Sie den Anleitungen auf dem Bildschirm.

5.         Wenn Sie gefragt werden, ob es sich bei dem Produkt um einen lokalen Drucker oder einen Netzwerkdrucker handelt, wählen Sie "Lokaler Drucker" (Standardauswahl). Deaktivieren Sie das Kontrollkästchen Plug & Play-Drucker automatisch ermitteln und installieren. Klicken Sie auf "Weiter".

6a.         Wenn die Installation über USB erfolgt und der Drucker der HP LaserJet P1100/P1560/P1600 Serie zuvor noch nicht installiert war, ist noch kein USB-Anschluss installiert. Das ist normal. Wählen Sie LPT1: aus, und klicken Sie auf Weiter.

6b.         Wenn die Installation über USB erfolgt und der Drucker der HP LaserJet P1100/P1560/P1600 Serie zuvor installiert und angeschlossen wurde, ist bereits ein USB-Anschluss installiert. Wählen Sie diesen USB-Anschluss aus, und klicken Sie auf Weiter.

Hinweis:    Stellen Sie sicher, dass Sie den Drucker der HP LaserJet P1100/P1560/P1600 Serie an denselben physischen USB-Anschluss wie zuvor anschließen.

7.         Wählen Sie Datenträger, und geben Sie an, wo der Treiber der HP LaserJet P1100/P1560/P1600 Serie installiert werden soll.

8.         Gibt es mehrere Auswahlmöglichkeiten, so wählen Sie Ihr Druckermodell und klicken auf Weiter.

9.         Folgen Sie den Anleitungen auf dem Bildschirm.

10.        Wenn Sie aufgefordert werden, den Drucker anzuschließen, schließen Sie den Drucker der HP LaserJet P1100/P1560/P1600 Serie an die Steckdose und den Computer an.

 

Hinweis:    Wenn Sie nicht zum Anschließen des Druckers der HP LaserJet P1100/P1560/P1600 Serie aufgefordert werden, schließen Sie den Drucker an die Stromversorgung an und zum Abschluss der Installation an Ihren Computer.

 

11.        Folgen Sie den Anleitungen auf dem Bildschirm, bis die Installation abgeschlossen ist.

 

Wenn die Installation abgeschlossen ist, haben Sie möglicherweise zwei Kopien des Druckertreibers. Wenn dies der Fall ist, löschen Sie bitte den zuerst erstellten Treiber. Er hat in der Regel denselben Namen wie Ihr Druckermodell. Befolgen Sie die nachfolgende Anleitung, um zu überprüfen, wie viele Treiber erstellt wurden und um den ersten zu löschen.

 

1.         Zeigen Sie im Menü "Start" auf "Einstellungen", und klicken Sie auf "Drucker".

2.         Wenn Sie sowohl über "HP LaserJet P1100/P1560/P1600 Serie" als auch "HP LaserJet P1100/P1560/P1600 Serie (Kopie 2)" verfügen, löschen Sie nicht den "Kopie 2"-Treiber. Löschen Sie stattdessen den ersten Treiber mit dem Namen "HP LaserJet P1100/P1560/P1600 Serie".

 

Jetzt können Sie mit dem Treiber "HP LaserJet P1100/P1560/P1600 Serie (Kopie 2)" drucken.
