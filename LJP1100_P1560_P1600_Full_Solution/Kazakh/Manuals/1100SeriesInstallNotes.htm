I. КІРІСПЕ

БҰЛ ҚҰЖАТТЫҢ МАҚСАТЫ

Бұл құжатта арнайы орнату бойынша нұсқаулар мен басып шығару жүйесін орнатудың және пайдаланудың алдында сіз білуге тиісті басқа ақпарат берілген. HP LaserJet бағдарламалық құралы төмендегі операциялық жүйелермен жұмыс істейді:

·         Windows XP (32 биттік және 64 биттік)

·         Windows Server 2003 (32 биттік және 64 биттік)

·         Windows Vista (32 биттік және 64 биттік)

·         Windows Server 2003 (32 биттік және 64 биттік)

 

Ескертпе: Бұл HP LaserJet бағдарламалық құралы Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000 және Windows Server 2000 жүйелерін қолдамайды.

Басқа жүйелерді орнатуға қатысты нұсқауларды байланысты орнату туралы ескертулерінен, оқып шығуға арналған файлдардан немесе сол жүйеге қатысты өнім ақпаратынан қараңыз.

 

Бұл файлда арнайы орнату, үйлесімділік және ақаулықтарды жою туралы ақпарат берілген. Құрылғы мүмкіндіктері мен драйвер мүмкіндіктері туралы қосымша ақпарат алу үшін құрылғының пайдаланушы нұсқаулығын және әрбір бағдарламалық құрал компонентімен берілетін электрондық анықтаманы қараңыз.

БҰЛ БАҒДАРЛАМАЛЫҚ ҚҰРАЛ КІМГЕ ҚАЖЕТ?

Барлық пайдаланушыларға HP LaserJet Professional P1100/P1560/P1600 басып шығарғыш серияларын қолдану үшін осы басып шығару жүйесінің бағдарламалық құралы қажет болады.

БАСПА ЖҮЙЕСІНІҢ КОМПОНЕНТТЕРІНЕ ЖАЛПЫ ШОЛУ

CD-ROM дискісінде IBM үйлесімді компьютер жүйелеріне және Apple Macintosh (және үйлесімді) жүйелеріне арналған бағдарламалық құрал бар. Белгілі бір драйвер немесе бағдарламалық құрал туралы қосымша ақпарат алу үшін оның байланысты электрондық анықтамасын қараңыз.

Дискіден орнатуға арналған соңғы пайдаланушы бағдарламалық құрал компоненттері

·         HP LaserJet Professional P1100/P1560/P1600 басып шығарғыш сериялары басты компьютерге негізделген баспа драйвері

 

Осы құрылғыға арналған кейбір драйверлер интернеттен жүктеуге қолданылады. Бұл драйверлер:

 

·         Windows (тек қана P1600 басып шығарғыш сериялары үшін) жүйесіне арналған PCL5 әмбебап баспа драйвері

·         HP LaserJet Professional P1100/P1560/P1600 басып шығарғыш сериялары XPS баспа драйвері

II. БЕЛГІЛІ МӘСЕЛЕЛЕР ЖӘНЕ ЖҰМЫС ҮРДІСТЕРІ

Жалпы

 

·         Белгілі бір жоғары жылдамдықты USB 2.0 карталары қол қойылмаған басты бақылау драйверлерімен бірге беріледі. Егер сіз USB 2.0 бағдарламалық картасын сатып алып, қол қойылмаған драйвер болса, сізге драйверді орнату мәселелері немесе құрылғы мен HP бағдарламалық құралы арасындағы байланыс мәселелері кездесуі мүмкін. Сіздің әрдайым Microsoft WHQL сертификатталған басты бақылау драйверін USB аппараттық құралымен пайдалануыңыз ұсынылады.

·         Қосу және ойнату баспа драйверінің пакеттерін интернетте төмендегі URL мекенжайынан алуға болады:

·         Ең төменгі жүйе талаптарына ғана сәйкес келетін жүйлердегі функциялар шектеулі болуы мүмкін. Көптеген кескіндерді немесе бейнелерді пайдалану үшін рұқсат етілетін жұмысқа және сенімділікке қойылатын жүйелік талаптарға сай жүйе қажет.

·         Ақаулықтарды жою және орнату

Егер құрылғы танылмаса:

  • Кабельдің USB 2.0 жоғары жылдамдығына сәйкес келетінін тексеріңіз.
  • Кабельдің дұрыс қосылғанын тексеріңіз. USB кабелін шығарып, кабельдің ұшын қараңыз, содан кейін оны USB портындағы үлгіге сәйкестендіріңіз. Оның жоғарғы жағының төмен қарамағанын тексеріңіз. USB кабелінің бір ұшын HP құрылғысындағы USB портына, екінші жағын компьютердің артына толығымен кіргізіңіз.
  • Перифериялық құрылғының қосылғанын тексеріңіз. Өшірулі болған жағдайда перифериялық құрылғылар танылмайды.
  • USB кабелінің ұзындығының 2 метрден немесе 6,5 футтан аспайтынына көз жеткізіңіз.
  • USB портының қосылғанын тексеріңіз. Кейбір компьютерлер USB порттары ажыратылған күйде келеді.
