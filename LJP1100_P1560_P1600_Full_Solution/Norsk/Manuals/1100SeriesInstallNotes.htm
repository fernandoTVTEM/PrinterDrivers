I. INNLEDNING

FORMÅLET MED DETTE DOKUMENTET

Dette dokumentet inneholder spesifikke installeringsinstruksjoner og annen informasjon du bør kjenne til før du installerer og bruker utskriftssystemet. Programvaren til denne HP LaserJet-en fungerer med følgende operativsystemer:

·         Windows XP (32-bit og 64-bit)

·         Windows Server 2003 (32-bit og 64-bit)

·         Windows Vista (32-bit og 64-bit)

·         Windows Server 2008 (32-bit og 64-bit)

 

Merk: Denne HP LaserJet-programvaren støtter ikke Windows 95, Windows 98, Windows ME, Windows NT, windows 2000 og Windows Server 2000.

Du finner instruksjoner om installering på andre systemer i tilhørende installeringsmerknader, viktig-filer eller annen produktinformasjon for det bestemte systemet.

 

Denne filen inneholder spesifikk informasjon om installering, kompatibilitet og feilsøking. Hvis du ønsker mer informasjon om enhets- og driverfunksjoner, kan du lese brukerhåndbøkene og den elektroniske hjelpen som følger med hver programvarekomponent.

II. KJENTE PROBLEMER OG LØSNINGER

Generelt

 

·         Enkelte høyhastighets USB 2.0-kort leveres med usignerte drivere for vertskontroller. Hvis du har kjøpt et USB 2.0-tilleggskort og har en usignert driver, kan du få problemer med driverinstalleringen eller med kommunikasjonen mellom enheten og HP-programvaren. Vi anbefaler at du alltid bruker en driver for Microsoft WHQL-sertifisert vertskontroller sammen med USB-maskinvare.

·         Plug-and-Play-skriverdriverpakker er tilgjengelige fra Internett på følgende adresse:

http://www.hp.com/support/LJP1100-serien

http://www.hp.com/support/LJP1560-serien

http://www.hp.com/support/LJP1600-serien

 

·         Systemer som bare oppfyller de laveste systemkravene, vil få begrenset funksjonalitet. Bruk av mange bilder eller video krever et system som oppfyller de anbefalte systemkravene for å oppnå akseptabel ytelse og pålitelighet.

·         Feilsøking i konfigurasjon og installasjon

Hvis systemet ikke registrerer enheten:

  • Kontroller at kabelen er kompatibel med høyhastighets USB 2.0.
  • Kontroller at kabelen er riktig tilkoblet. Trekk ut USB-kabelen, se på enden og sammenlign med mønsteret i USB-porten. Kontroller at den ikke er satt inn opp ned. Sett den ene enden av USB-kabelen helt inn i USB-porten på HP-enheten, og sett den andre enden inn i datamaskinen.
  • Kontroller at enheten er slått på. Systemet registrerer ikke andre enheter hvis strømmen er slått av.
  • Kontroller at USB-kabelen ikke er lengre enn 2 meter eller 6,5 fot.
  • Kontroller at USB-porten er aktivert. Noen PC-er leveres med deaktiverte USB-porter.

Oppgradering av Windows

·         HP anbefaler at du avinstallerer HP-enheter før du oppgraderer til en nyere, støttet versjon av Windows.  Hvis du for eksempel skal oppgradere fra Windows XP til Windows Vista, avinstallerer du HP-enheten ved hjelp av Avinstallering eller Legg til eller fjern programmer, og oppgraderer deretter til nytt operativsystem. Når operativsystemoppgraderingen er fullført, installerer du HP-enheten på nytt med den nyeste programvaren fra følgende Internett-adresse:

http://www.hp.com/support/LJP1100-serien

http://www.hp.com/support/LJP1600-serien

III. INSTALLERINGSINSTRUKSJONER

Hvis du bruker Windows, registrerer konfigurasjons- og installeringsprogramvaren på CD-ROM-en versjonen og språket for operativsystemet du bruker.

INSTALLERINGSPROGRAMVARE FOR WINDOWS med HP Smart Install – ANBEFALT installeringsmetode

MERK: Du må ha tilgang som systemansvarlig for å installere utskriftssystemet på operativsystemene Windows XP eller Windows Vista.

1.       Sett USB-kabelen inn i datamaskinen og skriveren.

2.       HP Smart Install-programmet skal kjøre automatisk. Hvis HP Smart Install-programmet ikke starter i løpet av 30 sekunder, utfører du trinn 3–5.

3.       Velg Kjør på Start-menyen på oppgavelinjen.

4.       Klikk på Bla gjennom. En ny CD-ROM-stasjon som heter HP Smart Install vil bli tilgjengelig. Bla til denne stasjonen.

5.       Velg filen SISetup.EXE, klikk på Åpne og klikk deretter på OK.

6.       Følg instruksjonene på skjermen.

 

Installering av PROGRAMVARE for HP Smart Install via nettverket

1.       Skriv inn IP-adressen for enheten i adresselinjen i en nettleser.

2.       Klikk på HP Smart Install-fanen.

3.       Klikk på Last ned-knappen.

4.       Følg instruksjonene på skjermen.

 

INSTALLERINGSPROGRAMVARE FOR WINDOWS Med CD-ROM

MERK: Du må ha tilgang som systemansvarlig for å installere utskriftssystemet på operativsystemene Windows XP eller Windows Vista.

Installering av utskriftssystem fra CD-ROM

1.       Ikke koble USB-/parallellkabelen til datamaskinen.

2.       Lukk alle programmer.

3.       Sett inn CD-ROM-en.

4.       Hvis installeringsprogrammet for HP-utskriftssystemet ikke starter i løpet av 30 sekunder, utfører du trinn 5–7.

5.       Velg Kjør på Start-menyen på oppgavelinjen.

6.       Bla til rotkatalogen på CD-ROM-en.

7.       Velg filen Setup.EXE, klikk på Åpne og klikk deretter på OK.

8.       Følg instruksjonene på skjermen.

 

Tilleggsinstruksjoner for installering av HP Laserjet P1100w-printerserien

HP LaserJet Professional P1100w-skriverserien støtter Wi-Fi 802.11 b/g.  Kontroller at den trådløse ruteren eller tilgangspunktet er satt opp korrekt før du begynner å installere skriveren.  Ha konfigureringsinformasjonen til det trådløse nettverket tilgjengelig i tilfelle du trenger den.

 

Deaktiver HP Smart Install for å bruke en driver som ikke er vertsbasert

Hvis du ønsker å bruke en annen driver via USB enn standard vertsbasert driver som følger med HP Smart Install og CD-en (ekskl. PCL5E eller XPS), må du først deaktivere HP Smart Install på datamaskinen eller skriveren.

1.       Sett USB-kabelen inn i datamaskinen og skriveren.

2.       HP Smart Install-programmet skal kjøre automatisk. Avslutt dette programmet uten å installere den vertsbaserte driveren.

3.       Sett inn CD-ROM-en

4.       HP Smart Install-programmet skal kjøre automatisk. Avslutt dette programmet uten å installere den vertsbaserte driveren. 

5.       Bla til UTIL-katalogen på CD-en.

6.       Velg filen HPSmartInstallConfig.EXE, klikk på Åpne og klikk deretter på OK.

7.       Det finnes to alternativer i denne funksjonen. Ett av alternativene vil deaktivere HP Smart Install på datamaskinen, men den vil fortsatt kjøre hvis skriveren er koblet til andre datamaskiner. Det andre alternativet vil deaktivere HP Smart Install på skriveren. Hvis du velger dette alternativet, vil skriveren ikke kjøre HP Smart Install på noen av datamaskinene hvis ikke den aktiveres på nytt med denne funksjonen.

 

 

Deaktiver HP Smart Install på nettverksprodukter

HP Smart Install kan deaktiveres på nettverksskrivere av administrator ved behov.

1.       Skriv inn IP-adressen for enheten i adresselinjen i en nettleser.

2.       Klikk på fanen Innstillinger.

3.       Klikk på menyen Skriverinnstillinger.

4.       For å deaktivere HP Smart Install via USB, sett Aktiver USB HP Smart Install-alternativet til Nei.

5.       For å deaktivere HP Smart Install via nettverket, sett Vis HP Smart Install-nedlastingsalternativ til Nei.

Installering via Legg til skriver

HP LaserJet kan også installeres via Microsoft-funksjonen Legg til skriver i mappen Skrivere. Denne funksjonen kalles også Veiviser for skriverinstallasjon. Instruksjonene i veiviseren kan variere noe på ulike Windows-versjoner. For å installere via APW, må du først deaktivere HP Smart Install ved hjelp av instruksjonene ovenfor.

 

Installering med veiviseren i WinXP 

1.         Kontroller at HP LaserJet P1100-/P1560-/P1600-skriverserien ikke er koblet til strømnettet eller datamaskinen.

2.         Åpne Start-menyen, velg Kontrollpanel og klikk på Skrivere og telefakser.

3.         Klikk på Legg til skriver og klikk på Neste i skjermbildet Velkommen til veiviseren for skriverinstallasjon.

4.         Følg instruksjonene på skjermen.

5.         Når du får spørsmål om produktet er en lokal skriver eller en nettverksskriver, velger du Lokal skriver (standardvalget). Fjern avmerkingen for Finn og installer Plug and Play-skriveren automatisk.  Klikk på Neste.

6a.         Hvis du installerer via USB og ikke tidligere har installert HP LaserJet P1100-/P1560-/P1600-skriverserien, vil det ikke være installert en USB-port ennå. Dette er normalt. Velg LPT1: og klikk på Neste.

6b.         Hvis du installerer via USB og tidligere har installert og koblet til HP LaserJet P1100-/P1560-/P1600-skriverserien, vil det være installert en USB-port. Velg denne USB-porten og klikk på Neste.

Merk:    Kontroller at du kobler HP LaserJet P1100-/P1560-/P1600-skriverserien til den samme fysiske USB-porten som den tidligere har vært koblet til.

7.         Velg ”Har diskett”, og angi plasseringen av driveren for HP LaserJet P1100-/P1560-/P1600-serien som skal installeres.

8.         Hvis det vises flere alternativer, velger du din skrivermodell og klikker på Neste.

9.         Følg instruksjonene på skjermen.

10.        Når du får beskjed om å koble til skriveren, kobler du HP LaserJet P1100-/P1560-/P1600-skriverserien til strømnettet og deretter til datamaskinen.

 

Merk:    Hvis du ikke får beskjed om å koble til HP LaserJet P1100-/P1560-/P1600-skriverserien på dette tidspunktet, bør du koble den til strømnettet og datamaskinen etter at installeringen er fullført.

 

11.        Følg instruksjonene på skjermen til installeringen er fullført.

 

Når installeringen er ferdig, kan det hende at du har opprettet to eksemplarer av skriverdriveren. I så fall bør du slette den første som ble opprettet. Den har vanligvis samme navn som skrivermodellen. Følg trinnene nedenfor for å kontrollere hvor mange drivere som ble opprettet, og for å slette den første.

 

1.         Åpne Start-menyen, velg Innstillinger og klikk på Skrivere.

2.         Hvis du har både HP LaserJet P1100-/P1560-/P1600-seriene og HP LaserJet P1100-/P1560-/P1600-seriene (kopi 2), må du ikke slette den med "kopi 2" i navnet. Isteden sletter du den første driveren som ble opprettet, kalt ”HP LaserJet P1100-/P1560-/P1600-serien”

 

Nå er du klar til å skrive ut ved hjelp av driveren ”HP LaserJet P1100-/P1560-/P1600-serien” (kopi 2).

Skrive ut med PCL5e

PCL5a er bare støttet på HP LaserJet Professional P1600-skriverserien.

PCL5e-driveren finner du på Internett på følgende adresse:

http://www.hp.com/support/LJP1600-serien

 

Skrive ut med XPS

Du finner XPS-driveren på programvare-CD-en som fulgte med skriveren, eller på Internett på følgende adresse:

http://www.hp.com/support/LJP1100-serien

http://www.hp.com/support/LJP1560-serien

http://www.hp.com/support/LJP1600-serien

Bruk veiviseren for skriverinstallasjon til å installere XPS-driveren.
