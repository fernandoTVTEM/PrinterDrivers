I. 개요

본 문서의 목적

이 문서에서는 인쇄 시스템을 설치하여 사용하기 전에 알아야 할 설치 방법 및 기타 정보에 대해 설명합니다. 이 HP LaserJet 소프트웨어는 다음 운영 체제에서 작동합니다.

·         Windows XP(32비트 및 64비트)

·         Windows Server 2003(32비트 및 64비트)

·         Windows Vista(32비트 및 64비트)

·         Windows Server 2008(32비트 및 64비트)

 

주: HP LaserJet 소프트웨어는 Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000 및 Windows Server 2000을 지원하지 않습니다.

다른 시스템을 설치하는 방법에 대한 지침은 특정 시스템과 관련된 설치 메모, Readme 파일 또는 기타 제품 정보를 참조하십시오.

 

이 파일에는 특정 설치, 호환성 및 문제 해결 정보가 나와 있습니다. 장치 기능 및 드라이버 기능에 대한 자세한 내용은 각 소프트웨어 구성 요소에 포함된 장치 사용자 설명서 및 온라인 도움말을 참조하십시오.

소프트웨어 사용 대상

이 인쇄 시스템 소프트웨어는 HP LaserJet Professional P1100/P1560/P1600 프린터 시리즈를 작동하려는 모든 사용자에게 필요합니다.

II. 알려진 문제와 해결 방법

일반 사항

 

·         특정 고속 USB 2.0 카드에는 서명하지 않은 호스트 컨트롤러 드라이버가 제공될 수 있습니다. USB 2.0 애드인 카드를 구입한 경우, 서명하지 않은 드라이버가 있으면 장치와 HP 소프트웨어 사이에 드라이버 설치 문제 또는 통신 문제가 발생할 수 있습니다. USB 하드웨어에는 항상 Microsoft WHQL 인증 호스트 컨트롤러 드라이버를 사용할 것을 권장합니다.

·         플러그 앤 플레이 인쇄 드라이버 패키지는 다음 웹사이트에서 구할 수 있습니다.

http://www.hp.com/support/LJP1100series

http://www.hp.com/support/LJP1560series

http://www.hp.com/support/LJP1600series

 

·         최소 시스템 요구사양만 갖춘 시스템에서는 기능이 제한될 수 있습니다. 많은 이미지나 비디오를 사용할 경우 안정적인 성능을 발휘하려면 권장 시스템 요구사양에 맞는 시스템이 있어야 합니다.

·         설정 및 설치 문제 해결

장치가 인식되지 않는 경우 다음과 같이 하십시오.

  • USB 2.0 고속 호환 케이블인지 확인하십시오.
  • 케이블이 제대로 연결되었는지 확인하십시오. USB 케이블을 뽑아서 케이블 끝이 USB 포트의 패턴과 일치하는지 살펴 보십시오. 케이블이 거꾸로 되지 않았는지 확인하십시오. USB 케이블의 한쪽 끝을 HP 장치의 USB 포트에 꽂고, 다른 쪽 끝은 컴퓨터 뒷면에 완전히 꽂습니다.
  • 주변 기기의 전원이 켜져 있는지 확인하십시오. 전원이 꺼져 있으면 주변 기기가 인식되지 않습니다.
  • USB 케이블의 길이가 2미터(6.5피트)를 넘지 않는지 확인하십시오.
  • USB 포트가 활성화되어 있는지 확인하십시오. 일부 PC에서는 USB 포트가 비활성화된 경우도 있습니다.

III. 설치 방법

Windows 환경에서 작동 시, CD-ROM 설정 또는 설치 소프트웨어는 실행 중인 운영 체제의 버전과 언어를 자동으로 감지합니다.

HP Smart 설치를 사용하여 WINDOWS에 소프트웨어 설치(권장 설치 방법)

주: Windows XP 또는 Windows Vista 운영 체제에 인쇄 시스템을 설치하려면 관리자 권한이 필요합니다.

1.       USB 케이블을 컴퓨터 및 프린터에 연결합니다.

2.       HP Smart 설치 프로그램이 자동으로 실행됩니다. 30초 후에도 HP Smart 설치 프로그램이 시작되지 않으면 3 ~ 5단계를 수행합니다.

3.       작업 표시줄의 시작 메뉴에서 실행을 선택합니다.

4.       검색하기를 누릅니다. HP Smart 설치라고 하는 새 CD-ROM 드라이브를 사용할 수 있습니다. 이 드라이브로 이동합니다.

5.       SISetup.EXE 파일을 선택하고 열기를 누른 다음 확인을 누릅니다.

6.       화면에 표시되는 지침을 따릅니다.

 

HP Smart 설치를 사용하여 네트워크를 통해 소프트웨어 설치

1.       웹 브라우저에서 브라우저의 주소줄에 장치에 대한 IP 주소를 입력합니다.

2.       HP Smart 설치 탭을 누릅니다.

3.       다운로드 단추를 누릅니다.

4.       화면에 표시되는 지침을 따릅니다.

 

CD-ROM을 사용하여 WINDOWS에 소프트웨어 설치

주: Windows XP 또는 Windows Vista 운영 체제에 인쇄 시스템을 설치하려면 관리자 권한이 필요합니다.

CD-ROM에서 인쇄 시스템 설치

1.       USB/병렬 케이블을 컴퓨터 시스템에 연결하지 않습니다.

2.       모든 프로그램을 닫습니다.

3.       CD-ROM을 넣습니다.

4.       30초 후에도 HP 인쇄 시스템 설치 프로그램이 시작되지 않으면 5 ~ 7단계를 수행합니다.

5.       작업 표시줄의 시작 메뉴에서 실행을 선택합니다.

6.       CD-ROM 드라이브의 루트 디렉토리로 이동합니다.

7.       Setup.EXE 파일을 선택하고 열기를 누른 후 확인을 누릅니다.

8.       화면에 표시되는 지침을 따릅니다.

 

HP LASERJET P1100W 프린터 시리즈 설치를 위한 추가 지침

HP LaserJet Professional P1100w 프린터 시리즈 프린터는 Wi-Fi 802.11 b/g를 지원합니다. 프린터를 설치하기 전에 무선 라우터 또는 액세스 지점이 설정되었는지 확인하십시오. 필요한 경우를 대비하여 무선 네트워크의 구성 정보를 알고 있어야 합니다.

 

HP SMART 설치를 비활성화하여 호스트 기반이 아닌 드라이버 사용

HP Smart 설치 및 CD에 포함된 표준 호스트 기반 드라이버가 아니라 USB를 통해 드라이버(예: PCL5E 또는 XPS)를 사용하려면 먼저 HP Smart 설치를 컴퓨터 또는 프린터에서 비활성화해야 합니다.

1.       USB 케이블을 컴퓨터 및 프린터에 연결합니다.

2.       HP Smart 설치 프로그램이 자동으로 실행됩니다. 호스트 기반 드라이버를 설치하지 않고 이 프로그램을 종료합니다.

3.       CD-ROM을 넣습니다.

4.       설치 프로그램이 자동으로 실행됩니다. 호스트 기반 드라이버를 설치하지 않고 이 프로그램을 종료합니다. 

5.       CD의 UTIL 디렉토리로 이동합니다.

6.       HPSmartInstallConfig.EXE 파일을 선택하고 열기를 누른 다음 확인을 누릅니다.

7.       이 유틸리티에는 두 개의 옵션이 있습니다. 한 개의 옵션은 컴퓨터에서 HP Smart 설치를 끄는 것으로, 프린터가 다른 컴퓨터에 장착된 경우 HP Smart 설치가 계속 실행됩니다. 다른 옵션은 프린터에서 HP Smart 설치를 끄는 것입니다. 이 옵션을 선택하면 이 유틸리티를 사용하여 HP Smart 설치를 다시 활성화하지 않는 한 어떤 컴퓨터에서도 프린터가 HP Smart 설치를 실행할 수 없습니다.

 

 

네트워크 제품에서 HP Smart 설치 비활성화

필요한 경우 관리자가 네트워크 프린터에서 HP Smart 설치를 비활성화할 수 있습니다.

1.       웹 브라우저에서 브라우저의 주소줄에 장치에 대한 IP 주소를 입력합니다.

2.       설정 탭을 누릅니다.

3.       인쇄 설정 메뉴를 누릅니다.

4.       USB를 통해 HP Smart 설치를 비활성화하려면 USB HP Smart 설치 활성화 옵션을 아니오로 설정합니다.

5.       네트워크를 통해 HP Smart 설치를 비활성화하려면 HP Smart 설치 다운로드 옵션 표시를 아니오로 설정합니다.

프린터 추가를 통한 설치

HP LaserJet 프린터는 프린터 폴더에서 Microsoft의 프린터 추가 기능을 통해서도 설치할 수 있습니다. 이 기능은 프린터 추가 마법사 또는 APW로도 알려져 있습니다. APW 지침은 Windows 버전마다 약간 다를 수 있습니다. APW를 통해 설치하려면 먼저 위의 지침을 사용하여 HP Smart 설치를 비활성화해야 합니다.

 

WinXP APW 설치 

1.         HP LaserJet P1100/P1560/P1600 프린터 시리즈의 전원이 연결되지 않았고 컴퓨터에 연결되지 않았는지 확인합니다.

2.         시작 메뉴에서 제어판을 가리킨 다음 프린터 및 팩스를 누릅니다.

3.         프린터 추가를 누른 다음 “프린터 추가 마법사 시작” 화면에서 다음을 누릅니다.

4.         화면의 지침을 따릅니다.

5.         제품이 로컬 프린터인지 네트워크 프린터인지 묻는 메시지가 표시되면 로컬 프린터(기본 선택임)를 선택합니다. 자동으로 플러그 앤 플레이 프린터 검색 및 설치 확인란을 선택 해제합니다. 다음을 누릅니다.

6a.         USB를 통해 설치 중이고 이전에 HP LaserJet P1100/P1560/P1600 프린터 시리즈를 설치한 적이 없으면 아직 USB 포트가 설치되지 않는 것이 정상입니다. LPT1을 선택합니다. 다음을 누릅니다.

6b.         USB를 통해 설치 중이고 이전에 HP LaserJet P1100/P1560/P1600 프린터 시리즈를 설치 및 연결한 적이 없으면 아직 USB 포트가 설치되지 않습니다. 이 USB 포트를 선택하고 다음을 누릅니다.

주:    HP LaserJet P1100/P1560/P1600 프린터 시리즈를 이전에 연결한 것과 동일한 물리적 USB 포트에 연결해야 합니다.

7.         "디스크 있음"을 선택하고 설치할 HP LaserJet P1100/P1560/P1600 시리즈 드라이버의 위치를 지정합니다.

8.         여러 개의 선택 사항이 나타나면 사용 중인 프린터 모델을 선택하고 다음을 누릅니다.

9.         화면의 지침을 따릅니다.

10.        프린터를 연결하도록 메시지가 표시되면 HP LaserJet P1100/P1560/P1600 프린터 시리즈의 전원을 연결하고 컴퓨터에 연결합니다.

 

주:    이 때 HP LaserJet P1100/P1560/P1600 프린터 시리즈를 연결하도록 메시지가 표시되지 않으면 설치가 완료될 때 프린터의 전원을 연결하고 컴퓨터에 연결해야 합니다.

 

11.        설치가 완료될 때까지 화면의 지침을 따릅니다.

 

설치 완료 시, 2개의 프린터 드라이버 사본이 생성되어 있을 수 있습니다. 이 경우 처음 생성된 사본을 삭제해야 합니다. 일반적으로 처음 생성된 사본은 프린터 모델과 동일한 이름을 사용합니다. 생성된 드라이버 수를 확인하고 처음 생성된 사본을 삭제하려면 아래의 단계를 수행합니다.

 

1.         시작 메뉴에서 설정을 가리킨 다음 프린터를 누릅니다.

2.         "HP LaserJet P1100/P1560/P1600 시리즈" 및 "HP LaserJet P1100/P1560/P1600 시리즈(사본 2)"가 모두 있는 경우 이름에 "사본 2"가 있는 드라이버를 삭제하면 안 됩니다. 대신에 "HP LaserJet P1100/P1560/P1600 시리즈"라고 하는 첫 번째 생성된 드라이버를 삭제합니다.

 

이제 "HP LaserJet P1100/P1560/P1600 시리즈(사본 2)" 드라이버를 사용하여 인쇄할 준비가 되었습니다.
